Установил параметр в конфиге /etc/pam.d/passwd (читал в статье про настойки безопасности что можно этот модуль включить)
password required pam_cracklib.so difok=2 minlen=8 dcredit=2 ocredit=2 retry=3
при попытке смены пароля выдает ошибку
root / #  passwd root
passwd: Неизвестный модуль
passwd: пароль не изменён
Почему неизвестный модуль? pam_cracklib.so, пакеты соответствующие для него установлены, должно работать, эта ж запись в конфиге есть но комментирована
