Linphone - протестировал в прошлом году.
вот это?
С этого момента, мы приступаем к конфигурированию программного телефона. Для этого нам необходимо знать:

    IP-адрес АТС;
    Имя (логин) пользователя;
    SIP пароль пользователя.

Ниже на рисунке отмечены номера полей которые необходимо заполнить.

    Идентификатор SIP - в нашем случае это имя (логин) пользователя+домен АТС с портом;
    Адрес SIP-прокси - в этом поле введите имя (логин) пользователя+домен АТС с портом;
    Продолжительность регистрации(сек) - поставить время регистрации 60 сек;
    После этого кликните на кнопку OK
шо за Идентификатор SIP,домен АТС с портом,Адрес SIP-прокси,IP-адрес АТС?
