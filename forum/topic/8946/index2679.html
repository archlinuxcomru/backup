Когда стоял pacman-3.1.4-1, все было нормально.
Ставлю новый pacman
===================
проверка
pacman -Q glibc pacman
glibc 2.8-3
pacman 3.2.1-2
====================
в /vat/pkg/ исполняю
repo-add /vat/pkg/custom.db.tar.gz /vat/pkg/*.pkg.tar.gz
========================
в /etc/pacman.conf добавляю

Server = file:///vat/pkg/
==================
исполняю
pacman -Sy
:: Синхронизируется база данных пакетов…
file:///vat/pkg/custom.db.tar.gz: Неподдерживаемая схема.
ошибка: не удалось обновить custom (непредвиденная ошибка)
ошибка: не удалось синхронизировать ни одну базу данных
