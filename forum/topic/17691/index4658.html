Странно. Проверил под разными DE (LXQT и XFCE4) результат один - показывает. Но показывает только те ресурсы, которые имеет общие папки или принтеры.

Предлагаю вернуть настройки по умолчанию в самбе sudo cp /etc/samba/smb.conf.default /etc/samba/smb.conf и внести только WORKGROUP, не трогая больше ничего.

Может арч где-то криво стал, или какой-то из пакетов - здесь может быть все что угодно.

Решил проблему следующим образом
Добавил в файл smb.conf в секцию Global Settings

username map = /etc/samba/smbusers
os level = 33
map to guest = Bad User
usershare allow guests = yes
guest ok = yes

cat /etc/samba/smbusers
nobody = guest pcguest

Пароль у nobody в sambe сделал пустым.

Все заработало после перезагрузки.
