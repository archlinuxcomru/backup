Итак, решение:

Зайти при загрузке в консоль, и там:

git clone https://aur.archlinux.org/downgrader.git
cd downgr* && makepkg
sudo pacman -U $(ls | grep xz)
downgrader xorg-server

Здесь выбрать в самом начале списка предыдущую перед установленной сейчас (installed) версию (у меня это был пункт 3 - 1.19-6), и она установится. Соглашаемся на удаление того, что предложат удалить. Ребут, профит.
