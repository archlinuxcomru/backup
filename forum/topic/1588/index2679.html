WM - WMFS2
В статусбаре и заголовках окон кириллица отображается каракулями.
Подозреваю, что шрифт, используемый wm по умолчанию не поддерживает кириллицу.
При смене шрифтов в wmfrc и перезапуске wm шрифт не меняется
Кусок wmfsrc: 
  [theme]
     # No name mean default
     # name = "default"
#     font = "-*-terminus-bold-*-*-*-12-*-*-*-*-*-*-*"
#     font = "terminus"
#     font = "profont"
Не один из трёх вариантов “font =” не изменяет шрифт.
Куда копать?
