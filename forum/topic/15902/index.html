Подключаюсь к wifi через утилиту iw, но есть вопросы ну и как всегда ошибки, ниже приведу порядок ввода команд:


#iw dev
##В ответ получаю доступный интерфейм wlo1

#ip link set wlo1 up
#iw dev wlp4s0 scan | less
##Тут выбираю свою сеть(ssid) которую раздаю через роутер с использование WPA/WPA2

# wpa_supplicant -i wlan0 -c <(wpa_passphrase ваш_SSID ваш_ключ)

А вот теперь вопрос и проблема. Обязательно ли генерировать запись подключения при помощи wpa_passphrase и редактировать /etc/wpa_supplicant/wpa_supplicant.conf?
Пробовал подключаться вот так

# wpa_supplicant -i wlan0 -c <(ваш_SSID ваш_ключ)
Выдает syntax error unexpected token ')'
Но ничего не вышло, возможно из-за того что в пароле к wifi имеется занк ')'? Как подключаться к wifi через wpa_supplicant с указание ssid и пароля без правки файлов.
