Отключается модуль wi-fi в Samsung N127

eugenez
# 10 лет, 1 месяц назад
Темы: 15
Сообщения: 355
Участник с: 17 августа 2011
Сменил провайдера и роутер (поставил вместо адсльного D-Link DSL-6240 D-Link DIR-615), после смены стал периодически отваливаться беспроводной интерфейс. Выглядит это так:
[35478.767001] ath: phy0: Could not kill baseband RX
[35478.894720] ath: phy0: Unable to reset channel, reset status -5
[35478.894759] ath: phy0: Unable to set channel
[35479.018002] ath: phy0: Could not kill baseband RX
[35479.145675] ath: phy0: Unable to reset channel, reset status -5
[35479.145722] ath: phy0: Unable to set channel
[35479.273254] ath: phy0: Unable to reset channel, reset status -5
[35479.273294] ath: phy0: Unable to set channel
Нетворкменеджер подключить беспроводное соединение не может. Беспроводный модуль (ath9k) из памяти не выгружается. Перезагрузка спасает на некоторое время.

Временное решение, подсмотреное здесь и выглядящее так, 
#!/bin/bash
sudo modprobe -rfv ath9k
sudo rfkill unblock all
sudo modprobe -v ath9k
помогает, но хотелось бы решить проблему радикально, вдруг кто-то сталкивался с подобным.

Настройки роутера, относящиеся к wi-fi, такие: 802.11 mode - mixed (g+n), auto channel scan enable, channel width 20 MHz, SSID broadcast disable, WPA2 personal/AES, Beacon period 100, RTS threshold 2347, fragmentation 2346, DTIM interval 1, Short GI on.
