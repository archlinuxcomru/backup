скрипт - текстовый файл содержаший команды, понятные той программе, которой он будет выполняться.Сохраняется в программе типа типа блокнот
(НИ В КОЕМ СЛУЧАЕ НЕ OPENOFFICE И НЕ ГРАФИЧЕСКОМ РЕДАКТОРЕ). первая строка указывает программу для которой файл написан (кто его выполнять будет). чтобы он выполнялся как программа, надо
1. убедиться что есть та программа которая будет читать/выполнять файл
2. поставить право на чтение и выполнение для пользователя
Да пребудет с вами знание ip адреса

Удаляете то что вручную собирали(volumed)
sudo pacman -R название пакета
Ставите утилиты для сборки 
sudo pacman -S base-devel
Ставим yaourt
Самый простой способ установить Yaourt - это добавить его репозиторий в Ваш /etc/pacman.conf:
Для i686:

Server = http://repo.archlinux.fr/i686
Для x86-64:

Server = http://repo.archlinux.fr/x86_64 
pacman -Syu yaourt
Устанавливаем систему нотификаций
по идеи должен стоять xfce4-notifyd если нет то ставим 
sudo pacman -S xfce4-notifyd
можно поставить и другие, например notify-osd https://wiki.archlinux.org/index.php/Notify_OSD#Xfce
Ставим xfce4-volumed
yaourt -S xfce4-volumed
Запускаем демон
xfce4-volumed
Должно работать
