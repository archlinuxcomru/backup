В tint2rc единственное подозрительное место: strut_policy = follow_size - обычно эта опция находится в секции “PANEL”:
# PANEL
...
...
strut_policy = follow_size
если это вообще имеет значение, т.е., это к тому, что конфиг читается сверху вниз, последовательно.
Можно ещё попробовать 
panel_size = 0 26
Вобщем, для экспериментов: http://code.google.com/p/tint2/wiki/Configure

На всякий случай, у меня так:
#---------------------------------------------
# PANEL
#---------------------------------------------
panel_monitor = all
panel_position = top center horizontal
panel_size = 100% 24
panel_margin = 0 0
panel_padding = 0 0 0
font_shadow = 0
panel_background_id = 1
wm_menu = 0
panel_dock = 0
panel_layer = top
# LTSBC define the items tint2 will show and the order of those items.
# panel_items = STC will show the Systray, the taskbar and the clock (from left to right). SVN ONLY.
#    * L to show Launcher
#    * T to show Taskbar
#    * S to show Systray (also called notification area)
#    * B to show Battery status
#    * C to show Clock 
panel_items = TSCL	
strut_policy = follow_size
