Запускать chromium с флагом “–allow-outdated-plugins”.
Или прописать этот флаг на постоянное место жительства в файл /etc/chromium/default
$ cat /etc/chromium/default 
# Default settings for chromium. This file is sourced by /usr/bin/chromium
# Options to pass to chromium
CHROMIUM_FLAGS="--allow-outdated-plugins"
Все остальные решения, которые я видел - костылеобразные. От случая к случаю и от обновления до обновления.
