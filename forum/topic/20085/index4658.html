Можно также переназначить клавишу KP_Enter на Return
sudo pacman -S xorg-xmodmap
xmodmap -pke > ~/.Xmodmap
nano ~/.Xmodmap
изменить строчку keycode 104 = KP_Enter
на
keycode 104 = Return NoSymbol Return
сохранить и применить изменения
xmodmap ~/.Xmodmap

вики
https://wiki.archlinux.org/index.php/Xmodmap
