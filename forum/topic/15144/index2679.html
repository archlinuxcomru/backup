awesome: правило для nomacs

lumberjack
# 7 лет назад (отредактировано 7 лет назад)
Темы: 165
Сообщения: 799
Участник с: 06 января 2012
не пойму, как прописать правило для просмотрщика изображений nomacs, чтобы все окна, кроме основного, были в плавающем режиме.
нарисовал такое:
{ rule_any = { class = { "nomacs", "Nomacs" } },
		except = { instance = "main" },
		properties = { tag = tags[1][6], floating = true } },
но правило не работает. пробовал несколько вариантов, но итог -- либо все в плавающем режиме, либо все в тайле, как на скрине.

xprop nomacs
xprop окна распечатки
пока вообще тупо
{ rule = { class = "Nomacs" },
		properties = { tag = tags[1][6] } },
бо разницы с вышеуказанным правилом никакой...
