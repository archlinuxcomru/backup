У вас модуль dbus есть значит.
Просто не python-dbus пакетом поставлен.

Если интересно можно разобраться откуда модуль
сделать скриптик helpdbus
#!/usr/bin/env python
import dbus
help (dbus)

И запустить → в секции FILE должен быть путь до модуль
python '/home/oleg/Desktop/helpdbus' |grep -A 20 FILE
Ну а там уже смотреть какому пакету принадлежит файл, или вспоминать когда через pip или по другому ставили

Для стандпртного поставляемого python-dbus
FILE
    /usr/lib/python3.7/site-packages/dbus/__init__.py

.
pacman -Qo  /usr/lib/python3.7/site-packages/dbus/__init__.py
/usr/lib/python3.7/site-packages/dbus/__init__.py принадлежит python-dbus 1.2.8-2
