Вообще на wiki арча есть прекрасное описание настройки шрифтов… Сглаживание в гноме и иксфце настраивается посредством встроенных демонов, во всех остальных боксах нужно использовать файл ~/.fonts.conf
+ <~/.Xresources>|</etc/X11/Xresources> следующего содержания:
!! fonts
Xft.dpi: 96
! hintnone, hintslight, hintmedium, hintfull
Xft.hintstyle: hintfull
Xft.hinting: true
! lcdnone, lcddefault, lcdlight, lcdlegacy
Xft.lcdfilter: lcddefault
Xft.rgba: rgb
тоже не помешает :)
