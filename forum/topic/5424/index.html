После перехода с Ubuntu у меня возникла проблема с коррекцией времени. Решил так: установил пакет ntp, а в /etc/cron.daily поместил скрипт с правом выполнения
#!/bin/bash
#
# Set time
#
/usr/bin/ntpdate -uBs 0.ru.pool.ntp.org
exit 0
Выполнение ежедневных заданий выбрал так, чтобы в это время компьютер как правило был включен и в Сети.
Демон не запущен, одной коррекции в сутки мне вполне хватает. Если недостаточно, можно поместить в /etc/cron.hourly :)
