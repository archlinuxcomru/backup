Гугли по mount: /mnt: unknown filesystem type 'LVM2_member'
... если диск был размечен под LVM, то вместо /dev/sda3 нужно указывать другое имя ...
Кроме того имеются и другие нюансы с разбивкой
Paratrupper
GPT PMBR size mismatch (1497751 != 1986557) will be corrected by write.
The backup GPT table is not on the end of the device.

В части chroot - если проблема только в установке нескольких пакетов, то можно и не использовать chroot для этого …. потом его освоишь.
А пока рекомендую скачать нужные пакеты отсюда (думаю разберешся), поместить их в любое место своей системы, например, /home/…./, затем загрузиться в свою систему и установить эти пакеты, используя команду sudo pacman -U /путь/к/пакету/<пакет>
Один минус - если большой промежуток времени между последним обновлением, то могут быть проблемы с зависимостями … но попробуй хотя бы только пакеты для настройки сети, может и проскочит ...
