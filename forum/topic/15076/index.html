У меня таже беда с ГТК3 приложениями. Появилось после смены темы с помощью lxapperance.
Наверно правильно будет менять темы только в конфигах вручную указывая тему, иконки и т.п. Или гуглить как отключить все эти рамки и заголовки окон, либо менять тему в ГТК3

Вот уже решил, спустя пару минут:

в файле .config/gtk-3.0/settings.ini добавить:

[Settings]
gtk-theme-name = Adwaita
gtk-fallback-icon-theme = gnome

А воттак можно отключить кнопки у gnome-декораций:
gtk-decoration-layout=menu:

Более правильное решение, работает с любыми темами:
~/.config/gtk-3.0/gtk.css
.window-frame, .window-frame:backdrop {
 box-shadow: 0 0 0 black;
 border-style: none;
 margin: 0;
 border-radius: 0;
}

.titlebar {
 border-radius: 0;
}

.window-frame.csd.popup {
  box-shadow: 0 1px 2px rgba(0, 0, 0, 0.2), 0 0 0 1px rgba(0, 0, 0, 0.13);
}
Взял отсюда.
