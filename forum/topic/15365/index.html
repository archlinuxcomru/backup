Правильное решение находится вот тут: wiki.archlinux.org/Backlight.
В моём случае помогло добавление следующего параметра ядра - acpi_backlight=vendor.

Исправляем ошибку:

1. Открываем в редакторе файл конфигурации загрузчика.
# sudo nano /etc/default/grub
2. Дописываем нужный параметр в строку:
> GRUB_CMDLINE_LINUX_DEFAULT="quiet acpi_backlight=vendor"

3. Обновляем загрузчик.
# update-grub
4. Перезагружаемся.

5. Убеждаемся, что ошибка пропала:
$ journalctl -b -p err
