После установки драйверов на radeon возникли проблемы с экраном...
1. После запуска системы у меня на экране открыто 2 рабочих стола (у меня i3wm). Точнее - eDP1 и HDMI1. Выключаю это только с помощью arandr. Отключал HDMI1 в arandr, reboot - картина не меняется. Как, почему такое могло случиться?
2. Разрешение экрана стало больше, хотя
 $ xrandr
Screen 0: minimum 320 x 200, current 1920 x 1080, maximum 16384 x 16384
eDP-1 connected primary 1920x1080+0+0 (normal left inverted right x axis y axis) 344mm x 193mm
...
HDMI-1 connected (normal left inverted right x axis y axis)
   1920x1080     60.00 +  50.00    59.94
Во время установки драйверов я не менял какие-либо параметры экрана. Почему они изменились?
Весь вечер читал вики по темам Xorg, xinit, radeon, в /etc/X11/xorg.conf.d/ создавал файлы 20-intel.conf, 20-radeon.conf - не помогло.
Смотрел, читал здесь темы о проблемах с разрешением экрана - вроде и близко, но не то...

$ xrandr --listproviders
Providers: number : 2
Provider 0: id: 0x44 cap: 0xf, Source Output, Sink Output, Source Offload, Sink Offload crtcs: 3 outputs: 2 associated providers: 1 name:modesetting
Provider 1: id: 0xde cap: 0xd, Source Output, Source Offload, Sink Offload crtcs: 0 outputs: 0 associated providers: 1 name:HAINAN @ pci:0000:01:00.0

я так понял, что провайдер изображения у ноута modesetting, а не radeon... Подскажите пожалуйста, что сделать, чтобы был radeon?
