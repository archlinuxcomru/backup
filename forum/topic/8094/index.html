Дело продвинулось.
У хоста eth0 имеет адрес 192.168.0.2, у гостя выбрал в настройках сети bridge адаптер, прописал у eth0 гостя ip 192.168.0.1, выставил в качестве шлюза ip хоста - 192.168.0.2. Теперь с гостя виден хост :) . То есть локальная сеть из гостя и хоста есть. Осталось понять как соединить гостя с интернетом через хост. Прошу подсказать в каком направлении копать дальше.

да дальше бери любое руководство как настраивать шлюз на линухе и настраивай хост :)

например такое:

на виртуалке
прописать gw на хост
можно вручную что-то вроде
route add default gw 192.168.0.2

на хосте
echo 1 >/proc/sys/net/ipv4/ip_forward
iptables -F
iptables -t nat -F
iptables -t nat -A POSTROUTING -i eth0 -o ppp999 -j MASQUERADE
iptables -P FORWARD ACCEPT

как-то таг, потом курить дальше.
systemd должен умереть.
