В ядре Linux (и в Арче) есть модули поддержки файловых систем qnx4 и qnx6. Так что по идее, они должны монтироваться обычной командой mount.

Средств проверки этих ФС под Linux я не нашел.

Для поиска и выгрузки уцелевших файлов с разрушенных ФС, безотносительно типа ФС, есть специальные программы, например photorec из пакета testdisk. Если файлы опознаваемого формата и не фрагментированы, есть хорошие шансы вытащить их без повреждений.
