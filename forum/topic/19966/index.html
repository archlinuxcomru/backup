Обратил внимание на долгую загрузку системы после создания своп-файла командами
sudo fallocate -l 8GB /swapfile
sudo swapon --discard /swapfile
И добавлением строки 
/swapfile none swap defaults,discard 0 0
в /etc/fstab. Позже обратил внимание на долгую загрузку системы. Отключил своп-файл, но скорость загрузки не изменилась.

Я создаю своп вот таким образом:
fallocate -l 8G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo "/swapfile none swap defaults 0 0" >> /etc/fstab
В разных системах, арч, центось, суся и тд, никогда проблем не было. Проверить наличие swap-файла можно так:
swapon -s
