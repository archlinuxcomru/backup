имеется ноутбук asus k53sm, гибридная графика интел + нвидиа.
установлен bumblebee + primus

при попытке запустить:
primusrun nvidia-settings
вылазит ошибка:
You do not appear to be using the NVIDIA X driver. Please edit your X configuration file (just run `nvidia-xconfig` as root), and restart the X server.

optirun nvidia-settings
показывает те же результаты.
проги запускаются с видюхой нвидии нормально.
менеджер настроек-же нивкакую...

optirun glxinfo |grep -i opengl
показывает nvidia карту. без оптирана показывает соответственно интелскую

видео ошибки
