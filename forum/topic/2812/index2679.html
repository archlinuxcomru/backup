Шрифт и его размер - 
dconf-editor
org.gnome.nautilus.desktop изменяете значение font на свое

Цвет шрифта на рабочем столе - правите CSS в теме Gtk 
/*текст на рабочем столе - пассивная иконка */
.nautilus-desktop.nautilus-canvas-item {
/* цвет текста*/ 
  
    color: #D4E0E0;
    
    /* тень текста*/
   text-shadow: 1 1  #000000;
  
}
/* текст на рабочем столе - активная иконка*/
.nautilus-desktop.nautilus-canvas-item:active {
      /* фон текста*/   
     background-image:-gtk-gradient (linear,
                   left top, right top,
                   from (shade (#DAE7E8, 0.98)),	
		   color-stop (0.50, shade (#DAE7E8, 1.1)),
                   to (shade (#DAE7E8, 0.98))); 
/* цвет текста*/
     color: #000000;
/* тень текста */
     text-shadow: 1 1 #D3DEEE;
    border-radius: 4;
}
/* текст на рабочем столе - выбранная иконка*/
.nautilus-desktop.nautilus-canvas-item:selected {
 /* фон текста*/ 
      background-image: -gtk-gradient (linear,
                   left top, right top,
                   from ( rgba(0, 120, 201, 0.7)),	
		   color-stop (0.50, shade (#DAE7E8, 1.1)),
                   to (rgba(0, 120, 201, 0.7))); 
 /* цвет текста*/
     color: #000000;
/* тень текста*/    
     text-shadow: none; 
    border-radius: 4; 
}

Как изменить “Файловая система …GB” - поставить метку (например видео или хлам) например с помощью gparted

В “gtk.css” обычно описываются переменные (вроде X=3 Y=4) для цветов, в конце же идут ссылки на css где описывается остальное ( в вашем случае @import url(“gtk-widgets.css”).

Откройте “/ваша тема/gtk3/gtk-widgets.css” в редакторе ( удобно использовать редактор с подсветкой синтаксиса и цветовыми инструментами - например Geany ).
Вбейте в поиск nautilus, если там описывается nautilus-desktop то измените значения на свои. Если нет - добавьте в конец файла то что я давал выше (цвета разумеется под себя). Надо еще проверить нет ли других ссылок в конце gtk-widgets.css .

Сохраните изменения и перезагрузите наутилус 
killall nautilus 
nautilus
