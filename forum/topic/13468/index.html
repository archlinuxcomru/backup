Проблема оказалась несколько глубже, чем предполагалось. Отметку "решено" это не снимает, но описать суть надо.

Дело в том, что указанное проявляется, если использовать класс документа extreport. Видимо там перекрываются некоторые табличные настройки. Если использовать класс report, то всё в порядке - ставим \caption{...} до тела таблицы - он будет вверху, ставим после - внизу. А в расширенном отчете можно переопределить стиль подписи как показано выше.

Гугление по этим и другим вопросам, а так же прочтение С. М. Львовского не по диагонали а внимательно дало следующий код стилевого пакета, который будем расширять по мере необходимости


%--------------------------------------------------------------------
%
%	Стилевой пакет для оформления диплома в LaTeX
%	(c) maisvendoo, 30.04.2014
%
%--------------------------------------------------------------------
\RequirePackage[T2A]{fontenc}

% Вражеская кодировка винды - надо так, иначе бибтех не пашет
\RequirePackage[cp1251]{inputenc}
% Руссиш и инглиш с переносами
\RequirePackage[english, russian]{babel}

% Формулы, математические шрифты, ссылки, списки и прочее
\RequirePackage{amssymb, amsfonts, amsmath, mathtext, cite, enumerate, float}

% Пакет для вставки рисунков
\RequirePackage{graphicx}

% Подключаем дополнительный пакет для шрифтов 14пт
\RequirePackage[14pt]{extsizes}

\RequirePackage[left=3cm, right=1cm, top=2cm, bottom=2cm]{geometry}

% Пакет для настройки интервалов
\RequirePackage{setspace}

% Команды преамбулы

% Переопределяем интервал
\renewcommand{\baselinestretch}{1.5}

% Нумерация частей арабскими цифрами
\renewcommand{\thepart}{\arabic{part}}

% Счетчик глав
% Переопределяем команду, чтобы счетчик глав зависел от счетчика частей
\renewcommand{\newcounter}{\newcounter{chapter}[part]}
% Устанавливаем счетчик
\setcounter{chapter}{1}

\renewcommand{\theenumi}{\arabic{enumi}.}
\renewcommand{\labelenumi}{\arabic{enumi}.}
\renewcommand{\theenumii}{.\arabic{enumii}.}
\renewcommand{\labelenumii}{\arabic{enumi}.\arabic{enumii}.}
\renewcommand{\theenumiii}{.\arabic{enumiii}}
\renewcommand{\labelenumiii}{\arabic{enumi}.\arabic{enumii}.\arabic{enumiii}.}

Используется вражья кодировка CP1251 из-за того, что предполагается юзать BibTeX, а он с utf8 к сожалению не дружит.
