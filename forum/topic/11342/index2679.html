Попробуйте так (пусть пользователь будет vova):
1. В /etc/hosts уберите четвертую строку (дубляж)
2. Переделайте в httpd-vhosts.conf
<VirtualHost *:80>
    ServerAdmin [email protected]
    DocumentRoot /home/vova/server
    ServerName server
    <Directory /home/vova/server>
      DirectoryIndex index.php index.html index.htm
      Options -Indexes FollowSymLinks MultiViews +Includes
      AllowOverride All
      Order allow,deny
      Allow from all
</Directory>
</VirtualHost>
3. Создайте папку /home/vova/server
4. Сделайте для этой папки
$ su -c 'chown -R vova:http /home/vova/server'
5. В папке создайте тестовый файл index.php или index.html
#index.php
<?php
phpinfo();

Перезапустите apache.
Попробуйте открыть http://server/

P.S. Уберите из httpd.conf
<Directory /home/server/>
Options FollowSymLinks
Options Indexes FollowSymLinks
AllowOverride None
Order deny,allow
Deny from all
</Directory>
