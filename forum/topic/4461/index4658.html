Да, надо будет поставить из репов пакет ntfs-3g. Вручную. Примерно так:

menu.lst
# (2) Windows
title Windows
rootnoverify (hd0,0)
makeactive
chainloader +1
fstab
/dev/sda1    /mnt/win    ntfs-3g    noatime,users    0    0
