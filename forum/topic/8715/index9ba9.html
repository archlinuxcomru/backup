1. поднятие waln0
2. прописывание машрутов до vpn: ip route add 10.30.0.0/255.255.255.0 via 192.168.1.1
3. удаление маршрута по умолчанию
4. проверить что в текущей конфигурации доступен vpn-server
5. запуск vpn

Ментрику на лету изменить нельзя, только удалив маршрут и создав заново с указанием новой метрики. Короче, тот умолчательный маршрут который у тебя есть по умолчанию, до установки соединения нужно обработать примерно таким образом:
ip route del default via ###.###.###.###
ip route add default via ###.###.###.### metric 2

т.е. новый маршурт будет с метрикой 0, и более приоритетным. как-то так. пробывать нужно.
