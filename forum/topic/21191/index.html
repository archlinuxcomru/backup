не получается убрать панель с мониторов..

бреу стандартный конфиг rc.lua меняю

     -- Create the wibox
     s.mywibox = awful.wibar({ position = "top", screen = s })

на

     -- Create the wibox
     s.mywibox = awful.wibar({ position = "top", screen = 1 })

и получается (картинку если тут вставлять она слишком большая и всё портит)
