[РЕШЕНО] Ядро 3.10.2-1 и VMware 9.0.2

maisvendoo
# 9 лет, 1 месяц назад
Темы: 68
Сообщения: 1142
Участник с: 10 октября 2012
Тем кто пользуется VMware будет интересно, я вот только вчера настроил удаленную отладку на связке VMware + gdb + eclipse, и тут на тебе - свежее ядро на котором предполагался крах этой ВМ, по многочисленным слухам. Моему огорчению не было предела, когда я обнаружил что это так и есть - не работают дрова блочных устройств и сеть и сама варька не стартует.

Но к счастью на арчевики оперативно решили этот вопрос, приведя следующую последовательность команд для патчинга и пересборки модулей VMware

$ cd /tmp
$ curl -O http://pkgbuild.com/git/aur-mirror.git/plain/vmware-patch/vmblock-9.0.2-5.0.2-3.10.patch
$ curl -O http://pkgbuild.com/git/aur-mirror.git/plain/vmware-patch/vmnet-9.0.2-5.0.2-3.10.patch
$ cd /usr/lib/vmware/modules/source
# tar -xvf vmblock.tar
# tar -xvf vmnet.tar
# patch -p0 -i /tmp/vmblock-9.0.2-5.0.2-3.10.patch
# patch -p0 -i /tmp/vmnet-9.0.2-5.0.2-3.10.patch
# tar -cf vmblock.tar vmblock-only
# tar -cf vmnet.tar vmnet-only
# rm -r vmblock-only
# rm -r vmnet-only
# vmware-modconfig --console --install-all

Создал эту тему сразу с пометкой "решено", дабы народ с проблемой этой видел сразу как решить )
