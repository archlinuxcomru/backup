vadik
tune2fs -r 0 /dev/sda1
Отменит резервирование блоков в файловой системе диска sda1. Для корневого раздела естественно этого делать не стоит, а вот для /home и файлопомоек очень даже помогает. Потому как резервировать место под нужды root на дисках, в которые root писать все равно ничего не будет, смысла попросту нет, а на современных дисках такое резервирование может откушать десятки гиг.
Справедливо только для файловой системы ext3/4.
