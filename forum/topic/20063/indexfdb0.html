Клавиатуры с подсветкой бывают двух типов
- подсветка зависит от внешнего освещения и, вроде бы, не имеет ручного управления
- подсветка (вообще то имеет другое название, точно не помню) светится постоянно, но возможно ручное регулирование.
Оба варианта имеют клавишу для отключения.

Согласно описания в данном ноуте имеется три режима подсветки, значит должны быть комбинации клавиш для управления/выбора режима. Кроме этого должны быть комбинации клавиш для включения/выключения подсветки. В связи с чем вопрос - эти комбинации клавиш функционируют?
Частенько в linux эти мультимедийные клавиши не работают. Чтобы они заработали приходится применять разные способы (например, прописать acpi_backlight=vendor и другие).
Кроме того, можно попробовать отследить к каким файлам обращается система при управлении подсветкой. Правда есть нюансы, делают клавиатуры, в которых управление подсветкой не связано с системой.

EDIT 1 - интересно посмотреть вывод команды setleds (в консоле) - будут ли показаны светодиоды, ответсвенные за подсветку?
setleds
Current default flags:  NumLock off   CapsLock off   ScrollLock off
Current flags:             NumLock off   CapsLock off   ScrollLock off
Current leds:              NumLock off   CapsLock off   ScrollLock off
