Аудио плеер - удаление играющего трека с hdd по хоткею

vbrv
# 12 лет назад
Темы: 3
Сообщения: 8
Участник с: 10 августа 2010
Часто качаю сборники и потом нещадно удаляю с них бутор, которого обычно не меньше 90%.
Для этих дел ищу плеер с возможностью удаления играющего трека с hdd по хоткею.
В винде для этих целей отлично подходит AIMP.

Знаю, что подобное можно реализовать в MPD - но в нём только один живой плейлист, а хотелось бы хотя-бы два.
Видел решение для Amarok, но он мне не нравится.

Приглянулись два плеера: Rhythmbox и Exaile.

Решение (не идеальное) для Rhythmbox нашёл, для Exaile ещё не искал (подскажите, если кто знает).

Плагин Rhythmbox delete current song Plugin.
sudo pacman -S python-xlib libnotify
yaourt -S trash-cli
Кидаем плагин в папку /home/юзер/.gnome2/rhythmbox/plugins/

Последняя версия 0.0.3 в Арче из коробки не работает.
Плагин писался под Убунту, там команда (или симлинк?) удаления в мусорку trash, в арче trash-put. Можно поправить скрипт, можно сделать симлинк.
Также автор реализовал одну сомнительную фичу, которая в Арче не работает: для каждого удаляемого трека, в папке /tmp создаётся скрипт, который позволяет восстановить удалённую песню в зад, туда где и была.
Не понимаю зачем это, когда можно просто из корзины вернуть файл туда где он был, без скриптов.
Но в Арче trash-put не имеет прав удалять из папки /tmp. Видимо в Убунте имеет. Кстати, может быть можно trash-put дать права удалять из /tmp ?
$ trash-put test.mp3.sh
trash-put: trash: cannot trash regular file `test.mp3.sh': Permission denied: ‘/.Trash-1000’

В общем правим скрипт, меняем trash на trash-put и комментим сомнительную фичу.
vi /home/юзер/.gnome2/rhythmbox/plugins/deletefile/__init__.py
os.system("trash \""+file+"\" \""+recoversh+"\"")
заменям на
os.system("trash-put \""+file+"\"")
комментим:
#os.system("echo \"mv \\\""+filename+"\\\" \\\""+file+"\\\" && rm \\\"\$0\\\"\" > \""+recoversh+"\"")
#os.system("chmod +x \""+recoversh+"\"")

Перезапускаем плеер, включаем плагин Правка -> Модули - Delete current song from harddisk.

С Rhythmbox есть одна неприятность, он не даёт через консоль удалить песню из плейлиста или сделать его рефреш, поэтому удалённые треки будут болтаться в плейлисте.

UPD: хоткей Win+Up. Как поменять хоткей написано в README.
