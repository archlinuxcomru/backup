Всем привет!

После обновления до KDE 4.7 не могу залогиниться из за ошибки.
…
Подскажите, пожалуйста, что можно сделать для того чтобы определить источник ошибки?

Была точно такая же ошибка, исправил ошибку комплексно:
1. Отключил репозиторий (зеркало) яндекса в настройках pacman (вместо него поставил первый из списка для USA)
2. Удалил KDE c зависимостями (pacman -R kde) и папку ~/.kde
3. Очистил кэш пакмэна (pacman -Scc)
4. Установил следующие пакеты KDE: pacman -S kdebase kdebase-workspace

Теперь тьфу-тьфу все работает.
