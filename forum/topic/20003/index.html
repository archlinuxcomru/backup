В связи с этой проблемой пришёл к выводу, что под катом что-то перестроили при взаимодействии с сенсорами железа. То есть началось это раньше, а явно проявилось в ещё больших случаях после обновления.

Добавить виджеты - Сведения о системе - Memory Usage (для примера) - Настроить виджет - Вкладка Sensors Details. Видим 3 серых окна и задаёмся вопросом: а что дальше?

Берём другой виджет - Disk Usage - и обнаружим, что он устроен точно так же. Во вкладке Appearance меняем в поле Load Preset на Memory Usage. Вуаля, назначение виджета поменялось. И содержание настроек всех других вкладок.

Однако попытка запустить виджет в моём случае для любой настройки из Load Preset закончилась неудачей. Одним словом надо разбираться.
