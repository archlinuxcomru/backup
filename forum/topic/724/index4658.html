А потом нашёл и (вроде как) решение - сменить любую тему на дефолтную Adwaita:
$ gsettings set org.gnome.desktop.wm.preferences theme 'Adwaita'
$ gsettings set org.gnome.desktop.interface gtk-theme 'Adwaita'

После этого сессия Gnome завелась. Традиционно все расширения отпали, часть обновил через extensions.gnome.org, часть погибла (погодный индикатор).

Cinnamon, пересобранный вчера, даже после коррекции тем в cinnamon-settings не завёлся.
