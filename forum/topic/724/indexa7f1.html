Кстати, с NumLock интересная ситуация: при переключении раскладок индикатор гаснет, но само состояние не меняется – NumLock остается включенным. Поэтому удобно использовать вместо индикатора расширение Lock Keys, которое, к тому же, оповещает об изменении состояния не только NumLock, но и CapsLock.
Сейчас опробуем, об индикаторе сообщал выше, но никто не помог, сейчас попробуем.(Жму установить, но он его не устанавливает, что-то неверно делаю? он для 3.6 пригоден?)
Тогда попробуйте поставить ручками из git ;)
Чтобы не заморачиваться, скопируйте и выполните в терминале простую команду:
wget https://github.com/kazysmaster/gnome-shell-extension-lockkeys/archive/master.zip && unzip master.zip && cp -vR gnome-shell-extension-lockkeys-master/[email protected]/ ~/.local/share/gnome-shell/extensions/ && rm -vR master.zip gnome-shell-extension-lockkeys-master/

Исправляем безобразную область уведомлений
Самая некрасивая часть нового ГНОМа 3.6., ИМХО, – ужасная область уведомлений внизу экрана.
Задача: сделать ее меньше размерами (как в старые добрые времена).
РЕШЕНИЕ. Правим 2 файла:
1. /usr/share/gnome-shell/theme/gnome-shell.css
Не забываем сделать резервную копию:
sudo cp -iv /usr/share/gnome-shell/theme/gnome-shell.css /usr/share/gnome-shell/theme/gnome-shell.css.bak
затем открываем в любимом текстовом редакторе: например, 
sudo nano /usr/share/gnome-shell/theme/gnome-shell.css
Находим следующую секцию:
/* Message Tray */
#message-tray {
    background: #2e3436 url(message-tray-background.png);
    background-repeat: repeat;
    transition-duration: 250;
    height: 72px;
}
и исправляем значение height на свое (например, 36px)
2. Таким же образом создаем резервную копию, а затем открываем файл /usr/share/gnome-shell/js/ui/messageTray.js, находим секцию
const Source = new Lang.Class({
    Name: 'MessageTraySource',
    SOURCE_ICON_SIZE: 48,
и исправляем значение SOURCE_ICON_SIZE на свое (у меня 24).
