Всем доброго времени суток!
Вчера, с помощью друга, поставил arch linux себе на компьютер( acer aspire v5-561G)
все шло хорошо, пока не начали настраивать видеокарту

и вот чудом получилось все нормально запустить установкой драйвера intel
P.S:когда пытался установить радеоновский, то разумеется редактировал /etc/mkinitcpio.conf и вписывал в MODULES "radeon"
и вот уже совсем отчаявшись сделал вот что, по совету моего друга, вот решил попробовать:
sudo pacman -S xf86-video-intel
sudo pacman -S mesa
sudo nano /etc/mkinitcpio.conf (указал в MODULES="i915")
sudo mkinitcpio -p linux
sudo reboot

и ЧУДО!все заработало!
Но все бы хорошо…но не запускаются игры..(пытался запустить urbanterror)
