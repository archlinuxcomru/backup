cd ~/.local/share/Steam/ubuntu12_32
file * | grep ELF | cut -d: -f1 | LD_LIBRARY_PATH=. xargs ldd | grep 'not found' | sort | uniq
У меня был вывод:
	libdbus-glib-1.so.2 => not found
	libnm-glib.so.4 => not found
	libnm-util.so.2 => not found
	libopenal.so.1 => not found
	libudev.so.0 => not found
Ищем:
sudo pacman -Fsq libdbus-glib-1.so.2 libnm-glib.so.4 libnm-util.so.2 libopenal.so.1 libudev.so.0
Если не работает поиск, делаем синх:
sudo pacman -Fy
Ну и собственно устанавливаем 32 либы которые покажет.

Теперь steam-native спокойно стартует.
з.ы. steam-runtime работает без этих танцев.
