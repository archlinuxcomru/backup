Дошел до момента Установки qemu и создание образа

systemctl start libvirtd

sudo virsh sysinfo

Пробую создать виртуалку
sudo qemu-system-x86_64 -nographic -m 1024 --enable-kvm -machine smm=off -cdrom CentOS-7-x86_64-Minimal-1503-01.iso -boot order=d image_file
Пробовал с разными ключами, в ответ получаю
(process:15989): GLib-WARNING **: gmem.c:482: custom memory allocation vtable not supported

sudo modprobe virtio
