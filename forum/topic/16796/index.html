Здравствуйте. Подскажите пожалуйста альтернативу.
На debian я устанавливал драйвера для клавиатуры при помощи этого:
apt install g++ libusb-1.0-0-dev
git clone https://github.com/milgner/k290-fnkeyctl
cd k290-fnkeyctl
g++ -std=c++11  k290_fnkeyctl.cpp -lusb-1.0 -o k290_fnkeyctl
install -o root -g root -m 0755 ./k290_fnkeyctl /usr/local/sbin
install -o root -g root -m 0644 ./99-k290-config.rules /etc/udev/rules.d
udevadm control --reload-rules
udevadm trigger --subsystem-match=input
Как быть на arch? Просто без этих драйверов не работают F1-12 клавиши.
