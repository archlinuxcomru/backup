Девушка попросила, "че я не мужик что ли" (с). Написал небольшую программку с гуями. Из текущих возможностей - посчитать БЖУ, калорийность, гликемический индекс. Возможность составить меню с сохранением в файл. Также прилагается текстовая база данных (около 3к наименований, правда без гликемического индекса). И всякие плюшки для "юзер-френдли".
Из зависимостей для скрипта: python2, python2-pyqt, qt. Бинарник вроде не должен зависимостей иметь, но он тяжелый, ибо бинарник из питона. Для сборки нужен git.
Имеется в наличии в ауре: food_gui (дефолтная сборка из бинарников, иначе - чуть чуть поправить PKGBUILD - там все написано).
Исходники, бинарник под венду и прочие мелочи доступны на гитхабе

Мб кому нибудь вдруг полезна будет.
ЗЫ: текущей англ.базой данных лучше не пользоваться :D
ЗЗЫ: ток чур ногами не бить за код) моя специальность вообще даже близко с IT не стоит, потому там все ужасно ><
