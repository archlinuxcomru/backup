Есть два абсолютно одинаковых монитора Sumsung E1920
Видео карта GeForce 8500 GT
Оба монитора имеют только VGA вход, на видео карте VGA и DVI
При подключении монитора по через переходник DVI->VGA установить нужное разрешение невозможно,
Как решить данную проблему?
Просьба показать конфиг для двух мониотров.
Подключил нужный монитор, сохранил бинарный файл, и добавил его в конфиг.
http://bpaste.net/show/156567/

Только мониторы, не включаются с таким конфигом.

Кроме CustomEDID, есть ещё другой способ задания параметров режима – через Modeline в секции Monitor: 
    ModeLine       "1280x1024_60.00"  108.00 1280 1342 1450 1692 1024 1025 1028 1066 +HSync +VSync
#   Modeline       "1280x1024_60.00"  108.00 1280 1328 1440 1688 1024 1025 1028 1066 +hsync +vsync
Обе строки указывают вполне правильный режим для вашего монитора, только с разным сдвигом, только на 60 Гц. Погуглив, вы можете найти аналогичные строчки и на 75 Гц.

И насколько я помню, чтобы заставить драйвер nvidia в одной конфигурации два раза использовать одну и ту же видеокарту, раньше требовалось добавить в КАЖДУЮ из двух секций Device по одной строчке вида 
    BusID          "PCI:2:0:0"
Только цифры заменить на реальный адрес вашей видеокарты на шине PCI, который показывает команда lspci

В секциях Screen в такой конфигурации вроде бы ещё требовались такие опции: 
    Option         "NoTwinViewXineramaInfo" "true"
    Option         "TwinView" "0"
    Option         "DynamicTwinView" "false"
хотя возможно, некоторые из них лишние.

Документация по параметрами конфига для nvidia:
http://us.download.nvidia.com/XFree86/Linux-x86/325.15/README/xconfigoptions.html
