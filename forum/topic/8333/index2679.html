Пытаюсь прошить свой adsl модем через tftp. Для этого установил tftp-hpa. Далее в /etc/hosts.allow добавил
tftpd: ALL
in.tftpd: ALL
в /etc/rc.d/tftpd изменил
TFTPD_ARGS="-l -v -v -v -c -s /var/tftpboot"
запустил сервер, далее
$ sudo chmod 777 /var/tftpboot

Преблема 1: не могу залить на него файлы через tftp. только копированием через cp
$ tftp -v 192.168.1.4
Connected to 192.168.1.4 (192.168.1.4), port 69
tftp> put test
putting test to 192.168.1.4:z [netascii]
Error code 0: Permission denied

Проблема 2: если вручную копирую файл прошивки в /var/tftpboot, то с самого компьютера я ее скачать могу, а вот на модеме я ее никак получить не могу >:( 
[[email protected]:~]$ tftp -v 192.168.1.4

tftp> get firmware

Received 1977896 bytes in -2.7 seconds [-5772182 bit/s]
tftp> q
