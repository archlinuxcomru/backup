лечится отключением автозагрузки видео и гифок
плюс отключением анимации
top_ram.sh
#!/bin/bash
ps axo rss,comm,pid \
| awk '{ proc_list[$2]++; proc_list[$2 "," 1] += $1; } \
END { for (proc in proc_list) { printf("%d\t%s\n", \
proc_list[proc "," 1],proc); }}' | sort -n | tail -n 25 | sort -rn \
| awk '{$1/=1024;printf "%.0fMB\t",$1}{print $2}'
Постоянно на телефоне включенный, на компе открыл закрыл
На старте
155MB	telegram-desktop
