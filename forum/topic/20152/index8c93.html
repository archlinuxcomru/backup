если нужно постоянно, то нужно хук создать, например такой

/etc/pacman.d/hooks/del-telergam-icons.hook
[Trigger]
Type = Package
Operation = Install
Operation = Upgrade
Target = *

[Action]
Description = del telegram icons
When = PostTransaction
Exec = /bin/sh -c 'find /usr/share/icons/|grep -i telegram|xargs rm'

наверно можно сделать
Type = Path
Target = /usr/share/icons/*

Но вы уверены что дело в иконках?
Нет, конечно. Уверенность может дать только длительное тестирование программы, на которое нет ни времени, ни необходимых навыков. Просто есть три факта, имевшие место конкретно на моей системе:
1. до удаления иконок Telegram падал несколько раз за полчаса-час; при этом нужно исключить утечку памяти, поскольку запускался он буквально перед этим, ОЗУ у меня 16 Гб, из них свободны были гигабайт 14-15; плюс падала программа при открытии именно тех подписок, которые имели более 7-10 новых сообщений, и падал именно при подгрузке этих сообщений;
2. после удаления иконок за неделю использования не упал ни разу;
3. переустановка приводит к возвращению пункта 1.
