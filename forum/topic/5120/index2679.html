Stand-alone window manager
To run Openbox on its own, simply add the following to the bottom of ~/.xinitrc:
exec openbox-session
You may also start OpenBox from the shell using xinit:
$ xinit /usr/bin/openbox-session
If you have used another window manager before, such as Xfwm, and Openbox won't start after logging out of X, try moving the autostart folder:
mv ~/.config/autostart ~/.config/autostart-bak
Using D-Bus or similar, use this instead:
exec ck-launch-session openbox-session
If you also use polkit and D-Bus (e.g. for automount drivers in nautilus from Gnome 2.28 / 2.30) use:
exec ck-launch-session dbus-launch openbox-session
Note: pyxdg is required for Openbox's xdg-autostart
