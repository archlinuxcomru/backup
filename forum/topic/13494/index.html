Во время установки с флешки использовал WiFi-Menu, все работало отлично. После того, как флешку вынял и загрузился с установленной системы НИКАК не могу подключится к своему WiFi. Устанавливал на ноутбук.

Пытался подключатся я к WiFi сети с SSID: senpos и БЕЗ пароля. Интерфейс: wlo1

Что я пробовал делать:

  • Настраивал через netcl и профили вручную
    • sudo cp /etc/netctl/examples/wireless-open /etc/netctl
sudo nano /etc/netctl/wireless-open
    И отредактировал его таким орбазом:

    DESCRIPTION='A simple opened wireless connection'
    CONNECTION='wireless'
    INTERFACE='wlo1'
    SECURITY='none'
    ESSID='senpos'
    IP='dhcp'
    # Uncomment this if your ssid is hidden
    #HIDDEN=yes
    После этого 
    sudo netctl start wireless-open
    Результат

    Job for [email protected]\x2dopen.service failed. See 'systemctl status [email protected]\x2dopen.service' abd 'journalctl -xn' for details
    В этих файлах ничего толкового не нашел.
  • Почему бы и из системы не запустить сеть через wifi-menu? Потому что нет пакета dialog. Но ладно, я сделал arch-chroot с установочной флешки и установил его. Перезагрузился обратно в систему, выполнил
    • sudo wifi-menu
    Из списка выбрал свою сеть, но меня встретила та же ошибка что и в первом случае.
  • Почитав мануалы, узнал что нужно установить три пакета: wireless_tools, wpa_supplicant, wpa_actiond. Опять, arch-chroot, установил, перезагрузился.. Результата не дало.
  • Мне посоветовали пойти другим способом и завести сеть так:
    • sudo ip link set wlo1 up
sudo wpa_supplicant -B -i wlo1 -c <(wpa_passphrase "senpos" "")
sudo dhcpcd wlo1
    После второй команды консоль сказала мне, что нет папки /dev/fb* (не записал весь путь).

Что посоветуете сделать? Почему не запускается сеть WiFi на установленной базовой системе? Работать через Ethernet не могу, т.к. провайдер просит ввести логин\пароль их сервиса для открытия доступа, после смены железа. А через консоль это сделать я не знаю как.
