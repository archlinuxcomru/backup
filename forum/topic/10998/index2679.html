1. Устанавливаем файловый менеджер Thunar
pacman -S Thunar
2. Устанавливаем дополнительные пакеты
sudo pacman -S libvirt ebtables bridge-utils openbsd-netcat qemu radvd pm-utils dnsmasq dmidecode
(может что лишнее - извеняйте)
3. Запускаем Thunar, переходим в меню->Правка->Особые действия:
а) Нажимаем на кнопку с изображением плюса, что бы добавить новое действие
б) Заполняем поля в окне :
Имя: 
ISO образ
(можно произвольно)
Описание: 
Открыть ISO образ в KVM
(можно произвольно)
Команда: 
qemu-system-x86_64 -enable-kvm -m 1024  -cdrom %f  или qemu-system-i386 -enable-kvm -m 1024  -cdrom %f
(количество оперативной памяти в данном примере указано 1024Мб, у меня при большем значении iso не запускаются , хотя оперативы 6 ГБ )
в) Теперь переходим на вкладку "Условия появления"
Где "Шаблон имени файла" пишем : 
*.iso
Где "Появляться, если выделение содержит" отмечаем: 
Другие файлы
Нажимаем на "ОК" и закрываем окно "Особые действия"
4. Переходим в каталог c iso файлами для проверки. Теперь, только при клике на файл с расширением iso, в меню будет появляться пункт " ISO образ " по выбору данного пункта меню будет запускаться виртуальная машина

Р.S. Очень удобно , пожалуйста пользуйтесь .
