Последние несколько дней перестал запускаться conky. Поверхностный анализ выявил, что конфиг старый и его пора приводить к новому виду. А новшество, как я понял, заключается в переходе к парсингу конфига при помощи LUA. Коротко о главном:
1. комментарии ( -- или --[[ ]] );
2. конфигурация начинается с ( conky.config = { );
3. переменные, для примера (
alignment = 'top_right',
background = true,
font = '8x13',
) больше нет никаких yes и no, они меняются на true и false;
4. содержательная часть вывода начинается с ( conky.text = [[ );
5. переменной xftfont больше нет, нужно использовать просто font = '...'.

Для создания дефолтного конфига пишем (!!WARNING - сохранить старый конфиг не забудьте!!!):
$ conky -C > ~/.conkyrc
