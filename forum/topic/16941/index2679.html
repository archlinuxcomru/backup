https://gist.github.com/cryzed/e002e7057435f02cc7894b9e748c5671#gistcomment-1965568
Вот одной командой:
sudo pacman -S freetype2 lib32-freetype2 fontconfig lib32-fontconfig
Предложит заменить и соглашаетесь.
Если от infinality у вас стояли ещё cairo-infinality-ultimate, jdk8-openjdk-infinality, jre8-openjdk-infinality - то добавьте ещё к команде выше и эти шрифты:
cairo lib32-cairo jdk8-openjdk jre8-openjdk jre8-openjdk-headless

Потом перезапускаетесь. Если всёё норм - удаляете ключи и зеркала Infinality.
1) Удаляете ключ командой: sudo pacman-key -d 962DDE58
2) Удаляете зеркала Infinality из файла /etc/pacman.conf

А поднастроить малехо не пробовали? У меня векторный шрифт только в браузере. Все нормально без особых наворотов:
/etc/profile.d/freetype2.sh
—————
# truetype:interpreter-version=35  # Classic mode (default in 2.6)
# truetype:interpreter-version=38  # Infinality mode
# truetype:interpreter-version=40  # Minimal mode (default in 2.7)
export FREETYPE_PROPERTIES="truetype:interpreter-version=38"
~/.config/fontconfig/fonts.conf
—————
<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
	<match target="font">
		<edit mode="assign" name="antialias">
			<bool>true</bool>
		</edit>
		<edit mode="assign" name="embeddedbitmap">
			<bool>true</bool>
		</edit>
		<edit mode="assign" name="hinting">
			<bool>true</bool>
		</edit>
		<edit mode="assign" name="autohint">
			<bool>false</bool>
		</edit>
		<!-- Hintstyle avaible: hintnone, hintslight, hintmedium, hintfull -->
		<edit mode="assign" name="hintstyle">
			<const>hintmedium</const>
		</edit>

		<!-- Lcdfilter avaible: lcdnone, lcddefault, lcdlight, lcdlegacy -->
		<edit mode="assign" name="lcdfilter">
			<const>lcdlight</const>
		</edit>

		<!-- RGB avaible: unknown, rgb, bgr, vrgb, vbgr, none -->
		<edit mode="assign" name="rgba">
			<const>rgb</const>
		</edit>
	</match>
</fontconfig>
