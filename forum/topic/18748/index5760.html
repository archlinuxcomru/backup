akorop, да все там есть. Добавьте же, к примеру, в grub:
menuentry "Boot SystemRescueCd using default options" {
    load_video
    insmod gzio
    insmod part_gpt
    insmod part_msdos
    insmod ext2
    search --no-floppy --label boot --set=root
    loopback loop /systemrescuecd.iso
    echo   'Loading kernel ...'
    linux  (loop)/sysresccd/boot/x86_64/vmlinuz img_label=boot img_loop=/systemrescuecd.iso archisobasedir=sysresccd setkmap=us
    echo   'Loading initramfs ...'
    initrd (loop)/sysresccd/boot/x86_64/sysresccd.img
}
menuentry "Boot SystemRescueCd and copy system to RAM"{
    load_video
    insmod gzio
    insmod part_gpt
    insmod part_msdos
    insmod ext2
    search --no-floppy --label boot --set=root
    loopback loop /systemrescuecd.iso
    echo   'Loading kernel ...'
    linux  (loop)/sysresccd/boot/x86_64/vmlinuz img_label=boot img_loop=/systemrescuecd.iso archisobasedir=sysresccd copytoram setkmap=us
    echo   'Loading initramfs ...'
    initrd (loop)/sysresccd/boot/x86_64/sysresccd.img
}
Да, iso переименовать в systemrescuecd.iso , ну или в grub выше исправить на systemrescuecd-6.0.0.iso
У раздела, в который закидывается исошник, должна быть метка boot

Да и вообще , исошник спасательной системы в корне спасаемой системы - бред.
Лучше уж кусок для этих livecd отрезать от "нижнего" раздела. Вдруг понадобится корень ресайзить, да ту же метку поставить и т.п.

to akorop - уточнение параметра archisobasedir=sysresccd
В нашем установочном archiso (ARCH) этого параметра нет, так как по дефолту стоит arch (archisobasedir=arch), а значит прописывать его и не нужно.
Привожу описание некоторых параметров archiso
archisolabel= Set the filesystem label where archiso files reside. Default: (unset)
archisodevice= Set the device node where archiso medium is located. Default: "/dev/disk/by-label/${archisolabel}"
archisobasedir= Set the base directory where all files reside. Default: "arch"
copytoram= If set to "y" or just "copytoram" without arguments, all SquashFS are copied to "RAM". Default: (unset)
Ошибки не исчезают с опытом - они просто умнеют
