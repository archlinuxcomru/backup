Доброго времени суток, уважаемые форумчане!
Решение данной проблемы искал, но усехом мои поиски не увенчались...
Монтирую ntfs в fstab по wiki:
/dev/sdb2 /mnt/WIN ntfs-3g uid=userid,gid=groupid 0 0
или так:
/dev/sdb2 /mnt/WIN ntfs-3g force,showexec,rw,user,uid=1000,locale=ru_RU.UTF-8 0 0
После монтирования диск в dolphin виден, но в папке /mnt ничего не показывает, а значит не виден в других программах, где требуется указать каталог.
Пробовал менять права на папки и на саму /mnt. Толку ноль.
Автомонтирование силами kde, не дает доступ, пока не зайдешь на диск через dolphin хотя бы раз.
Прошу направить в нужном направлении, очень хочется автоматически монтировать пару дисков.
Может я что то не правильно делаю? Может есть более простые способы автомонтирования?
Какие логи приложить?
