Имеется arch linux с plasma и nvidia.
Ставлю пакет linux-xanmod-anbox с AUR.
Запускаю систему с новым ядром, sddm не запускается, загрузка кончается на start hostname service, плюётся какими-то audit, в логах видно.
Перехожу в tty2, сохраняю логи, пытаюсь запустить sddm через systemctl start sddm и просто sddm, просит аутентификации и ничего не происходит, кроме тех же аудитов.
Иксы стартуют (startx), только с mesa драйвером(у меня nvidia)
На стандартном ядре всё запускается, пробовал запуск с fallback initramfs, пробовал переустанавливать иксы, sddm, nvidia, nvidia-dkms. Проблема уже была раньше и с linux-xanmod (без anbox).

Ставил LightDM, не запускается, в journalctl status ошибки "Start request repeated too quickly", "Failed with result exit-code".
И ещё доп. вопрос, есть ли пакеты уже собранного xanmod для арча? Мб собрать ядро вручную (без aur), как это сделать?
