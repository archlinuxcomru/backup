Вы gparted через sudo запускаете?
Если через sudo - то переменные окружения по умолчанию не передаются (соответственно не применяются настройки шрифтов)
Для передачи переменных запускайте sudo с опцией -E
sudo -E gparted
или отредактируйте /etc/sudoers Defaults env_keep += переменная
https://wiki.archlinux.org/index.php/Sudo#Environment_variables
Можно еще использовать gksu
gksu gparted

Настойки для рута находятся в папке /root
можно просто создать ссылки на настройки юзера
ln -s /home/oleg/.gtkrc-2.0 /root/.gtkrc-2.0
ln -s /home/oleg/.config/gtk-3.0/settings.ini /root/.config/gtk-3.0/settings.ini
ln -s /home/oleg/.config/fontconfig /root/.config/fontconfig
