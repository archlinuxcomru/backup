Сейчас в alsa dmix включен по дефолту.
В настройках плеера установите вывод звука в alsa default, а asound.conf приведите, например, к такому виду:
pcm.real {
type hw
card 0
device 0
}
ctl.real {
type hw
card 0
device 0
}

add.
Flashplayer ориентируется на этот конфиг и по идее должен работать и с таким видом:
pcm.!default {
type hw
card 0
}
ctl.!default {
type hw
card 0
}
но уж не помню почему я внес в него изменения. По-моему, именно в флешплеере с таким конфигом в начале воспроизведения были слышны щелчки.
