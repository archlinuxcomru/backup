vhd тут нафиг не нужен.
образ создается с помощью dd.
форматируется в линуксовую фс без разделов.
граб поддерживает загрузку из образа с помощью loopback.
при этом в начальный загрузчик граба можно встроить конфиг и нужные модули, чтобы его файлы не валялись в разделе.
в инитрам сначала должен монтироваться сначала раздел с файлом корень примонтируется сам, т.к. путь до файла должен передаваться из граб.
в инитрамфс должны быть встроены модули фс раздела и корня.

в принципе ничего сложного.

команда не даю, т.к. с собой нет компа.

Это ты читай мануал по драйверам к твоему VHD.

Например, для iso-образа параметр примерно вот так передается:
linux /linux iso-scan/filename=/install/mini32.iso
для ubuntu

либо вот так:
linux (loop)/arch/boot/i686/vmlinuz archisolabel=$isolabel
для арча

Разумеется, втупую копировать не надо, у тебя будет все по-другому.
