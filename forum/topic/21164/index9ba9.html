Про ядра ничего не увидел
-

Если rEFInd автоматически обнаружит ваше ядро, вы можете поместить файл refind_linux.conf, содержащий параметры ядра, в тот же каталог, что и ваше ядро. Первая незакомментированная строка refind_linux.conf будет параметрами по умолчанию для ядра. Последующие строки будут создавать записи в подменю, доступном с помощью +, F2 или Insert.

Вот чтобы не размазывать конфиги по куче мест лучше имхо использовать один основной refind.conf
