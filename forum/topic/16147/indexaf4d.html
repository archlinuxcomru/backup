man xkill
Xkill is a utility for forcing the X server to close connections to clients. This program is very dangerous, but is useful for aborting programs that have displayed undesired windows on a user's screen.

Xkill утилита для принудительного закрытия соединения клиента от X сервера. Эта программа очень опасна, но может быть полезна для прерывании программ, которые отображаются нежелательные окна на экране пользователя.
дело в том что программы перед закрытием могут выполнять какие нибудь действия, например сохранение той же сессии, а при киле они эти действия не выполняют, а тупо уничтожаются, что может привести к нежелательным, а иногда к фатальным последствиям!
