На вкус и цвет, конечно, все фломастеры разные, но лично я предпочитаю вместо Stylish использовать файл userContent.css в профиле фокса (~/.mozilla/firefox/profile_name/chrome/userContent.css) - тот же самый эффект, но на один аддон в фоксе меньше.
Просто прописать первой строкой
@namespace url(http://www.w3.org/1999/xhtml);
для применения стиля глобально, а дальше обычный css-код
Чтобы применять к определённому сайту после первой строки пишем так
@-moz-document domain("archlinux.org.ru") {
css-код
}
Я себе так забабахал тёмную тему для убунтовского форума.

Единственный недостаток - для применения изменений требуется перезапуск браузера (в Stylish изменения применяются "на лету")
Кстати, стиль Textbox color fix состоит из одной единственной строки
body textarea, body input {background-color: white !important; color: black !important;}
)))
Я у себя в userContent.css прописал это несколько иначе
input,textarea {
  background-color: #eee !important;
  color: #000 !important;
}

P.S> кстати, там же (~/.mozilla/firefox/profile_name/chrome/) есть файлик userChrome.css, в котором можно прописать правила изменения для морды фокса. Хотя тут несколько сложнее, чем с контентом - нужно точно знать названия элементов, которые хочешь изменить. Но это легко решается с помощью DOM Inspector. Я одно време активно баловался с этим файлом - перекраивал морду по своему вкусу. Потом надоело.
