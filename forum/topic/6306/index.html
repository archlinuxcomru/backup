В настройке DHCP-сервера сложного нет.
Я проделал следующие шаги:
1) Установил pacman -S dhcp
2) Отредактировал фыйл /etc/dhcpd.conf
ddns-update-style none;
option domain-name "example.com";
option domain-name-servers 192.168.10.1;
default-lease-time 259200;
max-lease-time 259200;
authoritative;
log-facility local7;
subnet 192.168.10.0 netmask 255.255.255.0 {
range 192.168.10.25 192.168.10.254;
option domain-name-servers 192.168.10.1;
option domain-name "example.com";
option routers 192.168.10.1;
option broadcast-address 192.168.10.255;
default-lease-time 259200;
max-lease-time 259200;
}
3) сделал в автозапуск с системой, добавив в файл /etc/rc.cong
DAEMONS=(syslog-ng network dhcpd netfs crond)

и тут возник вопрос.
Нам необходимо, чтобы DHCP сервер обрабатывает запросы только с интерфейса eth0 (из внутренней сети), как это сделать??

у кого какие мысли есть на это? поделитесь!
