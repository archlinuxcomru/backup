Решение:
Откат cups на предыдущую версию
~]$ yaourt -S downgrade
~]$ downgrade cups
~]$ 2
В SystemGroup вместо sys ставим lp
~]$ sudo gedit /etc/cups/cupsd.conf
#
# "$Id: cupsd.conf.in 9407 2010-12-09 21:24:51Z mike $"
#
# Sample configuration file for the CUPS scheduler.  See "man
cupsd.conf" for a
# complete description of this file.
...
LogLevel warn
# Administrator user group...
SystemGroup lp root
# Only listen for connections from the local machine.
Listen localhost:631
...
К слову сказать в последнем cups вообще нет такого раздела в файле /etc/cups/cupsd.conf
# Administrator user group...
SystemGroup sys root
Думаю его можно было легко добавить, но из-за глюка с двойным меню не стал заморачиваться и просто откатил на предыдущую версию...
