Доброго времени суток!
Во время установки линукс арч не удается подключиться к wifi, а именно:

Проблема решена:
подключил телефон через USB (usb модем) интернета все равно не было (проверял через пингование DNS сервера ping 8.8.8.8) выдавало нет доступа. Посмотрел драйвер - r8169 Realtek. Далее сделал следующее:
----modprobe r8169
----ip link set dev eth0 up
----dhcpcd eth0

После этого снова проверил пинг DNS сервера и все заработало, всем спасибо.
