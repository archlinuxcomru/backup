Такое костыльное решение ( впрочем как и программа )
sudo pacman -R skypeforlinux-bin
rm -R  ~/.config/skypeforlinux
yaourt -S  skypeforlinux
skypeforlinux
логинимся вводим пароль затем закрываем программу не выходя из логина.
Потом
sudo pacman -R skypeforlinux
yaourt -S skypeforlinux-bin
skypeforlinux

Должно работать пока не выйдете из логина
А вообще зачем вам этот пожиратель оперативки. Есть же skype ( aur/skype 4.3.0) , на худой конец вкладка в браузере
