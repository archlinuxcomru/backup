/etc/polkit-1/rules.d/49-nopasswd_limited.rules

/* Allow members of the wheel group to execute the defined actions
 * without password authentication, similar to "sudo NOPASSWD:"
 */
polkit.addRule(function(action, subject) {
    if ((action.id == "org.archlinux.pkexec.gparted" ||
	 action.id == "org.libvirt.unix.manage") &&
        subject.isInGroup("wheel"))
    {
        return polkit.Result.YES;
    }
});

Ошибки надо устранять последовательно.
Если я говорю, что параметры (или звёздочка) необходимы, значит без этого точно не заработает.
Синтаксис такой:
юзер хостнейм=NOPASSWD:/путь/имя параметры
Запуск, соответственно:
sudo /путь/имя параметры
Команда после sudo должна быть написана точно так, как в sudoers, включая путь (если есть в sudoers) и все параметры, иначе не разрешит.
