а где (как) эти параметры можно менять?
Например, последние 2 параметра (tuner и card)
tuner:tuner type (array of int)
card:card type (array of int)
при загрузке модуля saa7134 передать эти параметры, если, конечно у тебя задействован точно этот модуль. Список значений этих параметров, соответствующих конкретному железу, можно узнать в исходниках ... или погуглить. Частенько это помогает при решении проблем.
Кстати, там же есть и другие параметры, например, отладочные - audio_debug:enable debug messages [tv audio] (int) и др. Но нужно знать, что они обозначают.

EDIT 1 - например, для BeholdTV 507 FM (VID:PID=5ace:5070) ..... card=127 ..... вот только не уверен твое ли это?
