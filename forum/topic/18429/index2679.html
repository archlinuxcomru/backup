Всем привет!
Установил arch linux на новый laptop и столкнулся с ошибками которых раньше не наблюдал.
А именно: Система НЕ может выключиться, перезагрузиться, уйти в ждущий режим и т.д. (появляется черный экран, без признаков жизни)
Периодически виснут Kosole и xfce4-terminal, причем тянут за собой всю систему помогает только hard reset.
Перезагружаться пробовал и через терминал, команда sudo reboot, иксы виснут намертво (без перехода к черному экрану), приходится перезагружаться с "кнопки"
Установка чистая, ничего "лишнего" не устанавливал кроме DE
Kernel: x86_64 Linux 4.16.5-1-ARCH
DE: KDE 5.45.0 / Plasma 5.12.4
CPU: Intel Core i7-7700HQ
GPU: Mesa DRI Intel(R) HD Graphics 630 (Kaby Lake GT2)
Arch установлен рядом с widows10:
boot - boot/uefi - ssd диск,
swap - 4Gb - ssd диск,
root - 46Gb - ssd диск,
home - ~420Gb - hdd диск

Драйвера на nvidia еще не ставил, стоят только xf86-video-intel

При загрузки системы замечено следующее системное сообщение:
ACPI Error: [^^^PEG0.PEGP.EASP]

Отчаянно прошу помощи!!!
