В nginx это выглядит как:
server_name archlinux.org.ru *.archlinux.org.ru;
Алиасов нет.

В секции сайта сделайте как было: 
server_name archlinux.org.ru;
Для редиректа добавьте отдельную секцию: 
server {
 listen       80;
 server_name *.archlinux.org.ru;
 rewrite .* http://archlinux.org.ru/;
}
