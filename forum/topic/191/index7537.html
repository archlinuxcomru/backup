Пакман вроде как без глаза был, но я решил от заявки не отклоняться. Вот что получилось:

<!doctype html>
<html>
	<head>
		<meta charset="utf-8" />
		<style>
			.pacman-body {
				width: 20px;
				height: 20px;
				border-radius: 10px;
				background: #1E90FF;
			}
			.pacman-eye {
				width: 4px;
				height: 4px;
				border-radius: 2px;
				position: relative;
				top: -16px;
				left: 6px;
				background: #fff;
			}
			.pacman-mouth {
				width: 0;
				height: 0;
				border-top: 4px solid transparent;
				border-right: 12px solid #fff;
				border-bottom: 4px solid transparent;
				position: relative;
				top: -16px;
				left: 8px;
			}
		</style>
	</head>
	<body>
		<div class="pacman-body"></div>
		<div class="pacman-eye"></div>
		<div class="pacman-mouth"></div>
	</body>
</html>
