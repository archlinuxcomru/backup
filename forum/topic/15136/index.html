Доброго времени суток!Являюсь новеньким в ваших рядах,и только ступил на путь Арча.В процессе установки возникла проблема с распаковкой базовой системы.Тобишь с командой:
pacstrap /mnt base base-devel
По выполнению команды pacstrap находит нужные пакеты и при попытке установить выбивает следующее:

error:could not open file: /etc/mtab: No such file or directory
error:could not determine filesystem mount points
error:failed to commit transaction(unexpected error(
Error occurred,no packages were upgraded.
ERROR: Failed to install packages to new root
Из сего я понял,что у команды не выходит найти файл mtab,который содержит в себе все смонтированные точки.Поискал я эти файлы.Нашел в корне /etc/mtab и /mnt/etc/mtab,оба файла при команде ls светятся красным цветом,и при этом нечитаемы,и ничего я с ними сделать не могу.По поиску ничего не нашел,в вики аналогично.Надеюсь на вашу помощь:)
