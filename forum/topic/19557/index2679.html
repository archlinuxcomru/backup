У меня был wifi-адаптер с чипом rtl8192eu. Драйвера кажись из AURа ставил тогда.
Я уже давно не пользуюсь этим wifi-адаптером и не планирую в будущем.

Уже просто достало при обновлении через pacman сообщение
(10/27) Install DKMS modules
==> dkms install rtl8192eu/1.0 -k 5.3.8-arch1-1
Error! Bad return status for module build on kernel: 5.3.8-arch1-1 (x86_64)
Consult /var/lib/dkms/rtl8192eu/1.0/build/make.log for more information.

Модуль не загружен (lsmod | grep rtl8192). Пакета с таким названием тоже в системе нет.

Команду 
dkms remove..
пробовал. Пишет что удалено.
# dkms remove rtl8192eu/1.0 --all

------------------------------
Deleting module version: 1.0
completely from the DKMS tree.
------------------------------
Done.
Но при очередном обновлении вижу, что результата - ноль.

Где и что нужно почистить чтобы избавиться навсегда от остатков этого пакета и соответственно от этих сообщений в консоли?
