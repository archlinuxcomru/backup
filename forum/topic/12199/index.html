Добрый день! Сегодня наконец настроил авто-подключение к интернету через netctl после загрузки ОС, как все сломал :)

Установил android-sdk. Для adb прописал в ~/.bashrc
PATH="${PATH}:/opt/android-sdk/platform-tools/"

Посмотрел netctl status wireless-wpa
12:15 [[email protected]]$ ~: netctl status wireless-wpa
[email protected]\x2dwpa.service - A simple WPA encrypted wireless connection
   Loaded: loaded (/etc/systemd/system/[email protected]\x2dwpa.service; enabled)
   Active: failed (Result: exit-code) since Sun 2013-08-18 12:15:35 FET; 21s ago
     Docs: man:netctl.profile(5)
  Process: 216 ExecStart=/usr/lib/network/network start %I (code=exited, status=1/FAILURE)
Попробовал sudo netctl reenable wireless-wpa, но не помогло.

Теперь приходися пользоваться sudo netctl start wireless-wpa, что не комельфо.
