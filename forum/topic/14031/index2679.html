Надо посмотреть выхлоп xprop на предмет класса окна. Если окна ворда и экселя идентифицируются по разному, то можно

Upd.: Выхлоп xprop. Первое значение - instance, второе - class

Excel

WM_CLASS(STRING) = "EXCEL.EXE", "Wine"

Word

WM_CLASS(STRING) = "WINWORD.EXE", "Wine"

Далее настраиваем так

~/.config/awesome/rc.lua

{

    rule = { class = "Wine", instance = "EXCEL.EXE"},
    properties = { tag = tags[1][1], maximized_horizontal = true, maximized_vertical = true, floating = false}

  },

{

    rule = { class = "Wine", instance = "WINWORD.EXE"},
    properties = { tag = tags[1][2], maximized_horizontal = true, maximized_vertical = true, floating = false}

  },
И будет эксель на 1м теге, а ворд на 2м
