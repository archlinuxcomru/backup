Локаль настроили?
Дайте вывод
locale
и
locale -a

Для настройки смотрите вики
https://wiki.archlinux.org/index.php/Locale#LANG:_default_locale

В /etc/locale.gen
# Configuration file for locale-gen.
en_US.UTF-8 UTF-8
ru_RU.UTF-8 UTF-8
#
выполнить
sudo locale-gen
В /etc/locale.conf
LANG="ru_RU.UTF-8"
Перезагрузиться
