Проблема в самом установщике. Решение нашел тут: http://www.google.com/support/forum/p/e … d4e2&hl=en

Надо распаковать файлы из .run файла и заставить установщик использовать старую версию гтк:
./GoogleEarthLinux.bin –target /tmp/ge
cd /tmp/ge/setup.data/bin/Linux/x86/
mv setup.gtk setup.gtk2
cd /tmp/ge
LC_ALL=en_EN ./setup.sh
