Сегодня другу установили Arch Openbox,всё нормально но в браузере firefox очень бледный текст,с трудом читается.Замена и настройка шрифтов в lxapparance-obconf и в obconf естественно на шрифты в браузере не влияют.Был с собой USB HDD с установленным Debian openbox ,там всё нормально.
Просмиотрел в чём разница- в debuan есть папки /usr/share/fonts и /usr/share/fontsconfig а в arch только папка /usr/share/fonts/.
Когда внаглую скопировал папку /usr/share/fontsconfig из debian в arch шрифты в браузере стали намного лучше.
Никогда не заморачивался с шрифтами,мне лиш-бы было всё чётко и понятно и всегда так и было сколько раз ни устанавливал Arch,не знаю что сегодня получилось.

Vadim, у меня тоже openbox и тоже ubuntu fonts. Шрифты в Арче настраиваются элементарно: надо просто создать симлинки из каталога /etc/fonts/conf.avail в /etc/fonts/conf.d. Вот настройки, которые я добавлял:
[~]» ls -la /etc/fonts/conf.d
...
lrwxrwxrwx 1 root root   30 июн 22 05:21 10-autohint.conf -> ../conf.avail/10-autohint.conf
lrwxrwxrwx 1 root root   34 июн 22 05:21 10-no-sub-pixel.conf -> ../conf.avail/10-no-sub-pixel.conf
...
lrwxrwxrwx 1 root root   35 июн 22 05:21 10-sub-pixel-rgb.conf -> ../conf.avail/10-sub-pixel-rgb.conf
lrwxrwxrwx 1 root root   30 июн 22 05:21 10-unhinted.conf -> ../conf.avail/10-unhinted.conf
lrwxrwxrwx 1 root root   39 июн 22 05:21 11-lcdfilter-default.conf -> ../conf.avail/11-lcdfilter-default.conf
...
Еще укажите предпочтительные шрифты в файле /etc/fonts/conf.avail/60-latin.conf:
[~]» cat /etc/fonts/conf.avail/60-latin.conf
...
		<family>serif</family>
		<prefer>
			<family>Liberation Serif</family>
			<family>DejaVu Serif</family>
...
		<family>sans-serif</family>
		<prefer>
			<family>Ubuntu</family>
			<family>DejaVu Sans</family>
...
		<family>monospace</family>
		<prefer>
			<family>Ubuntu Mono</family>
			<family>DejaVu Sans Mono</family>
...
