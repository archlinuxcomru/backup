pethead,
sudo используется в консоли/терминале, поэтому ввести пароль можно только запуская сабж в терминале
чтобы иметь возможность запускать любое приложение с повышенными привилегиями без использования терминала, нужно к команде запуска добавлять gksu/gksudo или kdesu, в зависимости от окружения
типа вот так
gksu cpu-g
в десктоп-файле, который Вы обзываете ярлыком, нужно подправить строку Exec=команда_запуска, тогда при запуске будет появляться окно для ввода пароля
