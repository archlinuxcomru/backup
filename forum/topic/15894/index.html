Здравствуйте! Господа, помогите разобраться, пожалуйста. Дело в том,что Arch установлен из back-up имиджа с помощью Redobackup. Система вполне рабочая с необходимым набором программ. Но работа сети в системе только через DHCP, настроить на статический адрес не получается. Проделал следующее:

То есть интерфейс присутствует и запущен
Далее пробую назначить адрес руками #ip addr add 192.168.x.x broadcast 192.168.x.x dev eth0 и #ip route add default via 192.168.x.x
прописываю DNS, пингуется только шлюз. После reboot настройки теряются.

Спасибо большое pethead, решено следующим образом:
в /etc/dhcpcd.conf меняем на

interface eth0
static ip_address= 192.168.x.x
static routers= 192.168.x.x
static domain_name_servers= 192.168.x.x

перезапускаем
#systemctl restart dhcpcd.service

и вуаля!
