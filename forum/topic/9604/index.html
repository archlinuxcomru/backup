Здравствуйте. Первый раз ставлю Арч, ситуация такая:
1. Скачал CORE-образ, проверил md5 - совпало.
2. Прожёг на CD-RW.
3. Гружусь с диска, всё проходит отлично до, собственно, меню установщика.
4. Успешно монтирую разделы, захожу в меню выбора пакетов.
5. В списке категорий получаю что-то вроде:
“- base
- input
- file”
Короче говоря, почему-то категории некорректно отображаются.
6. Выбираю, допустим, base. Жму, Ok, выбираю пакеты по умолчанию - yes. Меня выбрасывает в главное меню.
7. Жму install packages, оно мне говорит, что нужно сначала эти пакеты выбрать…

Если нажать no в вопросе о выборе пакетов по умолчанию - тоже самое.

В гугле и по форуму ответов не нашел. Что посоветуете? Такое впечатление, что почему-то некорректно читается какой-то файл со списком категорий и пакетов… Грешу на криво прожженный образ.
