Стоит последнее ядро linux-pf. Настраивал TuxOnIce по вики. При выполнении команды
# hibernate -F /etc/hibernate/tuxonice.conf
ровно через двадцать секунд происходит ошибка:
hibernate: TuxOnIce reported the following errors:
 - TuxOnIce was aborted (see dmesg).
 - Freezing processes failed. Wacky driver problems :( (see dmesg)
Вот вывод dmesg|grep TuxOnIce:
[    0.772676] TuxOnIce 3.3 (http://tuxonice.net)
[    1.108357] TuxOnIce: Can't translate "/dev/sda1" into a device id yet.
[    3.041833] TuxOnIce: No image found.
[  430.831323] TuxOnIce: No image found.
[  430.939170] TuxOnIce 3.3, with support for usm, compression, block i/o, swap storage, file storage, userui.
[  451.931094] TuxOnIce debugging info:
- TuxOnIce core  : 3.3

В чем может быть проблема? Уже все гуглы перелопатил, ничего не помогло. Обычная гибернация с обычным ядром работает без проблем. Хук resume в mkinitcpio выставлен, параметр ядра resume=/dev/sda1 в загрузчике выставлен.
