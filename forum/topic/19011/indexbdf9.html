Свежайшая версия IceArch 0.999 доступна для скачивания :
http://tiny.cc/9061tz
контрольная сумма : 9e423aca9b67148c692e2fc6316082fe
Из изменений по сравнению с предыдущей версией:
1) переезд сборочницы на новую версию archiso
2) В IceDM добавлены новые полезные функции :

3) i3 в сборке переведен на панель tint2 с возможностью прямо в графическом конфигураторе добавлять на нее
кнопки запуска приложений, настроено jgmenu :

4) 20% оперативки выделено под zram ( пакет zramswap ):

5) В сборку добавлены icedisk ( для операций с разделами - надстройка над cfdisk c опцией форматирования ) и обновленная
montirovka .
6) добавлен скрипт klishe.sh (в Icewm-е на хоткее "Win+o") - dialog-скрипт запуска юзер-контекст-скриптов применительно к вхождению(буферы-обмена/"ручной ввод") [# Иксы, Консоль] Можно сказать что это "контекстное меню"( по аналогии например с файл-менеджером) для "вхождения".
(подробности - ~/.config/klishe/Readme.txt - https://www.hostedredmine.com/attachments/379795)
7) В сборку добавлен отладчик gdb и соответствующий ему плагин для Vim :
https://www.dannyadam.com/blog/2019/05/debugging-in-vim/
Из vim : этот плагин запускается :

:packadd termdebug
:Termdebug
