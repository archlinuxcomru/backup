После установки Арча стоял и успешно работал экранный менеджер lxdm. Захотел поставить lightdm. Предпринятые действия:

sudo pacman -Rds lxdm
sudo rm /etc/systemd/system/display-manager.service
sudo pacman -S lightdm
systemctl enable lightdm.service

Вроде всё проходит успешно, хотя при попытке перезагрузиться, с рабочего стола исчезает всё, кроме обоев, и система замирает наглухо, ребутиться приходится кнопкой Reset. После перезагрузки загрузка замирает наглухо на следующих надписях:

.............
FAILED to start Light Display Manager
OK Reached target graphical interface

И всё, однако, никакого тебе lightdm. Как говорится, что я делаю не так? Обратные же, точно такие же, действия по установке lxdm обратно на место, приводят к результату сразу и без проблем. Действовал строго по https://wiki.archlinux.org/index.php/Display_manager_(%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9)#.D0.97.D0.B0.D0.BF.D1.83.D1.81.D0.BA_.D1.8D.D0.BA.D1.80.D0.B0.D0.BD.D0.BD.D0.BE.D0.B3.D0.BE_.D0.BC.D0.B5.D0.BD.D0.B5.D0.B4.D0.B6.D0.B5.D1.80.D0.B0
