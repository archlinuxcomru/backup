Я в системе новичок. Поэтому могу многого не понимать. Извиняюсь если предоставляю информацию не в полном объёме или не ту.
Теперь по порядку.
Стоит chromium. При открытии нескольких вкладок одновременно жутко тормозит систему. Проц 90-100%, ОЗУ 60-70%.
В связи с этим решил попробовать другие браузеры. Опера не понравилась, Мозилла тоже. Разница в потребляемости ресурсов в сравнении
с хромиумом оказалась незначительной. Остановился на qupzilla.
qupzilla ставилась через yaourt. Какое комюнити - без понятия. А какое используется по умолчанию?
Запуск браузера в терминале:
[[email protected] ~]$ qupzilla
AdBlockSearchTree: Inserting rule with filter len <= 0!
QupZilla: 0 extensions loaded
SpellCheck: Dictionaries for "/usr/share/hunspell/ru_RU" doesn't exists!

attempt to connect to server failed
