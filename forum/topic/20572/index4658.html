azaqthoth
fdisk: cannot open /dev/sdd: Permission denied
Для начала неплохо бы посмотреть выводы (из вывода lsblk узнаем, что USB есть /dev/sdN - замени N на свое)
sudo fdisk -l /dev/sdN
sudo hexdump -C -s 446 -n 66 /dev/sdN
sudo hdparm -Tt /dev/sdN (скорость чтения ... скорость записи через dd, но можно пока не тестировать)

PS - если hdparm не установлен, то рекомендую установить, заодно посмотришь полную инфу о флэшке - sudo hdparm -I /dev/sdN

EDIT 1 - в части проверки
azaqthoth
запустил команду sudo badblocks -vws /dev/sdd
рекомендую протестировать утилитой whdd - покажет всю картинку по блокам в части скорости доступа .... описано в этом топике
Ошибки не исчезают с опытом - они просто умнеют
