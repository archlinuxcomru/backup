Здравствуйте,
установил на ноутбуке Arch (Core ISO i686).
Всё делал по инструкции из вики.
Установка прошла успешно, правда, активация свопа при загрузке фэйлилась (но это победил mkswap).
Проблема в том, что никак не получается настроить сеть.
Дома adsl-модем D-Link, настройки pppoe прописаны в нём, dhcp отключен, IP модема 192.168.1.198.
Компьютеры подключены к свитчу, а свитч к модему.
В /etc/rc.conf прописал hostname и настройки для статического IP:
eth0="eth0 192.168.1.25 netmask 255.255.255.0 broadcast 192.168.1.255"
INTERFACES=(eth0)
gateway="default gw 192.168.1.198"
ROUTES=(gateway)
В /etc/resolv.conf указал dns
nameserver 212.19.2.1
nameserver 212.19.3.1

При попытке пропинговать шлюз, выдает Destination Host Unreachable. Пинг любого интернет ресурса не выдаёт вообще ничего.
Искал по форуму, гуглил… решение найти не удалось… может кто подскажет, в чем может быть проблема?
