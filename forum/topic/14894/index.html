Ни один эквалайзер не меняет звук.

Доброго времени суток!
Настроил по дефолту свою карту в asoundconf. Теперь везде слышен звук. Поставил alsaequal, настроил его, запускается. Вот конфиг файл (/etc/asound.conf):

ctl.equal {
    type equal;
}

pcm.plugequal {
    type equal;
    # Modify the line below if you do not
    # want to use sound card 0.
    #slave.pcm "plughw:0,0";
    # by default we want to play from more sources at time:
    slave.pcm "plug:dmix";
}

# pcm.equal {
# If you do not want the equalizer to be your
# default soundcard comment the following
# line and uncomment the above line. (You can
# choose it as the output device by addressing
# it with specific apps,eg mpg123 -a equal 06.Back_In_Black.mp3)
pcm.!default {
    type plug;
    slave.pcm "plugequal";
}
Проблема в том, что при изменении показателей в эквалайзере нем звук он не меняет. Прочел множество тем по этому сабжу, вроде все правильно сделал (в конфигах точно). Причем не только alsaequal не меняет ни чего, даже в foobnix в своем эквалайзере не меняет ничего. Совсем пичаль. Есть какие-нибудь предложения что попытаться сделать?

P.S.
Не сижу под рутом. Еще интересует как поменять частоту дискретизации и настроить 7.1 Virtual Speaker Shifter ну и долби всякие.
