Привет всем. Я случайно выключил Journal в Systemd. Обратно не могу вернуть:
sergey ~ $ sudo systemctl enable systemd-journald.socket
The unit files have no [Install] section. They are not meant to be enabled using systemctl.
sergey ~ $ sudo systemctl enable systemd-journald.service
The unit files have no [Install] section. They are not meant to be enabled using systemctl.
Помогите, пожалуйста.

ln -sr /usr/lib/systemd/system/systemd-journald.socket /usr/lib/systemd/system/sockets.target.wants/systemd-journald.socket
ln -sr /usr/lib/systemd/system/systemd-journald.service /usr/lib/systemd/system/sysinit.target.wants/systemd-journald.service
