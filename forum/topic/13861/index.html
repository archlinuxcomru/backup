Grub2 на btrfs завелся без проблем, но моя цель завести на нём syslinux, благо недавно вроде как разрабы запилили данную возможность начиная с версии 6.03pre13-1.
btrfs создаётся на самом устройстве, а не на отдельном разделе:
mkfs.btrfs /dev/sda
содержащий два подтома:
btrfs subvolume create /mnt/root
btrfs subvolume create /mnt/home
устанавливается загрузчик syslinux:
pacman -r /mnt -Sy --noconfirm syslinux
syslinux-install_update -iam -c /mnt
