netctl start lan-config

Пробую через dhcpcd. В /etc/dhcpcd.conf указываю
interface enp3s0
static ip_address=xxx.xx.xxx.xx/24
static routers=xxx.xx.xxx.1
В /etc/resolv.conf
nameserver=8.8.8.8
nameserver=8.8.4.4
