то такие вещи в пакете base ест
Он то есть но его запускать надо.
Гляньте может и запущен, а не работает
systemctl status dhcpcd.service
Если не запущен
активируйте сервис
systemctl enable dhcpcd.service
systemctl start dhcpcd.service
для вайфая может еще хук понадобится прописать, кто пользуется может подскажут нужен ли или гляньте вики по тому чем вайфай запускаете
см https://wiki.archlinux.org/index.php/Wireless_network_configuration
https://wiki.archlinux.org/index.php/Dhcpcd#10-wpa_supplicant
