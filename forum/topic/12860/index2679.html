Пример
 {
    rule = { class = "qemu-system-x86_64" },
    properties = { floating = true, maximized_horizontal = false, maximized_vertical = false, ontop = true, geometry = {x = 1100, y = 500} }
  },
Параметр geometry определяет координаты левого верхнего угла окна. Взято из моего конфига - надо было располагать эмулятор в определенном месте экрана
