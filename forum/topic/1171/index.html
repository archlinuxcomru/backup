Firefox (Правка) –> Настройки –> Дополнительно –> Общие –> “Системный настройки” –> Установить галочку “Всегда проверять при запуске, является ли Firefox браузером по умолчанию” –> Нажать “Проверить сейчас” –> “Да”

Параметры системы –> Привязки файлов –> Поиск по “html” –> Application –> Для всех типов первым приоритетом выставить Firefox (если Firefox в списке приоритета приложений отсутствует - добавить)
Arch x86_64, Linux 5.16.7, Plasma 5.23.5
