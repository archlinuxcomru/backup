огромное спасибо,сделал через case-работает как часики.Файл выключалки теперь вообще маленький
#!/bin/bash

action=$(yad --center --on-top --undecorated \
     --button="!./icons/cancel.png! ОТМЕНА:1"\
    --button="!./icons/exit.png! ВЫЙТИ:2"\
    --button="!./icons/lock.png!БЛОКИРОВКА:3"\
    --button="!./icons/restart.png!ПЕРЕЗАГРУЗКА:4 "\
    --button="!./icons/shutdown.png!ВЫКЛЮЧИТЬ:5 ")
ret=$?

case "$ret" in
2)  killall --user $USER -TERM &;;
3) xscreensaver-command -lock & ;;
4) sudo reboot & ;;
5) sudo shutdown -h now & ;;
esac
exit 0
Команду killall --user $USER -TERM для выхода взял из Arch вики.
