Не знаю кто-как,но я читая,потому что раз тоже столкнулся с проблемой
Xdg-open и Default applications
Смотрю,что за файл
xdg-mime query filetype Music/Test/1.mp3
audio/mpeg
Потом смотрю кто открывает по умолчанию этот файл,т.е
смотрю файл cat /home/user/.local/share/applications/mimeapps.list
А можно быстрее 
xdg-mime query default audio/mpeg                                                                          [15:38~]
userapp-mpg123.desktop
Потом создаю файл например:userapp-mpg123.desktop
[Desktop Entry]
Encoding=UTF-8
Version=1.0
Type=Application
NoDisplay=true
Exec=mpg123 %f
Name=mpg123
Comment=Особое определение для mpg123
Потом в файл /home/user/.local/share/applications/mimeapps.list добавляю строки
В секции [Default Applications] и [Added Associations] строки
audio/mpeg=userapp-mpg123.desktop
И проверяю кто будет открывать файл по умолчанию
xdg-mime query default audio/mpeg
userapp-mpg123.desktop
Если есть FM,то там еще проще.
У меня так работает в OpenBox,Awesome,I3
Так же и ссылки из приложений открываются,тем чем мне нужно.
Последний крик моды - эхо 30-летней давности.
