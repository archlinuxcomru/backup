Счастливый обладатель Intel Core I5 и Nvidia 950M. Поставил Арч, работает нормально с mesa-дровами, 950 не используется. Попытался установить дрова nvidia. История команд
sudo pacman -S nvidia
sudo nvidia-xconfig
Попытался удалить nouveau дрова, но sudo pacman -Q | grep nouveau не выдал ничего. Хотя lspci -k | grep -A 2 -E "(VGA|3D)" пишет что по умолчанию используется nouveau драйвер.
После сделал mkinitcpio -p linux и reboot, но после перезагрузки Xorg не поднялся. Как сделать чтобы по умолчанию юзался не nouveau?

У вас двойная графика? А разве не нужен тут bumblebee (или как оно сейчас называется правильно)?

По поводу используемого драйвера - тут лучше посмотреть не в lspci, а в

glxinfo | grep ender

там будет строка "OpenGL Renderer string", в ней пишется используемый драйвер иксов для DRI/3D. А lspci пишет, что устройство есть и драйвер для него загружен.

Еще можно посмотреть в /var/log/Xorg.0.log, но я не уверен, как оно будет себя со сдвоенной графикой вести.
