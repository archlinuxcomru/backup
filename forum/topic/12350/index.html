Затем отключаю netctl disable wifi-home, и пытаюсь включить автоматическое подключение.
sudo systemctl enable [email protected]

Так netctl же сам через systemd активирует профиль (вики). То есть disable делать не надо, можно один раз сделать netctl enable - профиль будет добавлен в автозагрузку.

А для разового запуска можно использовать netctl start.
