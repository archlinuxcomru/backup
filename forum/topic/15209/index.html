Важно: Проект SliM заброшен (домашняя страница проекта не работает, осталось только зеркало для скачки) и SLIM не полностью совместим с systemd, в том числе с logind сеансами. Рекомендуется использовать другой Экранный менеджер или Xinitrc.

Попробуйте к примеру LightDM

Рекомендую через Xinitrc:
Для автозапуска X при входе в систему:
В ~/.bash_profile добавить: - если используете Bash (эта оболочка установлена по умаолчанию)

[[ -z $DISPLAY && $XDG_VTNR -eq 1 ]] && exec startx

Для автологина в систему:
Установить пакет:
yaourt -S xlogin-git

И выполнить:
sudo systemctl enable [email protected]
в ~/.xinitrc прописать
exec startxfce4
