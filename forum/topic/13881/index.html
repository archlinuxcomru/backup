yaourt -Sb spacefm

редактировать PKGBUILD, добавить '--with-gtk3' сюда:

build() {
  cd $pkgname-$pkgver
  ./configure --prefix=/usr \
    --disable-pixmaps --with-gtk3
  make
}

и собрать.
