На всякий случай сделайте себе какой нибудь лайв линукс на флешке или диске чтобы можно было подправить систему если сильно накосячите.
Дальше

Сперва удалите что на редактировали на веселе
sudo rm -R /etc/default/grub
sudo rm -R /etc/grub.d/

Затем переустановите груб
sudo pacman -S grub

Потом отредактируйте /boot/grub/grub.cfg
для вашего случая должно быть только
set gfxmode=auto
set timeout=0
menuentry 'Arch Linux' {
	linux	/boot/vmlinuz-linux root=UUID=ad4103fa-d940-47ca-8506-301d8071d467 rw  quiet
	initrd  /boot/intel-ucode.img /boot/initramfs-linux.img
}

Сохраняете изменения. grub-mkconfig -o делать нельзя!!!

Перезагружаетесь
