Предлагаю поднять на сервере новый сервис: поиск по неофициальным репозиториям.
Принцип работы примерно такой: админы/модеры добавляют через веб-морду неофициальные репозитории в систему. Раз в определенный период времени (например, 1 день) система забирает из репозиториев <repo name>.db.tar.gz и составляет у себя список пакетов.
Плюс вем-морда для поиска пакетов, которая генерит страничку с таким содержанием: репозитории в котором находится пакет (или несколько репозиториев), версия, зависимости
