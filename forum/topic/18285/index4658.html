как называется именна ента хрень, которую надо изменить, или цвет шрифта ентой хрени?

Как называется можно посмотреть в инспекторе (parasite для gtk2)

В sylpheed фон(светлосерый) хрени прописан намертво в приложении,поэтому от темы не зависит.

Можно изменить цвет шрифта на контрасный:
Копируем папку темы /usr/share/themes/Breeze-Dark в ~/.themes
Для удобства переименовать в Breeze-Dark2
Открыть в редакторе ~/.themes/Breeze-Dark2/gtk-2.0/gtkrc
и добавить в конец

#для Sylpheed
style "sylpheed" {
  fg[NORMAL] = "#27A5E6"
  fg[PRELIGHT] = "#3EB2ED"
}

#class "e - Sylpheed 3.7.0"  style "sylpheed"
widget_class "*GtkVBox.GtkScrolledWindow.GtkTextView.GtkEventBox.GtkHBox.GtkLabel" style "sylpheed"
Цвета можно прописать по вкусу (голубой вроде не выбивается из темы и на сером виден)
Сохранить файл и в lxappearance установить тему breeze-dark2

По идеи можно прописать только для sylpheed, но что то у меня не выходит ( может из за имени "e - Sylpheed 3.7.0" с пробелом)
Можно просто пускать отдельно sylpheed с этой темой, если где то будет мешать

Получится так. Если где еще надо, посмотрите в инспекторе название виджита и пропишите для него
