не могу подкчючить ни вай-фай ни медь…тупо ничего не выдает по dhcpcd, зависает (сама система не висне, dhcpcd входит в ступор, айпишки не рисует)и молчит
Смотрим правильное название сервиса и устройств и их статус, проверяем что бы одновременно не работали несколько конфликтующих сервисов
systemctl |grep  -i net
#и
systemctl |grep -i dhcp
# или полный
systemctl

Смотрим статус устройства и dhcpcd - может будут видны ошибки
примерно (ставим свои значения)
systemctl status sys-devices-pci0000:00-0000:00:1c.1-0000:03:00.0-net-enp3s0.device
systemctl status dhcpcd@enp3s0.service

Смотрим журнал
journalctl -b

dhcpcd также можно заменить на systemd-networkd + systemd-resolved
https://wiki.archlinux.org/index.php/Systemd-networkd
Пример (свое название сервиса ставим)
systemctl stop dhcpcd@enp3s0.service
systemctl disable dhcpcd@enp3s0.service
sudo nano /etc/systemd/network/20-dhcp.network
# пишем (вместо enp3s0 свой)
[Match]
Name=enp3s0
[Network]
DHCP=ipv4
#сохраняем
sudo nano /etc/systemd/network/25-wireless.network
# пишем (вместо wlp3s0 свой)
[Match]
Name=wlp3s0
[Network]
DHCP=ipv4
#сохраняем
sudo rm /etc/resolv.conf
sudo ln -s /run/systemd/resolve/resolv.conf /etc/resolv.conf
systemctl enable systemd-networkd
systemctl enable systemd-resolved
systemctl reboot
