[РЕШЕНО] Thunar не копирует файлы с примонтированной NFS-шары

bujhm666

Ситуация такая, NFS-шары монтируются через fstab при загрузке. Строка монтирования из fstab:
192.168.10.102:/mnt/data/video /mnt/video nfs auto,rw,nolock 0 0
Так вот Thunar не хочет копировать файлы с примонтированной шары. Ошибок не выдает, просто появляется окно прогресса копирования и больше ни чего не происходит, отменить его можно только убийством процесса.
В то же время копирование по 'cp' или в Midnight Commander происходит нормально. Если зайти на шару по Sambe, то тоже Thunar нормально копирует, проблема тольк в NFS. Ни кто не сталкивался с таким? Не пойму куда копать.

PS. Забыл добавить, что это произошло после какого-то обновления (когда, не могу сказать), раньше все было нормально.
