Берём за основу system rescue cd, инструкция по изготовлению флешки тут.oo
Далее в лепим каталоги под каждую систему и прописываем их в grub'е.
По крайней мере у меня на 32 GB флэхе спокойно жил кроме реск'ю сиди (кстати сказать - готовый stage3 для gentoo) ещё инсталлятор арча, дебиана, партед мэджик, DrWeb live CD, BartPE XP, [email protected] live cd, и hiren boot cd.

В некоторых дисках еще и этот initrd приходилось править и переупаковывать - там, местами надо поправить, чтоб искали не строго на CD продолжение, а конкретный файл/каталог на всех носителях, до которых можно дотянуться.

Наиболее простым вариантом оказался следующий (флешка у меня sdf):
yaourt -S mbr
yaourt -S syslinux
cfdisk /dev/sdf
Здесь нужно создать раздел, установить загрузочный флаг и выбрать тип W95 FAT32 (LBA) .
Далее установка mbr:
install-mbr /dev/sdf
Создание файловой системы:
mkfs.vfat -F 32 -n disk /dev/sdf1
Затем я перекопировал структуру папок и файлов из флешки, специально для этого созданной в программе XBoot (Программа для Windows. Загрузчик выбранный в Xboot должен быть syslinux), никаких программ в ней установлено не было. Только файлы необходимые для работы самого загрузчика.
Далее выполнил:
syslinux -d /boot/syslinux /dev/sdf1
Заработало. Остальное решается добавлением образов в папку images и редактированием конфигов.
