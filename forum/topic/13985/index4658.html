Ну таки напишу как сделано у меня на ноуте, который с uefi

1. Посредством gdisk выполнена разметка

NAME   MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
sda      8:0    0 298,1G  0 disk
├─sda1   8:1    0     1G  0 part /boot/efi
├─sda2   8:2    0    60G  0 part /
├─sda3   8:3    0     8G  0 part [SWAP]
└─sda4   8:4    0 229,1G  0 part /home

sda1 - раздел для загрузчика на FAT
sda2 - корень
sda3 - swap
sda4 - хомяк

использование gdisk ничем не отличается от fdisk

В /boot/efi установлена efi-вариация GRUB2, о чем красочно и на русском языке написано у Natrio (правда я ставил по ингишу, этой статьи на русском тогда ещё не было)
