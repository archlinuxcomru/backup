Если не появляется, см. ck-launch-session и т.д. в вики по этому поводу.

У меня появляется, но через udisks монтировать отказывается, то есть udisks почему-то не воспринимает устройство MemoryStick и не хочет его монтировать, хотя и видит. SD монтируется нормально.

Если вам надо смонтировать один раз, просто зайдите под рута и смонтируйте
mkdir /mnt/tmp
mount /dev/mspblk0p1 /mnt/tmp
Если надо иногда – можете установить pmount, добавить в файл /etc/pmount.allow строчки 
/dev/mspblk?p?
/dev/mmcblk?p?
и монтируйте от пользователя 
pmount /dev/mmcblk0p1
отмонтирование 
pumount /dev/mmcblk0p1
Если вам требуется добиться монтирования через GUI (udisks) – придётся покурить его конфиги, чтобы заставить признать сие устройство.

Кстати, интересный факт:
SD: 
$ lsblk /dev/mmcblk0
NAME        MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
mmcblk0     179:0    0   244M  0 disk 
└─mmcblk0p1 179:1    0   244M  0 part
MemoryStick: 
$ lsblk /dev/mspblk0
NAME        MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
mspblk0     254:0    0  59.5M  0 raid 
└─mspblk0p1 254:1    0  59.5M  0 part
Получается, SD имеет TYPE=disk, а MemoryStick TYPE=raid
Возможно, тут-то собака и зарыта.
