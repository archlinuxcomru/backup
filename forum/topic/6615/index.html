Если хард большой, то можно поставить 2 системы рядом ;-) а потом в ГРУБе выбирать что грузить - линукс или венду.

1. убери пробелы - wpa_supplicant -Dwext -iwlan0 -c/etc/wpa_supplicant.conf (лично у меня из-за этого были глюки), добавь ключ -В для бэкграунда.
2. поставь приоритет повыше в конфиге wpa_supplicant.conf (например priority=1 )
3. Если IP получаешь по DHCP, то
# dhcpcd wlan0

Собстно вроде все. Мой рабочий конфиг:
ctrl_interface=/var/run/wpa_supplicant
eapol_version=1
ap_scan=1
fast_reauth=1
network={
	ssid="MySSID"
	scan_ssid=1
	eap=PEAP
	key_mgmt=WPA-EAP
	phase2="auth=MSCHAPV2"
	identity="User"
	password="password"
	priority=1
}
