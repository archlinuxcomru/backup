Нашёл на Git Hub скрипт для просмотра видео с EX.UA и fs.to.
Устанавливается одной командой 
curl -L https://git.io/juKf | sh
,как пользоваться написано если перейти по ссылке.
Всё работает отлично.
Если видео с ex.ua можно было и так смотреть ,скопировав и вставив адрес в vlc,в fs.to этот номер не проходил.С этим скриптом прекрасно работает.
