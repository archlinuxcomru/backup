https://aur.archlinux.org/packages/sway-xkb-switcher/
Работает, правда питоновских зависимостей тянет 15 метров…
В общем, удалил пакет вместе с зависимостями, достаточно python-i3ipc и самого скрипта:
# pacman -S python-i3ipc
$ cd /tmp/
$ git clone https://github.com/nmukhachev/sway-xkb-switcher
$ mv sway-xkb-switcher/sway_xkb_switcher/switcher.py ~/.config/sway/
Запуск:
[ -f /tmp/sway-xkb-switcher.lock ] && rm /tmp/sway-xkb-switcher.lock
$HOME/.config/sway/switcher.py --background -D "English (US, menu-super_r)"
ну или подправить дефолтную раскладку исходя из вывода:
$ swaymsg -rt get_inputs | grep -m1 -A2 xkb_layout_names
