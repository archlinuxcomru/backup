Эх, ну чтоб по вики долго не лазать:

1. Копируем файл конфигурации:
$ mkdir -p ~/.config/openbox
$ cp /etc/xdg/openbox/{rc.xml,menu.xml,autostart,environment} ~/.config/openbox
2. Открываем ~/.config/openbox/autostart, пишем:
setxkbmap -layout us,ru -variant -option grp:alt_shift_toggle,terminate:ctrl_alt_bksp &
3. Для отображения раскладки установить xxkb и создать его конфигурационный файл в домашней директории:
~.xxkbrc
Ну и настроить там как-нибудь :)

Как вариант
/etc/X11/xorg.conf.d/01-keyboard-layout.conf
Section "InputClass"
        Identifier "keyboard-layout"
        Driver "evdev"
        MatchIsKeyboard "yes"
        Option "XkbLayout" "us, ru"
        Option "XkbVariant" ","
        Option "XkbOptions" "grp:ctrl_shift_toggle,grp_led:scroll"
EndSection
