Продолжая изучать материал нашел две, на мой взгляд, нужные опции для сборки пакетов
  --enable-shared[=PKGS]  build shared libraries [default=yes]
и

  --enable-static[=PKGS]  build static libraries [default=no]
Но ни товарищ Гугл ни Вики по ним ничего сказать не могут. Если кто пользовался данными опциями или имеет ссылочку на материал поделитесь пожалуйста информацией.
