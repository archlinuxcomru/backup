Отгруженно в вики http://wiki.archlinux.org/index.php/Rc.conf_( Русский)


этот файл представляет собой базовую настройку системы, локаль/часовой пояс/демоны/ и тд.
далее приведен rc.conf с переводом комментариев и добавлениями.

#
# /etc/rc.conf - Основная конфигурация Arch Linux
#
#
# -----------------------------------------------------------------------
# ЛОКАЛИЗАЦИЯ
# -----------------------------------------------------------------------
#
# LOCALE: доступные языки могут быть просмотрены с помощью 'locale -a'
# HARDWARECLOCK: установите "UTC" или "localtime" (во что выставлять часы биос)
# TIMEZONE: доступные временные зоны /usr/share/zoneinfo (к примеру для Москвы выставите Europe/Moscow)
# KEYMAP:  доступные раскладки клавиатуры можно найти/установить в  /usr/share/kbd/keymaps
# CONSOLEFONT: консольный шрифт /usr/share/kbd/consolefonts (только для не-US в консоли)
# CONSOLEMAP: кодировка консоли (если отлична от locale) /usr/share/kbd/unimaps
# USECOLOR: использование ANSI цветов в сообщениях при запуске
#
LOCALE=
HARDWARECLOCK="localtime"
TIMEZONE=Europe/Moscow
KEYMAP=us
CONSOLEFONT=
CONSOLEMAP=
USECOLOR="yes"
#
# -----------------------------------------------------------------------
# ЖЕЛЕЗО
# -----------------------------------------------------------------------
#
# Сканировать железо и загрузить необходимые модули при загрузке
MOD_AUTOLOAD="yes"
# Черный список модулей - они никогда не будут загруженны hwdetect
MOD_BLACKLIST=()
#
# Модули для загрузки при запуске (в этом порядке)
#   - поставьте ! перед модулем, чтобы занести его в черный список
#
MODULES=(!usbserial)
# Сканировать LVM тома при загрузке. необходимо если вы используете LVM
USELVM="no"
#
# -----------------------------------------------------------------------
# СЕТЬ
# -----------------------------------------------------------------------
#
#имя хоста. введите сюда имя машины.
HOSTNAME="myhost"
#
# Интерфейсы для запуска при загрузке (будут запущены в этом порядке)
# Введите каждый интефейс в список INTERFACES
#   - префикс ! в  INTERFACES дезактивирует запуск
#
#Замечание: чтобы использовать DHCP, выставьте "dhcp" (eth0="dhcp")
#
lo="lo 127.0.0.1"
eth0="eth0 192.168.0.2 netmask 255.255.255.0 broadcast 192.168.0.255"
INTERFACES=(lo eth0)
#
# Роутеры добавляемые при запуске (в этом порядке)
# Введите каждый роутер в список ROUTERS
#   - префикс ! в ROUTES дезактивирует добавление
#
gateway="default gw 192.168.0.1"
ROUTES=(!gateway)
#
# Активировать следующие сетевые профили. Это пользено только
# если вы используете различные сетевые комбинации (к примеру на ноутбуке)
#   - выставьте 'menu' чтобы показать меню при запуске для выбора профиля
#   - префикс ! дезактивирует загрузку
#
# Профили сети могут быть найдены в /etc/network-profiles
#
#NET_PROFILES=(main)
#
# -----------------------------------------------------------------------
# DAEMONS
# -----------------------------------------------------------------------
#
# Демоны для запуска при загрузке. (будут запущены в этом порядке)
#   - префикс ! дезактивирует загрузку
#   - префикс @ означает старт в фоновом режиме (скрипт в /etc/rc.d/ будет запущен, и загрузка продолжится не ожидая его завершения)
#
DAEMONS=(syslog-ng !hotplug !pcmcia network netfs crond)
# Конец файла

если никаких замечаний не будет - пихаем в вики.

Ошибки:
1) доступные языки могут быть просмот*р*енны с помощью ‘locale -a’
2) к примеру для *М*осквы выставите Europe/Moscow
3) Модули для загрузк*и* при запуске (в этом порядке)
4) Интерфейсы для запуска при загрузке (бу*д*ут запущены в этом порядке)
5) Демоны для запуска при загрузке. (будут запу*щ*ены в этом порядке)
6) - префикс ! де*з*активирует загрузку
7) - префикс @ означает старт в фоновом режиме (скрипт в /etc/rc.d/ будет запу*щ*ен, и загрузка продолжится не ожидая его завершения)
8) # Конец фа*й*ла

PS: Да, и Canada/Pacific как-то не к месту. Раз в качестве примера приводишь Europe/Moscow то можно и в TIMEZONE ее запихнуть.
