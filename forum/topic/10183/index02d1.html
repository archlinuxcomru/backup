На бубнофоруме высвечиваются 1 2 3 ... последняя страница. При нажатии на троеточие вся строка в поле сабжа заполняется номерами страниц. Если тема большая снова нажимаем на троеточие и т.д.
Тоже вариант, но при большом количестве страниц - долго придется добираться до нужной, а прямой ввод номера может сократить количество лишних кликов.

Форум, стараниями GeyseR-а, переехал на django 1.7, поэтому теперь он легко может бы запущен на arch linux:
Форкнуть https://github.com/hovel/aor
git clone [email protected]:MYREPO/aor.git
systemctl start mysqld
mysql -u root -p
CREATE DATABASE aor DEFAULT CHARSET utf8 COLLATE utf8_general_ci;
cd aor/aor
vi settings_local.py
DATABASES = { 
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'aor',
        'USER': 'root',
        'PASSWORD': '',
        'HOST': '',
        'PORT': '',
        'TEST_NAME': 'aor_test',
        'TEST_CHARSET': 'utf8',
        'TEST_COLLATION': 'utf8_general_ci'
    }   
}
cd ~
pacman -S python2-pip python2-virtualenv libmysqlclient libfreetype6-dev
virtualenv -p /usr/bin/python2 ~/.virtualenv/aor
source ~/.virtualenv/aor/bin/activate
cd aor 
pip install -U -r build/pipreq.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
127.0.0.1:8000
127.0.0.1:8000/admin
