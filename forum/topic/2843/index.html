Смотрели мою ссылку? Для моей Logitech C170 достаточно было:

% cat /etc/asound.conf
pcm.usb
{
    type hw
    card C170
}
pcm.!default
{
    type asym
    playback.pcm
    {
        type plug
        slave.pcm "dmix"
    }
    capture.pcm
    {
        type plug
        slave.pcm "usb"
    }
}
