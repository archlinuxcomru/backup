а также есть возможность установки из yaourt дополнительных апплетов командой yaourt -S budgie-desktop-git
в корне неверное понимание
budgie-desktop-git это не дополнительные апплеты, а это тот же budgie-desktop только git версии, поэтому команда yaourt -S budgie-desktop-git и просит разрешение сначала удалить budgie-desktop:
budgie-desktop-git и budgie-desktop конфликтуют. Удалить budgie-desktop? [Y/n]
Y
после удаления, yaourt автоматом установит budgie-desktop-git.

как тогда так сделать?
Что именно у вас не получается?
Удаляете старый пакет 
pacman -Rdd budgie-desktop
Ставите новый 
yaourt -Syu budgie-desktop-git
или pacman -U пакет  если он у вас в темп сохранился
Правите .xinitrc
##!/bin/sh

#
# ~/.xinitrc
#
# Executed by startx (run your window manager from here)
if [ -d /etc/X11/xinit/xinitrc.d ] ; then
    for f in /etc/X11/xinit/xinitrc.d/?*.sh ; do
        [ -x "$f" ] && . "$f"
    done
    unset f
fi
export XDG_CURRENT_DESKTOP=Budgie:GNOME
exec budgie-desktop

Перезагружаетесь
