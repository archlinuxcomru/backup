Читай внимательнее Wiki Touchpad Synaptics .... также обрати внимание на synclient
В Wiki все расписано подробно, например,
The primary method of configuration for the touchpad is through an Xorg server configuration file. After installation of xf86-input-synaptics, a default configuration file is located at /usr/share/X11/xorg.conf.d/70-synaptics.conf. Users can copy this file to /etc/X11/xorg.conf.d/ and edit it to configure the various driver options available. Refer to the synaptics(4) manual page for a complete list of available options. Machine-specific options can be discovered using #Synclient.

EDIT 1 - в принципе можно обойтись и без конфига, если настроек не много, например, у меня всего 2 параметра, которые я просто прописал в .bashrc
cat .bashrc | grep sync
# synclient FingerHigh=40
synclient VertEdgeScroll=1
и как видно из двух прописанных параметров используется один, второй закоментирован.
Хотя, конечно, правильнее прописать это в конфиге .... но было лень, проще прописать в .bashrc
Ошибки не исчезают с опытом - они просто умнеют

startx не работает
настройте ~/.xinitrc правильно, у вас дефолтный с программами которых у вас нет и ненужны
#!/bin/sh

userresources=$HOME/.Xresources
usermodmap=$HOME/.Xmodmap
sysresources=/etc/X11/xinit/.Xresources
sysmodmap=/etc/X11/xinit/.Xmodmap

# merge in defaults and keymaps

if [ -f $sysresources ]; then
    xrdb -merge $sysresources

fi

if [ -f $sysmodmap ]; then
    xmodmap $sysmodmap
fi

if [ -f "$userresources" ]; then
    xrdb -merge "$userresources"

fi

if [ -f "$usermodmap" ]; then
    xmodmap "$usermodmap"
fi
#
# start some nice programs

if [ -d /etc/X11/xinit/xinitrc.d ] ; then
 for f in /etc/X11/xinit/xinitrc.d/?*.sh ; do
  [ -x "$f" ] && . "$f"
 done
 unset f
fi

exec i3
