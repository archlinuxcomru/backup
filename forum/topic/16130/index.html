gtk.css - ради эксперимента я изменил цвет внутренней части окна transmission-gtk (часть окна в которой отображаются закачки) через значение "background-color"
.view {
  color: #31363b;
  background-color: #000000; }
- это секция /* Base States */

Секция /* Default color scheme */ задаёт типа "настройки по-умолчанию", а ниже уже подкручиваются отдельные параметры, их перекрывающие (или дополняющие?). Как-то так. В общем, понятного мне мануала по настройке gtk 3.0 я не нашёл, действую методом тыка.
