А разве нельзя установить систему и полностью её настроить, а потом убрать из хуков в mkinitcpio.conf autodetect
и выполнить
mkinitcpio -p linux
без autodetect это получается initramfs-linux-fallback
если его уберешь, то будет два одинаковых initramfs. initramfs-linux и initramfs-linux-fallback.
не нужно убирать из /etc/mkinitcpio.conf autodetect, а нужно просто поправить /etc/mkinitcpio.d/linux.preset
# mkinitcpio preset file for the 'linux' package

ALL_config="/etc/mkinitcpio.conf"
ALL_kver="/boot/vmlinuz-linux"

PRESETS=('default')

default_image="/boot/initramfs-linux.img"
default_options="-S autodetect"
и будет генерироваться только один initramfs без autodetect...
