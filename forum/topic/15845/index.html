Предыстория
С утра запустил машину, загузился - всё путём. Через напродолжительное время комп завис насмерть, не реагируя ни на мышь, ни на клавиатуру. Никакие REISUB не помогали. Пришлось ткнуть кнопку перезагрузки на системнике.
При загрузке вылезла ошибка
Root device mounted successfully, but /sbin/init does not exist.
Запустил ноут, полез в гугл, нашёл кучу советов, в том числе и с этого форума.
Взял свою заветную флешку (там archbang, SystemRescueCD, Lubuntu) Запустился, смонтировал корень и прочие директории, croot /mnt
chroot failed to run command ‘/bin/bash’
Приплыли! Ладно, бывало такое.
Пробуем варианты
явно указываем путь к исполнительному файлу - пофиг!
cp /bin/bash /mnt/bin/bash
пофиг! наверное либ каких-то на хватает (вычитал)
pacman -r /mnt -S bash
error: failed to initialize alpm library
(datebase is incorrect version: /mnt/var/lib/pacman/)
error: try running pacman-db-upgrade
всё! приплыли!
pacman-db-upgrade -r /mnt
сделано
pacman -r /mnt -S bash
сделано
chroot /mnt
опять не видит bash, даже если явно указываю путь!
у меня больше вапиантов нет, кроме
pacman -r /mnt -S base base-devel
