…систем-монитор ставьте из AUR gnome-shell-extensions-git отредактировав PKGBUILD добавив systemMonitor в строку с перечислением дополнений.

Не получится – систем-монитор для сборки нужна версия libgtop старшая, чем в природе существует, поэтому при конфигурировании скриптов сборки systemMonitor будет отключено и не станет собираться. Там надо перед строчкой с ./autogen добавить ещё строчку “sed -e ‘s:2.28.4:2.28.3:g’ -i ./configure.ac”, тогда соберется. Но в индикаторах его все равно не видно :(

Теперь в GNOME3 можно использовать PackageKit, который по сути выступает в качестве GUI для Pacman'а. Он позволяет устанавливать/удалять пакеты, при этом разбираясь с зависимостями; искать пакеты по имени и описанию; искать пакеты – владельцы файлов и т.д. Увидеть скриншоты можно здесь.
