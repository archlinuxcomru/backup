Всем привет. Подскажите где могут быть ошибки. У меня была такая ситуация. Я поставил систему с скрипта там раздел /home отдельно не был предусмотрен. Решил присоединить раздел /home. Таким способом:
# Монтируем раздел /home : mount /dev/sdaY /mnt/ home
Cоздаем папку /home : mkdir /mnt/home
# Создаем папку пользователя в /home : mkdir username
# Смотрим инфу о разделе /home: blkid /dev/sdaY
# Редактируем fstab: sudo nano /etc/fstab
# Добавляем инфу о разделе /home, взятой из команды blkid.
# Генерируем fstab: genfstab -U /mnt/etc/fstab
или правильней genfstab -pU /mnt >> /mnt/etc/fstab ?

в итоге fstab сгенерировался не совсем правильно. Поменяв разделы местами.
Очистил его. И перезагрузился. Как ни странно система загрузилась без ошибок.

Переделал по схеме :
#Монтируем /
mount /dev/sdaY /mnt

#Создаем директорию boot и home в mnt
mkdir /mnt/{boot,home} -это не вышло.

#Монтируем boot
mount /dev/sdaY /mnt/boot

#Монтируем swap
swapon /dev/sdaY

#Монтируем /home
mount /dev/sdaY /mnt/home
genfstab -pU /mnt >> /mnt/etc/fstab

Полет нормальный. Какой вариант правильней?
