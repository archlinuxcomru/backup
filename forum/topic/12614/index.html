B570. В каталог /etc/acpi/actions подложил:

для уменьшения яркости:
# /etc/acpi/actions/bl_down.sh:
#
#!/bin/sh
bl_device=/sys/class/backlight/intel_backlight/brightness
echo $(($(cat $bl_device)-200)) >$bl_device

для увеличения яркости:
# /etc/acpi/actions/bl_up.sh
#
#!/bin/sh
bl_device=/sys/class/backlight/intel_backlight/brightness
echo $(($(cat $bl_device)+200)) >$bl_device

Сделал исполняемыми и настроил в баше алиасы. Костыль, возможно, но работает.

пробовал sudo echo "0" >> /sys/class/backlight/intel_backlight/brightness, но терминал даже пароля не спрашивает :(
Так работать в любом случае не будет.. sudo действует только на echo, а пишется в конвейер уже с правами пользователя надо делать как-то так:
sudo sh -c 'echo "0" > /sys/class/backlight/intel_backlight/brightness'
