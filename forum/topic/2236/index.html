Выложи пожалуйста свои конфиги
У меня один монитор. Оставил только опции, которые действительно нужны - таких оказалось не много: пришлось явно указать DPI. Вообще опции те же, что и в xorg.conf.
/etc/X11/xorg.conf.d/20-monitor.conf
Section "Device"
	Identifier  "Device0"
	Driver      "nvidia"
	Option	    "NoLogo" "True"
	Option      "UseEdidDpi" "False"
	Option      "DPI" "90 x 90"
EndSection
Section "Screen"
	Identifier "Screen0"
	Device     "Device0"
	Monitor    "Monitor0"
	Option     "DynamicTwinView" "False"
#	SubSection "Display"
#		Depth     24
#	EndSubSection
EndSection
Переключение раскладки настроено на правые Ctrl+Shift, но это легко меняется.
/etc/X11/xorg.conf.d/20-keyboard.conf
Section "InputClass"
        Identifier "keyboard-layout"
        MatchIsKeyboard   "on"
        Option "XkbLayout"  "us,ru(winkeys)"
        Option "XkbOptions" "grp:rctrl_rshift_toggle,grp_led:scroll,terminate:ctrl_alt_bksp,compose:lwin"
EndSection
