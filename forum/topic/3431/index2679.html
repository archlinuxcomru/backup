[РЕШЕНО] Не стартуют иксы после выхода из ждущего режима

Oliver

Всем привет! Такая проблема нарисовалась: после выхода из ждущего режима появляется просто консоль, startx говорит “xinit: connection to X server lost”. После перезагрузки компьютера иксы стартуют нормально. В чем может быть проблема? Настраивал ждущий режим так: установил pm-utils, добавил в /boot/grub/menu.lst в параметры ядра resume=/dev/sda3 (путь к swap), в /etc/X11/xorg.conf в секцию “Device” добавил "Option “nvAGP” “1”".
Семь бед - один reset.

Для вывода X с драйвером nvidia из спящего режима нужен файл /etc/pm/config.d/nvidia 
HIBERNATE_RESUME_POST_VIDEO=yes
SUSPEND_MODULES="nvidia"
Это даст вам предварительную загрузку модуля “nvidia” перед “разморозкой” программ.
