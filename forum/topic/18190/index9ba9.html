Не понятно вифи интерфейс есть а в ifconfig его не показывает. Может конфликт сервисов.
Гляньте что у вас запускается 
 systemctl --type=service
смотрите netctl-блабла.service может еще какие вроде нетворг менеджера

остановите их пока (только те что интернета касаются!)
sudo systemctl disable netctl-блабла.service
sudo systemctl disable dhcpcd.service
Перезагрузитесь
и поднимите днс только на проводном интернете
sudo dhcpcd enp2s0
или какое у вас будет название в ls /sys/class/net
проверьте ифконфиг и пинг и 
networkctl status

Посмотрите поднят ли интерфейс enp2s0
ip a
state DOWN или state UP
ели не поднят поднимите
sudo ip link set enp2s0 up
попробуйте пингануть роутер
ping -c 5 192.168.0.1
есть ли пинг?
sudo dhcpcd enp2s0
и посмотрите
systemctl status dhcpcd.service
networkctl status
