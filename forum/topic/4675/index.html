Из этого https://bbs.archlinux.org/viewtopic.php?id=115507 обсуждения понял, что на данный момент решение проблемы только в даунгрейде. Откатил следующие пакеты
xorg-server (1.10.0.902-1 => 1.9.4.901-1)
xf86-input-evdev (2.6.0-3 => 2.6.0-2)
xf86-video-vesa (2.3.0-5 => 2.3.0-4)
и всё завелось (ждущий, спящий, перезагрузка, выключение).
Для тех, кто почистил кэш, на http://schlunix.org/archlinux/extra/ такие версии есть, проверено.
