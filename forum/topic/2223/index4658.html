если используется Oxygen, то:
1. System Settings -> Applicationa appearance -> Style -> Widget Style Configure -> убрать галочку Enable Animations
2. System Settings -> Workspace Appearance -> Configure Decoration -> убрать галочку Enable Animations

Надеюсь, поможет. После этого кедами со спецэффектами можно пользоваться даже на gma950.
