Если есть желание самостоятельно собрать функционал под свои потребности и возможности, то, на мой взгляд, самый идеальный вариант - motion. Можно сохранять данные в различных форматах, можно прикручивать запуск скриптов при возникновении различных событий, детектор движения, работа с различными видами камер: через lin4video и по ip.
Для готового решения - ZoneMinder. Сам не пробовал. Коллега жаловался, что хорошо “грузит” процессор, особенно при количестве камер больше двух (возможно, могли иметь место субъективные причины).
ffmpeg использовать, конечно, можно, но надо думать о том, где хранить данные. Без детектора движения запись будет идти постоянно. Хотя, может к ffmpeg есть какой-нибудь плагин для этой цели? Я не встречал.
