Внезапно сегодня оказалось, что для перезагрузки или выключения надо ввести пароль.
Также, если раньше при выборе пункта меню "Выход" или по команде exit снова запускался Xorg и Xfce, то теперь просто происходит логин и Иксы не запускаются сами.
В системе настроен автологин по ArchWiki.
/etc/systemd/system/[email protected]/autologin.conf:
[Service]
ExecStart=
ExecStart=-/usr/bin/agetty --autologin my_username --noclear %I 38400 linux
Type=idle
Правда, сейчас в АрчВики указан файл /etc/systemd/system/[email protected]/override.conf и в нем:
ExecStart=-/usr/bin/agetty --autologin username --noclear %I $TERM
Это существенно?
~/.bash_profile:
#
# ~/.bash_profile
#

[[ -f ~/.bashrc ]] && . ~/.bashrc
if [[ ! $DISPLAY && $XDG_VTNR -eq 1 ]]; then
  exec startx
fi
~/.xinitrc:
#!/bin/sh

# start some nice programs

if [ -d /etc/X11/xinit/xinitrc.d ] ; then
 for f in /etc/X11/xinit/xinitrc.d/?*.sh ; do
  [ -x "$f" ] && . "$f"
 done
 unset f
fi

exec startxfce4
До сего дня всё работало.
Вчера,правда, обнаружил, что не работает команда xkill, но это так, к слову.
