При очередном обновлении granite-bzr получил следующую ошибку:

==> Building and installing package
==> Сборка пакета: granite-bzr 555-1 (Вт апр  9 08:18:46 NOVT 2013)
==> Проверяю необходимые для запуска зависимости...
==> Проверяю необходимые для сборки зависимости...
==> Получение исходных файлов...
==> ОШИБКА: Не указан агент для обработки URL bzr+https. Проверьте /etc/makepkg.conf.
    Преждевременный выход...
==> ERROR: Makepkg was unable to build granite-bzr

Соотв. строка в PKGBUILD

source=("granite::bzr+https://code.launchpad.net/~elementary-pantheon/granite/granite")

Раньше собиралось. Гугель ничего не подсказал. Много думал над makepkg.conf - не уловил, в чём ошибка :( Читка https://wiki.archlinux.org/index.php/PKGBUILD#source тоже ясности не прибавила.

Проблема скорее всего дурацкая, просто знаний не хватает. Помогите разобраться плз ;)

Попробуй замени старый конфиг новым
sudo cp -f /etc/makepkg.conf.pacnew /etc/makepkg.conf

а пакет собрать так
makepkg -s

если от рута то
makepkg --asroot -s

Только что проверил, все собралось из зависимостей установило
