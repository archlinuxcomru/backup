Если нет жуя, то естественно не нужно такой зависимости цеплять.
Удалить пакет со всеми неиспользуемыми зависимостями: pacman -R -s (если правильно тебя понял)

насчет строчки что ты привел, во первых заводи баг-репорт на этот пакет, ибо вместо mv там нужно использовать install и вообще сама эта строка на хак похожа. А суть того что она мешает сборке без жуя: когда делается make install не создается директория skins/default соответственно саму тему переместить просто некуда, догадываюсь что на экране надпись типа “No such file or directory”, просто если сам собираешь, подразумевается что ты ДЕЙСТВИТЕЛЬНО ПОНИМАЕШЬ что делаешь.
systemd должен умереть.
