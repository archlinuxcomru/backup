На недобуке лучше остановиться на i686, ибо
1) Меньше потребление памяти;
2) Для редких i686-only программ не требуются дополнительные 32х битные библиотеки;
3) Для задач нетбука преимуществ x86_64 нету;
4) Есть пакеты i686-only. С x86_64 only я не сталкивался.
