И вот включил сейчас 10 минут назад -Загрузилось опять так же как на фото с планшета сфоткал...А сделал так и исправилось:
Вошёл кнопками в терминал: CTRL+ALT+F2 (он мигал и я долго пытался вбить логин, а потом мучился пароль вбивал. Перемигивалось с одного на другой терминал)
У меня обновления из терминала, быстро стрелками по истории выбрал: pacaur -Syu && sudo pacman -Syu && paccache -r && sudo pacman -Scc --noconfirm && sudo pacman -Rsn $(pacman -Qdtq) --noconfirm && sudo rm -rf ~/.cache/thumbnails/* --noconfirm
И сразу после обновы там в F2-на F1 иксы загрузились!
