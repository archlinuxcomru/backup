здравствуйте. суть такова: есть игра, установки не требует, на запуск есть скрипт, но при его попытке его запустить пишет:

bash: /run/media/ataq/data/game/PT/start.sh: Отказано в доступе

chmod 755 или chmod +x никак не повлияли на результат, но если запускать так:
sh 'путь к скрипту' то ошибка уже немного другая:

Running Planescape: Torment Enhanced Edition
/run/media/ataq/data/game/PT/start.sh: строка 22: ./Torment64: Отказано в доступе

в 22 строке он запускает файл который без расширения, а dolphin его видит как выполняемый. вот собственно и все, вопрос в том как запустить? спасибо всем откликнувшимся.

собсвтенно содержание скрипта:

#!/bin/bash
# GOG.com (www.gog.com)
# Planescape: Torment Enhanced Edition

# Initialization
CURRENT_DIR="$( cd "$( dirname "${BASH_SOURCE[0]}" )" && pwd )"
cd "${CURRENT_DIR}"
source support/gog_com.shlib

# Game info
GAME_NAME="$(get_gameinfo 1)"
VERSION="$(get_gameinfo 2)"
VERSION_DEV="$(get_gameinfo 3)"

# Actions
run_game() {
 echo "Running ${GAME_NAME}"
 arch="$(uname -m)"
 cd game
 if [ "$arch" == "x86_64" ]
 then
    ./"Torment64"
 else
    ./"Torment"
 fi
}

default() {
  run_game
}

# Options
define_option "-s" "--start" "start ${GAME_NAME}" "run_game" "$@"

# Defaults
standard_options "$@"
