По итогу при выборе есть только "Reboot Info Firmware interface"
Уточните сам загрузчик или груб грузятся с меню?
Чтобы знать проблема в загрузке самих загрузчиков или в загрузке из них системы
Грузитесь с установочного iso и проверяйте

efibootmgr -v
Наличие загрузочных записей с linux boot и grub и пути к ним
Монтируете ефи раздел и проверяете наличие по этим путям загрузчиков и наличие vmlinuz-linux и initramfs-linux.img

Исправьте /boot/loader/loader.conf
default  arch.conf

В /boot/loader/entries/arch.conf
options root=/dev/sda1 rw пропишите по LABEL
options root="LABEL=Arch" rw
