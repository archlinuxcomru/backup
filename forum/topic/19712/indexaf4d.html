Автор пакета предпринял дополнительные шаги.

Отредактировано: в качестве альтернативы использованию ключа API YouTube, я создал форк youtube-viewer, называемый straw-viewer, который использует API invidio.us для поиска видео на YouTube. Проект находится на ранней стадии разработки, и в нем могут отсутствовать некоторые функции, но для базовых вещей, таких как поиск видео и плейлистов, он должен работать без особых проблем.

Новый проект уже доступен в AUR.
