Возможна ли установка базовой системы без интернета?

Всем привет!
Ранее (июль 2012) ставил arch с образа записанного на cd. Рабочую систему получил без подключения к инету (создал пользователя и т.д.).
Далее на другом компе скачал несколько утилит wvdial и т.д. перенёс это в арч, установил эти утилиты без инета.
С помощью утилит насроил инет, обровил pacman, обновил систему - всё хорошо.
Был перерыв, захотел вновь заняться арчем.
Теперь же как вижу офлайн рабочую систему установить не возможно, т.к. имеем не установочный cd (всей операционки), а
только установщик (скачанный по ссылке Загрузки), с помощью которого только при наличии инета можно поставить рабочую операционку.
Т.е. сначала - инет, затем - рабочая система. Это и есть арч вей?
Почему пакеты base и base-devel не включили в образ установщика, чтобы можно было получить рабочую систему без инета?
или я чтото пропустил?
