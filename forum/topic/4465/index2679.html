Похоже, для nautilus 3 (без GNOME 3)
gconftool-2 -s "/desktop/gnome/thumbnailers/[email protected]/enable" -t boolean "true"
gconftool-2 -s "/desktop/gnome/thumbnailers/[email protected]/command" -t string "/usr/bin/ffmpegthumbnailer -s %s -i %i -o %o -c png -f -t 10"
присутствие или отсутствие этих ключей уже ни на что не влияет, причём, в dconf'е, если “вчитаться”, это включено по-умолчанию, только не работает.
Так что, да, yaourt -S ffmpegthumbnailer и /usr/share/thumbnailers/ffmpeg.thumbnailer проблему решает, остаётся только гадать, какого … это снова приходится делать своими руками.
