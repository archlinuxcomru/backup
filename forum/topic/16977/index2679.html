Aivar, все описал .... но если кому то нужно поподробнее, то я делаю так ...
Смотрим устройства и их id (например, mouse)
$ xinput list
↳ SIGMACHIP Usb Mouse id=9
Смотрим параметры мыши (id=9), например, связанные с ускорением
$ xinput list-props 9 | grep 'Accel Speed'
libinput Accel Speed (274): 0.000000
libinput Accel Speed Default (275): 0.000000
Изменям параметр, например замедляем (тип - float, диапазон -1.0 … 1.0)
$ xinput set-prop 9 274 -0.5
(274 - идентификатор libinput Accel Speed (274) - …. это чтобы меньше писать ...)
Проверяем
$ xinput list-props 9 | grep 'Accel Speed'
libinput Accel Speed (274): -0.500000
libinput Accel Speed Default (275): 0.000000
И так же для других параметров …...
PS … насчет Option "AccelProfile" (не везде работает) …. Not all devices support this option or all profiles. If a profile is unsupported, the default profile for this device is used. For a description on the profiles and their behavior, see the libinput documentation (PS... например, это ...)
Ошибки не исчезают с опытом - они просто умнеют
