Такс, решение по драйверу по ссылке - https://iamjagjeetubhi.wordpress.com/2016/06/19/install-realtek-rtl8723be-wifi-drivers-in-arch-linux/ есть решение.
либо еще тут https://bbs.archlinux.org/viewtopic.php?id=208472

Продублирую.

Сливаем реп.
$ git clone https://github.com/lwfinger/rtlwifi_new.git
$ cd rtlwifi_new
$ git checkout rock.new_btcoex

Собираем
$ make clean && make
$ sudo make install

Обновляем initramfs

$sudo mkinitcpio -p linux

Далее проверка порта антены - я этого не делал.
$cat  /etc/modprobe.d/rtl8723be.conf
options rtl8723be fwlps=N ips=N swlps=N swenc=Y disable_watchdog=1 ant_sel=X

Ну и как ее выбрать
$ sudo rmmod rtl8723be
$ sudo modprobe rtl8723be ant_sel=1
$ sudo iwlist wlan0 scan
$ sudo rmmod rtl8723be
$ sudo modprobe rtl8723be ant_sel=2
$ sudo iwlist wlan0 scan

А ну и еще
ВАЖНОЕ
Плосле обновления ядра нужно будет повторять операцию с самого начала.
