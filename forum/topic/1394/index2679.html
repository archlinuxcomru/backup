Всем доброго дня. Прошу помощи у знатоков по сборкам пакетов…
В теории все должно быть просто, но у меня к сожалению ни чего не выходит… До этого ни когда пакеты не собирал…
Дело вот такое…
Установил себе awesome (устанавливал через pacman), и для отображения раскладки решил использовать kbdd, так сказать чтоб было кошерное (щас раскладка переключается через xorg.conf.d/20-bla_bla.conf).
В AUR лазил там всего два пакета, один из них уже числится устаревшим, а второй git-й. Но они оба не ставятся.
Ну думаю соберу свой.. Скачал исходники (вот эту версию kbdd-0.6.2.tar.gz). Скачал для примера PKGBUILD ( этого и этого) и вдумчиво читал Wiki и прочие ссылки…
Делал все по Wiki (abs и base-devel установлены)
Создал каталог, где буду собирать… подправил PKGBUILD
# Maintainer: Bla_bla
pkgname=kbdd
pkgver=0.6.2
pkgrel=1
pkgdesc="Simple daemon and library to make per window layout."
arch=('i686' 'x86_64')
url="https://github.com/qnikst/kbdd"
license=('GPL3')
depends=()
source=("https://github.com/downloads/qnikst/$pkgname/$pkgname-$pkgver.tar.gz)
md5sums=('df2ad5cf9f9c723fd24fbf0577a2b4af')
build() {
    cd $startdir/src/$pkgname-$pkgver
    ./configure --prefix=/usr
    make || return 1
    make prefix=$startdir/pkg/usr install
}
