Как раскрасить текст на панели
На какой панели? Нет в i3 никакой панели, если только не влепили что-то из крысы и пр.
Если вопрос таки по i3status, то возможности вывода цветов ограничены для каждого блока, но хорошо описаны в документации.
В двух словах цвет строки состояния может иметь 4 значения:
 дефолтный (задается директивой statusline в конфиге i3-wm);
 color_good (дефолт - зеленый);
 color_degraded (желтый);
 color_bad (красный).
Доступные цвета для блоков:
Disk:
  low_threshold - color_bad

 Path_exists:
  format - color_good
  format_down - color_bad

 Ethernet:
  format_up - color_good
  format_down - color_bad

 CPU Usage
  degraded_threshold - color_degraded
  max_threshold - color_bad

 CPU-Temperature
  max_threshold - color_bad

 Memory
  threshold_degraded - color_degraded
  threshold_critical - color_bad

 File Contents
  format - color_good
  format_bad - color_bad
Т.е. вы можете глобально или локально для каждого указанного блока установить желаемые цвета, но не для volume. Тут будет только дефолтный цвет. Неразрешенные переменные игнорируются.
