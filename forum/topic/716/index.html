Как уже озвучил Bujhm666 - лучше использовать французские зеркала, это в /etc/pacman.d/mirrorlist:
## Score: 1.1, France
Server = http://mir.archlinux.fr/$repo/os/$arch
Остальное закомментировать

Чтобы работало автодополнение в pacman:
pacman -S bash-completion
После этого в файл “~/.bashrc” добавить:
complete -cf sudo
complete -cf man
