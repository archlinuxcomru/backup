в лисе - страница назад/страница вперед
Из коробки по идеи должно на libinput
cat '/var/log/Xorg.0.log' |grep  "input driver"
Using input driver 'libinput'

можно забиндить с xbindkeys
https://wiki.archlinux.org/index.php/Xbindkeys
https://wiki.archlinux.org/index.php/Mouse_buttons#xvkbd_and_xbindkeys
в ~/.xbindkeysrc
# Mouse Buttons
"xvkbd -xsendevent -text "\[XF86Back]""
m:0x0 + b:8
"xvkbd -xsendevent -text "\[XF86Forward]""
m:0x0 + b:9
:8 :9 заменить на те что у вас по нажатию показывает в xev

p.s. xvkbd пожалуй лучше поменять на xautomation
примеры
https://wiki.archlinux.org/index.php/Logitech_MX_Master#Xbindkeys
