Ноутбуки Lenovo Yoga
Некоторые новые ноутбуки Yoga (например, Yoga 530) поставляются с rtl8821ceчипом Wi-Fi / Bluetooth. Но ideapad-laptopмодуль, который может входить в ваш дистрибутив, может конфликтовать с этим драйвером.
Отключить:
sudo modprobe -r ideapad_laptop

BlueTooth не работает
Это может быть связано с тем, что ядро ​​загружает неправильный файл прошивки для этой карты. Взгляните на руководство @wahsot по адресу https://github.com/tomaspinho/rtl8821ce/issues/19#issuecomment-452196840, чтобы узнать, поможет ли это вам.
