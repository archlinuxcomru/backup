Линукс не трогал, стоит уже года наверное 2. После очередного обновления появилась проблема: при загрузке пишет какие то ворнинги красным что у меня хост не указан и таймзона. Все на самом деле указано. А после загрузки экран логина - все нормально. Но захожу в свое имя - все некликабельно, те никак вообще не реагирует ни на мышку ни на клаву. И записки на рабочем столе сами по себе есть, но пустые (а у меня они забиты текстом). Если перезагрузиться резетом раз 5 - оможно залогиниться нормально.

#
# /etc/rc.conf - Main Configuration for Arch Linux
#
# See 'man 5 rc.conf' for more details
#
# LOCALIZATION
# ------------
HARDWARECLOCK="localtime"
TIMEZONE="Europe/Kiev"
KEYMAP="ru"
CONSOLEFONT=
CONSOLEMAP=
LOCALE="ru_RU.UTF-8"
DAEMON_LOCALE="yes"
USECOLOR="yes"
# HARDWARE
# --------
MODULES=()
USEDMRAID="no"
USEBTRFS="no"
USELVM="no"
# NETWORKING
# ----------
HOSTNAME="arch"
interface=eth0
address=
netmask=
broadcast=
gateway=
NETWORK_PERSIST="no"
# DAEMONS
# -------
#
DAEMONS=(@syslog-ng @network dbus @crond @sensors gdm)
