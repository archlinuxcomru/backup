babay2004, по-идее ntfs-3g сам определяет локаль, так что ничего указывать не надо.

Теперь заноси в /etc/fstab строку
/dev/sda1 /media/SYSKA ntfs-3g defaults 0 0

Создавай в /media каталог SYSKA и выставляй на него права доступа.

У тебя всё правильно, за исключением точек монтирования. Я не знаю чем ты создавал свои папки, но если не mkdir - то права могут отличаться. Я просто создавал директории /mnt/sda3, /mnt/sda4 из под root командой mkdir. В /etc/fstab всё как у тебя:
/dev/sda3 /mnt/sda3 ntfs-3g defaults,locale=ru_RU.UTF-8 0 0
/dev/sda4 /mnt/sda4 ntfs-3g defaults,locale=ru_RU.UTF-8 0 0
Всё всегда работает. Ни права, ни пользователя никогда не переназначал.
Можно ещё и кодовую страницу выставлять, для того, чтобы ntfs-3g не мучался её определять.
