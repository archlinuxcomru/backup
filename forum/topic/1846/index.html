1. Установил арчлинукс с CD, попробовал выполнить обновление: pacman -Syu. Получаю ошибку "Не удалось завершить запрос (конфликтующие файлы):
glibs: /usr/bin/tzselect уже существует в файловой системе
glibs: /usr/sbin/zdump уже существует в файловой системе
glibs: /usr/sbin/zic уже существует в файловой системе
При установке любой из программ по pacman -S появляется сообщение, что сначала будет обновлен pacman, выдается список библиотек и опять появляется ошибка, если ответить “да” на вопрос об обновлении. Если ответить “нет” на вопрос об обновлении, программа скачивается. Однако после скачки wicd и попытке запуска wicd-curses появляется сообщение о неправильно версии библиотеки glibs. Подскажите, пожалуйста, как исправить эту ошибку.
2. Команда 
rankmirrors -t /etc/pacman.d/mirrorlist
выдает ошибку, что не задан curl и выдает список зеркал без указания времени. Причем не очень понятно, какие из серверов самые быстрые - которые вверху списка или внизу. Так, например, при загрузке нескольких пакетов у меня появлялась надпись о том, что обращение к серверу вверху списка завершилось ошибкой. Подскажите, пожалуйста, какие параметры нужно указать этой программе для получения максимума информации.

Перед обновлением сделайте синхронизацию.
# pacman -Syy

Зеркала. Лучше используйте немецкие - в стандартном mirrorlist раскомментируйте соответствующие строки. Пока хватит. Эти зеркала лучше работают в Европе.
Потом уже делайте 
# pacman -Syu
