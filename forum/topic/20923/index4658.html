Так регулировка частоты работает:
не совсем
без опции частота при простое ядер снижается незначительно
с опцией, снижается до минимальной частоты
к примеру у меня, без опции 2400 минимум, а с опцией, 1200

ещё у опции есть 3 варианта, для всех логических ядер, для всех одинаково и для всех физических ядер
