xtishka,
у меня такого не наблюдается
в качестве проигрывателя использую mpd + mpc + ncmpcpp, вывод звука через alsaequal (эквалайзер для алсы)
mpd.conf
mixer_type "software"
audio_output {
type		"alsa"
name	"equal"
device	"plug:plugequal"
}
все сторонние звуки воспроизводятся без проблем - и видео в mpv, и в браузере, и звуки событий в sublime text, и все остальные тоже

про alsaequal и mpd - http://muhas.ru/?p=174 && http://zenux.ru/articles/26/
