UPD ... Но, мое мнение, с появлением умных утилит, у молодежи пропадает интерес к изучению, наверное, точнее сказать пропадает стремление к освоению чего то нового.

Multibootusb автоматом устанавливает Grub2 - можно пользоваться им, например для загрузки образов с других разделов, т.к syslinux этого не умеет.
Создаем в корне папку grub4dos, копируем в нее файлы скачанные из любых источников (можно здесь) - у нас появляется возможность запускать с помощью Grub4Dos со своими уже наработанными конфигами (menu.lst я кидаю в корень)
### Загрузка из /multibootusb/syslinux.cfg
# Boot GRUB4DOS
LABEL GRUB4DOS BootManager
МЕНЮ LABEL GRUB4DOS BootManager
KERNEL /grub4dos/grub.exe --configfile=/menu.lst

Ну а дальше осваиваем, настраиваем, украшаем...

Единственный ньюанс -
При проверке работы загрузчиков указывать опцию cache=none, чтобы не кешировался старый конфиг

$ sudo qemu-system-x86_64 -boot c -drive file=/dev/sdd,cache=none -m 2048
