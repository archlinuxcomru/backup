Установка Grub4Dos из Linux такая же простая, как из винды и есть несколько способов. Давно не делал, а потому забыл - смысл то понятен, но есть нюансы, пришлось пошерстить свою базу, и нашел.
Привожу рабочую версию, которую сам пробовал несколько раз. Что нужно - всего 1 файл из grub4dos - grldr.mbr
Описываю с самого начала, чтобы не было проблем.
Перед началом чистим таблицу разделов командой (или сделать это из fdisk - параметр "d")
# dd if=/dev/zero of=/dev/sdb bs=512 count=1

1. # fdisk /dev/sdb
- вкючить опцию "c" (это чтобы начать с 63 сектора)
- "d" - удаляем все разделы, если перед началом не удалили таблицу разделов
- "n" - создаем новый раздел и далее до файловой системы все по умолчанию (желательно - начало с 63 сектора)
- "t" далее "7" - создаем файловую систему ntfs (но не забываем, что фактически ее еще нет)
- "a" - bootable flag
- "p" - проверяем, что получили
- "w" - сохраняем и выходим
2. # gparted (или что другое) - форматируем в ntfs
3. # dd if=/dev/sdb bs=1 skip=439 count=72 2>/dev/null | dd of=/path/grub4dos/tmp/grldr.mbr bs=1 seek=439 conv=notrunc
4. # dd if=/path/grub4dos/tmp/grldr.mbr of=/dev/sdb
5. Копируем нужные файлы на флешку и, разумеется, правим menu.lst
Все.
UPD ... нашел и первоисточники - это и это

PS ... забыл предупредить, что используемый файл grldr.mbr нужно скопировать в другое место, например /tmp
cp /path/grub4dos/grldr.mbr /tmp/grldr.mbr
Ошибки не исчезают с опытом - они просто умнеют

Стоял вопрос быстро создать загрузочную флешку для установки Windows.

1. Устанавливаешь и открываешь multibootusb
2. Втыкаешь флешку (например sdd)
3. В левом верхнем углу выбираешь sdd1
4. Выбираешь Install Syslinux (с копированием файлов)
5. В Select Image нажимаешь Browse (выбираешь нужный образ, в данном случае Windows...iso)
6. Нажимаешь Install distro

Получаем multiboot флешку с установочным образом windows
Проверяем загрузку: $ sudo qemu-system-x86_64 -boot c -drive file=/dev/sdd,cache=none -m 1024
Радуемся... Устанавливаем челу Windows...

Только, что проверил - 10 минут (время распаковки образа на флешку)

Бывают заморочки:
Иногда приходится форматировать в Windows ну и т.д
