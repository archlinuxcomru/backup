После обновления перестал запускаться Stardict (stardict segmentation fault) и вылечить его можно так:
sudo rm /usr/lib/stardict/plugins/stardict_spell.so

Кстати вот здесь есть все словари из Lingvo x3 + озвучка для нескольких языков.
