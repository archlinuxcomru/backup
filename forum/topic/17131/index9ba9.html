pcmanfm --help
Использование:
  pcmanfm [ПАРАМЕТР…] [FILE1, FILE2,...]

Параметры справки:
  -h, --help                   Показать параметры справки
  --help-all                   Показать все параметры справки
  --help-gtk                   Показать параметры GTK+

Параметры приложения:
  -p, --profile=PROFILE        Имя профиля конфигурации
  -d, --daemon-mode            Запустить PCManFM как демон
  --no-desktop                 Нет функции. Для совместимости с Nautilus
  --desktop                    Запустить менеджер рабочего стола
  --desktop-off                Выключить менеджер рабочего стола, если он запущен
  --desktop-pref               Открыть диалог параметров рабочего стола
  --one-screen                 Использовать параметр --desktop только на одном экране
  -w, --set-wallpaper=FILE     Установить обои рабочего стола из файла FILE
  --wallpaper-mode=MODE        Установить режим для обоев рабочего стола. РЕЖИМ=(color|stretch|fit|crop|center|tile|screen)
  --show-pref=N                Открыть диалог параметров на странице N
  -n, --new-win                Открыть новое окно
  -f, --find-files             Открыть поиск файлов
  --role=ROLE                  Роль окна в оконном мендежере
  --display=ДИСПЛЕЙ            Используемый X-дисплей

параметр --desktop не работает / вызывает сбой X-сервера
Убедитесь что вы являетесь владельцем и имеете право на запись в ~/.config/pcmanfm
Установка обоев с помощью параметра --desktop-pref или путем отредактирования ~/.config/pcmanfm/default/pcmanfm.config решает проблему.
