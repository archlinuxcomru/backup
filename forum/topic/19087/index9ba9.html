l1onsun
типа systemctl start dhcpcd.service сделать?
Лучше для конкретного интерфейса.
Смотрите имена
ls /sys/class/net
#или
ip link
Включаете сервис системд (для примера интерфейс enp3s0)
sudo systemctl enable dhcpcd@enp3s0.service
Запускаете сервис
sudo systemctl start  dhcpcd@enp3s0.service
Смотрите состояние
systemctl status dhcpcd@enp3s0
