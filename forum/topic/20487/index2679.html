Стандарт предписывает uefi грузить файл /EFI/BOOT/bootx64.efi на fat32 разделе.
Достаточно создать на диске фат раздел. На нем папку /EFI/BOOT и скопировать туда файл ефи приложения ( например загрузчика, груб, рефинд, систембут,ядра некоторые) переименовав его в
bootx64.efi ( для 32битного ефи имя другое)
Выставив в ефибиос загрузку с этого диска будет грузиться этот файл
