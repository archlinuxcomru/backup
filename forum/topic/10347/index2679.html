Новое ядро 32-й серии наконец-то добралось до !
Основные изменения

  • Изменения в системе KMS:
    intel kms включён по умолчанию
  • radeon kms выключен по умолчанию. (radeon.modeset=1 для включения)
  • На картах radeon и nvidia “ранний” kms на текущий момент не работает. Исправление ожидается со следующим релизом mkinitcpio.

  • Прочие Арч-специфичные изменения:
    включена опция CONFIG_PM_DEBUG
  • включена опция CONFIG_MMIOTRACE
  • Выделен пакет kernel-headers. Для сборки внешних модулей ставьте kernel26-headers! Добавьте его в makedepend ваших PKGBUILDов!
  • Добавлена поддержка xen для 64-битного ядра
  • Новая подсистема firewire
такие дела.
