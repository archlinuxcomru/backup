Перезагружаться не надо, достаточно было запустить compton с желаемыми опциями.

xtishka
для чего вообще нужен композитный менеджер
Тени, прозрачность, fade in-out etc, однако compton умеет включать вертикальную синхронизация, чего для некоторых видеокарт не умеет стандартный nvidia-setting (а если и умеет, то криво).
убейте compton и поэкспериментируйте с vsync:
killall compton
compton –vsync opengl # это например
Не копипастьте, введите руками. На данном этапе с тегом code проблемы (загляните в man compton)
