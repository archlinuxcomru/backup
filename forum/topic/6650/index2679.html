Постепенно перебираюсь в более удобные окружения. Решил начать с awesome (из-за названия), настроил основные компоненты, виджеты, тэги, назначения тэгов, в общем освоился неплохо. Но возникла проблема с хот кеями…
Для начала вот конфиг

Секция с моими хот кеями:
    awful.key({}, "#107", function () exec( "scrot -q 96 -s 'shot-%m-%d-%H%M%S-$wx$h.png' -e 'mv $f ~/pic/screenshot/scrot'" )    end),
    awful.key({}, "#171", function () exec("mpc next")    end),
    awful.key({}, "F12",  function () exec("urxvt") end),

Работают все хоткеи, кроме scrot. Пробовал писать:
"scrot -q 96 -s shot-%m-%d-%H%M%S-$wx$h.png -e 'mv $f ~/pic/screenshot/scrot'"
(без первых кавычек) тоже не работает.
Команду копипастил в консоль, отлично работает. Кейкоды подставлял из рабочих строчек - не работает. Понятия не имею как тут может быть ошибка (((
