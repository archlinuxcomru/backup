Вот пакет который меня, на данный момент интересует.
Как его собрать вообще возможно?
Нажать кнопку clone oк download нажать download ZIP, скачать zip архив, распаковать его.
Далее в файле INSTALL черным по белому:

Linux/Unix]
* Default install prefix /usr/local
* By default libvacuumutils.so is copied to /usr/local/lib. On some distributives
* you may receive message that libvacuumutils.so is not found. In this case configure
* your system to search libraries in /usr/local/lib.
qmake CONFIG-=debug_and_release CONFIG-=debug -recursive vacuum.pro
make
make install

RusWolf, уже не однократно черным по белому говорилось: не надо так делать! Вы устанавливаете пакет в обход пакетного менеджера, в следствии чего теряете контроль зависимостей, версий и прочих плюшек, что pacman делает автоматически. Потом у вас появляются конфликты и прочие радости, которые у неокрепших умов, скорее всего, приведут к банальной переустановке системы.

hoz, в такой способ, для теста, можете собрать пакет, но устанавливать не надо. Впрочем, хозяин - барин...
