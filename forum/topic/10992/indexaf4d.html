Огромный каталог, в котором собрано огромное количество видео, музыки, изображений, программ, игр и множества другого. Также есть возможность закачивать свои файлы для хранения либо обмена ими по сети. Отличается возможностью закачки очень больших файлов. На Ex.ua можно смотреть фильмы онлайн, слушать музыку на сайте через встроенный плеер.

К сожалению, этот сервис доступен только гражданам Украины, поэтому Рунет пользователям приходится искать аналог ex.ua в России либо заморачиваться с подменой ip.
