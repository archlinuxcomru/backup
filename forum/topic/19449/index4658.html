В fstab синтаксис такой: _что монтируем_куда монтируем_опции монтирования_ проверка

в вашем же fstab вы пытаетесь примонтировать четыре диска в одну точку ( / корень) еще и с неправильными опциями.

Создайте разные точки монтирования (папки) и монтируйте диски туда.

Не знаю, связана моя бывшая проблема с вашей, но было такое: типа not authorized при входе. Решение нашел такое

Надо в /etc/polkit-1/rules.d/ создать файл с именем 99-my.rules (название особой роли не играет, важно только расширение и желателен номер) и прописать туда:

polkit.addRule(function(action, subject) {
if (action.id.indexOf("org.freedesktop.udisks2.") == 0 && subject.isInGroup("storage")) {
return polkit.Result.YES;
}
}
);

Как сами видите, юзер должен быть в группе storage.
Вдруг поможет. Сам на таком скрипте сижу, со всеми этими энтфэсами-3джи плюс какая-то бигудина ещё.
