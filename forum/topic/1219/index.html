А git запоминается просто

войти в каталог /etc под рутом
cd /etc/
выполнить, что бы инициализировать git
git init
git config –global user.email “[email protected]”
git config –global user.name “Your Name”
git add -A
git commit -m $(date +%Y-%m-%d-%H%M%S)
Вот и создан начальный репозиторий.

Потом после каждого обновления или изменения конфигов
git add -A && git commit -m $(date +%Y-%m-%d-%H%M%S)

Что бы посмотреть изменения в графическом режиме (так удобнее)
gitk
Я например юзаю dolphin там измененные файлы сразу видны и diffuse он нормально с гитом работает, что бы посмотреть текущие не закомиченные изменения в файле.
