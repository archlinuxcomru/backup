 Разметка и выставление типов на разделы
fdisk /dev/sda
форматирование
mkfs.vfat -n UEFI /dev/sda1
mkfs.ext2 -L Boot /dev/sda2
mkfs.ext4 -L Root /dev/sda3
mkfs.ext4 -m 0 -L Home /dev/sda4
mkswap -L Swap /dev/sda6

sdaX не забудь поменять на свои!!!
