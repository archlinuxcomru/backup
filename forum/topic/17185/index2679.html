ВД-шный мастер-пассворд известен, только вот ввести бы его куда-нибудь.
А точно залочен??? ............... Лучше для начала проверить - привожу мой вывод, не залоченного
# hdparm -I /dev/sda | grep -A9 Security: .... (9-ая строчка лишняя, если залочен, то ее не будет, а появится другая - смотри ниже)
Security:
	Master password revision code = 65534
		supported
	not	enabled
	not	locked
	not	frozen
	not	expired: security count
		supported: enhanced erase
	158min for SECURITY ERASE UNIT. 158min for ENHANCED SECURITY ERASE UNIT.
Logical Unit WWN Device Identifier: 500003933a784117
Если залочен, то в выводе исчезнет часть not и появится дополнительная строчка о Security level ….... что очень важно — там будет написано какой уровень безопасности установлен

PS .......... Если нет блокировки HDD BIOS (о чем говорит наличие в выводе not frozen), то можно попробовать применить --security erase (man hdparm) …. как я понял, данные уже не важны … есть также опция --security-disable --security-unlock PWD (man hdparm в помощь ....... давно не читал, похоже перепутал ..... одним словом смотри man ...)
Есть и другие способы и утилиты, но сначала пробуй hdparm, утилита хорошая - но возможно придется диск цеплять к другому ноуту
Ошибки не исчезают с опытом - они просто умнеют
