Hdparm - не выставляет режим 255 на ноуте

golan
# 10 лет, 4 месяца назад
Темы: 8
Сообщения: 107
Участник с: 06 октября 2011
Добрый день друзья.
Имеется такая проблема.
Хочу отключить совсем парковку головок диска.
Что сделал:
1. Создал файл /etc/hdparm.conf (изначально его там не было), добавил туда сдедующий код:
/dev/sda {
    apm = 255
    spindown_time = 0
    }
Не помогает
2. Добавил строку в /etc/rc.local
#!/bin/bash
#
# /etc/rc.local: Local multi-user startup script.
#
hdparm -M 254 -B 255 /dev/sda
exit 0
Не помогает.

При загрузке вижу сообщения, что rc.local отрабатывает своё…но потом в рабочем окружение результат такой:
[[email protected] ~]$ sudo hdparm -B /dev/sda
/dev/sda:
 APM_level	= 254

Специально поставил последнюю убунту для эксперимента, там изначально имеются следующие файлы и папки:
/lib/hdparm/hdparm-functions
/etc/hdparm.conf
При настройке убунты выше приведенным способом результат тот что мне нужен…, что же здесь не так?
