Хорошо бы ещё убрать зазор между "областями", но мне пока не приходит в голову, как это можно сделать.
Выяснилось, между тем, что увеличенный static const unsigned int borderpx создаёт проблемы с некоторыми приложениями, отображаемыми в режиме "окно без рамки" на весь экран - они смещаются за край экрана на величину borderpx.
Проблема решилась установкой static const unsigned int borderpx в один пиксель, а static const unsigned int snap равным нолю: граница хорошо видна и в то же время, совершенно ненавязчива.
