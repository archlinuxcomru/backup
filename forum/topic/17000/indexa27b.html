На xmonad.org есть приписка по этой фишке: WARNING: This prompt won't display all possible layouts, because the code to enable this was rejected from xmonad core. It only displays layouts that are actually in use. Also, you can only select layouts if you are using NewSelect, rather than the Select defined in xmonad core (which doesn't have this feature). So all in all, this module is really more a proof-of-principle than something you can actually use productively.
Я так понял, что ее суть сводится к: Таким образом, в целом этот модуль действительно более доказательство принципа, чем то, что вы можете эффективно использовать. (перевод)
У меня оно лайоуты показывает только назначенные текущему воркспейсу и как-то не сразу, невнятно, т.е. символы вводишь, потом начинаешь затирать и появляются варианты, но по выбору и ентер - ничего не происходит. Закомментил.
