Сообщаю огромный секрет.
VIM поддерживает синтаксис python "из-коробки". Не нужно специальных установок, флагов, и прочей лобуды.
Например, найдите у себя любой скрипт python
$ find / -name '*.py' 2>/dev/null
Откройте любой из них в vim, и вы все увидите.

Откройте любой из них в vim, и вы все увидите.
вим собирается с различными модулями. Вим, содержащийся в гвиме, по сути собран с максимальными возможностями (если я не ошибаюсь). Что влечет немного мусора (кстати, про руби забыл). И ранее, арчевский вим был собран БЕЗ поддержки питона, что заставило знакомого гентушника, который попробовал поставить арч на замену, плеваться.
Если вы посмотрите PKGBUILD для вима, то вы увидите:
./configure --prefix=/usr --localstatedir=/var/lib/vim \
     --with-features=big --with-compiledby=ArchLinux \
     --enable-gpm --enable-acl --with-x=no \
     --disable-gui --enable-multibyte --enable-cscope \
     --disable-netbeans --enable-perlinterp --disable-pythoninterp \
     --disable-python3interp --disable-rubyinterp --disable-luainterp
это именно под сам вим (питон в последней и предпоследней строчках). Для гвима - будет флаг --enable-pythoninterp.
