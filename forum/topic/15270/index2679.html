когда-то свой смартфон с кастомной прошивки на офф.прошивал, думаю для планшета будет похоже:
1. Скачать и установить FlashTool
1.1. прописать правило и перезагрузится:
cat /etc/udev/rules.d/51-sony.rules
SUBSYSTEM=="usb", ACTION=="add", ATTRS{idVendor}=="0fce", ATTRS{idProduct}=="*", MODE="0777"
 % sudo reboot
2. Скачать прошивку
3. Файл прошивки с расширением *.ftf кидаем в папку ~/.flashTool/firmwares/
4. Запускаем FlashTool:
4.1. нажимаем кнопку "Flash device" и выбираем режим "Flashmode"
4.2. выбираем из списка свою прошивку
4.3. подтверждаем нажимая кнопку "Flash"
Телефон:
1. Выключить телефон
2. Подключить телефон к компьютеру через USB-кабель при этом зажимая клавишу УМЕНЬШЕНИЯ громкости до момента когда на телефоне не загорится ЗЕЛЁНЫЙ светодиод
3. Ждём, пока FlashTool увидит телефон
4. Ждём окончания прошивки
5. Отключить кабель и включить телефон

А для прошивки на кастомные больше нравится использовать android-sdk:
1. Подготовка компьютера
1.1. Установить и настроить Android SDK
% yaourt -S android-sdk android-udev
1.2. Скачать прошивку
1.3. Из архива прошивки извлечь файл загрузчика boot.img
2. Подготовка телефона
2.1. Разблокировать загрузчик если ещё не разблокирован
2.2. В корне SD-карты телефона создать папку firmwares в которую скопировать архив с прошивкой
2.3. Выключить телефон
3. Установка CWM (ClockWorkMod Recovery)
3.1. Подключить телефон к компьютеру через USB-кабель при этом зажимая клавишу УВЕЛИЧЕНИЯ громкости. На телефоне должен загореться СИНИЙ светодиод.
3.2. В терминале:
% cd /opt/android-sdk/platform-tools
% ./fastboot flash boot /путь/к/boot.img
% ./fastboot reboot
3.3. Отсоединяем USB-кабель
4. Прошивка телефона
загружаемся в установленный CWM и перепрашиваем
...
