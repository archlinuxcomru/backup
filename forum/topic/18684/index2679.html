dhcpcd enp2s0

- пишет, что "...already running". Чего ему не хватает, что делать? Просто дайте, плз, конкретный совет, без отсылания к викам, я только что оттуда, только ещё больше запутался в этом всём.

dhcpcd в арчисо уже при старте запущен, так что интерфейс должен быть поднят.
Вместо ifconfig теперь ip
Так что смотрите какие есть интерфейсы
ip link
Ну и поднять отключить соответственно
# ip link set интерфейс up
# ip link set интерфейс down

Ну а почему гугл не пингуется могут разные причины быть.

Может у провайдера привязка к MAC address
Попробовать пингануть шлюз провайдера. сменить мак https://wiki.archlinux.org/index.php/MAC_address_spoofing

Может проблемы с dhcpcd
https://wiki.archlinux.org/index.php/Dhcpcd#Troubleshooting
