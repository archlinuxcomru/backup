Когда еще использовал ubuntu у меня была проблема с загрузкой проца при использовании mpd… Проблема решилась указанием следующих параметров в /etc/mpd.conf:
audio_output {
        type                    "alsa"
        name                    "My ALSA Device"
        period_time             "50000"
        device                  "hw:0,0"     # optional
        format                  "44100:16:2" # optional
}
audio_output_format             "44100:16:2"
Важным оказалось именно указание device и format. Без этих параметров mpd работало, но при этом процессор загружался до 50 процентов…
Попробуйте, есть вероятность, что именно в этом проблема.
