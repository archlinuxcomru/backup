пусть не относится к теме, но советую убрать “а”:
#DHCP:     Set your interface to "dhcp" (eth0="dhcp")
a
#
 
#Static IP example
потом
#Static IP example
eth0="eth0 92.55.27.205 netmask 255.255.255.0 broadcast 92.55.27.255"
#eth0="dhcp"
INTERFACES=(eth0)

убери broadcast 92.55.27.255, это, фактически, не нужно.

# Routes to start at boot-up (in this order)
# Declare each route then list in ROUTES
#   - prefix an entry in ROUTES with a ! to disable it
#
gateway="default gw 92.55.27.1"
ROUTES=(gateway)

ROUTES=(gateway) замени на ROUTES=(!gateway) (вики рекомендует это делать только с динамическим айпи, но по моему опыту - всё наоборот.)

и, наконец, - pppoe-setup настраиваешь на eth0? DNS'ы указываешь? и ещё совет: подключайся через pppoe-connect, он покажет тебе всю картину подключения и ошибки.
