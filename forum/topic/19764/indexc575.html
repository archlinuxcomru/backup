Pale Moon можно скачать с сайта программы http://linux.palemoon.org/download/mainline/ распаковать в ~/
создать симлинк
ln -s ~/palemoon/palemoon /usr/bin/palemoon
для значков
ln -s ~/palemoon/browser/chrome/icons/default/default16.png /usr/share/icons/hicolor/16x16/apps/palemoon.png
ln -s ~/palemoon/browser/chrome/icons/default/default32.png /usr/share/icons/hicolor/32x32/apps/palemoon.png
ln -s ~/palemoon/browser/chrome/icons/default/default48.png /usr/share/icons/hicolor/48x48/apps/palemoon.png
ln -s ~/palemoon/browser/icons/mozicon128.png /usr/share/icons/hicolor/128x128/apps/palemoon.png
Запустить файл palemoon в распакованной директории или
создать текстовым редактором файл palemoon.desktop в /usr/share/applications/palemoon.desktop след. содержания
[Desktop Entry]
Version=1.0
Name=Pale Moon Web Browser
Comment=Browse the World Wide Web
Keywords=Internet;WWW;Browser;Web;Explorer
Exec=palemoon %u
Terminal=false
X-MultipleArgs=false
Type=Application
Icon=palemoon
Categories=Network;WebBrowser;Internet
MimeType=text/html;text/xml;application/xhtml+xml;application/xml;application/rss+xml;application/rdf+xml;image/gif;image/jpeg;image/png;x-scheme-handler/http;x-scheme-handler/https;x-scheme-handler/ftp;x-scheme-handler/chrome;video/webm;application/x-xpinstall;
StartupNotify=true
Обновить кэш значков
gtk-update-icon-cache -f /usr/share/icons/hicolor
В системе должны быть gtk2 gtk-engines gtk-engine-murrine
Русская локаль https://addons.palemoon.org/releases/pm-langpack-ru/
Если нет желания возиться с about:config в строке адреса, установите переключатель локалей https://addons.palemoon.org/addon/locale-switcher/
Обновление в программе - автопроверка и установка новой версии или удаление директории ~/palemoon и распаковка туда нового архива. Подробнее http://linux.palemoon.org/help/installation/

Если сложно, то просто скачать архив с http://linux.palemoon.org/download/mainline/
распаковать в ~/ и в распакованной директории запустить файл palemoon.
Без gtk2 не эапустится, без gtk-engines gtk-engine-murrine не выглядит, как приложения gtk3.
Удаление
файловых ассоциаций
rm /home/*/.local/share/applications/userapp-Pale\ Moon-*.desktop /home/*/.local/share/applications/mimeinfo.cache
и остальных файлов
rm -rf /usr/bin/palemoon ~/palemoon /usr/share/applications/palemoon.desktop /usr/share/icons/hicolor/*/apps/palemoon.png
Источник - сайт программы http://linux.palemoon.org/help/installation/
