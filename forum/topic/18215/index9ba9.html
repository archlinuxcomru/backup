Я бы с удовольствием, но его нет:
$ ls -1 .i3/
config_panel
config_sidemon
config_winlap
graph.sh
menu.sh
mouse.sh
msg.sh

По сути использую только возможность i3bar выводить поступающую к нему информацию в JSON:
~/.i3/config_panel
--------
bar {
font pango:DejaVu Sans Condensed 20
status_command $HOME/.i3/panel.sh
position bottom
tray_output none
workspace_buttons no
colors {
background #121212
statusline #d0d0d0
}
}
т.е. возможности куда шире стандартного i3status'а, хоть и отъедает чуть больше процессорного (и пользовательского :)) времени.
