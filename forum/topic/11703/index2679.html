[РЕШЕНО] Ничего не собирается из AUR

Установил yaourt, прописал следующий репозиторий в /etc/pacman.conf
[archlinuxfr]
Server = http://repo.archlinux.fr/$arch

Затем обновился с помощью sudo pacman -Syu
Но не один пакет не хочет собираться.
Приведу два примера сборки:

1. Попытка установить rosa-media-player
Все доходит до приведенных ниже строк и на этом останавливается
==> Проверка исходных файлов с помощью sha1sums...
    f3a2060983e794f2a8f168ddec8a7a9a5d81a28d ... Готово
==> Распаковка исходных файлов...
  -> Извлечение f3a2060983e794f2a8f168ddec8a7a9a5d81a28d с помощью bsdtar
==> Запускается build()...
ROMP Version is #define ROMP_VERSION "1.6"
ROMP Release is #define ROMP_RELEASE "8"
/tmp/yaourt-tmp-improovizator/aur-rosa-media-player/./PKGBUILD: line 36: make: команда не найдена
==> ОШИБКА: Произошел сбой в build().
    Преждевременный выход...
==> ERROR: Makepkg was unable to build rosa-media-player.
==> Restart building rosa-media-player ? [y/N]
==> ------------------------------------------
==>
Взглянул на указанный документ на строку 36. В ней следующий кусочек кода:
{
  cd "$srcdir/$pkgname"
  ./get_romp_version.sh $pkgver $_upstream_rel
  make PREFIX=/usr QMAKE=/usr/bin/qmake-qt4 LRELEASE=/usr/lib/qt4/bin/lrelease no-midi-support
}
make PREFIX - это и есть 36-ая строчка.

2. Установка аудиоплеера muine
==> Распаковка исходных файлов...
  -> Извлечение muine-0.8.11.tar.bz2 с помощью bsdtar
==> Запускается build()...
/tmp/yaourt-tmp-improovizator/aur-muine/./PKGBUILD: line 40: patch: команда не найдена
==> ОШИБКА: Произошел сбой в build().
    Преждевременный выход...
==> ERROR: Makepkg was unable to build muine.
==> Restart building muine ? [y/N]
==> ------------------------------
==>
Открыл указанный в консоли файл и нашел строчку с ошибкой:
build() {
  cd "$srcdir/$pkgname-$pkgver"
  patch -Np1 -R -i "$srcdir/use-coloredcellrendererpixbuf.patch"
  patch -Np1 -i "$srcdir/fix-multimedia-key-support.patch"
  patch -Np1 -i "$srcdir/use-notify-sharp.patch"
  patch -Np1 -i "$srcdir/fix-deprecations.patch"
  patch -Np1 -i "$srcdir/use-standard-icons.patch"
  patch -Np1 -i "$srcdir/use-dbus-sharp.patch"
  patch -Np1 -i "$srcdir/fix-db-build.patch"
  patch -Np1 -i "$srcdir/drop-gnome-sharp-dependency.patch"
  autoreconf -fi
  ./configure --prefix=/usr --sysconfdir=/etc --localstatedir=/var \
              --disable-static --disable-schemas-install \
              --with-gconf-schema-file-dir=/usr/share/gconf/schemas
  make
patch -Np1 -R -i "$srcdir/use-coloredcellrendererpixbuf.patch" это и есть та 40-ая строчка
И такая хрень происходит при попытке собрать любое приложение. Из официальных репозиториев все устанавливается и работает нормально. Надеюсь на помощь, Спасибо!
Всё меняется.
