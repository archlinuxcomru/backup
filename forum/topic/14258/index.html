5. Хочу также перенести в РАМ все временные файлы, /tmp, и логи (/var/log) - вот здесь пока не понял как это сделать, подскажите, кто знает
Переносить логи в RAM - плохая затея, как потом будешь "разбор полётов" делать? Ведь после перезагрузки все данные логи будут пустыми.
Вот для примера мой fstab для SSD

# /etc/fstab: static file system information
#
# <file system>	<dir>	<type>	<options>	<dump>	<pass>
# /dev/sdb6 LABEL=ROOT
UUID=248a0e64-28df-465a-877f-7dec69ca888a /  ext4  rw,relatime,discard 0 1
tmpfs  /tmp         tmpfs nodev,nosuid			            	0 0      #монтируем в RAM
tmpfs  /var/tmp   tmpfs nodev,nosuid                                    0 0
