УРА!!! У меня заработало) Всем спасибо) Вот таким образом всё запускается:

1) Выполнить:
nano /etc/ppp/ip-pre-up
Вписать:
#!/bin/sh
/sbin/route del default
Выполнить:
chmod +x /etc/ppp/ip-pre-up
2)Выполнить:
nano /etc/resolv.conf
Вписать:
# Generated by resolvconf
nameserver 8.8.8.8
3)Выполнить:
systemctl stop [email protected]
4)Выполнить:
nano /etc/ppp/peers/provider
Вписать:
plugin rp-pppoe.so
# network interface
eth0
# login name
user логин
persist
defaultroute
hide-password
noauth
usepeerdns
Выполнить:
nano /etc/ppp/chap-secrets и nano /etc/ppp/pap-secrets
Дописать:
логин * пароль
5) Выполнить:
systemctl enable [email protected]
systemctl start [email protected]
