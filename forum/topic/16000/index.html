alainm, не читали конечно...

В принципе для этих целей polkit можно не устанавливать - выключать, перезагружать комп, напр., так:
$ sudo systemctl poweroff
$ sudo systemctl reboot
отключив предварительно запрос пароля для этих действий в /etc/sudoers.
