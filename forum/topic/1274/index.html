Сегодня решил попробовать LightDM вместо Slim - нелегкая дернула :)
systemctl disable slim.service
systemctl stop slim.service
и включил lightdm.service. Запуститься с ним не сумел.
Теперь пытаюсь включить slim.service и получаю
$ sudo systemctl enable slim.service
Failed to issue method call: File exists
Перезагрузка не помогла. В /etc/systemd/system отсутствует папка graphical.target.wants, куда должна быть помещена ссылка.
Заранее благодарен.
