Как-то так:
import XMonad.Layout.PerWorkspace
import XMonad.Layout.SimplestFloat
...
myLayout = onWorkspace "4" simplestFloat (tiled ||| Full ||| Mirror tiled ||| ...
На 4ом столе будет компоновка с плавающими окнами.
