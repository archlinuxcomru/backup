# 4 года, 3 месяца назад (отредактировано 4 года, 3 месяца назад)
Темы: 2
Сообщения: 10
Участник с: 04 июня 2018
В общем достало меня уже это ядро. Прошу помощи. Сильно не пинайте, я в Арче всего месяц.
1. Что хочу.
Хочу ядра под мои нужды, с теми опциями в ядре, которые мне нужны, а не которые включены по умолчанию.
Хочу BFQ по умолчанию, хочу попробовать новый фраймбуфер для nvidia, хочу четырёх желтеньких пингвинчиков при загрузке, отзывчивую Desktop-систему c 1000Гц. Мышь свою A4TECH USB (7 кнопок) нормально работающую хочу. Да мало ли ещё чего хочу...
Поскольку ABS приказала долго жить, пришлось разбираться с установкой в Арче ядра в слепую. До этого долгие годы был на Debian - прекрасная система. Решил попробовать Арч.
С Debiana вынес конфиг ядра, который хорошо на нём работал. На Debiane устанавливал требуемое мне отзывчивое ядро, драйвера Nvidia, запускал игрушки, вроде не плохо жил. В WOT с модами fps был равен 50. Захотел того же на арчи и даже большего.

2. Что делал.
Ниже приводится примерный вариант действий (чтобы понятно что да как в системе).

Ядро которое в системе:
# uname -a
Linux Arch 4.16.13-1-ARCH #1 SMP PREEMPT Thu May 31 23:29:29 UTC 2018 x86_64 GNU/Linux

Установил библиотеки Nvidia:
pacman -Syu  nvidia-390xx lib32-nvidia-390xx-utils nvidia-settings lib32-libpulse lib32-opencl-nvidia-390xx

С ванильным ядром всё работает нормально. Система загружается, nvidia своё отрабатывает.
Собираю ядро:
cd ~/
mkdir build
cd build/
asp update linux
asp checkout linux
cd ./linux/repos/core-x86_64/

Далее, подменяю конфиг идущий в составе на свой.

Генерирую хэш config и всвтавляю его в нужное место в PKGBUILD.
# sha256sum config
496e4c21891c44fee51a35d48e376d1875f3bb051957c2bff39efa199755a85c  config

Запускаю ядро на сборку:
makepkg -s

Во время сборки проверяются хэши всех source файлов для PKGBUILD в данной папке - ошибок нет. Ядро собирается без ошибок.
