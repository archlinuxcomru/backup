У меня есть небольшой тестовый линукс на основе арча, загружаемый через PXE. Я для него рисовал скриптик, который прямо при загрузке определяет какая у нас видюха и ставит нужный драйвер. Так вот для 7300GS (вообще для всей 7-й серии, просто 7300GS тестовой видюхой лежит, потому с уверенностью говорю о ней) ставится и прекрасно работает именно последний драйвер. 173xx только для 5-й серии.
