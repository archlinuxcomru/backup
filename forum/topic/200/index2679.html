Поддержу Mashin, SublimeText хороший редактор. Что касается vim, его нужно малость поднастроить:
Для тегов в .vimrc прописываем:
"Автодополнение кавычек/тегов
:inoremap " ""<LEFT>
:inoremap ' ''<LEFT>
imap [ []<LEFT>
imap ( ()<LEFT>
imap { {}<LEFT>
imap < <><LEFT>
так можно настроить любые теги, которые вам нужно дополнять.

Расстановка отступов:

set tabstop=4 
set shiftwidth=4
set smarttab
set expandtab "Ставим табы пробелами
set softtabstop=4 "4 пробела в табе
"Автоотступ
set autoindent

Используйте шаблоны для необходимых расширений файлов. Ставим плагин vim templates, в каталоге .vim/templates/ создаём свои шаблоны, которые vim будет подгружать при создании нового файла с расширением из templates. Например для html:
.vim/templates/template.html
<!DOCTYPE HTML>
<html>
<head>
/*Прописываем мета-теги для поисковых систем, указываем кодировку*/
    <meta http-equiv="Content-type" content="text/html; charset=utf-8" />
    <meta name="author" content="автор" />
    <meta name="keywords" content="ключевые слова" />
    <meta name="description" content="описание" />
    <link rel="stylesheet" type="text/css" href="style.css"/>
 /*Заголовок страницы*/
<title>Заголовок</title>
</head>
<body>
</body>
</html>
и дописать в .vimrc
:autocmd BufNewFile  *.html      0r ~/.vim/templates/template.html
И не нужно каждый раз писать базовые теги. Удобно.
