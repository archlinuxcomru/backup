У меня 403fm работает нормально в Debian Wheezy, если мне не изменяет память, мне пришлось прописать в файле
/etc/modprobe.d/tvcard.conf
options saa7134 card=27 tuner=5 alsa=1 secam=d
После этого нормально всё заработало. Не могу сказать про pci звук, у меня шнурочком к бластеру подключен выход тюнера.

#!/bin/sh

--------------

Нашел в недрах отдела старенький тюнер на чипе Philips SAA7133/7134/7135. Что бы запустить пришлось порядком погуглить.
В итоге для просмотра тв используется tvtime, для прослушивания радио gnomeradio.
Скрипт запуска gnomeradio, так как в чистом виде завести у меня не получилось.... И громкость только общий мастер... В перспективе надо пробовать исправить:
#!/bin/sh
if [ -f /usr/bin/arecord ]; then
arecord -q -D hw:2,0 -r 32000 -c 2 -f S16_LE | aplay -q - &amp;
fi
if [ -f /usr/bin/gnomeradio ]; then
gnomeradio -m --mixer=/dev/mixer:pcm
fi
pid=`pidof arecord`
if [ $pid ]; then
kill -9 $pid
fi
Скрипт запуска tvtime:
#!/bin/sh
if [ -f /usr/bin/arecord ]; then
arecord -q -D hw:2,0 -r 32000 -c 2 -f S16_LE | aplay -q - &amp;
fi
if [ -f /usr/bin/tvtime ]; then
tvtime -m --mixer=/dev/mixer:pcm
fi
pid=`pidof arecord`
if [ $pid ]; then
kill -9 $pid
fi
