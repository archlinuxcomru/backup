Темы: 29
Сообщения: 447
Участник с: 05 января 2021
Обновил:
Добавлено принятие параметров ком-строки для Find_GUI, чтобы использовать в контекстном меню файлового менеджера.
В качестве параметров может быть путь к папке и путь к конфигу, причём при совместном использовании двух параметров первым должен быть путь к папке, иначе второй параметр будет игнорироваться.

Для установки использовать PKGBUILD, который скачает архив, соберёт и установит. Пока у меня нет арч-ОС чтобы сделать zst.

В архиве есть пример для Nemo как добавить в конт. меню.
