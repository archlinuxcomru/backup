Дайнгрейдиться не так то просто.
Заход на версии PHP Version 7.0.9 пакетов php, php-apache, php-gd не слишком удачен.
httpd не стартует.

Syntax error on line 1 of /etc/httpd/conf/mods-enabled/php.conf: Cannot load modules/libphp7.so into server: libreadline.so.6: cannot open shared object file: No such file or dir

и далее

warning: downgrading package readline (7.0.001-1 => 6.3.008-4)
resolving dependencies...
looking for conflicting packages...
error: failed to prepare transaction (could not satisfy dependencies)
:: bash: installing readline (6.3.008-4) breaks dependency 'readline>=7.0'

Сделал cсылку.

sudo ln -s /usr/lib/libreadline.so /usr/lib/libreadline.so.6.0
sudo ln -s /usr/lib/libreadline.so /usr/lib/libreadline.so.6

Заработало!
