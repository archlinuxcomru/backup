PRIME render offload
NVIDIA driver since version 435.17 supports this method.

It needs a specific set of patches to the xorg-server that are present since version 1.20.6-1 on Arch. Also, starting from Xorg 1.20.7, the Xorg configuration is not needed anymore, since the needed options are already present on the driver directly.

As per the official documentation, it works with the modesetting driver over Intel graphics card and the AMDGPU driver over AMD graphics card (since version 450.57); success has been had with the Intel driver instead regardless. Refer to Intel graphics#Installation for more information.

The nvidia-prime package provides a script that can be used to run programs on the NVIDIA card.

If, for some reason automatic configuration does not work, it might be necessary to explicitly configure X with a Xorg#Using xorg.conf file:

/etc/X11/xorg.conf
Section "Device"
Identifier "iGPU"
Driver "modesetting"
EndSection

Section "Screen"
Identifier "iGPU"
Device "iGPU"
EndSection

Section "Device"
Identifier "dGPU"
Driver "nvidia"
EndSection
