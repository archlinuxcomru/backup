Поставил, без той команды не работает всё равно
так для проверки нужно выбрать соответствующий файл icd при загрузке программы
pacman -Ql amdvlk
pacman -Ql vulkan-amdgpu-pro
pacman -Ql vulkan-radeon
и, соответственно, вставлять указанные icd файлы в качестве переменной окружающей среды.
на данный момент по умолчанию в качестве файлов icd загружаются icd из admvlk
