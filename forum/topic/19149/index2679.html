Возможно кеш иконок не корректно создался.
Проверьте на всякий случай наличие иконок, при установке фокса должны были поставиться
ls -R /usr/share/icons/hicolor/ |grep firefox

И проверьте что бы без ошибок отработало создание кеша
sudo gtk-update-icon-cache -f -t /usr/share/icons/hicolor
sudo gtk-update-icon-cache -f -t /usr/share/icons/ваша используемая тема иконок
#или если используются иконки из домашней папки
gtk-update-icon-cache -f -t  ~/.icons/ваша используемая тема иконок
должно отработать без ошибок gtk-update-icon-cache: Cache file created successfully

Путаете boot и efi раздел. Когда читали.
efi-раздел монтируется в boot, который может быть папкой в корне, а может быть отдельным разделом (как вам лучше).
Если UEFI- то раздел efi (в fat32 формате) отдельный, смонтированный в boot, обязателен.
Если Legacy BIOS, то все можно на одном разделе, папка boot в корне, efi раздел не нужен. В зависмости, конечно, от нюансов,- будете ли вы шифровать диск и т.д.
