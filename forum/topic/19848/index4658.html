Если не ошибаюсь нет, я не включал службу dhcpcd перед exit и reboot
ну так включите, на livecd он автоматически подымается
выполните:
systemctl enable dhcpcd
systemctl start dhcpcd
enable - добавляет в автозапуск
start - запустить прямо сейчас
Unit file dhcpcd.service does not exist Вот это пишет когда пишу команду systemctl enable dhcpcd и на systemctl start dhcpcd тоже так

Unit file dhcpcd.service does not exist Вот это пишет когда пишу команду systemctl enable dhcpcd
значить не установлен пакет dhcpcd, устанавливайте
загрузитесь с livecd с которого ставили, примонтируйте раздел с системой и установите
mount /dev/sdX1 /mnt
pacstrap /mnt dhcpcd
вместо sdX1 ставьте свой раздел с корнем который прописывали при установке
