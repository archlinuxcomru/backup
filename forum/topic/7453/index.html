Судя по инету, известный баг в Эклипсе при работе c gtk > 2.18.0

Предлагается добавить
export GDK_NATIVE_WINDOWS=true
перед запуском эклипса (например в .bashrc и перелогиниться).

А где вы прописываете переменную?
По идее если в .bashrc то должно работать везде (после логина). Проверить негде - Эклисп не пользую.
Если совсем уж все плохо, можно отредактировать файл запуска Эклипса в /usr/share/applications заменив там в строке Exec
eclipse
на
export GDK_NATIVE_WINDOWS=true && eclipse
например.
