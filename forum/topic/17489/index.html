Ну как ты и посоветовал может кому пригодится

# git config --system http.sslVerify false
# git config --global http.sslVerify false
$ curl-config --ca
**/etc/ssl/certs/ca-certificates.crt** может быть такой путь **/etc/ca-certificates/extracted/ca-bundle.trust.crt**
# git config --system http.sslcainfo /bin/curl-ca-bundle.crt
