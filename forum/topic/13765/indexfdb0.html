Провел эксперимент, как простым способом менять ассоциации файлов, не используя для этого существующие утилиты.
Воспользовался для этого тем моментом, что файл /usr/share/applications/mimeinfo.cache отвечает при открытии файла за опцию «рекомендуемые программы».
Плюнул на все теории, что файл mimeinfo.cache не редактируется, на всякий случай сделал копию этого файла и дал команду
$ sudo nano /usr/share/applications/mimeinfo.cache
закомментировал строчку # application/x-cd-image=org.gnome.FileRoller.desktop;brasero.desktop;
ответственную у меня за ассоциацию iso файлов (назначена соответствующая прога)
перегрузил гномовский shell — и, образы iso вижу, но открыть не могу, вернул все на место, на всякий случай сделал update базы
$ sudo update-mime-database /usr/share/mime
Можно смело применять этот простой метод, но нужно знать приложение, ответственное за обработку файла
