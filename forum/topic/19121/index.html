Найдите свой сканер по lsusb :
 $  lsusb
Bus 002 Device 001: ID 1d6b:0003 Linux Foundation 3.0 root hub
Bus 001 Device 003: ID 04f2:0833 Chicony Electronics Co., Ltd
Откройте файл /lib/udev/rules.d/49-sane.rules и впишите в него:
# Kyocera Color M5521cdw
ATTRS{idVendor}=="xxxx", ATTRS{idProduct}=="xxxx", MODE="0664", GROUP="scanner", ENV{libsane_matched}="yes"
Где первые хххх ваш ID из lsusb, второй хххх ( после двоеточия : ) idProduct.
