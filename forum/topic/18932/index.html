xfce\gedit

По умолчанию, текстовые gedit-документы - во всяком случае, "обычные" файлы - открываются в одном и том же окне в разных вкладках. Мне это для работы очень неудобно - нужно, чтобы по умолчанию они открывались в разных окнах, как открываются окна Тюнара. Где чего для этого подкрутить надо? Разумеется, в Preferences gedit-a в его окнах подобной настройки нет.

P.S. Другие текстовики вроде традиционно горячо любимого mc не предлагать )
