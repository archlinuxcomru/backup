Имеется дуал бут диск с виндой:
NAME   MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT FSTYPE ROTA
sda      8:0    0 111.8G  0 disk                      0
|-sda1   8:1    0   500M  0 part            ntfs      0
|-sda2   8:2    0  64.4G  0 part            ntfs      0
`-sda3   8:3    0  46.9G  0 part /          ext4      0
Нужно переформатировать его в btrfs и оставить один лишь линукс.
По идее, алгоритм такой:
  1. С помощью dd или cp переместить систему во временное хранилище.
  2. Оставить на диске один раздел и отформатировать его в btrfs
  3. Тем же инструментом закинуть систему обратно
  4. Установить grub
  5. Сгенерировать fstab
Сработает?
Можно просто dd'шкой все скопировать из ext4 в ntfs, и из ntfs в btrfs?
Как загрузчик ставить? По-дефолту grub-install ; grub-mkconfig, и ничего не сломается?
