Только написал, и тут же нашёл и причину и решение...
Ну может кто-то тоже страдает, тогда думаю не помешает эта тема на русскоязычном форуме.

Собственно причина в gtkmm. Вернее в его версии 2.24.3-1. Если откатить до 2.24.2-2, вся трабла пропадает.
Решение для x86_64:
sudo pacman -Suy #Полное обновление
sudo mkdir -p /usr/local/lib/gtkmm && cd /usr/local/lib/gtkmm
sudo wget http://arm.konnichi.com/2013/01/05/extra/os/x86_64/gtkmm-2.24.2-2-x86_64.pkg.tar.xz
sudo tar -xvf ./gtkmm-2.24.2-2-x86_64.pkg.tar.xz
LD_LIBRARY_PATH=/usr/local/lib/gtkmm/usr/lib/ vmplayer #Стартуем виртуалку со старой библиотекой
Решение для i686:
sudo pacman -Suy #Полное обновление
sudo mkdir -p /usr/local/lib/gtkmm && cd /usr/local/lib/gtkmm
sudo wget http://arm.konnichi.com/2013/01/05/extra/os/i686/gtkmm-2.24.2-2-i686.pkg.tar.xz
sudo tar -xvf ./gtkmm-2.24.2-2-i686.pkg.tar.xz
LD_LIBRARY_PATH=/usr/local/lib/gtkmm/usr/lib/ vmplayer #Стартуем виртуалку со старой библиотекой

Всё взято отсюда.
