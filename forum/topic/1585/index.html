man pacman
Architecture = auto | i686 | x86_64 | …
If set, pacman will only allow installation of packages of the given architecture (e.g. i686, x86_64, etc).
The special value auto will use the system architecture, provided by in “uname -m”. If unset, no architecture
checks are made. NOTE: packages with the special architecture any can always be installed, as they are
meant to be architecture independent.
