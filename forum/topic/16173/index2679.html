Доброго дня!

Кнопки управления окном находятся справа,расположение заголовка По центру. В этом варианте есть небольшое смещение заголовка,вполне приемлемое.
Перетаскиваю кнопки влево.Заголовок сильно смещается вправо.Как поправить это?

https://www.dropbox.com/s/cd05q44uyhzgn6m/1.png?dl=0
https://www.dropbox.com/s/nznfjdofjywc364/2.png?dl=0

Изменив значение "title_horizontal_offset" в редакторе настроек можно добиться результата,но только для развернутых окон.

Поставьте справа три неиспользуемые вами кнопки (меню,прикрепить,свернуть в заголовок).
Можете сделать их прозрачными отредактировав изображения в /usr/share/themes/Arc-Dark/xfwm4 вроде ваша тема
