Решил задачу нужно с начало добавить запись в /etc/apt/sources.lists
deb http://deb.debian.org/debian/ stretch main
При обновлении зеркал apt update вылезает ошибка о ключах их добавил командой apt-key adv --keyserver keys.gnupg.net --recv-keys EF0F382A1A7B6500
Потом все заработало apt update обновился
Только если ставить пакеты через apt, то кучу зависимостей тянет

Зеркала обновляются, но как я понял debian пакеты таким способом нужно устанавливать в отдельный каталог, иначе система сломается
тестирую в вируалке в каталог /mnt
туда с начало надо установить pacman командой pacstrap -i /mnt, потом yaourt -S -r /mnt apt, добавить зеркала в /mnt/etc/apt/sources.list, добавить ключи apt-key adv --keyserver keys.gnupg.net --recv-keys EF0F382A1A7B6500
apt -o Dir=/mnt update обновить базу, но пакеты никак не поставить например apt -o Dir=/mnt mc выдает ошибку зависимостей и зависимости если ставить тоже ошибка зависимостей )) как то по кругу
