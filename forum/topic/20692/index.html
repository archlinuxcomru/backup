добрый день, ув. форумчане. Прошу вашей помощи. Имею Arch x64 на своём ПК. На Виндовс продолжительное время пользуюсь браузером SlimJet, на Линукс не пробовал его ставить, не подумал что у них есть версия для Linux. Сегодня находясь в оболочке ArchLinux зашёл на сайт этого браузера чтобы обновить на флешке инсталлятор для Виндовс. и тут мне высветило что есть версии для Линукс. Сразу возникла мысль поискать в AUR. Поискал - нашёл. Ставлю -
yay -S slimjet
всё поставилось, всем доволен. Включаю видео на Ютуб а оно не работает. Оказалось в этом браузере нет декодеров для проигрывания MP4 файлов (лицензионные нарушения)
по этой ссылке решение вопроса - https://www.slimjet.com/en/libffmpeg.php
зашёл туда, сделал как просят, скачал файл с библиотекой, распаковал. Запускаю браузер, включаю видео на Ютуб - мне пишет "ошибка воспроизведения аудио"
прошу помогите пожалуйста. Если кому не сложно - поставьте этот браузер, подмените указанную библиотеку и посмотрите будет ли у вас такое самое?
