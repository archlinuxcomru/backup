Чтобы включить «песочницу» для повышения уровня безопасности, необходим доступ администратора. Выберите «Использовать команду sudo», если у вас есть доступ sudo к таким системам, как ubuntu. Выберите «Использовать команду su», если у вас есть доступ к рут-аккаунту. Если у вас нет доступа с правами администратора, выберите «Выход» и добавьте переключатель «–не-песочница» для запуска Slimjet без «песочницы» (менее безопасно).
1) Использовать команду sudo
2) Использовать команду su
3) Выход
Пожалуйста, выберите один из вариантов для продолжения:
