есть мнение, что арч сложный в освоении, потому что там всё в консоли, и поэтому многие убунтоводы боятся на него переходить.
на самом деле ничего сложного нет, и сейчас вы в этом убедитесь =)

ВНИМАНИЕ: основная задача этого хауту не научить новичка работать в арче/линуксе, а просто предоставить ему готовую рабочую операционку ubuntu-like, в которой он сначала смог бы работать, а уже потом, если есть желание, учиться, осваивать вики и тд.

быстрая установка арча для тех кому лень читать вики. распечатать на бумажке.
перед установкой убедитесь что вы не идиот

Установка с диска:
рекомендации - ставить на reiserfs, без свапа
предполагается что с установкой с диска вы справились
в конце установки вам будет предложено настроить текстовые конфиги, а также придумать пароль рута. можете сделать это сейчас или позже, после логина

Поднятие инета:
<тут должна идти телега про настройки инета и 2 часа мороки с каким-нибудь юсб-модемом>

Обновление системы:
# pacman -Syu

Добавление юзера:
добавляем простого смертного пользователя:
# adduser
устанавливаем пакет sudo:
# pacman -S sudo
дальше добавляем юзера в группу wheel (этой группе можно использовать судо):
# usermod -aG wheel ваш_юзер
дальше разрешаем это в файле sudoers (раскомментируем строку с wheel):
# nano /etc/sudoers

Русификация:
# nano /etc/locale.gen
ищем строчку с ru_RU.UTF-8 и раскомментируем её, затем
# locale-gen
# nano /etc/rc.conf
прописываем:
LOCALE="ru_RU.UTF-8"
HARDWARECLOCK="UTC"
TIMEZONE="Europe/Moscow"
KEYMAP="ru"
CONSOLEFONT="cyr-sun16"
CONSOLEMAP=
USECOLOR="yes"

Поднимаем иксы:
устанавливаем дрова на видео + xorg + gnome + шрифты + флэш:
# pacman -S xorg nvidia gnome gnome-desktop gnome-system-tools gnome-extra ttf-ms-fonts ttf-dejavu flashplugin
на вопрос “ставить ли всё скопом?” отвечаем нет и отсеиваем кучу лишнего говна
ждём, пока установится, затем запускаем демоны:
# /etc/rc.d/hal start
# /etc/rc.d/fam start
конфигурируем xorg специальной утилитой от nvidia:
# nvidia-xconfig
запускаем gdm:
$ gdm

не пугаемся, привыкаем.
логинимся обычным юзером.

Настройка rc.conf:
# nano /etc/rc.conf
пихаем наши демоны и гдм в секцию daemons чтоб грузились автоматом
@hal @fam @portmap @gdm

для успешного монтирования ntfs делаем:
# pacman -S ntfs-3g


Установка программ, которых нет в репозиториях:
вариант 1. идём на сайт производителя, качаем исходники, собираем и тд.
вариант 2. в арче есть замечательный репозиторий AUR, в котором есть практически всё. 
# pacman -S firefox
заходим на сайт http://aur.archlinux.org , ищем, качаем pkgbuild, устанавливаем его:
$ makepkg -p ваш_скрипт
сборка пакета может занять долгое время, готовый пакет имеет вид “имя-версия-i686.pkg.tar.gz”
# pacman -U пакет_который_получился

установка Оперы:
качаем тар с сайта оперы, распаковываем, затем:
# /путь_до_папки_c_оперой/install.sh
Оперу также можно поставить из AUR.

Примечание для тролллей: вместо fam можно поставить gamin, вместо nvidia - ati, вместо гнома кеды, вместо рейзера ext2, вместо оперы lynx, вместо арча генту. <далее 10 способов собирания дистрибутива на коленке используя канцелярские скрепки, клей Момент и пассатижы.>
спасибо за внимание.
