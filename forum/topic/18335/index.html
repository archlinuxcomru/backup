в пакете есть какой то скрипт для сборки модуля.
# catalyst_build_module all

и есть hook fglrx который можно в раздел hooks mkinitcpio.conf прописать. (не в modules а hooks)
после чего:
# mkinitcpio -p linux

проверить собран модуль или нет можно так:
# catalyst_build_module check

пробуйте а там видно будет
