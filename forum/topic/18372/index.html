Сперва проверьте сеть, могли поставить что то изменившее настройки вроде network-manager
ping -c3 google.com
#если не работает
ping -c3 8.8.8.8
Если не пингуется google.com настраиваем сеть.
Вики
https://wiki.archlinux.org/index.php/Network_configuration
Судя по картинке сеть или dns не рабочие
(картинки пожалуйста выкладывайте превьюшками лучше на http://itmages.ru или https://imgur.com https://ultrapic.net )
Если пингуется
Пробуем пинговать сервер репозитория записанный первым в /etc/pacman.d/mirrorlist
например для http://mirror.yandex.ru/archlinux/$repo/os/$arch
ping -c3 mirror.yandex.ru
Если пинга нет заменяем в /etc/pacman.d/mirrorlist на другой рабочий
Например для России
## Russia
Server = http://mirror.aur.rocks/$repo/os/$arch
Server = https://mirror.aur.rocks/$repo/os/$arch
Server = http://mirror.rol.ru/archlinux/$repo/os/$arch
Server = https://mirror.rol.ru/archlinux/$repo/os/$arch
Server = http://mirror.truenetwork.ru/archlinux/$repo/os/$arch
Server = http://mirror.yandex.ru/archlinux/$repo/os/$arch
Server = https://mirror.yandex.ru/archlinux/$repo/os/$arch
Server = http://archlinux.zepto.cloud/$repo/os/$arch
Все остальное удаляем или комментим, первым используется сервер из верха списка
Сохраняем файл и
pacman -Syyu
Посмотреть актуальные зеркала тут
https://www.archlinux.org/mirrorlist/
Вики
https://wiki.archlinux.org/index.php/Mirrors#Official_mirrors
