Templainer и shpinog, я ни тем, ни другим не пользуюсь, у меня обычная, 3-х кнопочная. Как я понял из той статьи, можно указать геймпаду "место" через переменную 'SDL_JOYSTICK_DEVICE=/dev/input/js0' или, может быть, для нужной вам расстановки девайсов составить соответствующее правило udev, типа (лишь в кач-ве примера, вам нужны, само собою, названия именно ваших девайсов):

# Set joystick device priority 
KERNEL=="js?", ATTRS{name}=="SHENGHIC 2009/0708ZXW-V1Inc. PLAYSTATION(R)3Conteroller", NAME="input/js1", RUN+="/usr/bin/startjoy1"
KERNEL=="js?", ATTRS{name}=="PLAYSTATION(R)3 Controller (hidraw)", NAME="input/js0"
подробнее здесь: http://ubuntuforums.org/showthread.php?t=2186521 или у гугла.
