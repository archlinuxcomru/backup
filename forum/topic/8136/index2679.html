Обновлена библиотека extra/libjpeg до версии 7-1. Зависимый софт пересобран не весь. Поэтому, если встретили подобную ошибку:
error while loading shared libraries: libjpeg.so.62: cannot open shared object file: No such file or directory
рекомендуется сделать подобный симлинк:
ln -s /usr/lib/libjpeg.so /usr/lib/libjpeg.so.62
или попробовать откатить пакет на предыдущую версию.
Разберемся, голубчик!
