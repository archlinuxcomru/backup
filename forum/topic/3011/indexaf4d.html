Минусы - скорее даже вопрос - на сколько стабильна ветка 3.6 ядра?
3.6 - для сервера вообще не нужно.
Для этого есть stable longtime supported kernel package suitable for servers.
~ $ yaourt linux-lts
1 testing/linux-lts 3.0.53-1
    The linux-lts kernel and modules - stable longtime supported kernel package suitable for servers
2 testing/linux-lts-headers 3.0.53-1
    Header files and scripts for building modules for linux-lts kernel
3 core/linux-lts 3.0.52-1
    The linux-lts kernel and modules - stable longtime supported kernel package suitable for servers
4 core/linux-lts-headers 3.0.52-1
    Header files and scripts for building modules for linux-lts kernel
Тестинг для сервера вообще не советую, да думаю вы и сами это знаете.
У меня два сервера на арче. Один на “чистом” арче, а второй на ArchServer-e.
Вот уже ~ два года как. Глюков не замечено.

У меня дома уже год в качестве шлюза работает сервер на ARCH с самомобранным LTS-ядром.
Ничего лишнего не устанавливал, только необходимый минимум программ. Все нужные сервисы в docker-контейнерах.
