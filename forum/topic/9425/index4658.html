я вообщем то хотел под /var/cache/pacman выделить, чтобы вытяеутые с инета пакеты не потерять (или можно по другому)
Что бы не следить сразу за двумя важными разделами в системе /home и /var/cache/pacman/pkg, следите за только за одним /home. А второй просто слинкуйте на /home. Будет это выглядеть примерно так:
// создаем каталог в домашнем, где будут хранится кэшированные пакеты пакмана
mkdir /home/d33/pkg
// копируем уже находящиеся в кэше пакеты в новое моесто хранения
cp /var/cache/pacman/pkg/*.pkg.tar.gz /home/d33/pkg/
// удаляем, теперь уже, ненужный каталог
rm -rf /var/cache/pacman/pkg
// линкуем новый каталог на старое место 
ln -s /home/d33/pkg /var/cache/pacman
При этом пакман не заметит подмены, и Вам не придется создавать дополнительные разделы.
Разберемся, голубчик!
