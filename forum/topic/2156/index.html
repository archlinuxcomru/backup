Забыли. что такое Arch…
1. Идем по адресу - http://cgit.freedesktop.org/pm-utils/
2. Скачиваем архив с исходниками.
3. Из папки /pm-utils-1.4.1/pm/sleep.d/ берем файл 55NetworkManager.
4. Кладем последний в /etc/pm/sleep.d
5. Делаем файл исполняемым.
6. Пробуем.
7. Если результата нет - идем https://wiki.archlinux.org/index.php/Pm-utils - курим… по поводу Hooks.

А такая команда 
$ dbus_send --system                        \
		--dest=org.freedesktop.NetworkManager \
		/org/freedesktop/NetworkManager       \
		org.freedesktop.NetworkManager.wake && \
	    echo Done. || echo Failed.
после выхода из suspend срабатывает?
Есть некоторые особенности NetworkManager, когда для некоторых карт он не перезапускается, пишут, что это иногда решается перезагрузкой соответствующего модуля. Например - 
modprobe -r ath9k && modprobe ath9k
Это, в случае успеха, внести в вышеупомянутый файл. При выходе из suspend проверьте наличие модуля карты (lsmod).
