Вы страшилки какие-то рассказываете... О каких "местах" речь? N-цать раз менял одно на другое. Меняется беспроблемно.
(Из заметок) Пакеты для
nvidia 340хх:
nvidia-340xx
nvidia-340xx-utils
lib32-nvidia-340xx-utils (опционально для wine)
nouveau:
xf86-video-nouveau (опционально для иксов)
lib32-libglvnd (опционально для wine)
nouveau-fw (опционально для hwdec)
mesa-vdpau (опционально для vdpau)
libva-mesa-driver (опционально для vaapi)
Т.е. в случае с нуво (кроме mesa, разумеется, который и так установлен) можно вообще ничего не устанавливать. У меня сейчас:
$ pacman -Qqs nouveau
nouveau-fw
$ pacman -Qqs mesa
glu
libva-mesa-driver
mesa
