1. в пакете openssh есть серверная часть.
2. ssh позволит только получить доступ к консоли, для чего гораздо удобнее использовать какой нибудь терминал, чем какуюто непонятную программу.
3. для работы 'кнопочек' нужен xbmc
4. magic packet тоже можно слать через терминал
мне кажется лучше установите какой нить vnc если хотите визуально чем то управлять, либо же используйте какие нибудь webui элементы.
p.s. программа вообще какая то стрёмная удалите её, вместе с гугл маркетом и полу гигабайтом непонятного проприетарного шила!!!
