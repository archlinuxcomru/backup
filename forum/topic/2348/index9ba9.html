Без сомнения! Но только для отностительно продвинутых индивидуальных пользователей.
Для массового использования на рабочих станциях и серврерах - я не уверен. Есть отзывы рискнувших? На поддержку уходит больше времени чем на “готовые” декстопные убунты альты и прочие?

На мой взгляд на сегодняшний день Арч это чисто десткопная система. На десктопе с Арчем работать просто удобно и приятно.
Для серверов думаю он мало пригоден. Максимум домашний севрачек, для свои нужд.

А для продакшена конечно нет. Иначе это уже мазохизм. Для продакшена главное стабильность и своевременные обновления безопасности.
Есть заточенные серверные дистры с долгосрочными циклами поддержки. Например:
  • Debian
    Ubuntu Server LTS
    CentOS
