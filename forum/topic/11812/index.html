Но, думаю, будет правильнее, если ссылка будет такой: https://wiki.archlinux.org/index.php/Xdg-open , исходя из чего файловый менеджер по умолчанию лучше прописать в секцию [Default Applications]

Сравните:
[Added Associations]
.....
inode/directory=pcmanfm.desktop;
$ xdg-mime query default inode/directory
audacious.desktop
и
[Default Applications]
.....
inode/directory=pcmanfm.desktop
$ xdg-mime query default inode/directory
pcmanfm.desktop

add. Заместо правки конфига можно:
$ xdg-mime default pcmanfm.desktop inode/directory
