Судя по https://bugs.archlinux.org/task/48554?project=5&cat[0]=33&string=virtualbox-host-dkms WIKI по виртуалбоксу перепишут.
После очередного обновления виртуалбокса, включая псоледний, уже не надо прописывать virtualbox.conf в /etc/modules-load.d/.

Теперь по-умолчанию файл virtualbox-host-dkms.conf находится в /usr/lib/modules-load.d/, который автоматически подгружается также с помощью сервиса systemd-modules-load (https://www.freedesktop.org/software/systemd/man/modules-load.d.html).

PS:Не нужно создавать virtualbox.conf и активировать DKMS. )
