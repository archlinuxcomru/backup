Здравствуйте.
При установки новых пакетов часто появляются файлы .pacnew.
После полного апдейта системы приходится прокручивать всю консоль, смотреть какие конфиги изменились, и мержить файлики вручную, как правило какой нибудь файлик смержить забываю.
По этому хочется найти решения, что бы после апдейтов, автоматически открывалась какая нибудь мержилка файлов и предлагалось внести новые изменения.
Например делаешь полный апдейт системы, какой нибудь пакет создает .pacnew по окончанию установки этого пакета, открывается мержилка и установка дальше не идет пока не закроешь окошко мержилки, т.е. что то типа редактирования PKGBUILD в yaourt, только для мержа файлов.
Есть ли что то подобное?

Без vim…

Если только посмотреть изменения, то
$ fldiff /etc/rc.conf{,.pacnew}

Не только смотреть
$ sudo meld /etc/rc.conf{,.pacnew}
$ sudo mcdiff /etc/rc.conf{,.pacnew}
