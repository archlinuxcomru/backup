выбор ориентация окон для открытия нового приложения
super + alt + {Up,Down,Left,Right} - (north,south,west,east)
Да, есть такая штука - преселект, с которой можно открывать новое окно в любом месте, она подсвечивает где откроется окно. По сути это делает ненужным множество лайоутов (как, например, "решетка", "спираль" и пр. в xmonad), поскольку в любой момент это можно решить вручную по-своему. Хотя, для этого потребуется лишний раз применить хоткей. Это тоже одна из подкупающих фишек, присущих только bspwm.
