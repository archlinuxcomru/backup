Всем привет, ситуация у меня такая: имею локальную сеть и интернет через 1 сетевую карту, сеть состоит из 3-х подсетей (192.168.128.0, 10.0.0.0, 172.16.0.0), интернет идёт через роутер который находиться на другом конце сети ;)
Мой ip адрес в сети находиться в зоне: 10.0.0.0
Ip адрес роутера: 10.0.3.2
Ip адрес сетевого шлюза для других подсетей: 10.0.5.2

Прописаны route's:
172.16.0.0      10.0.5.2        255.255.248.0   UG    1      0        0 eth1
192.168.128.0   10.0.5.2        255.255.224.0   UG    1      0        0 eth1
10.0.0.0        *               255.255.192.0   U     0      0        0 eth1
default         10.0.3.2     0.0.0.0         UG    3      0        0 eth1

resolv.conf
nameserver 192.168.128.254
domain ionitcom.ru
domain jagry.ru
nameserver 192.168.128.251
domain ionitcom.ru
domain jagry.ru
nameserver 10.0.3.2

Проблема следующая: Все гуёвые приложения работают как надо по всем маршрутам кроме Gajim и через консоль(помимо ping'a) вообще в интернет достучаться не могу.

Подкину немного лулзов :) 
root ~ # pacman -Syu
:: Синхронизируются базы данных пакетов...                                      
ошибка: не удалось получить файл 'core.db.tar.gz' из mir1.archlinuxfr.org : No address record
