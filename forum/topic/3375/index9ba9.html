Имена файлов на рабочем столе трудновато прочитать (не знаю, можно ли как-то цвет шрифта на рабочем столе наутилуса поменять)
Как то так:
~/.themes/ваша тема/gtk-3.0/gtk.css
бла бла бла
@import url("gtk-widgets.css");
@import url("nautilus.css");

~/.themes/ваша тема/gtk-3.0/nautilus.css
 /* Настройки наутилуса */
/* радиус рамки текста  */
.nautilus-canvas-item {
    border-radius: 4;
}
/* текст на рабочем столе - пассивная иконка */
.nautilus-desktop.nautilus-canvas-item {
/* цвет текста */   
    color: #D4E0E0;
    /* тень текста*/
    text-shadow: 1 1 black;
}
/* текст на рабочем столе - активная иконка */
.nautilus-desktop.nautilus-canvas-item:active {
      /* фон текста */   
     background-image:-gtk-gradient (linear,
                   left top, right top,
                   from (shade (#DAE7E8, 0.98)),	
		   color-stop (0.50, shade (#DAE7E8, 1.1)),
                   to (shade (#DAE7E8, 0.98))); 
/* цвет текста */
     color: #000000;
/* тень текста */
     text-shadow: 1 1 #D3DEEE;
}
/* текст на рабочем столе - выбранная иконка */
.nautilus-desktop.nautilus-canvas-item:selected {
 /* фон текста */   
     background-image:-gtk-gradient (linear,
                   left top, right top,
                   from (shade (#DAE7E8, 0.98)),	
		   color-stop (0.50, shade (#DAE7E8, 1.1)),
                   to (shade (#DAE7E8, 0.98))); 
/* цвет текста */
     color: #000000;
/* тень текста */    
     text-shadow: none;  
}

Перед этим посмотреть чтобы не конфликтовали настройки - проверить по ссылкам import url (в gtk.css ) поиск по nautilus.
