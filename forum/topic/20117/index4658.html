Purpursarkans, уточнение в части SysRq - System Request - а точнее, системный запрос, единственная связь с ядром при зависании системы ... и если связь имеется, значит ядро живое.
В части Alt + SysRq + R, дословно, согласно DOC
R - Turns off keyboard raw mode and sets it to XLATE.
расширенно
Pressing Alt+SysRq+R restores the usual (outside X) ASCII mode where you can type characters.
Ошибки не исчезают с опытом - они просто умнеют
