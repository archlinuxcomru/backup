Рекомендуют alias вынести в отдельный файл ~/.bash_aliases, а в ~/.bashrc прописать следующее:
# alias
if [ -f ~/.bash_aliases ]; then
    . ~/.bash_aliases
fi
#
Как пишут - при такой организации, во-первых, ~/.bashrc не перегружается дополнительными записями, а во-вторых, снижается риск его повреждения в результате случайной ошибки.

alias c1='command1'
