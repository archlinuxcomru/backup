Перемонтируем на запись корневую ФС
mount / -o remount,rw
Приводим /etc/fstab к виду
# 
# /etc/fstab: static file system information
#
# <file system>	<dir>	<type>	<options>	<dump>	<pass>
tmpfs		/tmp	tmpfs	nodev,nosuid	0	0
/dev/sda2 none	swap defaults	0	0
/dev/sda7           	/         	ext4      	rw,relatime,data=ordered 0 1
Только с разделом подкачки (swap) сами смотрите, что у вас там.
Все так сделал.swap у меня /dev/sda6 поэтому fstab подправил как надо. Потом reboot. После прописал в 
nano /etc/ppp/peers/имя_подключения
defaultroute
