Лучше так:
Сейчас у нас все пакеты стоят, надо просто загрузить уже, можно сказать, установленную систему.
Теперь достаточно выполнить pptpsetup с параметрами (–start в этом случае можно не вбивать). Он создаст файл с настройками подключения (я рекомендую задать provider как имя подключения) в /etc/ppp/peers/ Надо этот файл открыть, вписать туда defaultroute и… перезагрузиться. После перезагрузки можно просто запускать подключение через pon имя_подключения

Если будет и дальше ругаться на фс только для чтения - сначала разберитесь с этим, интернеты потерпят.
