Осталось подождать, когда эта автоматизация понадобится – как всё сработает.

Итак, сегодня boost-libs 1.75.0-1 -> 1.75.0-2.

Обновление системы
pikaur -Syu --noconfirm
в окончании процесса выдало:
(5/9) Updating icon theme caches...
(6/9) Runs pacdiff utility
(7/9) Check for packages that need to be rebuild
>>> Some packages need to be rebuilt. See list below:
>>> easystroke-git repkg
>>> Run repkg to automatically rebuild all packages that need one.
(8/9) Updating the desktop file MIME type cache...
(9/9) Updating the MIME type database...

Таким образом при вышеназванных текущих условиях (pikaur, а не yay; и в режиме –noconfirm) пакет repkg работает в режиме полуавтомата. Напоминая, что необходимо пересобрать указанный пакет в соответствии с ранее определённым правилом (в данном случае изменение boost-libs).

Выполнение команды
repkg
пересобрало пакет: easystroke-git. Так это работает.
