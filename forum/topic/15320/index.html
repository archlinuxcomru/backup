Потом я подумал, что не нужно мудрить, а пробовать самый простой вариант:
На хосте:
# systemctl enable systemd-networkd
# systemctl start systemd-networkd
В контейнере:
Создал файл /etc/systemd/network/MyDhcp.network

[Match]
Name=host0

[Network]
DHCP=ipv4

И

# systemctl enable systemd-networkd
# systemctl start systemd-networkd

И все завелось.
