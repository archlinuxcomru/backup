KDEшным конфигуратором. Который systemsettings. Почему недостаточно - фиг знает.

В арче KDE не пользовался, в кубунте отконфигурённый гуёвым настройщиком ~/.gtkrc-2.0 выглядит так:
$ cat .gtkrc-2.0
# File created by KDE Gtk Config
# Configs for GTK2 programs
include "/usr/share/themes/oxygen-gtk/gtk-2.0/gtkrc"
include "/etc/gtk-2.0/gtkrc"
style "user-font"
{
        font_name="Ubuntu"
}
widget_class "*" style "user-font"
gtk-font-name="Ubuntu 9"
gtk-theme-name="oxygen-gtk"
gtk-icon-theme-name="oxygen"
gtk-fallback-icon-theme="oxygen"
gtk-toolbar-style=GTK_TOOLBAR_ICONS
gtk-menu-images=1
gtk-button-images=1

И ещё делается симлинк на этот файл, по имени ~/.gtkrc-2.0-kde4.

В моём случае иконки цепляются. Может, пример натолкнёт на какие-то мысли.
