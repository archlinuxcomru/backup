1) Не могу избавиться от назойливого окна после загрузки КДЕ с просьбой ввести пароль. Не помогли рекомендации типа:
echo 10240 > /proc/sys/fs/inotify/max_user_watches && rm -rf ~/.kde/share/apps/nepomuk/ (даже установлен akonadi-fake) или
echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -p
Я вам уже давал ссылку, там про /etc/sysctl.conf ничего нет, это устарело. Попробуйте создать /etc/sysctl.d/90-inotify.conf с содержимым:
#increase inotify watch limit
fs.inotify.max_user_watches = 10000

sharddin
2) Как настроить сочетание клавиш Alt+Ctrl+BackSpace для ручного перезапуска Х-сервера??
У меня для этих целей создан файлик /etc/X11/xorg.conf.d/90-keyboard.conf 
Section "InputClass"
    Identifier             "Keyboard Defaults"
    MatchIsKeyboard        "yes"
    Option                 "XkbLayout" "us, ru"
    Option                 "XkbOptions" "grp:caps_toggle, grp_led:caps, terminate:ctrl_alt_bksp, lv3:ralt_switch, misc:typo"
EndSection
Ключевая настройка terminate:ctrl_alt_bksp, остальное по вкусу.

sharddin
3) Не работает применение изменений настроек системы КДЕ, требующие рут-прав.
Попробуйте в этом окошке ввести свой пароль, а не пароль рута.
