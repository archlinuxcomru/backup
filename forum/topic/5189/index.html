там пишут:
For ALSA 1.0.9rc2 and higher you don't need to setup dmix for analogue output. Dmix is enabled by default for soundcards which don't support hardware mixing. You still need to set it up for digital outputs.
Выход у меня аналоговый, значит эта штука включена, да и в гноме нормально все было. Помнится нужно было пакет какой-то установить, чтоб алса одновременно из нескольких источников воспроизводить могла. Вроде бы что-то с esd связано..Хотя может ошибаюсь…


ПС Решается созданием файла .asoundrc в /home/user с вот таким содержимым
 pcm.softvolPhonon {           
    type softvol              
    slave.pcm "default:CARD=0"
    control {                 
        name "Phonon"         
        card 0                
    }                         
    min_dB -51.0              
    max_dB 0.0                
    resolution 100            
    hint {                    
        show on               
        description "My Soundcard with extra Volume Control"
    }                                                       
}
