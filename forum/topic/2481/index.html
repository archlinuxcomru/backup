Здравствуйте, есть несколько проблем с мультимедиа клавишами.

1. Как заставить включать/выключать тачпад с помощью клавиши XF86TouchpadToggle (keycode 191). В КДЕ при попытке выставить эту клавишу, мне написало, что она не поддерживается Qt. Временно решил это с помощью xbindkeys. Выключаю нажатием XF86TouchpadToggle, командой synclient TouchpadOff=1. Включаю нажатием Shift+XF86TouchpadToggle, командой synclient TouchpadOff=0. Но я не знаю как можно повесить одну команду одновременно и на включение и на выключение.

2. Абсолютно не работают клавиши регулировки яркости, showkey не показывает никакого кода для них. При этом в биосе и в убунте, что была предустановлена, все работало.

Можно вот такой скриптик заюзать
#!/bin/bash
touchpad_status=`synclient | grep TouchpadOff | awk '{print $3}'`
if [ $touchpad_status == "0" ];
        then
                synclient TouchpadOff=1
                echo "TouchpadDisabled"
        else
                synclient TouchpadOff=0
                echo "TouchpadEnabled"
fi
