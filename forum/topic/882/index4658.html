 да git для таких простых вещей легче не придумаешь…
в папке в которой нужно создать репозиторий выполняем
git init
Для добавления всех файлов в этой папке
git add -A
Применить
git commit -m ‘тут коментарий’

Когда что то изменилось в папке, самое простое это опять выполнить
git add -A
git commit -m ‘тут коментарий’

Так же самое простое посмотреть историю изменений в графическом режиме выполнив
gitk


Например в dolphin есть дополнение для работы с гитом. там сразу помечаются изменившиеся файлы и папки в репозитории, и в менюшке можно выбрать нужное действие - добавить, закомитить и т.п.

А еще прога diffuse умеет работать с git, в очень удобном графическом режиме показывает, вплоть до буквы, что именно изменилось в файле.

Сложности никакой!!! Выше описал простейшие основы для линейного ведения истории изменений…
