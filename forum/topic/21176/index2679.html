Обычно это происходит при первом запуске системы, через пару секунд (5-15 секунд), или при запуске приложения через primusrun (сразу я запускаю Chrome через эту команду)
Я юзаю GT 525M с процессором Intel Core i7 2670QM (Intel HD Graphics 3000) через bumblebee. Система фактически работает через встройку, я просто стартую некоторые приложения через primusrun/optirun.
Интересно, что если система работает уже 30-60 секунд без падения, то при запуске любого приложения, падения не происходит.
