Все заработало, разобрался:
Нужно было исправить конфиг wvdial.conf на такой:
[Dialer defaults]
Modem = /dev/rfcomm0
Baud = 115200
Init2 = AT+CGDCONT=1,"IP","internet.tele2.ru"
Phone = *99#
Username = *
Password = *
Stupid Mode = 1
New PPPD = yes
Auto DNS = 1
Поставить в глубине настроек телефона галочку "раздавать интернет по bluetooth" и включить мобильный интернет на самом телефоне.

Остался, правда, еще один вопрос: соединение устанавливается, но через какое-то время рвется и пытается соединиться снова. При этом каждый раз на телефон приходит запрос о разрешении компьютеру использовать мобильный интернет. Подскажите, может быть есть какие-то опции в wvdial.conf для обеспечения устойчивости связи? Или попытаться в телефоне настроить отключение таких запросов?
