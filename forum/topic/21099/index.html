FileSizesList

Скачать с yandex


Назначение
Список размеров файлов в виде дерева. Упрощает поиск больших файлов и папок
Использование
1. В архиве есть пример для добавления в контекстное меню файлового менеджера Nemo.
2. При запуске без параметров открывается выбор каталога.
3. Выбрать любой файл/папку и вызвать контекстное меню с 4 пунктами (открыть, удалить, расположение, копировать адрес).
4. В ini-файл можно указать отображение чекбоксов для группового удаления папок
спасибо Петру за отличный пример генерации структуры файлов.
