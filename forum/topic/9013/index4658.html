In Kernel 2.6.30 the modules fail to compile out of the box again. I followed these instructions over here

Download this script and this patch:

# cd /tmp
# wget http://communities.vmware.com/servlet/J … .30-fix.sh
# wget http://communities.vmware.com/servlet/J … -fix.patch
# chmod +x vmware-6.5.2-modules-2.6.30-fix.sh
# ./vmware-6.5.2-modules-2.6.30-fix.sh
