YaWP и другие, расположение виджетов на панель?
YaWP - это погода, да, конечно.
Долго не гадал, но, возможно, думаю, что...
Это Yasp-Scripted. На панели их три, каждый с соответствующим конфигом.
Например, CPU:
default title:color="#a0a0a0" title:font="PT Sans Caption,8,-1,5,75,0,0,0,0,0" title:shadow="Plain" title:alignment="Center"
default value:color="#c8c8c8" value:font="Terminus, 9" value:alignment="Left"
default interval="2000"
sensor name="Cpu0Load" type="engine" cmd="systemmonitor:cpu/cpu0/TotalLoad:value" math="int $0"
sensor name="Cpu1Load" type="engine" cmd="systemmonitor:cpu/cpu1/TotalLoad:value" math="int $0"
value key="" use="Cpu0Load" format="$1%" 
value key="" use="Cpu1Load" format="$1%"
