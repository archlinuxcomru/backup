Смержили скрипты? По логике вещей получается нечто подобное:
9       Видео для Android
        set %t
        if [ "%d" = "%D" ]; then
        echo "Входная и выходная папки должны быть разными"
        exit 1
        fi
        while [ -n "$1" ]; do
        ext=`basename "$1"`
        ext="${ext##*.}"
        newname=`basename "$1" .$ext`.mp4
	ffmpeg -i %d/"$1" -vn -y %D/in.wav
	sox %D/in.wav %D/fast.wav tempo 1.6
	ffmpeg -r 40 -i %d/"$1" -i %D/fast.wav -map 0:0 -map 1:0 -f mp4 -vcodec h264 -s 852x480 -r 25 -vb 500k -strict -2 -acodec aac -ab 128k -y %D/"$newname"
        rm %D/in.wav %D/fast.wav
        shift
	done
Ничего не скармливаю МС, поэтому не проверял.
