Возьми исходники клюттера с гномовского фтп. Выбери себе какую-нибудь версию здесь.
./configure --prefix=/usr
make
make install
Внимательно читай вывод каждой команды. Если чего не хватает, пробуй искать сначала в репах, потом в ауре.
