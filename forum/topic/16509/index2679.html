Рекомендую использовать загрузчик grub вместо syslinux
sudo pacman -R syslinux
sudo pacman -S grub
если есть другие операционные системы, кроме арча, тогда ещё понадобится пакет os-prober
установка и конфигурация GRUB:
sudo grub-install /dev/sdX
sudo grub-mkconfig -o /boot/grub/grub.cfg
sudo reboot
вместо sdX устройство, на котором установлена система
