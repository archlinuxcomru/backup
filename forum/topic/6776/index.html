У компании Acronis есть пакет Snap Deploy, который включает в себя средства для снятия образов дисков, сервер dhcp, сервер tftp плюс служба (и гуй к ней) для массовой равертки образов по сети. Сутьв том, что на клиентах запускается специальный агент (через pxe или cd), который находит в сети сервер acronis и сигнализирует о готовности к получению образа. На сервере выбираются нужные клиенты, определяются их будущие плоскиеимена (допускаются шаблоны), выбирается образ. Далее начинается процесс развертки образа на клиенты. Особенность - данные идут мультикастом, т.е. есть управляющий канал по tcp для каждого клиента, а сами данные передаются мультикастом по udp, что ускоряет весь процесс. Существует ли подобное по в linux?
P.S. с DHCP и TFTP все ясно. Интересует именно массовая установка.
