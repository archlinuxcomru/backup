Пожалуйста не отправляйте в поиск, соответствующие темы искал и читал, но есть вопросы на которые ответа видимо не нашёл и интересно услышать мнение людей на ответы использующих разные оконные менеджеры.
Среда рабочего стола у меня xfce4 - считаю наиболее удобной для меня и быстрой в работе для моего железа.

По большей части в compiz мне понравилось:
- переключение между окнами используя для активации угол рабочего стола + сам эффект представления окон при переключении когда они показаны, это достаточно удобно
- красивое сворачивание и разворачивание окон, использовал прозрачность с изменением масштаба

Собственно всё что требуется, может быть сообщество подскажет как этого добиться в xfce без compiz

В автозапуск (требуется пакет kwin):
if [ "$XDG_SESSION_DESKTOP" = "XFCE" ]; then
	if [[ -n $(pidof -s xfwm4) ]]; then # XFCE + kwin
		if [[ -n $(which kwin_x11 2>/dev/null) ]]; then
			(kwin_x11 --replace)&	# менеджер окон
		fi
	fi
fi
Кроме того, может понадобиться инициализация среды plasma, например, для захвата окна курсором в "Специальных параметрах окна"
# Plasma startup. We need to create config folder so we can write startupconfigkeys
[[ -n $XDG_CONFIG_HOME ]] && configDir=$XDG_CONFIG_HOME || configDir=${HOME}/.config # this is the default
mkdir -p $configDir
[ -r $configDir/startupconfig ] && . $configDir/startupconfig
# XCursor mouse theme needs to be applied here to work even for kded or ksmserver
if test -n "$kcminputrc_mouse_cursortheme" -o -n "$kcminputrc_mouse_cursorsize"; then
    kapplymousetheme "$kcminputrc_mouse_cursortheme" "$kcminputrc_mouse_cursorsize"
    if test $? -eq 10; then
        XCURSOR_THEME=breeze_cursors
        export XCURSOR_THEME
    elif test -n "$kcminputrc_mouse_cursortheme"; then
        XCURSOR_THEME="$kcminputrc_mouse_cursortheme"
        export XCURSOR_THEME
    fi
    if test -n "$kcminputrc_mouse_cursorsize"; then
        XCURSOR_SIZE="$kcminputrc_mouse_cursorsize"
        export XCURSOR_SIZE
    fi
fi

Включение при старте прописал в файле:
/etc/xdg/xfce4/xfconf/xfce-perchannel-xml/xfce4-session.xml

  <property name="Client0_Command" type="array">
   <value type="string" value="kwin_x11"/>
   </property>
