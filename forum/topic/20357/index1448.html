А можно глянуть поддержку определенного процессора( Intel Atom z2760 ) в ядре?

из форума 2014года
Atom Z2760 использует печально известный графический процессор PowerVR. Единственная поддержка, которая существует, - это драйвер настройки режима ядра gma500_gfx. Нет X и нет драйвера mesa для обеспечения 2D-ускорения, 3D-ускорения или ускоренного воспроизведения видео.
Потом драйвер завезли и вроде даже работало без gpu ускорения, а в 4.15 опять выкинули
вики
PowerVR-based graphics (GMA 3600 series) are not supported by open source drivers.
–
The Linux kernel has support since version 3.5, but since version 4.15 the relevant kernel driver, uvesafb, has not been included in the Archlinux kernel so using the DKMS version of the driver is necessary. See uvesafb for more information.
Наверно vall самое разумное и предложил вначале.

Можно попробовать другой линукс 32 с ядром до 4.15 или собирать https://aur.archlinux.org/packages/uvesafb-dkms/ и добавлять в исошку (или вообще сразу систему на флешку минимальную поставить и потом с нее грузиться)
