Pidgin IMs Gnome Shell integration

muffinmad
# 8 лет, 6 месяцев назад
Темы: 1
Сообщения: 45
Участник с: 17 июля 2012
Сначала было взялся править существующие решения для поддержки gnome-shell 3.10, но потом сделал свой велосипед.

Расширение показывает сообщения pidgin в gnome shell message tray.

Что умеет:
  • Пока только сообщения. Ни чатов, ни передач файлов, ни atention.
  • Аватары пользователя + иконка статуса
  • Отправляет состояние "пользователь печатает" абоненту
  • Показывает количество новых сообщений
  • Search provider, чтоб в Overview отображались результаты поиска контакта по алиасу

Проблемы:
  • Сообщения остаются непрочитанными в Pidgin. Не нашел dbus-метода для отметки сообщений прочтенными
  • Даже если в настройках уведомлений указано "показывать при заблокированом экране", уведомлений не будет. Это потому что gnome-shell отключает все расширения при блокировке экрана. Расширение после разблокировки пытается сообщать о сообщениях, которые пришли пока экран был заблокирован.

Тестировал на gnome shell 3.10, pidgin 2.10.9, jabber

Тут: https://aur.archlinux.org/packages/pidgin-im-gnome-shell-extension/
И тут: https://extensions.gnome.org/extension/782/pidgin-im-integration/
