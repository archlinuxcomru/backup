Надо же, сам этого не читал...
Это работает с оговоркой:
By default, the binding will only run when you click on the titlebar of the window. If the --whole-window flag is given, it will run when any part of the window is clicked. If the --release flag is given, it will run when the mouse button is released.
Т.е. по умолчанию бинд работает если кликнуть по заголовку клиента. Иначе добавляете ключ --whole-window. Напр.:
bindsym --whole-window button3 и т.д.
Но оно вам надо? В первом случае - это работа только в стековом режиме и режиме вкладок, во втором - клик захватывается до софтины, в окне которой вы елозите мышью.
И то и другое считаю неудобным. У меня, поскольку компом пользуюсь не только я, вызов меню (см. мой линк выше) вызывается по $mod+p.
