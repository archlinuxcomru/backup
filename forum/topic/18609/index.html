Нужно установить пакет thunar-archive-plugin. Он содержит плагины для архиваторов: ark.tap, engrampa.tap, file-roller.tap, gnome-file-roller.tap. И поменяйте nemo-fileroller на обычный file-roller. Не знаю чем там конкретно отличается nemo-fileroller от обычного file-roller-а, но чтобы тунар увидел nemo-fileroller придется под него колхозить tap файл.

gsettings set org.gnome.desktop.interface font-name 'SF Pro Display 10'
gsettings set org.gnome.desktop.interface monospace-font-name 'Ubuntu Mono 12'
gsettings set org.gnome.desktop.wm.preferences titlebar-font 'SF Pro Display Medium 10'

Поднял с 96 до 99 - шрифты на нижней панели из мизерных стани нормальными, причём при уменьшении DPI обратно до 96 уже мизерными не стали. Шрифты в Тюнаре добил до нужных в Settings -- Appearance -- Fonts, а на десктопе они меняются вообще независимо ни от чего, так что и их поменял без проблем. Вот так "пасьянс" и "сошёлся".

Теперь все шрифты - в точности как были при Тюнаре 1.6. Благодарю за помощь, проблема решена!
