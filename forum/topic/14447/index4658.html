ну и совсем уж в качестве оффтопа - советую установить ещё package control channel
он добавляет список репозиториев и каналов для установки/обновления плагинов
1) устанавливаешь package_control
2) Preferences -> Package control -> Add repository -> https://github.com/wbond/package_control_channel
3) Preferences -> Package control -> Install package -> package_control_channel
4) профит - у тебя богатейший выбор плагинов для установки

я не программист и не вебдизайнер, чтобы в полной мере оценить преимущества сабли, но из всех испробованных мною редакторов этот мне понравился более всего
