modprobe vboxdrv

modprobe vboxnetflt
Потом интерфейс:
192.168.56.1/24
dhcp off
Потом:
ip link set dev vboxnet0 up
ip addr add 192.168.56.1/24 dev vboxnet0
ip route add 192.168.56.0/24 dev vboxnet0
И в гостевой системе тоже локалку с такими параметрами надо будет запустить.
