имеется плата OrangePI с установленным серверным ARM линуксом (ARMBIAN 5.35 user-built Ubuntu 16.04.3)
есть программа которую запускаю вручную через ./ArchiSteamFarm.
Захотел автоматизировать чтобы при запуске платы программа сама запускалась, в случае каких либо проблем перезапускалась, и можно было получить доступ к консоли. Через crontab достичь этого не удалось.
Опыта работы с linux крайне мал, по большей части ищу нужную информацию через интернет. Вроде как написал нужный сервис

[Unit]
Description=ArchiSTEAMfarm

[Service]
ExecStart=/ASF/ArchiSteamFarm
WorkingDirectory=/ASF/
Restart=always

[Install]
WantedBy=multi-user.target
положил его в /etc/systemd/system/
Пробовал включить

root@orangepipcplus:~# systemctl enable ASF
root@orangepipcplus:~# systemctl -l status ASF
