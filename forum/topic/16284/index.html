Ух ты, большое спасибо indeviral

только не понятно: у меня микросхема ds3231sn, но чтобы все работало приходится ставить ds1307

в итоге RPI2 B+ вместе с X205:

1.
sudo pacman -S i2c-tools
sudo pacman -S lm_sensors

2.
/boot/config.txt
dtparam=i2c_arm=on

3.
/etc/modules-load.d/raspberrypi.conf
i2c-dev
i2c-bcm2708

4. reboot

5. i2cdetect -y 1 - проверяем подключение

6.
sudo modprobe i2c-bcm2708
sudo modprobe i2c-dev
sudo modprobe rtc-ds1307

7.
su
echo ds1307 0x68 > /sys/class/i2c-adapter/i2c-1/new_device
exit

После этого работает hwclock

P.S.: Не знаю ставить ли вопрос решенным из-за echo ds1307 0x68 в п.7, так как у меня все же микросхема ds3231sn
