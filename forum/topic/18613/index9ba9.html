Можно глянуть файлы не принадлежащие пакетам
по идеи покажет установленные файлы через make install
find /etc /opt /usr | sort > all_files.txt
pacman -Qlq | sed 's|/$||' | sort > owned_files.txt
comm -23 all_files.txt owned_files.txt
или
sudo pacman -S lostfiles
sudo lostfiles
