Хочу добавить на панель свою иконку для запуска файла в gedit.
Создал директорию ~/.applications в ней файл update.desktop с содержимым:

[Desktop Entry]
Type=Application
Encoding=UTF-8
Exec=gedit /run/media/zenia/Archive/tekychka/update
Icon= /home/zenia/.icons/nabor/zametki.png

В файл /home/zenia/.config/lxpanel/default/panels/panel добавил:

Button {
id=/home/zenia/.applications/update.desktop
}
Перезапустил панель но созданный файл на панели не появился, хотя если запустить ~/.applications/update.desktop, откроется gedit с указанным файлом.
Подскажите где допускаю ошибку и не могу закрепить иконку на панель.

У вас случайно не LXDE ? Там вроде другой путь до конфига.
Попробуйте запустить в терминале lxpanel может ошибка в конфиге. Можно добавить программу через гуй (киньте update.desktop в ~/.local/share/applications будет в меню "прочие" ) . В крайнем случае запретить запись в конфиг.
Бывает и такое что просто забываешь файл сохранить после редактирования или редактируешь при запущенной программе.
