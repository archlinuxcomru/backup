Доброго времени суток!
Помогите настроить простейшую самбу на Raspberry Pi 3. Делаю:
sudo pacman -S samba
sudo mkdir /home/temp
sudo chmod -777 /home/temp
sudo pdbedit -a -u m
sudo systemctl enable smbd.service nmbd.service
sudo systemctl start smbd.service nmbd.service
Конфиг:
[global]
workgroup = WORKGROUP
netbios name = RP3
server string = Samba
security = user
browseable = yes
map to guest = Bad User
[temp]
comment = temp
path = /home/temp
read only = No
guest ok = Yes
При доступе из Windows XP пишет "Доступ запрещен"
Подобные темы читал, ничего не помогает.
