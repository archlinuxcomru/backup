В общем просто переустановка не помогла.
sudo pacman -Suy
sudo pacman -S librsvg mesa potrace

При переустановке выдовало ошибку
g_module_open() failed for /usr/lib/gdk-pixbuf-2.0/2.10.0/loaders/libpixbufloader-svg.so: /usr/lib/libharfbuzz.so.0: undefined symbol: FT_Get_Var_Blend_Coordinates

через команду pacman -Qo /usr/lib/libharfbuzz.so.0
увидел что файл принадлежит пакету harfbuzz

переустановка не дала результата. Поэтому пришлось откатить до версии harfbuzz 1.3.4-1

нашел в /var/cache/pacman/pkg предыдующие версии этого пакета и через pacman -U
В конце sudo /usr/bin/gdk-pixbuf-query-loaders --update-cache
и перезапуск иксов.

Если это так для исправления
sudo pacman -Suy
sudo pacman -S freetype2 fontconfig harfbuzz
#для 64 бит мультилиб
sudo pacman -S lib32-freetype2 lib32-fontconfig lib32-harfbuzz
#далее
sudo pacman -S librsvg
#для 64 бит мультилиб
sudo pacman -S lib32-librsvg
