Уфф… Не, ну по pacman -Sy он воссоздаст /var/lib/pacman/sync, установленные пакеты можно найти в /var/cache/pacman, по ним можно попробовать восстановить /var/lib/pacman/local копированием описаний установленных (которые есть в /var/cache/pacman) пакетов из /var/lib/pacman/sync в /var/lib/pacman/local. Только мороки немало. Умнее ничего в голову пока не приходит. Переустановка всего с pacman -Sf перепишет все конфиги, что неприемлемо, поэтому только руцями растыкивать все по местам теперь.
