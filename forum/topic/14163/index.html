1. Лучше использовать df -h , будет человеческий вид.
2. Считаем в твоих данных
- Объем раздела — 37352488K /1024/1024 = 35,6Г
- Использовано — 30710552K /1024/1024 = 29,3Г
- Доступно — 4721452K /1024/1024 = 4,5Г
- зарезервировано системой (5% ) - 35,6*0,05= 1,8Г

3. ИТОГО = 29,3 + 4,5 + 1,8 = 35,6
Ошибки не исчезают с опытом - они просто умнеют
