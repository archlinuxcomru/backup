Кали на btrfs у вас стоит
Проверьте что на монжаре стоят btrfs-progs
pacman -Qs  btrfs-progs
если нет поставьте и попробуйте опять ос пробер запустить, покажет ли кали.

Если стоят, попробуйте добавить insmod btrfs грубу, а лучше все те параметры что в кали стоят
menuentry "Kali linux" {
 insmod  btrfs
 set root=UUID=67531b9c-d8e0-48f0-8b32-ec67ba7c0d2a
 linux /boot/vmlinuz-5.5.0-kali2-amd64
 initrd /boot/initrd.img-5.5.0-kali2-amd64
}
menuentry "Kali linux boot config" {
insmod btrfs
set root=UUID=67531b9c-d8e0-48f0-8b32-ec67ba7c0d2a
configfile /boot/grub/grub.cfg
}
### END /etc/grub.d/10_linux ###

На os prober с btrfs есть багрепорт https://bugs.archlinux.org/task/64281

Но груб в любом случае должен конфиг свой использовать и пункты в меню быть.
А тут что то не то

Можно проверить просто изменив название пункта который показывается
например
menuentry 'Manjaro Linux'.....
на
menuentry 'Manjaro1'.....
если название не изменилось значит явно грузится груб с другим конфигом.
если изменилось то может ошибка синтаксиса в конфиге
