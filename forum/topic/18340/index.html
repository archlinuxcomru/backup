вместо букв какие-то квадраты
Похоже на шрифт без кириллицы, если конечно не забыли запустить
sudo locale-gen
дайте вывод
locale -a
должно быть
C
POSIX
en_US.utf8
ru_RU.utf8

Попробуйте поменять в настройках шрифт на dejavu
sudo pacman -S ttf-dejavu
затем в настройках xfce шрифт задать dejavu и перезапустить приложение или перелогиниться

1 Средствами xfce


2 С помощью setxkbmap в автозагрузке
setxkbmap -layout us,ru -variant ,winkeys -option  grp:lwin_toggle grp_led:scroll

3 Записать в xorg.conf
localectl --no-convert set-x11-keymap us,ru pc105  "grp:lwin_toggle, grp_led:scroll"

Смотрите вики русский

Для переключения раскладки еще есть хорошая программка созданная нашим форумчанином
