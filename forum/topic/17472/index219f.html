Сегодняшнее большое обновление питона (предполагаю источник в этом), которое pikaur удачно провёл, после перезагрузки привело к:
$ pikaur -Syu
Traceback (most recent call last):
  File "/usr/bin/pikaur", line 5, in <module>
    from pikaur.main import main
ModuleNotFoundError: No module named 'pikaur'
Обычно автор оперативно обновляется, на опережение. Кстати, впервые за всё время использования в моей практике возникла проблема с этим пакетом. Пока временно на yay.

P.S. И питон не хотел обновляться пока не удалил pkgbrowser. Это приложение традиционно уже создаёт ненужные проблемы перед апдейтом питона.
