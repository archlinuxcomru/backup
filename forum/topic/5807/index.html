Пример
pacman -Syu
S - (установка/обновление) Устанавливает или обновляет пакеты. Пакеты устанавливаются непосредственно с ftp серверов, включая все зависимости, необходимые для запуска.
y - Загрузить свежую копию основного списка пакетов с серверов, определенных в pacman.conf.
u - Обновит все устаревшие пакеты.
Т.е. даем пакману команду на установку/обновление пакетов, для этого сначала загружаем список пакетов, а потом обновляем все устаревшие пакеты.

Есть “операции” и есть “опции”. Первый список опций относится к -S (установка/обновление), но с опцией это уже поиск: pacman -Ss pkgname. Одни и те-же, если можно так выразиться, “маленькие буковки”, в сочетании с разными операциями (S, R, …) будут менять свой “смысл”, применительно только к этой операции.
pacman -Ss pkgname - запрос к удалённой базе данных (поиск),
pacman -Rs pkgname - рекурсивное удаление пакета со всеми зависимостями (при соблюдении определённых условий),
pacman -Qs pkgname - запрос и поиск (+вывод определённой информации) среди установленных.
Нужно проникнуться=)
