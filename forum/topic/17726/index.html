Можно по крестьянски:
1. Загружаешь Live Usb
2. В Gparted создаешь нужные разделы (Root, Home)
3. Монтируешь в /mnt новые и сохраненные разделы
4. Запускаешь mc и копируешь.
5. Затем входишь в новый Root (например /mnt/arch)
$ sudo arch-chroot /mnt/arch /bin/bash
- Правишь fstab
- Пересобираешь ядро:
# mkinitcpio -p linux
# exit
Ну, а по загрузке - у всех по-разному.
