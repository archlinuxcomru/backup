Volldemar
Could not locate dkms.conf file
Указывает на то, что в /var/lib/dkms есть dkms модули не имеющие файла dkms.conf в поддиректориях source - возможно не удаленные или не правильно установленные
Для пояснения
- смотрим сколько в /var/lib/dkms поддиректорий source
tree /var/lib/dkms | grep source
    │   └── source -> /usr/src/sysdig-0.27.1
… одна
- смотрим наличие файлов dkms.conf
find -L /var/lib/dkms -name 'dkms.conf'
/var/lib/dkms/sysdig/0.27.1/source/dkms.conf
… один и версии совпадают - ничего лишнего нет.
Один спец предложил очень хорошую команду для анализа таких ошибок - поиска проблемных модулей, которые не имеют файла dkms.conf
for i in /var/lib/dkms/*/[^k]*/source; do [ -e "$i" ] || echo "$i";done
… у меня этот вывод пустой - то есть ничего лишнего нет.
Если вывод не пустой, то нужно разбираться и, скорее всего, удалять не нужное ...
Ошибки не исчезают с опытом - они просто умнеют
