У меня проблемы с Vulkan'ом, GT 525M карточка.
В моём случае настройка сводилась к этим пунктам:
1. yay -S nvidia-390xx-dkms nvidia-390xx-utils (Установка dkms версии драйвера для ядер отличных от linux, например linux-hardened или linux-zen)
Вместо yay можно использовать любой AUR-helper, я предпочитаю paru.
2. sudo pacman -S bumblebee
3. sudo mkinitcpio -p linux (Пересборка initramfs)
4. sudo systemctl enable bumblebeed.service (Активация демона bumblebee)
5. sudo usermod -a -G bumblebee {Имя пользователя} (Добавление в группу)

Без этого, у меня ничего не работало:
Раскомментировать BusID в файле /etc/bumblebee/xorg.conf.nvidia
Возможно у тебя заведётся и без этого, но если нет, попробуй это в первую очередь.

Программы стартуют через команду optirun (можно юзать primusrun, придётся установить пакет primus), по умолчанию используется встройка.
Если хочешь, чтобы дискретка работала всегда, установи bbswitch-dkms (но во избежании проблем, категорически не рекомендую его ставить)
