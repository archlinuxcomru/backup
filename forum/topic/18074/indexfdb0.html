Скрипт:
screenlock-f

#!/bin/sh
if ! [ -f "$HOME/.config/bspwm/lockscreen.png" ]; then
	cp /usr/share/backgrounds/lockscreen.png $HOME/.config/bspwm/lockscreen.png
fi

B='#00000022'  # blank
C='#ffffff22'  # clear ish
D='#7D7D7Dcc'  # default
T='#1B1B1Cee'  # text
W='#880000bb'  # wrong
V='#202020ee'  # verifying

i3lock \
--keyhlcolor=1B1B1C22 \
--insidevercolor=$C \
--ringvercolor=$V \
\
--insidewrongcolor=$C \
--ringwrongcolor=$W \
\
--insidecolor=$B \
--ringcolor=$D \
--linecolor=$B \
--separatorcolor=$D \
\
--textcolor=$T \
--timecolor=$T \
--datecolor=$T \
--layoutcolor=$T \
--keyhlcolor=$W \
--bshlcolor=$W \
\
--screen 0 \
--blur 5 \
--clock \
--indicator \
--indpos="800:465" \
--radius=60 \
--layoutsize=12 \
--layoutfont="Digital Play St" \
--timestr="%H:%M:%S" \
--timefont="a_LCDNova" \
--timesize=18 \
--timepos="ix-200:iy-115" \
--datestr="%a, %d %b. %Y" \
--datefont="a_LCDNova" \
--datepos="ix-200:iy-90" \
--datesize=12 \
--keylayout 2 \
--textsize=10 \
-i $HOME/.config/bspwm/lockscreen.png

# --veriftext="Drinking verification can..."
# --wrongtext="Nope!"
# --textsize=20
# --modsize=10
# --timefont=comic-sans
# --datefont=monofur
# etc
Обратите внимание на строку 3 - где у вас будет картинка,
значениями
--indpos="800:465" \
...
--timepos="ix-200:iy-115" \
...
--datepos="ix-200:iy-90" \
установите круг и надписи чтобы они попали на картинке в нужное место
установите пакет i3lock-color из AUR
шрифты
Digital Play St
a_LCDNova

Команда в автозагрузку (если надо):
xautolock -time 10 -locker screenlock-f &
Заблокировать хоткеем - просто
screenlock-f
