Только что проделал это.

1. Загрузился с установочного диска
2. Вставил модем
3. Настроил ppp как указано в Вики
4. Прописал в /etc/resolv.conf установочного CD сервер DNS гугла 8.8.8.8

nameserver 8.8.8.8

иначе не будут резолвится имена зеркал и прочее-прочее

5. Запустил systemctl start [email protected]
6. Пошел пинг! А значит пойдет установка

Примечание: Модем предварительно переведен в режим "только модем". Собственно плод недавних трудов...

P.S.: Автору спасибо за поднятый вопрос - сам потренировался так поднимать сетку - может пригодится при крупномасштабной аварии системы )
