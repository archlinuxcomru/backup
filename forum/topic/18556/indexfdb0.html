Стоп увидел у вас два chipsec
первый наверно с blackarch стоит. Но теперь и он нормально установился

==> dkms install chipsec/1.3.6rc1 -k 4.16.13-2-ARCH
==> dkms install sysdig/0.21.0 -k 4.16.13-2-ARCH
==> dkms install chipsec-dkms/git

Гляньте откуда он у вас стоит
yaourt  chipsec
И кому нужен
pacman -Qi chipsec
вместо chipsec полное название пакета

По идее его теперь можно удалить, оставив chipsec-dkms/git он быстрее обновляется чем из blackarch
