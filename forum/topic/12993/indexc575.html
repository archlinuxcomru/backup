Может кто подсказать, возможно ли этот livecd (со всеми параметрами и настройками) перенести на desktop?
Грузитесь, открываете терминал:
su
mkfs.ext4 /dev/sda4
mount /dev/sda4 /mnt
cp -a -t /mnt /{boot,etc,home,opt,root,srv,usr,var}
mkdir -m777 /mnt/tmp
mkdir -m755 /mnt/{dev,mnt,proc,run,sys}
cd /mnt
ln -s usr/bin bin
ln -s usr/bin sbin
ln -s usr/lib lib
cp -a /run/archiso/bootmnt/mate/boot/i686/vmlinuz /mnt/boot/vmlinuz-linux
mount --bind /dev /mnt/dev
mount --bind /sys /mnt/sys
mount --bind /proc /mnt/proc
genfstab -p /mnt >>/mnt/etc/fstab
chroot /mnt
mkinitcpio -p linux
Устанавливаете загрузчик. Offline уже доступен Syslinux
exit
reboot
Получаете канонический Арч установленный на жесткий диск без доступа к инету. Естественно вместо /dev/sda4 используете свой коорневой раздел.
Может чего и упустил, благо поправить есть кому :)
