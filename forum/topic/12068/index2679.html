Подскажите пожалуйста,
как, при помощи pacman осуществить поиск среди установленных пакетов по дате установки?
Пакман вроде такого не предусматривает, однако каждый установленный пакет сопровождается каталогом с его метаданными. Можно вывести список этих каталогов с сортировкой по времени (это будет время модификации каталога): 
ls -ltr /var/lib/pacman/local/
Дальше будет уже несложно найти в списке нужную дату.
