KDE4: привязка действий к removable media devices

Добрый день.

В kde3 настроил для определенного типа устройства (например, для фотокамеры) определенное действие, которое вызывалось в момент подключения камеры к ПК. Для флешки сделал другое действие, которое было доступно только для смонтированной флешки (запуск mc в каталоге флешки).

Вопросы (применительно к kde4.2.1 archlinux):
- как реализовать (хотя бы где копать?) настройку действий для removable media devices;
- как размонтировать флешку (читай - безопасно удалить)?

P.S. на текущий момент флешка нормально автомонтируется и открывается в Dolphin-e.
