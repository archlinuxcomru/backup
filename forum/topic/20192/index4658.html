Для тех кто видит надпись мельком и желает ее увидеть во все красе, может усыпить grub и по любоваться надписью - для чего добавить в grub.cfg
set default=0
sleep --interruptible --verbose 20
где 20 - количество секунд сна .... слева будет идти отсчет оставшихся секунд ... прервать сон - esc
Ошибки не исчезают с опытом - они просто умнеют
