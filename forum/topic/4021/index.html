Я решил эту проблему таким образом. Скрипт для Mute Unmute привязал к клавише XF86AudioMute:

#!/bin/bash
unmute=$(amixer get Master | grep 'Mono: Playback' | awk -F '[' '{print $4}' | tr -d ']')
if [ $unmute = "on" ]
	then
		amixer set Master mute > /dev/null
	else
		amixer set Master unmute > /dev/null
fi
А для повышения и понижения звука привязал к клавишам команды:

Клавиша XF86AudioRaiseVolume команда amixer set Master 1%+

Клавиша XF86AudioLowerVolume команда amixer set Master 1%-

Я решил эту проблему таким образом. Скрипт для Mute Unmute привязал к клавише XF86AudioMute:

#!/bin/bash
unmute=$(amixer get Master | grep 'Mono: Playback' | awk -F '[' '{print $4}' | tr -d ']')
if [ $unmute = "on" ]
	then
		amixer set Master mute > /dev/null
	else
		amixer set Master unmute > /dev/null
fi

Как всё сложно. А ведь в вики есть простое решение:
 <keybind key="XF86AudioMute">
     <action name="Execute">
       <command>amixer set Master toggle</command>
     </action>
   </keybind>
The “Mute” key should unmute the Master control if it is already in mute mode.
