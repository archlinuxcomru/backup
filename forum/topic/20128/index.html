В systemd'шном
Попробуйте отключить, а то он говорит что /usr/bin/systemd-swap запущен и по тайм ауту пытается его снова запускать
systemctl disable systemd-swap.service
перезагрузиться и вручную запустить проверить
sudo /usr/bin/systemd-swap

--
правильней
sudo /usr/bin/systemd-swap stop
sudo /usr/bin/systemd-swap start

сначала что бы проверить что он еще где не запускается просто
sudo /usr/bin/systemd-swap start
