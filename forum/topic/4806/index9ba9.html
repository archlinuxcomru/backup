Где/как можно задать приложения для данного протокола?

1. about:config
2. создать ключ(логическое) - network.protocol-handler.expose.sop с значением false
3. открываешь страницу где есть ссылка на sop:// .. жмёшь ссылку, появляется запрос, чем открывать .. указываешь нужную софтину

то-же самое для любого протокола, например magnet

далее изменить софтину можно уже в Меню -> Настройки -> вкладка
