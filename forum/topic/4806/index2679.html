Для ФФ4 из АУР прокатывали эти два способа:

Способ 1

Откройте страницу со ссылками на русификации. Щёлкните кнопкой мыши по ссылке на русификацию для её установки. При появлении панели запроса на установку программного обеспечения с сайта http://www.mozilla-russia.org нажмите на кнопку «Allow».
Установите расширение Quick Locale Switcher.
Перезапустите Firefox.
Зайдите в настройки расширения Quick Locale Switcher с помощью пункта меню «Tools → Quick Locale Switcher → Options…» и установите флажок «User Interface Language (general.useragent.locale preference)».
Переключите язык на «ru-RU — Russian (Russia)» с помощью пункта меню «Tools → Quick Locale Switcher».
Перезапустите Firefox.

Способ 2

Откройте страницу настроек about:config.
Найдите параметр general.useragent.locale.
Смените его значение с en-US на ru.
Перезапустите Firefox.
