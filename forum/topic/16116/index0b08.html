Выхлопы консоли помещайте либо в текстовом виде (длинные - с тегом code), либо картинками с превьюшками. Предпочтительнее - первый вариант.
Сегодня ваши скрины по разным тема подчищу, с завтрашнего дня, если будете продолжать в том же духе - отправитесь на заслуженный отдых.

Добавь в /etc/pacman.conf строчки ниже, затем добавь ключ в системе 653C3094, и обнови репы
[catalyst]
Server = http://catalyst.wirephire.com/repo/catalyst/$arch
## Mirrors, if the primary server does not work or is too slow:
#Server = http://mirror.hactar.bz/Vi0L0/catalyst/$arch
Затем поставь дрова fglrx, а свободные удали
Необходимые пакеты
catalyst-hook
catalyst-utils
catalyst-libgl
lib32-catalyst-utils
lib32-catalyst-libgl
после установки выполни # aticonfig --initial скорее всего придется еще лезть в /etc/X11/xorg.conf
