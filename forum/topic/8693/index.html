Кстати, gxneur очень понравилось, правда он работает, только если на панели есть стандартный переключатель раскладок, сам не пойму почему :\
в /etc/X11/xorg.conf добавь если нету
Section "InputDevice"
	Identifier  "Keyboard0"
	Driver      "kbd"
	Option "XkbLayout"  "us,ru"
	Option "XkbVariant" ",winkeys"
	Option "XkbOptions" "grp:alt_shift_toggle,grp_led:scroll"
EndSection
раскладка будет меняться по alt+shift
