В Thunar открыть Правка -> Особые действия -> Добавить особое действие -> Имя : Создать файл -> далее в строке Команда: mousepad (или ваш любимый редактор). В "Условия появления" галочка на Каталоги. В контексном меню при выборе "создать файл" откроется сразу mousepad для записи. После внесения записей, не забыть - "Сохранить как"
