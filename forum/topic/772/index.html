Спасибо, нашёл. Вот рецепт, вдруг кому-то ещё пригодится:

Доступ к git из-за proxy
Для работы с yaourt частенько нужен доступ к GIT.

Один из вариантов решения проблемы, найденный на просторах Инета:

1. Ставим socat.

2. В каталоге ~/bin создаем файл proxy.sh с правами на выполнение и содержимым:
#!/bin/sh
_proxy=XXX.XXX.XXX.XXX
_proxyport=XXXX
exec socat STDIO PROXY:$_proxy:$1:$2,proxyport=$_proxyport
3. И перед запуском yaourt делаем
export GIT_PROXY_COMMAND=~/bin/proxy.sh
