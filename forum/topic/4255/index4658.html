Для slim (согласно вики):
В случае использования легковесного оконного менеджера совместно с SLiM, для избежания проблем с монтированием, необходимо для login_cmd после exec добавить ck-launch-session dbush-launch
Т.е. в файле /etc/slim.conf привести строку login_cmd exec бла-бла-бла к виду:
login_cmd exec ck-launch-session /bin/bash -login ~/.xinitrc %session
или, если не поможет: 
login_cmd exec ck-launch-session dbush-launch /bin/bash -login ~/.xinitrc %session
