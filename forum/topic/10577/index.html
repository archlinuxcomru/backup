такой вопрос, вернее два:
1.Нужно ли что-то доустанавливать чтобы были видны USB, и нужно ли править fstab в гостевой для автомонтирования USB.
2.У меня в гостевой Ubuntu не автомонтируется общая папка, правил fstab как в инструкции, в качестве общей папки была взята системная “Общедоступные” из хомяка, в гостевой она монтируется туда-же, через терминал проблем нет, все нормально подключается, а автоматом не получается…
Кто скажет в чем причина???
Для Ubuntu
Открываем терминал и вводим команду:
cat /etc/group | grep plugdev
Вам нужен id, который получился в результате выполнения команды. У меня это 46. Скорее всего, у вас будет тоже самое, но мало ли.
Так же, нужно создать саму директорию, в которую будет монтироваться расшаренная директория (для примера используется директория data). Для этого используйте команду:
sudo mkdir /media/data
Далее открываем на редактирование под суперпользователем файл /etc/fstab:
sudo gedit /etc/fstab
В появившемся файле вам нужно будет добавить в конец одну строчку, состоящую из 6 секций, разделённых пробелом.
Пример:
data /media/data vboxsf defaults,gid=46 0 0
data - имя директории, которое мы задали выше.
/media/data - директория, в которую монтируется наша директория.
vboxsf - файловая система, пишите именно так.
defaults,gid=46 - права на доступ и запись в расшаренную директорию. Число 46 мы получили выше, смотрите тот результат, который был у вас.
0 0 - два нуля через пробел, пишите именно так.
Для вступления изменений в силу, необходимо перезагрузить гостевую систему или выполнить следующую команду в терминале:
sudo mount -a
