Добый день, я представитель команды http://www.trackru.ru, ищем разработчика для разработки модулей отслеживания посылок, и это будет интересуется владельцы интернет магазинов. За вашу работу предлагаем денежное вознаграждение.

Trackru API- интеграционный сервис для рынка электронной коммерции, API для трекинга посылок, отслеживания почтовых отправлений. API документация: https://www.trackru.ru/api-index.html

Интеграция с сервисом Trackru API. который разрабатывает интеграционный сервис для рынка электронной коммерции, является универсальным протоколом для интеграции со всеми службами доставки в рамках "единого окна". Он поддерживает отслеживании 600+ служб доставки, включая внутренние и международные службы доставки, как Почта России, СДЭК, СПСР, Новая Почта, Aliexpress и т. д.

По вопросам по сотрудничестве, пишите на nicholas@trackru.org
