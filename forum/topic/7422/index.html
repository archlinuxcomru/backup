способа два:

1. если используется startx - как вариант прописать в ~/.xinitrc
2. во всех остальных, для кде: создать .desktop файл (в народе ярлык, или что-то там для запуска приложений) - в ~/.kde4/Autostart/
systemd должен умереть.

1. Левый верхний угол приложения - Дополнительно - Специальные параметры окна
2. Закладка “Поведение”: Галка “Пропустить панель задач” - “Применить изначально” - Поставить еще одну галку справа
3. Закладка “Геометрия”: Галка “Свернуто” - “Применить изначально” - Поставить еще одну галку справа
4. Перезапустить приложение

В свойствах самого приложения должно стоять “показывать значок в трее”.
