Подключаемый модуль Windows VHD Boot
Ventoy использует этот плагин для загрузки файлов VHD (x) с Windows 7+.
Поддерживаются как устаревший BIOS, так и UEFI. Поддерживаются как фиксированный, так и динамический VHD (x).
Ventoy использует этот плагин для загрузки файла виртуального диска (vhd / vdi / raw … с дистрибутивом Linux в нем) на реальной машине.
Поддерживаются как устаревший BIOS, так и UEFI.
