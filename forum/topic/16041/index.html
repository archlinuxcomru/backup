Есть репа с собранным archlinuxcn/firefox-gtk2
добавте в pacman.conf
[archlinuxcn]
SigLevel = Optional TrustAll
Server = https://cdn.repo.archlinuxcn.org/$arch

Да и GTK3 версия с обновленной гтк темой уже нормально работает - можете поставить тему aur/gtk-theme-arc-git или aur/vertex-themes-git
