Раньше использовал fontconfig-ubuntu, cairo-ubuntu, но последний год наблюдались явные проблемы с актуальностью обновлений, за которыми мейнтейнер не следил. Приходилось часто править PKGBUILD-ы вручную. Поэтому начал искать другие варианты, и нашел намного проще и намного лучше выглядящий вариант - Infinality. Установка вкратце:

Добавь в /etc/pacman.conf
[infinality-bundle]
SigLevel = Never
Server = http://bohoomil.com/repo/$arch

[infinality-bundle-fonts]
SigLevel = Never
Server = http://bohoomil.com/repo/fonts

[infinality-bundle-multilib]
SigLevel = Never
Server = http://bohoomil.com/repo/multilib/$arch

yaourt -Syua
yaourt -S infinality-bundle ibfonts-meta-extended ttf-ubuntu-font-family-ib

sudo fc-presets set
1) combi
2) free
3) ms
4) reset
5) quit
Enter your choice... 1
Тут можно поэкспериментировать.

В KDE выбери Ubuntu Regular и в качестве моноширинного Ubuntu Mono, размер подгони под свой dpi.
