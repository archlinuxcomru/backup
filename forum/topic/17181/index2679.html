Да, qtwebkit , чтобы не собиралась часа два, лучше скачайте отсюда (там ссылки в комментах ниже) и установите заранее:
$ sudo pacman -U qtwebkit-2.3.4-5-x86_64.pkg.tar.xz
----------------
Если потом будут проблемы c libappindicator, установите из ayatana (добавить эти строки в /etc/pacman.conf) :
[ayatana]
# Unity and related packages
# http://pkgbuild.com/~bgyorgy/ayatana.html
SigLevel = PackageRequired
Server = http://pkgbuild.com/~bgyorgy/$repo/os/$arch

sudo pacman -Syy
sudo pacman -S libappindicator
