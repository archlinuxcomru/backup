Просто выполни:
yaourt -S acestream-engine
sudo systemctl enable acestream-engine.service
sudo systemctl start acestream-engine.service
Затем покажи где спотыкнётся это:
yaourt -S acestream-player-data
А потом уже занимайся установкой 
yaourt -S acestream-mozilla-plugin
Всё!

upd: когда всё получится и установишь не забудь периодически очищать директорию с кешем
rm /home/acestream/.ACEStream/.acestream_cache/*
