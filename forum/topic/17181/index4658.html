Сначала я все нашел то, что установлено в моей системе и имеющее название acestream а так же подсветить надпись installed с помощью команды
yaourt -Ss acestream | grep installed --color=auto
У меня выдало этот список
[zerg@Swarm-PC ~]$ yaourt -Ss acestream | grep installed --color=auto
aur/acestream-engine 3.0.5.1-2 [installed] (85) (2,37)
aur/acestream-proxy-player 0.3.1-1 [installed] (11) (0,66)
[zerg@Swarm-PC ~]$
Я это все благополучно удалил т.к. не особо уверен в правильности последовательности установки и не помню как это все было установлено!
Удалял пакеты со всеми их зависимостями, с помощью команда ниже
sudo pacman -Rsc acestream-engine
sudo pacman -Rsc acestream-proxy-player
Сейчас у меня при вводе команды yaourt -Ss acestream | grep installed --color=auto Ни чего не выдает т.е. Ничего не установлено со словом acestream
Так же удалил из системы всех сирот на всякий случай с помощью команды
sudo pacman -Rsn $(pacman -Qdtq)

Вот, теперь с чего начать установку Acestream на ArchLinux Xfce4 x64bit?
Я правильно понимаю? Сначала устанавливаем
acestream-engine
А после устанавливаем
Aceproxy и репозиторий AceProxy с помощью команды
git clone https://github.com/ValdikSS/aceproxy.git
Или наоборот сначала Aceproxy а после acestream-engine?

вот так все скопом запускается (пути у тебя могут отличаться), запись логов можно упустить.....

STATE_DIR=/home/my/aceproxy/log
ACE_LOG=/home/my/aceproxy/log/ace.log
PROXY_LOG=/home/my/aceproxy/log/ace_proxy.log
acestreamengine --client-gtk --bind-all --login LOGIN --password PASSWORD --log-file $ACE_LOG 2>&1 &
echo $! > $STATE_DIR/ace.pid
python2 /home/my/aceproxy/acehttp.py > $PROXY_LOG 2>&1 &
echo $! > $STATE_DIR/proxy.pid

1 скачал актуальную на данный момент версию acestream
wget "http://dl.acestream.org/linux/acestream_3.1.16_ubuntu_16.04_x86_64.tar.gz"
скачалось в папку загрузки.
2 Разархивировал тарболом
tar zxvf acestream_3.1.16_ubuntu_16.04_x86_64.tar.gz
появилась папка с названием "acestream_3.1.16_ubuntu_16.04_x86_64" в дирректории /home/zerg/
3 переместил папку "acestream_3.1.16_ubuntu_16.04_x86_64" в /opt/acestream
sudo mv acestream_3.1.16_ubuntu_16.04_x86_64 /opt/acestream
Переместилось только содержимое папки "acestream_3.1.16_ubuntu_16.04_x86_64" в /opt/acestream а сама папка под названием "acestream_3.1.16_ubuntu_16.04_x86_64" исчезла!
4 Установил зависимости
sudo pacman -S python-apsw
sudo pacman -S python-setuptools
sudo pacman -S python2-crypto
5 добавить эти строки в /etc/pacman.conf
[ayatana]
# Unity and related packages
# http://pkgbuild.com/~bgyorgy/ayatana.html
