Yashalta, Вы так никогда не установите.
Не гоняйтесь Вы лучше за актуальной версией , а установите из aur
Еще раз: чтобы быстрее собиралось, установите для начала этот пакет
qtwebkit-2.3.4-5-x86_64.pkg.tar.xz
$ sudo pacman -U qtwebkit-2.3.4-5-x86_64.pkg.tar.xz
Далее:
yaourt -S acestream-engine acestream-mozilla-plugin acestream-player
Все остальное подтянется по зависимостям.
После сделайте это:
(добавить эти строки в /etc/pacman.conf) :
[ayatana]
# Unity and related packages
# http://pkgbuild.com/~bgyorgy/ayatana.html
SigLevel = PackageRequired
Server = http://pkgbuild.com/~bgyorgy/$repo/os/$arch

sudo pacman -Syy
sudo pacman -S libappindicator
Эти acestream-proxy acestream-proxy-player, если понадобятся потом установите.
