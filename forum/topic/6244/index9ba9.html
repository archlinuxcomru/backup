в windows7 все подключается нормально как с шифрованием так и без

при шифровании подключаюсь так:
1. 
sudo ifconfig eth1 up
sudo iwconfig eth1 essid mylink
sudo wpa_supplicant -B -Dwext -i eth1 -c /etc/wpa_supplicant.conf
2. или
sudo ifconfig eth1 192.168.1.7
sudo route add default gw 192.168.1.1
или
sudo dhcpcd eth1
