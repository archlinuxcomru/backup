Свежеустановленный Arch, Xorg, Gnome. (pacman -Syu)
По умолчанию гном открывает каждую папку в новом окне. Включив галку “Всегда открывать папки в обозревателе”, папки открываются в одном окне, но закрыв любое из этих окон, происходит непонятное.
Т.е. я закрываю крестиком любое окно наутилуса, и происходит следующее:
- Исчезают иконки с рабочего стола
- Курсор мыши принимает форму часиков
- Название окна, на панели окон, закрываемого окна сменяется с “Компьютер - обозреват…” на “Запускается 'File Manag…”
- Закрываются все окна наутилуса…
Спустя секунду - полторы, значки на раб. столе появляются.
Как изменить такое странное поведение файлового менеджера?

По ссылкам соседней темы, нашел решение, нужно выполнить команды:

gconftool-2 -s -t string /apps/nautilus/preferences/navigation_window_saved_geometry ‘400x200 +20+20’
gconftool-2 -s -t bool /apps/nautilus/preferences/navigation_window_saved_maximized false

Нехватало данных параметров, что и приводило к крэшу наутилуса…
