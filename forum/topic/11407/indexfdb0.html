Нижние:
За основу взято отсюда - http://freeazy.deviantart.com/art/Conky-Launchpad-186251285
Настроив, чтобы заработало и подумав, что в Гноме 3 зона для нотификации всё-таки снизу, поместил это дело вниз, поправив фон.
Архив - http://yadi.sk/d/DwqOhoHe41aLe
Требуется установка шрифтов Ubuntu - community/ttf-ubuntu-font-family ну и там внутри надо на свои железки править немного.
Идеально для разрешения - 1366х768 пикселей 16:9.

И еще добавлю: чтобы корректно работал статус, который следит за наличием обновлений, нужно заставить cron спрашивать об обновлениях:

$ sudo systemctl enable cronie && sudo systemctl start cronie
$ sudo systemctl status cronie
Если всё ок, то:
$ EDITOR=nano crontab -e
______________________
01 * * * * /usr/bin/sudo /usr/bin/pacman -Sy > /dev/null

Затем нужно разрешить проверку зеркал без пароля:
$ sudo EDITOR=nano visudo
_______________________
....
## User privilege specification
##
root ALL=(ALL) ALL
## Uncomment to allow members of group wheel to execute any command
%wheel ALL=(ALL) ALL
<username> ALL=(ALL) NOPASSWD: /usr/bin/pacman -Sy
....

Пока сonky показывает что обновления или есть или их нет, жду когда они появятся, чтобы научить показывать сколько именно пакетов нужно обновить. :)

Верхние - почти без изменений - http://www.deviantart.com/art/Flat-Weather-for-Conky-208130863
