lsusb

Bus 002 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
Bus 007 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub

Bus 003 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub

то есть появляется Bus 004 Device 003: ID 05e3:0741 Genesys Logic, Inc.
дело 100% не в контактах,драйвер в системе получается есть раз через 5 минут начинает работать(и работает прекрасно,быстро и без ошибок)
Если извлечь а потом опять подключить - то же самое,подключается минут через пять.
В общем даже не представляю куда копать.
