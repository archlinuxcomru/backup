[trinity] Server = http://repo.nasutek.com/arch/contrib/trinity/x86_64/
Пакеты этого репозитория работают на Arch не старше 08.2018, т.е. нужен Downgrade системы
пишем зеркало в /etc/pacman.d/mirrorlist и откатим все пакеты: pacman -Syyuu
# Server = http://archlinux.arkena.net/archive/repos/2018/08/01/$repo/os/$arch
Преимущества: на таком Trinity работают все темы оформлений: Plastic QTCurve и прочие…

Второй способ: просто добавить репозиторий Condres, получим плюшки Manjaro и TDE 14.0.5, в котором работают только старые темы.
[condresrepo]
Server = http://build.codelinsoft.it/mirror/

Сделал портированный (все в /opt) модуль из пакетов http://build.codelinsoft.it/mirror/
Портирование должно решить проблемы даугрейда. Недостающие либы кидаем в /opt/trinity/lib и это не влияет на имеющуюся систему

Протестировал в manjaro lxqt обновленном до сегодняшнего среза:
1. Распаковать : unsquashfs 089-kde3*.pfs
2. Скопировать в корень
3. sudo ldconfig
4. sudo killall Xorg
Работает. Тестировал мало
Известные проблемы :
а. Экран блокируется, но потом не разблокировать. В pra6408 такого нет
б. Настройщик wifi сети видит, но не подключает

Желающие просто посмотреть могут это сделать в pra6408 в виртуалке или установив простым копированием модулей
Прикручивать к рабочей системе рекомендовал бы с использованием initrd-AUFS-заморозки
