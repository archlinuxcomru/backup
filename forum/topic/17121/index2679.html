решил вот себе на ноут поставить LXQT. Очень нравится, только тиринг бесит. Почему то тут комптон не помогает.
На плазме5 все нормально, тиринга нету, в LXQT с запущенным комптоном тиринг остается при просмотре видео (при перетаскивании окон вроде тиринг пропадает)

конфиг интела:
cat /etc/X11/xorg.conf.d/30-intel.conf
Section "Device"
    Identifier     "Device0"
    Driver         "intel"
    VendorName     "NVIDIA Corporation"
    BusID          "PCI:0:2:0"
#    Option         "TearFree" "true"
#    Option         "AccelMethod" "sna"
EndSection

Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor0"
    DefaultDepth    24
    SubSection     "Display"
        Depth       24
    EndSubSection
EndSection

строка запуска комптона:
 compton -C -G -b --backend glx --vsync opengl --refresh-rate 60 -i 0.93

в настройках интела ставил разные способы акселерации, в строке запуска комптона также ставил разные типы --vsync, все безрезультатно.

тестовое видео для определения тиринга:
https://www.youtube.com/watch?v=cuXsupMuik4
