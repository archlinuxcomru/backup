В общем проблема решилась, решил написать здесь, вдруг кому-нибудь понадобиться ну или просто таких тугодумов не существует и в итоге буду писать для себя в общем проделал следующее:
1) Посмотрел вообще что есть у меня в системе(можно по другому :D)
командой :
aplay -l
card 0: Intel [HDA Intel], device 0: STAC92xx Analog [STAC92xx Analog]
2)потом пошел по адресу /etc/mobprobe.d/alsa-base.conf и там поменял, думаю собственно это и помогло
options snd-hda-intel index=0
прописал так
pcm.!default {
type hw card Intel
}
ctl.!default {
type hw card Intel
}
pcm.pulse {
type hw card Intel
}
ctl.pulse {
type hw card Intel
}

настройка чувствительности почему-то была в колонке Mux1.

после этого все стало просто прекрасно, но думаю в общем-то эти настройки делаются при настройке системы, ну что же я не знал, всем спасибо. Удачи)
