1. Вернуть содержимое /etc/inittab как было по умолчанию, а дальше повторюсь еще раз
В ~/.xinitrc добавить (раскомментировать) exec startkde
Запуск командой startx

2. Пример: useradd -m -s /bin/bash <username>
Создание пароля для пользователя username - passwd <username>
Насчет администратора - без комментариев
