Так, например
https://wiki.archlinux.org/index.php/X_keyboard_layout
Using X configuration files
/etc/X11/xorg.conf.d/00-keyboard.conf
Section "InputClass"
        Identifier "system-keyboard"
        MatchIsKeyboard "on"
       # ваши опции
EndSection
