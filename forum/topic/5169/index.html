:: Запускается полное обновления системы...
:: Заменить kernel-headers на core/linux-api-headers? [Y/n]
:: Заменить man на core/man-db? [Y/n]
:: Заменить klibc на core/mkinitcpio? [Y/n]
:: Заменить klibc-extras на core/mkinitcpio? [Y/n]
:: Заменить klibc-kbd на core/mkinitcpio? [Y/n]
:: Заменить klibc-module-init-tools на core/mkinitcpio? [Y/n]
:: Заменить klibc-udev на core/mkinitcpio? [Y/n]
:: Заменить pkgconfig на core/pkg-config? [Y/n]
:: Заменить procinfo на core/procinfo-ng? [Y/n]
предупреждение: vi: принудительно обновляю до версии 050325-3
:: pacman обнаружил свою новую версию.
:: Хотите прервать текущую операцию и
:: установить новую версию pacman'а сейчас? [Y/n]
разрешения зависимостей...
ошибка: не удается разрешить "openssl>=1.0.0", зависимость "libarchhive"
ошибка: не удается подготовить операцию (не удается удовлетворить зависимости)
:: libarchhive: требует openssl>=1.0.0
Обновил openssl вручную (pacman -S openssl). Теперь версия openssl:
[[email protected] ~]# pacman -Qi openssl | grep Версия

Но это ничего не поменяло. Почему пакмен не хочет обновлять, может кто подскажет?
