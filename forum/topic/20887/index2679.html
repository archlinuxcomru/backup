Мне не нужен сервис, который создает юзеров и группы без моего ведома.
Выдержка из DOC ... не использю code, чтобы показать выделение текста ...
systemd-sysusers использует файлы из каталога sysusers.d для создания системных пользователей и групп, а также для добавления пользователей в группы во время установки пакета или во время загрузки. Этот инструмент может использоваться только для распределения системных пользователей и групп, он бесполезен для создания несистемных (то есть обычных, «человеческих») пользователей и групп, поскольку он обращается к / etc / passwd и / etc / group напрямую, минуя любые более сложные пользовательские базы данных, например, любая база данных, включающая NIS или LDAP.
Ошибки не исчезают с опытом - они просто умнеют
