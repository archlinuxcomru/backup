Если выбираете метод через экспорт переменной export FREETYPE_PROPERTIES="truetype:interpreter-version= (в /etc/profile.d/freetype2.sh или настройках юзера) (38 # Infinality mode 35 # Classic mode),
то не забывайте что Chromium и остальные сандбокс ее игнорируют.
Нужно устанавливать
freetype2-old-hinting для 35  # Classic mode
freetype2-infinality для 38  # Infinality mode
ну и в самом Chromium liberation не забыть заменить

По поводу Телеграмма то он вообще свои настройки использует
дискуссия до сих пор открыта
https://github.com/telegramdesktop/tdesktop/issues/90
