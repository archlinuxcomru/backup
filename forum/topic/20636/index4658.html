mpv-k-prf - "выбор mpv-профилей (из предзаданных. "предзаданность" - в большей степени предполагается что пользователем)" [иксы, tty].

https://archlinux.org.ru/forum/post/224568
https://github.com/knstntnn/mpv-k-prf

---------
- из сбоки Ледяного (там повешено на хоткей 'win+y').
- не опакечено.
- "юзкейз": хоткей для выделенной ссылки(/файла) ; выбор из диалога mpv-профиля ( далее запуск mpv). [# для выделенных файлов, частенько бывает, нужно предварительно чистить буферы - на Ледяном 'win+Shift+x чистит все 3.]
- больше подробностей в видео по ссылке в "сообщении по первой ссылке " ( то же ужатое видео - https://www.hostedredmine.com/attachments/338825 ).
