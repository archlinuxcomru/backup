с –global вываливало ошибки
надо смотреть лог , может не обновили npm, может прав не хватает, может еще что
на всякий случай
sudo pacman -Syu nodejs  node-gyp  semver npm
#для общесистемной установки
sudo npm install --global webtorrent-mpv-hook
#или для локальной для юзера
npm install webtorrent-mpv-hook

--
amon
. You need to symlink a script file to your mpv scripts folder - на что именно симлинк (где скриптфайл нарисовался?) и в какую папку его тулить (такой нету)?

(такой нету):
Для юзера папка mpv скриптов ~/.config/mpv/scripts
Общесистемная /etc/mpv/scripts
если нет то создать разумеется
--
на что именно симлинк (где скриптфайл нарисовался?):
В папке скриптов надо создать линк на сам скрипт
если общесистемно ставили то /usr/lib/node_modules/webtorrent-mpv-hook/build/webtorrent.js
если для юзера ставили то ~/node_modules/webtorrent-mpv-hook/build/webtorrent.js
--
Run webtorrent-mpv-hook for instructions. You only need to do this once. - каким макаром и откуда этот скрип запускать?
Запустите просто от юзера в эмуляторе терминала 
webtorrent-mpv-hook
#или если для юзера ставили и нет в PATH ~/node_modules
~/node_modules/.bin/webtorrent-mpv-hook
вам покажет инструкцию как и на что сделать симлинк
--
amon
