Пипец, вынос мозга. Когда меня все задолбало - решил перезагрузить сервер. И о чудо, после этого форвардинг заработал хоть как-то - при добавлении тупого FORWARD ACCEPT. Теперь осталось понять 2 проблемы:
1) firefox работает через сквид, но когда форвардинг закрыт - сайты очень медленно прогружаются. Когда открыт - все нормально. Чудеса прям оО
2) как же правильно прописать порты (а главное - какие порты) для форвардинга в локальную сеть. Сейчас прописаны правила наподобие
-t nat -A PREROUTING –dst 90.189.142.25 -p tcp –dport 1234 -j DNAT –to-destination 192.168.0.3
-A FORWARD -d 192.168.0.3 -p tcp –dport 1234 -j ACCEPT
для каждого порта, которые не устраивают как минимум тем, что указаны на один компьютер, а не на всю локальную сеть. Но если не указывать конкертный IP - эти правила не работают.
