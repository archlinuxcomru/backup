Пытаюсь запустить установку Arch используя флешку.
Скачал последний образ дистрибутива.
Для записи образа на флешку, воспользовался инструментами предложенными в инструкции https://wiki.archlinux.org/index.php/USB_flash_installation_media#In_Windows

При выборе одного из вариантов, начинается загрузка, экран гаснет и на этом все. В результате - пустой, темный экран, сообщений об ошибках нет.

Компьютер: Sony VAIO. Модель: PCG-71C11V

Попробовал запустить флешку на другом ноуте, все отлично загрузилось
Тестовый был нетбук: Samsung. Модель:NP-N310

Была похожая ситуация. Проблема может быть в Secure Boot.
Поищите в BIOS изменение режима загрузки с UEFI на нативный. Если ноутбук может загружаться только через UEFI (с большой долей вероятности так оно и есть), то попробуйте записать образ на болванку и загружайтесь через внешний привод - встроенного, я так понимаю, у Вас нет?!

Проверил настройки BIOS, стоит какая-то урезанная версия InsydeH20 Setup Utility Rev. 3.5
В параметрах Boot есть только следующие пункты:
Boot Configuration
External Device Boot [Enabled]
Network Boot [Disabled]

Boot Priotity
External Device
Internal Optical Disc Drive
Internal Hard Disk Drive
Network

И это все. В других разделах тоже нет ничего интересного
На ноутбуке стоит Windows 7, и судя по разделу загрузки, система загружается в режиме BIOS-MBR.
