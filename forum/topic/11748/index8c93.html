Добавлю ещё фишку для любителей quake/guake-подобных терминалов (выпадающий из верхней панели поверх всех остальных окон)

1 - кладём в %HOME/.config/awesome файл quake.lua - http://pastebin.com/knhsUDTZ

2 - в rc.lua прописываем следующие строки (в самом начале)
local quake = require("quake")
local quakeconsole = {}
for s = 1, screen.count() do
   quakeconsole[s] = quake({ terminal = "urxvtc",
           height = 0.95,
           screen = s })
end
*** высота 95%, на всех экранах

3 - прописываем комбинацию клавиш на вызов/скрытие окна терминала (я привык к F12)
awful.key({    }, "F12",    function () quakeconsole[mouse.screen]:toggle()   end),

4 - профит!

***терминал запускается демоном при старте awesome - autostart("urxvtd -q -f -o &", 1)
вызывается клиент urxvtc
