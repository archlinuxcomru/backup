Попробуйте поставить xorg-xkbcomp версии 1.2.3. Делается это с помощью пакета downgrade из AUR. Насчет зазоров, нужно size_hints_honor = false в конфиге, в раздел правила. Примерно так:

awful.rules.rules = {
    -- All clients will match this rule.
    { rule = { },
      properties = { border_width = beautiful.border_width,
                     border_color = beautiful.border_normal,
                     focus = awful.client.focus.filter,
                     keys = clientkeys,
                     buttons = clientbuttons,
	     	     size_hints_honor = false } },
...
