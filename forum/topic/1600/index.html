Решение:

1) Установим yaourt

sudo nano /etc/pacman.conf

В конец файла дописываем:

[archlinuxfr]
Server = http://repo.archlinux.fr/$arch

После того, как добавили репозиторий…

sudo pacman -Syu
sudo pacman -S yaourt

2) Устанавливаем патч на флеш из AUR:

yaourt -S libvdpau-git-flashpatch
A.T.W.A.
