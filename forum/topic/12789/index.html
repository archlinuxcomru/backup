Имею ATI HD 5770, проблема такая, что в Firefox, лагает прокрутка (причем не только плавная), рабочий стол КДЕ. Особенно сильные тормоза, если на странице много элементов, скажем графики, либо куча ссылок, при этом такое чувство, что fps падает до 1-2 при прокрутке. Смотрел вывыд команды top в этот момент, дело точно не в процессоре, его нагрузка была максимум 25-35%. Ставил драйверы каталиста, проблема еще хуже, там тормоза уже на обычных страницах, походу проблема в 2D ускорении, т.к. использую КДЕ с включеным Open GL, все на столе работает быстро, а вот в браузере firefox беда. От firefox отказаться немогу, т.к. использую его для работы, в нем есть нужные плагины.
У кого аналогичная проблема, есть ли решения?

Может ли помочь, включение Glamour?
Glamor
С новыми версиями открытых драйверов ATI вы можете использовать новый метод ускорения под названием "glamor": это метод ускорения 2D реализуется через OpenGL и будет работать только с видеокартами на чипсете R300 и новее.
 Option	"AccelMethod"           "glamor"

Также есть вариант с wiki (https://wiki.archlinux.org/index.php/ATI):
2D performance (e.g. scrolling) is slow
If you have problem with 2D performance, like scrolling in terminal or browser, you might need to add Option "MigrationHeuristic" "greedy" into the "Device" section of your xorg.conf file.
Bellow is a sample config file /etc/X11/xorg.conf.d/20-radeon.conf:
Section "Device"
        Identifier  "My Graphics Card"
        Driver  "radeon"
        Option  "MigrationHeuristic"  "greedy"
EndSection

Что выбрать, что действительно поможет?
