Если ставлю среду xfce где по умолчанию только alsa просто создаю файл asound.conf и в его добавляю pcm.upmix51 {
type upmix
slave.pcm "surround51"
delay 15
channels 6

pcm.!default "plug:upmix51"

В таком случае саб работает даже если с браузера смотрю видео но только в одной вкладке.
В пульсе же я настраивал через /etc/pulse/daemon.conf где ; resample-method = speex-float-1
; enable-remixing = yes
; enable-lfe-remixing = no (поменял на yes)
; lfe-crossover-freq = 0
а так же default-sample-format = s16le
; default-sample-rate = 44100
; alternate-sample-rate = 48000
; default-sample-channels = 2 (выставил 5)
; default-channel-map = front-left,front-right (указал все выходы)
Но это не дало результатов, звук на саб не микшировался как в альсе.
