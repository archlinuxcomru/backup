Небольшое пояснение
В файле /proc/acpi/wakeup приведен список устройств, которые способны обеспечить выход из suspend
Мой пример
до ухода в suspend
grep -c enable /proc/acpi/wakeup
2
cat /sys/power/wakeup_count
2
sudo egrep 'success:|fail:' /sys/kernel/debug/suspend_stats
success: 0
fail: 0
после выхода из suspend по моему желаниею (кнопка power)
cat /sys/power/wakeup_count
3
sudo egrep 'success:|fail:' /sys/kernel/debug/suspend_stats
success: 1
fail: 0
и как видим - значение счетчика wakeup_count увеличилось на 1 ... а вот в твоем случае это значение не изменилось.
