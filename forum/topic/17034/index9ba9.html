Я правильно понимаю?
Сначала я в pacman.conf добавлю репозиторий для xorg117 т.к. Catalyst не поддерживает xorg-server 1.18 и более новые версии.
[xorg117]
Server = http://catalyst.wirephire.com/repo/xorg117/$arch
## Зеркала, если основной сервер не работает или работает, но слишком медленно:
#Server = http://mirror.rts-informatique.fr/archlinux-catalyst/repo/xorg117/$arch
#Server = http://mirror.hactar.bz/Vi0L0/xorg117/$arch
следом идет репозиторий для Catalyst
[catalyst]
SigLevel = Never
Server = http://mirror.hactar.xyz/Vi0L0/catalyst/$arch
Далее обновляю систему
sudo pacman -Syy
sudo pacman -Syu

Далее устанавливаю Catalyst, правильная последовательность?
