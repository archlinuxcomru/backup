Имеются: Звуковая карта Asus Xonar AE + колонки+наушники. Звук в наушниках есть (через обычный разъём jack), а вот колонки не в какую через s/pdif не хотят играть, настройки на скриншоте. Смущает скудность "столбиков" при выбранной звуковой карте в alsamixer.
раз звуковая видится системой, и звук есть в наушниках, почему его нету в колонках... Колонки поддерживают звук по оптике до 48000 герц и PCM, где можно этот вывод настроить... или в чём-то другом дело...

Решено.

sudo gedit /etc/asound.conf

всё что там есть удалить или закоментировать, и вставить:
pcm.!default {
type hw
card 0
}

ctl.!default {
type hw
card 0
}

Теперь можно выбрать s/pdif в настройках вывода звука, и звук будет)
