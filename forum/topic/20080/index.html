Всем привет!
Собираюсь поставить несколько linux дистрибутивов на один HDD,
и хотел бы узнать мнение опытных пользователей о том как лучше всего это сделать.

Цель: получить несколько полноценно функциональных linux дистрибутивов с возможностью выбрать нужную систему на этапе загрузки в меню GRUB

Доводилось читать об отчетах пользователей устанавливающих по 7 дистрибутивов линукс на один HDD. По их рассказам алгоритм установки такой:
1) создаем разделы под root для каждой из систем
2) последовательно устанавливаем системы

Так ли это? Или нужны еще какие то дополнительные манипуляции с GRUB?
