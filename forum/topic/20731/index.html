Даже на изменение /sys/…/brightness нет реакции:
echo "$BRIGHTNESS" > /sys/class/backlight/intel_backlight/brightness
Для начала дай вывод - ls /sys/class/backlight

EDIT 1 - у тебя точно только видеокарта Intel??? .... и смотри, что за ошибки есть в логах journal - journalctl -b | egrep -i 'backlight|brightness'

EDIT 2 - а вообще рекомендую погулить на BBS (воспользуйся там поиском по xbacklight) - много тем за последний год по данной проблеме - решения разные, зависит от железа
Ошибки не исчезают с опытом - они просто умнеют
