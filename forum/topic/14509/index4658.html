сделал так:

1) переустановил python: sudo pacman -S python
2) удалил hplip: pacman -R hplip
2) поставил основным python2: sudo cp /usr/bin/python2 /usr/bin/python
3) скачал с сайта http://hplipopensource.com исходники hp-lip версией 3.14, установил как scratch (пункт 14)
4) после этого в папке /usr/lib/python2.7/site-packages появился hpmudext
5) установил последний hplip: sudo pacman -S --force hplip
6) запустил настройку принтера: hp-setup -i
7) и все заработало
