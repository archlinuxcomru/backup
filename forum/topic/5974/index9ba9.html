error_file              "/home/su/.mpd/mpd.error"
pid_file                "/home/su/.mpd/pid"
state_file              "/home/su/.mpd/state"
default_permissions             "read,add,control,admin"
volume_normalization            "yes"
user                            "mpd"
bind_to_address      "127.0.0.1"
port                            "6600"
audio_output {
        type                    "alsa"
        name                    "My ALSA Device"
        device                  "hw:0,0"     # optional
        format                  "44100:16:2" # optional
}
mixer_type      "alsa"
mixer_device    "default"
mixer_control   "PCM"
filesystem_charset              "UTF-8"

есть еще способ запустить mpd - от своего пользователя
Для этого скопируй /etc/mpd.conf в ~/.mpdconf
Измени в ~/.mpdconf
user “mpd” на
user “su”
закомментируй device в audio_output - это чтобы со звуком было все норм.

И запускай его коммандой mpd от пользователя su
