wayland в системе установлен (как понимаю по умолчанию). Тормозов вроде не наблюдаю
Тормоза возможны только при определенных ситуациях. У меня одна прога притянула несколько зависимостей, связаных с xdg-desktop-portal.service, который не запускался и вызывал тормоза. Нашел решение для sway, сейчас все нормально - сервис запускается и тормоза исчезли.
Вот этот сервис, сейчас он нормально стартует
systemctl --user status xdg-desktop-portal.service
● xdg-desktop-portal.service - Portal service
     Loaded: loaded (/usr/lib/systemd/user/xdg-desktop-portal.service; static)
     Active: activating (start) since Fri 2022-05-06 17:37:14 MSK; 58s ago
   Main PID: 696 (xdg-desktop-por)
      ... и так далее ...

PS - вот только пока не стал глубоко копать, в части удаления этих xdg-desktop-portal - нужно смотреть их назначение, привязку и др. ... но понял одно, что это гадость, которую лучше убрать. Кстати, сразу после того обновления и reboot, стал тормозить и spectwm, но после все прошло, а вот на sway не прошло и пришлось искать лечение.
А потому предположу, если у кого появились тормоза, то сначала желательно проверить наличие пакетов xdg-desktop-portal-... и наличие ошибок в запуске сервиса. Вот эта гадость
pacman -Ss xdg-desktop-portal
extra/xdg-desktop-portal 1.14.4-1 [установлен]
    Desktop integration portals for sandboxed apps
extra/xdg-desktop-portal-gnome 42.1-1 [установлен]
    A backend implementation for xdg-desktop-portal for the GNOME desktop environment
extra/xdg-desktop-portal-gtk 1.14.0-1 [установлен]
    A backend implementation for xdg-desktop-portal using GTK
extra/xdg-desktop-portal-kde 5.24.5-1 (plasma)
    A backend implementation for xdg-desktop-portal using Qt/KF5
community/xdg-desktop-portal-lxqt 0.2.0-1
    A backend implementation for xdg-desktop-portal using Qt/KF5/libfm-qt
community/xdg-desktop-portal-wlr 0.5.0-1 [установлен]
    xdg-desktop-portal backend for wlroots
archlinuxcn/xdg-desktop-portal-lxqt-git 0.0.0.r13.9191efa-1
    A backend implementation for xdg-desktop-portal using Qt/KF5
Ошибки не исчезают с опытом - они просто умнеют
