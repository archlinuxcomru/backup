Место на дисках есть?
df

sudo pacman -Syu
конечно же, выполнялось?

Если ничего не помогает - на bbs есть такое грубое решение (архитектура x86_64 - “по вкусу”):
pacman -U /path/to/xz-utils-4.999.8beta-4-x86_64.pkg.tar.gz

И в совсем крайнем случае 
bsdtar -xf xz-utils-4.999.8beta-4-x86_64.pkg.tar.gz -C /

Затем не забыть
sudo pacman -S xz-utils
