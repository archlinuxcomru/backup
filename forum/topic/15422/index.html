Насколько я понял flashplugin для firefox постепенно устаревает и уже на некотрых сайтах не работает.
Неработоспособность проявляется в том, что не работают аудиоплееры на некоторых сайтах. В частности не работает плеер яндекс музыка.
Видео преимущественно работает.
Я пробовал в качестве альтернатив для flashplugin:
gnash
mozplugger + mpv
addwith
Ничего из этого не помогло. Поэтому когда на сайте не работает аудиоплеер вместо firefox прихожится использовать chromium. На chromium переходить не хотелось бы в силу привычки к firefox. Вопрос какие есть альтернативы flashplugin для firefox, чтобы можно было полноценно использовать firefox для видео и аудио?
