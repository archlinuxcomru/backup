В репах сейчас 2.4.1-1
попробуйте поменять зеркало на полностью синхронизированное
sudo pacman -S reflector
sudo reflector --verbose --latest 5 --sort rate --save /etc/pacman.d/mirrorlist
sudo pacman -Syu
или вручную новое зеркало в /etc/pacman.d/mirrorlist пропишите

Надо было заменить верхнее зеркало репозиториев в /etc/pacman.d/mirrorlist (сервер от куда пакеты качаются) иногда они бывают не полностью синхронизированы и часть пакетов на сервере устаревшие какое то время (от того и проблемы несовпадения версий ).
Зеркало по видимому уже полностью синхронизировалось.
Но рефлектор все же поставьте, чтоб всегда иметь свежие зеркала
https://wiki.archlinux.org/title/Reflector

AZJIO, рекомендую просто активировать эти немецкие сервера - проблем с ними не наблюдаю
grep -vE "(^#|^$)" /etc/pacman.d/mirrorlist
Server = http://mirrors.n-ix.net/archlinux/$repo/os/$arch
Server = https://mirrors.n-ix.net/archlinux/$repo/os/$arch
Server = http://mirror.23media.com/archlinux/$repo/os/$arch
Server = https://mirror.23media.com/archlinux/$repo/os/$arch

PS 1 - под "активировать" подразумеваю "раскоментировать"
Ошибки не исчезают с опытом - они просто умнеют
