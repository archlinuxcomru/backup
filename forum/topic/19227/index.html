GWM
С чем это может быть связано? Может это вопрос какой-нибудь безопасности, и просто выставив этот параметр в smtp я сделал что-то очень плохое?
Если почитать документацию, как было предложено выше, то там указано
default_transport = error:outside mail is not deliverable
The variable default_transport specifies which transport is used to deliver non-local mail (by default smtp). With this setting, any mail from the outside will bounce back with an error instead of being stuck in the mail queue forever.
Если сказать простым языком, то смысл такой - при установке параметра default_transport = error серевер перестает отправлять почту на внешние почтовые сервера.
В принципе этот же смысл и прописан в Wiki - просто нужно внимательно читать ...
Local mail
To only deliver mail to local system users …..
Uncomment, change, or add the following lines:
default_transport = error: outside mail is not deliverable
а дальше описано и про значение default_transport = smtp ….
И каждый выставляет то, что ему нужно ...
Ошибки не исчезают с опытом - они просто умнеют
