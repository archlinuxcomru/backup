Если глобально для всех пользователей, то смотрите в /usr/share/applications/mimeinfo.cache, если локально для одного пользователя, то в ~/.local/share/applications/mimeapps.list. Если этих файлов вдруг нет (что врядли), поставьте пакет gnome-mime-data, возможно поможет. )

Точно такое же окно будет появляться у вас там, где используется xdg-open просьбой открыть файл с незнакомым mime-type.
В том то и дело, что когда скачиваю, например, архив из AUR'a, для проверки, и нажимаю на него в загрузках, чтобы открыть, ничего не происходит. gnome-mime-data и xdg-utils установлены.

Решил проблему так: создал файл ~/.local/share/applications/defaults.list и записал туда необходимые MimeType и программы для их открытия. Всё заработало.

P.S MimeType, если кто не знает, можно посмотреть в /usr/share/applications/<название программы>.desktop
