При загрузке grub пишет красным
 FAILED Not running dhcpcd because there is already an interface specific instance
FAILED /var/run/dhcpcd.eth0.pid  FAILED
и 
FAILED VirtualBox Additions disabled, not in a Virtual Machine
.
После загрузки и интернет и виртуалбокс работают отлично, без сбоев.
Файл /var/run/dhcpcd.eth0.pid есть,там четыре цифры.
Что означают эти сообщения или просто на них забить и не обращать внимания?
Так уже давно,работе не мешает,просто решил спросить для общего развития .

FAILED Not running dhcpcd because there is already an interface specific instance
Наверно запущены несколько демонов.
Vadim
