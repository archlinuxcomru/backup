реальная скорость загрузки по гигабитному интерфейсу 10 метров в секунду
Чем проверяли?
Для проверки реальной скорости обмена в локальных сетях используется iperf.
Всё остальное - разные показатели, которые к скорости обмена имеют лишь посредственное отношение.
Например и windows и linux GUI сетевого соединения показывают скорость интерфейса а не скорость обмена. Скорость копирования информации по сетевым протоколам будет разной в зависимости от протокола (SMB, FTP, NFS), среды передачи (медь, оптика, радиоканал), для WIFI скорость передачи будет отличаться на разных расстояниях и разном железе.
В-общем, нужно больше конкретики:
архитектура сети,
железо,
операционные системы,
протоколы,
выхлоп iperf для локальной сети
и т.д
