1. флешка не стоит - нужно вводить пароль sudo
sudo pkexec nautilus
* pam_usb v0.5.0
* Authentication request for user "vasek" (sudo)
* Device "usb_16" is not connected.
* Access denied.
[sudo] пароль для vasek: …..
и все ... запускается nautilus, сразу после ввода пароля sudo

2. флешка стоит, пароль sudo вводить не требуется
sudo pkexec nautilus
* pam_usb v0.5.0
* Authentication request for user "vasek" (sudo)
* Device "usb_16" is connected (good).
* Performing one time pad verification...
* Access granted.
и все ... запускается nautilus без всякого ввода пароля

EDIT 1 - но есть файл /usr/share/polkit-1/actions/org.freedesktop.policykit.nautilus.policy
