Пытаюсь настроить acpid, есть желание настроить выключение компа по кнопке power. acpid нажатие отлавливает успешно, но вот выключить через терменал у меня не получается.
Гугление дало следующею команду:
dcop --all-sessions --all-users ksmserver ksmserver logout 0 2 0 && exit 0
На что система говорит что сервер dcop не запущен, где и как он должен запускаться я не знаю и почему не запускается тоже.
[[email protected] ~]$ dcop --all-sessions --all-users ksmserver ksmserver logout 0 2 0 && exit 0
DCOPClient::attachInternal. Attach failed Could not open network socket
DCOPClient::attachInternal. Attach failed Could not open network socket
ERROR: Couldn't attach to DCOP server!
Есть обходные пути?
