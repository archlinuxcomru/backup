Попробуй пересоздать swap разделы (поочередно загружаясь на каждом диске, если я правильно понял твою организацию систем на дисках)
1. Отключаем - $ sudo swapoff -a
2. Проверяем - $ swapon -s
3. Создаем - $ sudo mkswap /dev/sdaх
Перед вводом команды, чтобы не ошибиться, проверь какой свап раздел
$ sudo fdisk -l................тут же проверь ID раздела (82)
Указанный, после выполнения команды UUID, занеси в fstab (отредактируй)
4. Активируем - $ sudo swapon -a
5. Проверяем - $ swapon -s
6. Повторить для второго диска.
Надеюсь, все восстановится.
