# swapon /dev/sdb5
подключается без проблем.
Вот fstab 
# /etc/fstab: static file system information.

Что нибудь вроде systemctl enable swap ...... ?

У Вас GPT?
(В случае GPT у разделов также есть UUID, можно спутать).

Более того, systemd and swap

Note: If both generators create a systemd unit for the same swap partition, they are conflicting, resulting in an error message.

Т.е. запись в fstab может привести к конфликту.
