icedtea-web-java7 для этих целей не подходит, нужен java sun с сайте java.com

в chromium под линуксом начиная с версии 35 убрали поддержку жабы (сам был в шоке, может вернут обратно когда-нибудь), а под мазилой ставится так (имя файла jre-7u7-linux-i586.tar.gz заменить на свое)

скачиваете жабу с ссылки выше и распаковываете:
# cd /usr/java/
# tar zxvf jre-7u7-linux-i586.tar.gz

переходите в каталог мазилы:
# cd <каталог установки Firefox>/plugins

создаете ссылку:
- для 32 -битных ОС
# ln -s <каталог установки Java>/lib/i386/libnpjp2.so

- для 64-битных ОС:
# ln -s <каталог установки Java>/lib/amd64/libnpjp2.so

открываете мазилу и пишете в адресной строке about:addons , в списке должен появится java-tm(plugin), icedtea отключаете обязательно

сертификаты вносите в браузер по умолчанию и на всякий случай прописываете в жабе. для этого заходите /usr/java/ в папку с жабой, далее находите директорию bin в ней запускаете jcontrol. во вкладке security в список except site list вносите свой сайт банка (осторожнее с этим - угроза безопасности), затем жмете по кнопке Manage Sertificates выбираете тип сертификата Client Authentification (можно во все секции, кроме этой) - и вносите свои сертификаты.

небольшое уточнение: некоторые банки делают сайты заточенные под эксплорер. с этим уже ничего не поделать, эксплорер под вайном у меня не цепляет жабу, тут либо виртуалбокс либо винду ставить вместо пингвина
