mbr2gpt.exe
Зря делал встроенной утилитой - по ее использованию есть ограничения, какие точно не помню. Да и противоречивая информация по ее использованию - у кого то получается у кого то нет ... и советуют делать специально предназначенными для этого утилитами с Live CD, а не и самой системы.

PS - если не изменяет память, то ограничения по mbr2gpt.exe следующие
- диск должен быть загрузочным
- количество разделов - не больше 3-х
- не должно быть расширенного раздела.
Ошибки не исчезают с опытом - они просто умнеют
