Hello everyone!
хочу установить ArchLinux на уже существующие разделы.
fdisk -l
Device     Boot     Start       End   Sectors   Size Id Type
/dev/sda1  *         2048    206847    204800   100M  7 HPFS/NTFS/exFAT
/dev/sda2          206848 151427071 151220224  72,1G  7 HPFS/NTFS/exFAT
/dev/sda3       151429118 976771071 825341954 393,6G  5 Extended
/dev/sda5       151429120 159240191   7811072   3,7G 82 Linux swap / Solaris
/dev/sda6       159242240 295958527 136716288  65,2G 83 Linux
/dev/sda7       295960576 820568063 524607488 250,2G 83 Linux
/dev/sda8       820570112 976771071 156200960  74,5G 83 Linux

/dev/sda8 будет как /
/dev/sda7 будет как /home совместно с Debian(он на /dev/sda6)
если я сделаю так:
#mkfs.ext4 /dev/sda8
#mkswap /dev/sda5
потом:
#mount /dev/sda8 /mnt
#swapon /dev/sda5
#mkdir /mnt/home
#mount /dev/sda7 /mnt/home

будет правильно?
