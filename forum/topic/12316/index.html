Что такое "зарегистрированное пространство windows"?
виндовый boot-раздел, по сути нафиг не нужен ибо все равно половина boot-файлов висит в основном разделе с виндой

Имхо:
/boot - не нужен;
swap - 2 гб, но никак не больше
/ - 20гб

os-prober - ставится, затем стандартно grub-mkconfig -o /boot/grub/grub.cfg
винда автоматом в grub добавится

Раздел 100 метров для загрузчика windows при разбивке диска отдельно выделять - не нужно. При установке windows сам его выделит, если сочтёт его нужным.
Раздел под /boot я бы выделил ибо при реанимации системы с ним удобнее работать. Необходимость swap-раздела зависит от объема ram, 4 гига для домашнего/рабочего ПК ИМХО много. Никаких хитростей с os-prober нет, винду находит, добавляет загрузочную запись в меню grub.
