У меня на одном жестком диске уже установлены XP и Ubuntu. Хочу ещё поставить Arch. Только есть два вопроса.
1. После установки в новом грубе останутся XP и Ubuntu?
2. Уже есть один swap-раздел, который использует Убунту. Нужно ли для Arch'a создавать ещё один, или можно тот что есть юзать?

1. Если один груб уже есть, ставить второй совершенно не обязательно. Достаточно добавить загрузку арч в конфиг имеющегося. Подробности см. в руководствах по грубу.
2. Можно совершенно спокойно использовать имеющийся swap-раздел.
