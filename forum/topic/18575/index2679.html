WarThunder получил экспериментальную поддержку API Vulkan
для некоторых карточек (радеон) повышение FPS в два раза
Новость и инструкция на лоре
Войти в каталог со скачанной игрой.
Войти в каталог «content».
Создать файл pkg_vulkan.rq2
Запустить лаунчер, подождать пока будут скачано обновление.
Закрыть ланчер.
Отредактировать текстовый файл config.blk: в ключах renderer2:t=«auto» и driver:t=«auto» заменить «auto» на «vulkan».
Не запускать игру через лаунчер - он перепишет файл config.blk. Запускать игру как ./linux64/aces

Нвидиа меньше 600 серий к сожалению не поддерживает vulkan, так что не проверял
