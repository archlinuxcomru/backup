dhcpcd не дают никаких результатов, т.к этот сервис не найден, ровно как и NetworkManager, тоже на любые команды по его включению или отключния или чего бы то ни было еще это: Unit file NetworkManager.service does not exist.
Их надо было ставить при установке если нужны.

Какой у вас тип подключения ?

Для обычного проводного с dhcp

 sudo rm /etc/resolv.conf

sudo ln -s /run/systemd/resolve/resolv.conf /etc/resolv.conf
...
sudo nano /etc/systemd/network/20-dhcp.network
Пишем
[Match]
Name=en*
[Network]
DHCP=yes
Сохраняем
Включаем сервисы
 sudo systemctl enable systemd-networkd

sudo systemctl enable systemd-resolved

Ребут
