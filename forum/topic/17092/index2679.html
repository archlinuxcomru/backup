openbox + compton. В конфиге compton есть такая строка: inactive-dim = 0.4;.
Действительно притемняет,, то не все неактивные окна, а только окна неактивных процессов. Скажем, если открыто несколько документов LibreOffice, то при переключении на другое приложение все они притемняются, но при переключении между окнами LO неактивные окна LO не притемняются. При этом тот же самый compton тени от окон LO отображает без затей, так что это типа не глюк, а фича.
Так вот, нет ли средства притенять тупо все неактивные окна? Вместо compton годится что угодно другое, openbox менять ни на что не хочу.

Там еще есть несколько настроек для этого.

inactive-opacity = 1.0;                ## Opacity of inactive windows (0.1 - 1.0).
inactive-opacity-override = false;     ## Inactive opacity set by 'inactive-opacity' overrides value of _NET_WM_OPACITY.
inactive-dim = 0.0;                    ## Dim inactive windows (0.0 - 1.0, defaults to 0).
inactive-dim-fixed = true;             ## Do not let dimness adjust based on window opacity.

Можно поиграть с цифрами и буквами...
