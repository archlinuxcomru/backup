Вариант первый: Использовать сетевую загрузку (для простоты можно взять pixiecore)
  1. [Сервер] Установить pixiecore:
    2. % yaourt -S pixiecore
  2. [Сервер] Скачать образ ipxe.XXXX.lkrn (<1mb)
  3. [Сервер] Запустить:
    4. % sudo pixiecore boot ipxe.XXXX.lkrn -d
  4. [Клиент] Установить в BIOS загрузку по сети.
    5. Также необходимо быть в локальной сети с сервером и выходом в интернет через роутер.

Вариант второй: Собрать свой образ Арча
для простоты можно использовать, например, скрипт archiso-build, предварительно задав в скрипте переменную Platform которая определяет с поддержкой каких архитектур собирать образ. Для одной архитектуры размер образа будет ~ 400мб, оф.образ поставляется с поддержкой двух архитектур поэтому и весит почти в два раза больше.
После запуска, через 15-20 минут, получишь готовый образ.
