Взять загрузочный образ (я брал sysresccd, но подойдёт и другой) CD с syslinux или grub, добавить в образ ipxe.lkrn и пункт меню загрузки
например
http://tdkare.ru/sysadmin/index.php/Grub2
  menuentry "etherboot loader" {
     savedefault
     set root=(hd0,1)
     linux16  /boot/ipxe.lkrn
  }
прожечь образ на CD, загрузиться, выбрать нужный пункт.
(Сеть должна быть по проводу и с dhcp)

Варианты с записью образа на винт рассматривались, но, как я понимаю, тут надо иметь при себе какой-то загрузочный диск с грабом, дабы стартануть этот самый образ с винта.
Предлага 2 варианта
1. Загрузка с установочного образа ArchLinux, расположенного на HDD .... описано здесь (как то тренировался ради интереса)
2. Используя Plop Boot Manager — специально для таких случаев
Как делать - скачай с ихнего сайта plpbt-5.0.15 , в этом пакете тебе нужен образ plpbt.iso — запиши его на CD и грузись с него (флешка должна быть вставлена до загрузки) — после загрузки ищи USB , должен попасть сразу в меню загрузочной флешки.
UPD.....если последняя версия не сработает, можно попробовать старые версии....
Ошибки не исчезают с опытом - они просто умнеют
