UPD: Это проблема на стороне Arch Linux, решение находится в этом issue: https://github.com/pypa/pip/issues/10750
Кому лень заходить, вот пункты:

1. wget bootstrap.pypa.io/get-pip.py
2. Запустить get-pip.py с помощью версии Python, с которой у нас проблема. В моём случае это было: python get-pip.py
После этого обновляем pip по желанию и работаем.
