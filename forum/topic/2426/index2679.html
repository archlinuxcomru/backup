Соврал :) Стоял lxappearance2 не менял gtk3 темы . Поставил community/lxappearance 0.5.1-3 меняет - так что то что вам нужно. Старый конфиг нужно удалить 
rm -r ~/.config/gtk-3.0/
Изменения применяются после перезагрузки приложения например 
killall nautilus
nautilus /
На некоторых темах глючит - так что лучше по старинке cp -r /usr/share/themes/название_темы/gtk-3.0/ ~/.config/

The new lxappearance (from the Arch repo's) supports GTK3. Manually, you can tweak GTK3 by going to ~/.config/gtk-3.0/settings.ini. Here's mine:


gtk-theme-name = Zukitwo-Dark
gtk-icon-theme-name = elementary-mono-dark
#gtk-fallback-icon-theme-name = Faenza
# next option is applicable only if selected theme supports it
gtk-application-prefer-dark-theme = false
# set font name and dimension
gtk-font-name = Ubuntu 10
… хм возможно она глючит при переходе с 2 на 3,т.е. я помню както кто-то спрашивал об эдаком, но вроде он так и остался на утилите…я просто обычно что 2 что 3 руками меняю когда точно уверен какая мне тема нужна, но до этого lxapp.. не подводил. если кто проверит раньше напишите, я сравню потом.
