После обновления пакета dbus-openrc до версии 20160329-1 перестало работать автомонтирование.
После анализа распакованых пакетов старой и новой версии увидел что в новой версии нет файла /etc/X11/xinit/xinitrc.d/30-dbus
вот такого:
#!/bin/bash

# launches a session dbus instance
if [ -z "${DBUS_SESSION_BUS_ADDRESS-}" ] && type dbus-launch >/dev/null; then
  eval $(dbus-launch --sh-syntax --exit-with-session)
fi
Создал вручную файл-автомонтирование заработало.
Буду дальше разбираться.
