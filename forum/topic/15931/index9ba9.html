Решено!
В общем-то да, мать странная. Dual memory mode работает только если параллель идёт 2-4 слотами. Если плашки стоят в 1-3 слотах, то работает только Single memory mode.
Недавно делал в компе генеральную уборку, вот и решил поставить в 1-3 слоты. Вернул в зад, теперь всё огонь.
