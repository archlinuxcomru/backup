Проще показать.
$ nvidia-settings -q gpucoretemp

  Attribute 'GPUCoreTemp' (arch:0.0): 35.
    'GPUCoreTemp' is an integer attribute.
    'GPUCoreTemp' is a read-only attribute.
    'GPUCoreTemp' can use the following target types: X Screen, GPU.
  Attribute 'GPUCoreTemp' (arch:0[gpu:0]): 35.
    'GPUCoreTemp' is an integer attribute.
    'GPUCoreTemp' is a read-only attribute.
    'GPUCoreTemp' can use the following target types: X Screen, GPU.
$ nvidia-settings -q gpucoretemp -t
35
35
$ nvidia-settings -q [gpu:0]/gpucoretemp -t
35
Надо сказать, эта команда работает гораздо медленнее следующей цепочки:
$ nvidia-smi -q -d temperature | grep 'GPU Current Temp' | cut -d : -f 2 | sed s'/[ C]//g'
