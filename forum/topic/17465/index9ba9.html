Дошли руки до 32-битного железа. Обновился без разных примудростей - последовал примеру
Aivar
В конкретном случае я не использовал mirrorlist, а зеркало прописал в конфиг пакмана:
[core]
SigLevel = PackageRequired
Server = https://archlinux32.mirror.roelf.org/i686/$repo

[extra]
SigLevel = PackageRequired
Server = https://archlinux32.mirror.roelf.org/i686/$repo

[community]
SigLevel = PackageRequired
Server = https://archlinux32.mirror.roelf.org/i686/$repo
А потом сделал, как рекомендуют здесь :
# pacman -Syy archlinux32-keyring-transition
# pacman -Syuu
