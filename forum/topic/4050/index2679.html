frostmorne, запили у себя, локально. Проверь. Потом и в аур можно закинуть. Пересобрать можно через ABS, текущая версия: community/qbittorrent 2.8.2-1

ps. Сам сделал:
[email protected]:~$ /var/abs/local/yaourtbuild/qbittorrent/src/qbittorrent-2.8.2/src/qbittorrent
Peer ID: -UT2210-
HTTP user agent is uTorrent/2210(25302)

PKGBUILD - http://pastebin.ru/320129
rutracker.patch - http://pastebin.ru/320127

Если недоустановлены зависимости для сборки - выведет, чего не хватает ( у меня boost пришлось установить).
Почитай подробнее: ABS - вообще, рекомендуется для сборки использовать $HOME/abs или $HOME/build, но у меня это /var/abs/local/ с правами записи для меня.

pps. Вобщем, установил я его, но с рутрекера качаю редко, так что проверяйте сами. Назвал его qbittorrent-fake, чтобы не путать с официальным.
