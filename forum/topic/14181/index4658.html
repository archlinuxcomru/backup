У меня не установлен systemd-sysvcompat. Так что в грубе соответственно прописана переменная инит.

menuentry "Archlinux"{
linux /boot/vmlinuz-linux root=/dev/sda1 rw init=/sbin/init-openrc vga=791
initrd /boot/initramfs-linux.img
}

menuentry "Archlinux systemd"{
linux /boot/vmlinuz-linux root=/dev/sda1 rw vga=791
initrd /boot/initramfs-linux.img
}
Да пребудет с вами знание ip адреса

С версии 1.16 иксы запускаются без root-привилегий при наличии DRM/KMS-драйверов. Попробуйте запустить иксы по старинке, от рута:
$ cat /etc/X11/Xwrapper.config
allowed_users=anybody
needs_root_rights=yes

Взято отсюда:
https://bbs.archlinux.org/viewtopic.php?pid=1456369#p1456369
