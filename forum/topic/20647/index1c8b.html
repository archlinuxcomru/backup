Мне нужно что-либо показать?
Стандартный путь по вики
определяем железо и используемый драйвер
lspci -k
пробуем его поднять
ip link set wlan0 up
и смотрим
поднялся ли
ip link
state UP должно быть для начала

https://wiki.archlinux.org/index.php/Network_configuration/Wireless

если опять не будет видеть сети попробуйте антенну переключить
https://bbs.archlinux.org/viewtopic.php?id=208472
sudo rmmod rtl8723be
sudo modprobe rtl8723be ant_sel=1
sudo iwlist wlan0 scan
sudo rmmod rtl8723be
sudo modprobe rtl8723be ant_sel=2
sudo iwlist wlan0 scan
