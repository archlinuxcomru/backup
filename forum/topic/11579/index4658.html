P.S.: Понятно, слетели библиотеки
feh: error while loading shared libraries: libpng15.so.15: cannot open shared object file: No such file or directory
Это даже не смешно: 
$ pacman -Q libpng
libpng 1.6.2-3
$ pacman -Ql libpng|grep .so.
libpng /usr/lib/libpng16.so.16
libpng /usr/lib/libpng16.so.16.2.0
У вас feh ругался, потому что был связан с libpng15, которого уже нет, потому что новый libpng уже libpng16.

В любом случае, вместо того, чтобы упорно твердить "у меня ничего не работает" смотрите с чем связано, смотрите что пишет в консоли.
