Скрин бы для наглядности.

У меня тоже основной Liberarion Sans:size=10 + extra/cairo, extra/libxft, aur/fontconfig-lcd.
Сглаживание общесистемное, /etc/X11/Xresources:
Xft.dpi: 90
Xft.hintstyle: hintslight
Xft.hinting: true
Xft.antialias: true
Xft.lcdfilter: lcddefault
Xft.rgba: rgb
Проблем не наблюдаю. В кедах подхватываются общесистемные настройки, так что врядли они как-то “по-своему” шрифтами занимаются.

Вот что пишут в Wiki:
Например, для применения автохинтинга шрифтов Free Type необходимо создать в папке /etc/fonts/conf.d символическую ссылку на файл /etc/fonts/conf.avail/10-autohint.conf:

# ln -s /etc/fonts/conf.avail/10-autohint.conf /etc/fonts/conf.d/10-autohint.conf

Примечание: В настоящее время не все шрифты с автохинтингом выглядят приемлемо, поэтому рекомендуется его использовать либо для отрисовки шрифтов малого размера, либо не использовать вообще.
У меня выглядит очень здорово, только шрифт надо не больше 10 ставить.
И вот такой файл .fonts.conf, спасибо FIL
<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
 <match target="font">
  <edit mode="assign" name="rgba">
   <const>rgb</const>
  </edit>
 </match>
 <match target="font">
  <edit mode="assign" name="hinting">
   <bool>true</bool>
</edit>
</match>
</fontconfig>
