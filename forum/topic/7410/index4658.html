Вот у меня тоже чехарда была с интерфейсами при использовании стандартных имен. Пришлось обзывать их иначе:
tunguska% cat /etc/udev/rules.d/10-persistent-net.rules
# This file was automatically generated by the /lib/udev/write_net_rules
# program, run by the persistent-net-generator.rules rules file.
#
# You can modify it, as long as you keep each rule on a single
# line, and change only the value of the NAME= key.
# PCI device 0x1106:0x3106 (via-rhine)
SUBSYSTEM=="net", ATTR{address}=="00:19:5b:8a:fa:ed", NAME="internet"
# PCI device 0x10de:0x03ef (forcedeth)
SUBSYSTEM=="net", ATTR{address}=="00:24:21:f3:a5:1a", NAME="localnet"
# PCI device 0x1106:0x3106 (via-rhine)
SUBSYSTEM=="net", ATTR{address}=="00:1b:11:c3:f9:e8", NAME="empty"
Разберемся, голубчик!
