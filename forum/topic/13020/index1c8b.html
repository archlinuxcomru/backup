Размеры самой панельки (и другие параметры, такие как цвет, шрифт, итд) настраиваются в конфиге i3status?
Панели в обычном понимании нет, это - i3bar. Системная и др. информация. - i3status. Размер i3bar зависит от размера шрифта в конфиге i3:
~/.i3/config
————————————
# Font for window titles.
font -misc-fixed-medium-r-normal-*-15-140-75-75-c-90-iso10646-1
#font -misc-fixed-medium-r-normal-*-20-200-75-75-c-100-iso10646-1
#font -misc-dejavu sans-medium-r-*-*-0-*-*-*-*-*-iso10646-1
#font -misc-dejavu sans mono-medium-r-*-*-0-*-*-*-*-*-iso10646-1
#font pango:DejaVu Sans Condensed 11

Цвет рамки окна (если она включена) в том же конфиге:
# Colors
# class                 border  backgr. text    indicator
client.focused          #4c7899 #285577 #ffffff #2e9ef4
client.focused_inactive #333333 #5f676a #ffffff #484e50
client.unfocused        #333333 #222222 #888888 #292d2e
client.urgent           #2f343a #900000 #ffffff #900000
client.placeholder      #000000 #0c0c0c #ffffff #000000

Цвет самой "панели", кнопок (если они включены), цвет i3status в том же конфиге:
# Start i3bar to display a workspace bar (plus the system information i3status
# finds out, if available)
bar {
#	status_command i3status
status_command $HOME/scripts/i3stat-json.sh
        position top
#     separator_symbol " "
        workspace_buttons yes
        colors {
        background #000000
        statusline #d0d0d0

        focused_workspace  #f0f0f0 #285577
        active_workspace   #f0f0f0 #333333
        inactive_workspace #888888 #222222
        urgent_workspace   #f0f0f0 #900000
     }
    }

Цвет дефолтного i3status в зависимости от статуса информации можно определять в конфиге i3status:
~/.i3status.conf
————————————————
general {
	output_format = i3bar
        colors = true
	color_good= "#6bbe1a"
	color_degraded = "#c2c2c2"
	color_bad = "#b6212d"
#	color_separator = "#f0f0f0"
	interval = 1
}
Цвет разделителя можно менять при условии, что вы используете формат вывода отличный от i3bar или none.
Я не использую стандартный i3status, поэтому с выводимой инфой и ее цветом что хочу, то и ворочу. )
