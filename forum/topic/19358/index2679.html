Что копируешь? Откуда копируешь?
git clone https://github.com/ruped24/toriptables2.git
Клонирование в «toriptables2»…
cd ~/toriptables2
sudo cp -var toriptables2.py /usr/local/bin/
'toriptables2.py' -> '/usr/local/bin/toriptables2.py'
... и так далее ...
