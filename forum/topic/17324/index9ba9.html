Вы по таймеру проверяете количество обновлений?
По таймеру лишь выполняется pacman -Sy. В .zshrc прописано:

screenfetch
up=$(pacman -Qu) && [[ $(echo $up | wc -l) -gt 0 ]] && echo "\e[1;36m
Updates available:\e[1;37m
"$up
Если убрать таймер и прописать pacman -Sy в .zshrc, каждый раз сеанс будет запускаться очень долго
