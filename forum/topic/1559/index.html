заблокирован
 Шифровать пользовательские папки это проще простого.
Нужно загрузится в уровень 3, и под рутом делать:
Устанавливаем пакет ecryptfs-utils
Для шифрования загружаем модуль
modprobe ecryptfs
выполняем
ecryptfs-migrate-home -u yourname
и дальше действуем по инструкции которая будет выведена на экран, она на английском, там нужно будет ввести пароль того кого шифруем, и после шифрования не перезагружаясь войти тем пользователем которого шифровали в свободной консоле.
Если все прошло успешно, то можно удалить сохраняловку созданную скриптом…

Что бы расшифровывалась автоматом папка при входе пользователя
Нужно добавить в
/etc/pam.d/login
и
/etc/pam.d/kde
так же в другие файлы при необходимости

строки

auth        optional    pam_ecryptfs.so unwrap
password    optional    pam_ecryptfs.so
session     optional    pam_ecryptfs.so unwrap
