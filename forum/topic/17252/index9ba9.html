Поднял свои старые записи (из прошлой молодой жизни ...) ...... если в двух словах ...
protocol RTPROTO — протокол/идентификатор протокола маршрутизации …
Некоторые возможные значения — kernel, boot, static … смотри файл /etc/iproute2/rt_protos
static - маршрут устанавливается администратором для перезаписи динамической маршрутизации.
PS ... о статической маршрутизации нашел статью на русском без всяких заумностей (в 2-х словах)…
metric — метрика маршрута - отражает количество переходов, скорость прохождения пути, надежность пути и др.... вообщем имеет 5 параметров: bandwidth, delay, reliability, load, MTU
Расчитывается по сложной арифметической формуле, но упрощенно можно считать по формуле
metric = bandwidth + delay
