1.

xrandr --listproviders
Providers: number : 2
Provider 0: id: 0x88 cap: 0xb, Source Output, Sink Output, Sink Offload crtcs: 4 outputs: 5 associated providers: 0 name:Intel
Provider 1: id: 0x5f cap: 0x5, Source Output, Source Offload crtcs: 0 outputs: 0 associated providers: 0 name:nouveau
2. .bash_profile

export DRI_PRIME=1
3. .xinitrc

#set nvidia prime
xrandr --setprovideroffloadsink 0x5f 0x88
xcompmgr -n &

Дело в том, что наличие дискретной Nvidia еще не означает наличие Optimus.
Нужно проверить спецификацию лептопа.
Например, если взять Lenovo - не у всех, где есть 2 карты, есть Optimus.
Выше дали правильный совет : снесите всю графику до голой консоли.
Потом попробуйте завести автоматом на Intel.
После этого работайте с Nvidia.
Еще можете показать lspci.
У меня так

...
VGA compatible controller: Intel Corporation 4th Gen Core Processor Integrated Graphics Controller (rev 06)
...
3D controller: NVIDIA Corporation GK208M [GeForce GT 740M] (rev a1)
