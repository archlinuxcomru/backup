$HOME/.mozilla/firefox/profile_name/chrome/userChrome.css
вместо profile_name будет буквенно-цифровой код, уникальный для каждого профиля
типа pmt8zuv5.default
раньше каталог chrome присутствовал по дефолту и содержал два файла-образца - userChrome.example.css для морды и userContent.example.css для контента
сейчас надо создавать самому
в файле userChrome.css обязательно должен быть заголовок
@namespace url(http://www.mozilla.org/keymaster/gatekeeper/there.is.only.xul);
задающий пространство имён, для которого будут применяться прописанные в стиле правила
в данном случае there.is.only.xul - только для морды
