Haron_Prime, вобщем, сам спросил - сам отвечаю:
Лиса нынче читает настройки gtk-3.0/далее_по_списку.., в моём случае нужён файл scrollbar.css (не факт, что одноимённый есть в любой gtk-3.0-теме, точнее говоря, описание скроллбара непременно должно быть, но может быть включено в какой-нибудь другой, напр. в gtk-widget.css - если в файле gtk.css нет строки @import url("scrollbar.css"); - значит скроллбар надо искать в другом месте.)
Короче. Я сделал так (код только изменённых параметров, остальное не имеет отношения к вопросу):
~/.theme/$THEME/gtk.3-0/scrollbar.css
.scrollbar {
    -GtkRange-slider-width: 7;
    -GtkRange-trough-border: 1;
}

.scrollbar.slider,
.scrollbar.button {
        border-width: 0px;
	border-style: none;
        border-radius: 10px;
	background-color:  alpha (#000, 0.75); # цвет собственно слайдера, можно задавать, используя прозрачность, можно и напрямую: #XXXXXX;
}

.scrollbar.slider:prelight {
	background-color:   alpha (#000, 0.75); # цвет при наведении
}

PS. Я как-то упустил тот факт, что лиса теперь на gtk-3.0. С gtk-2.0 вопрос-то давно был решён, а тут на тебе...
