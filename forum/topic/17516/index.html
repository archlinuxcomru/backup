vacka
т.е. примерно как то так:
wget -r -k -l 7 -p -E -nc адрес сайта
Вроде бы все нормально - здесь важен параметр nc, который избавляет заново записывать загруженные файлы, если закачка была прервана, точнее, привожу уточняющее пояснение
-nc, — определяет поведение wget при загрузке, когда в каталоге назначения имеются ранее загруженные файлы из источника.
- когда wget запускается без ‘-N’, ‘-nc’, ‘-r’, или без ‘-p’, то при повторной загрузке файла FILE, будет загружена вторая копия с именем FILE.1 и т.д. Когда параметр ‘-nc’ определён, wget отклонит повторную загрузку файла FILE.
- когда wget запускается с параметрами ‘-r’ или ‘-p’, но без ‘-N’, ‘-nd’, или без ‘-nc’ то wget просто перезаписывает старый файл на новый. Добавление опции ‘-nc’ игнорирует копирование новых версий файлов (старые файлы не  перезаписываются).
PS ... насчет -l 7 - лучше перед закачкой проверить, может и без этой опции (по умолчанию 5), а может будет и больше
Ошибки не исчезают с опытом - они просто умнеют
