Для начала глянуть, что там в systemsettings на эту тему, у меня так:

Затем можно проверить в терминале выполнение команды (выключение без подтверждения):
qdbus org.kde.ksmserver /KSMServer org.kde.KSMServerInterface.logout 0 2 0
и с подтверждением:
qdbus org.kde.ksmserver /KSMServer org.kde.KSMServerInterface.logout 1 2 0
Не помню, нужно-ли это в DE, но проверить наличие юзера в группе power:
grep USER_NAME /etc/group|cut -f1 -d:
Пока всё.
