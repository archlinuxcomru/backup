Если роутер есть: можешь использовать бридж. В таком случае гостевая система будет доступна по сети, как любое сетевое устройство входящее в локальную сеть роутера (нужно только выяснить какой адрес гостевой системе выделил роутер).
Если роутера нет: лучше использовать "виртуальный адаптер хоста" (предварительно его необходимо создать: файл -> настройки -> сеть, вкладка виртуальный адаптер). По умолчанию виртуальный адаптер поднимает на хосте интерфейс 192.168.56.1, а на госте необходимо дать адрес 192.168.56.х.
