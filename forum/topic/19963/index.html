как видно, проблема туть:
error: Media directory "V,/home/warning/Videos" not accessible [Permission denied]

Выхлоп из /etc/systemd/system/minidlna.service.d/override.conf :

[Service]
ProtectHome=off

-----------------------------------

Часть /etc/minidlna.conf :
# specify the user account name or uid to run as
user=root

(Пробовал и warning , и minidlna)

media_dir=V,/home/warning/Videos

1. верни запуск сервиса от пользователя minidlna в /etc/minidlna.conf
2. добавь пользователя minidlna в группу которой принадлежит /home/warning/Videos (usermod -aG группа minidlna)
3. дай (проверь) права на чтение Videos и вложенного группе (chown -R )
