Все что писал ранее про резервирование - не читать и не осмысливать (запарился слегка).

В fstab замените пока:
none /dev/pts devpts defaults 0 0
none /dev/shm tmpfs defaults 0 0
на
devpts                  /dev/pts      devpts    defaults            0      0
shm                     /dev/shm      tmpfs     nodev,nosuid        0      0
Делу может и не поможет, но будет более правильно. И поднимите строчку монтирования корня выше точек /media/* (тоже на всякий случай).

Ядро самосборное или из репозиториев?
