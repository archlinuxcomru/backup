дополнения которые пользую -- https://addons.mozilla.org/ru/firefox/collections/5380554/linux/

так-же юзою такой стиль, который:
  1. перемещает адресную строку под вкладки (мне так интуитивно привычнее)
  2. Цвет надписи активной вкладки оранжевый + жирный (иначе активную вкладку трудно сразу найти глазами)
  3. убирает кнопки вперёд\назад когда они не актуальны
.mozilla/firefox/имя_профиля/chrome/userChrome.css
/*вкладки внизу*/
#nav-bar {
  -moz-box-ordinal-group: 1 !important;
  border-top-width: 0px !important;
}
#titlebar {
  -moz-box-ordinal-group: 2 !important;
}

/*цвета вкладок*/
.tabbrowser-tab[selected="true"] {
  font-weight: bold !important;
  color: #FF8800 !important;
}

/* Удаление кнопки назад  когда она недоступна */
#back-button[disabled="true"] { display: none; }
/* Удаление кнопки вперёд когда она не доступна */
#forward-button[disabled="true"] { display: none; }

browser.cache.disk.enable false
browser.cache.memory.enable true
browser.cache.memory.capacity -1
Можно сказать и B … (зачем нужно тогда хранить на диске?)
Запрет использования дискового кэша, если используется шифрованное SSL-соединение: browser.cache.disk_cache_ssl=false
Запрет создания кэша для просмотра страниц в офлайне: browser.cache.offline.enable=false
Запрет автоматического управления дисковым кэшем (если он включен): browser.cache.disk.smart_size.enabled=false
но если решились на установление browser.cache.disk.smart_size.enabled=false, то можно сказать и C - вместе с этим устанавливают и обнуление (запрет) объема дискового кэша, хранящегося на винчестере (в том числе - и офлайн-кэша):
 browser.cache.disk.smart_size.enabled=false
browser.cache.disk.capacity=0
browser.cache.offline.capacity=0

В части кэширования в RAM - есть несколько нюансов
1. Кэширование данных в RAM (browser.cache.memory.enable=true) не будет работать, если кэширование запрещено глобально (network.http.use-cache=false)
2. Есть user-ы, которые боятся использовать кэширование в RAM, так как есть предупреждение
Разрешение кэширования в оперативную память может привести к отслеживанию браузера по "цифровым отпечаткам", создаваемым при помощи E-Tag!
Ошибки не исчезают с опытом - они просто умнеют
