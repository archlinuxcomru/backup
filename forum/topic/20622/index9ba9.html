 Индикация выгруженных (спящих) вкладок
раскомментируйте любое из этих (как второе) и добавьте себе в userChrome.css

/* Прозрачность выгруженных вкладок
  .tabbrowser-tab[pending] {
  opacity: 0.3; !important;
} */

/* Серый цвет значков выгруженных вкладок */
  .tabbrowser-tab[pending] {
 filter: grayscale(100%) contrast(10%) !important;
}

/* Цвет выгруженных вкладок
  .tabbrowser-tab[pending] {
 background: darkgrey !important;
} */

/* Цвет шрифта выгруженных вкладок
  .tabbrowser-tab[pending] {
 color: darkgrey !important;
} */

/* Шрифт выгруженных вкладок
  .tabbrowser-tab[pending] {
  font-style: italic !important;
} */

ClearURLs

Это расширение автоматически удаляет элементы отслеживания из URL-адресов, чтобы защитить вашу конфиденциальность при просмотре страниц в Интернете.
Если вы ищете продукт на Amazon, вы увидите очень длинный URL, например:
https://www.amazon.com/dp/exampleProduct/ref=sxin_0_pb%3F__mk_de_DE=%C3%85M%C3%85%C5%BD%C3%95%C3%91&keywords=tea&pd_rd_i=exampleProduct&pd_rd_r=8d39e4cd-1e4f-43db-b6e7-72e969a84aa5&pd_rd_w=1pcKM&pd_rd_wg=hYrNl&pf_rd_p=50bbfd25-5ef7-41a2-68d6-74d854b30e30&pf_rd_r=0GMWD0YYKA7XFGX55ADP&qid=1517757263&rnid=2914120011

В действительности, большая часть указанного выше URL-адреса является кодом отслеживания. После того, как ClearURLs очистит адрес, он будет выглядеть следующим образом:
https://www.amazon.com/dp/exampleProduct

Все браузеры имеют встроенные функции и поисковые плагины, которые отслеживают Ваши предпочтения, местоположение, активность и многое другое … и часть информации периодически отсылается на сервер и храниться там определенное время.
Как один из примеров - в части отслеживание все браузеры используют так называемые supercookie (неудаляемые cookie), которые являются источником утечек и слежения за пользователем ... и просто так их не отключить (раньше было можно, сейчас затруднено, хотя почистить можно)
