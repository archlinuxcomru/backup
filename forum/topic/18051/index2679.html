[РЕШЕНО] Bash скрипт для обновления IP адреса у 6in4 брокера NetAssist

Всем привет.
Недавно открыл для себя 6in4. Настроил NetAssist на роутере (HE, tunnelbroker с удобным URL для обновления IP за NAT не работает).
Осталось для удобства "объединить" две команды в удобный скрипт.
Есть следующие команды для обновления IP:
$ wget -4 -qO- icanhazip.com
$ wget -qO- "http://tb.netassist.ua/autochangeip.php?l=login&p=pass&ip=айпишник")
Запрос к icanhazip возвращает внешний ip адрес за натом, этот ответ (айпишник) каким-то образом нужно вставить в запрос к netassist.
Есть добрые люди, которые помогут со скриптом или накидают ссылок на маны, как написать такой скрипт? Был бы очень признателен.
Dmitriy_Kniazev

Попробуй так(var-переменная):
var=$(wget -4 -qO- icanhazip.com);wget -qO- "http://tb.netassist.ua/autochangeip.php?l=login&p=pass&ip=""$var"
