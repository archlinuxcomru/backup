Добрый день! Достаточно популярная тема. И я знаю про UPnP и NAT, но что-то мне они не помогают. Торренты просто не раздаются, у меня карма падает на трекерах. На самих форумах люди недостаточно компитентные сидят, решил тут спросить.

Проблему видно из заголовка темы. Еще есть нюанс: в Ktorrent внизу есть кнопка UPnP, при её нажатии открывается список устройств. В этом списке у меня одно устройство “WANConnectionDevice” рядом с которым стоит значение Internal Server Error. KTorrent использует 6881 порт, еще есть порт трекера (8881), но я не понимаю что это. Какого трекера? Я же могу использовать не один трекер. В проки стоит “Использовать KDE как прокси”. Максимальное количество загрузок - 100. Загруженные данные хранятся на ext4-разделах.

Параметры сети:
Router D-Link DIR-300NRU

Роутер с провайдером соединяется по PPPoE
Параметры PPPoE:
http://upwap.ru/2883233
UPnP: Включено

Параметры локальной сети
Параметы LAN:
http://upwap.ru/2883236

Есть параметры vlan, но поменяв что либо там, потом просто не могу подключиться к своему роутеру ни по Ethernet ни по Wi-Fi. Прилагаю скриншот на всякий случай http://upwap.ru/2883269
DDNS, Маршрутизация и виртуальные серверы никак не настроены.
IGMP выключен

Параметры Wi-Fi:
MBSSID: выключено
Беспроводной режим: 802.11 b/g mixed
WPS: вылючено
WDS: вылючено

Параметры ПО:
Linux 3.6.2-1
KDE 4.9.2
KTorrent 4.3.0

Заранее спасибо за помощь!
