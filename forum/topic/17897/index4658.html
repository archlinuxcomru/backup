Ничего странного тут нет. Speedtest по умолчанию измеряет скорость общения с ближайшим сервером провайдера. Если при таком тесте полученные цифры будут сильно отличаться от тех, которые указаны у вас в тарифном плане, то можно смело напрягать провайдера о невыполнении условий договора. Запустите Speedtest с каким-нибудь удалённым от вашего места проживания сервером, например Хабаровск, или вообще забугорьем, цифры у вас будут совсем другие.

Пример:

1) До ближайшего сервера провайдера:

Retrieving speedtest.net configuration...
Testing from Rostelecom (хх.ххх.хх.хх)...
Retrieving speedtest.net server list...
Selecting best server based on ping...
Hosted by Rostelecom (Rostov-on-Don) [1.10 km]: 10.648 ms
Testing download speed................................................................................
Download: 22.25 Mbit/s
Testing upload speed....................................................................................................
Upload: 17.11 Mbit/s

2) Произвольный сервер. Выбрал Новую Зеландию, сервер 4118:

Retrieving speedtest.net configuration...
Testing from Rostelecom (хх.ххх.хх.ххх)...
Retrieving speedtest.net server list...
Selecting best server based on ping...
Hosted by Spark (Wellington) [16427.39 km]: 357.345 ms
Testing download speed................................................................................
Download: 12.51 Mbit/s
Testing upload speed....................................................................................................
Upload: 7.86 Mbit/s
