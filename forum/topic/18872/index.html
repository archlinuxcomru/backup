Приветствую всех!
Испытываю несколько проблем с работой графической среды Cinnamon. Перечислю перечень неисправностей:
1. Нет возможности сменить картинку рабочего стола. При попытки войти в контекстное меню - ничего не происходит.
2. Большая часть настроек через cinnamon-settings вываливается в ошибку сегментирования. Например, при попытке попасть Параметры системы -> Оборудование -> Дисплей выкидывает и выводит в консоль:
$ cinnamon-settings
True
Using pam module (python3-pampy)

(cinnamon-settings.py:1710): Gdk-WARNING **: 16:04:15.769:
gdkwindow-x11.c:5576 drawable is not a native X11 window
Ошибка сегментирования (стек памяти сброшен на диск)
Решается часть проблем через запуск:
$ GDK_BACKEND=x11 cinnamon-settings

3. Не могу добавить русскую раскладку, хотя локаль и региональные настройки настроены правильно.

В какую сторону смотреть? Система 2018.11.01 из этого релиза. Пакеты обновлены.
