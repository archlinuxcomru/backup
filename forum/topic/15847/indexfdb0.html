Была подобная проблема на manjaro после установки, выличилось заменой внутренностей /etc/pacman.d/mirrorlist
в mirrorlist было 3 зеркала, удалил сгенерил новые https://www.archlinux.org/mirrorlist/ и все разаботало.

## Russia
Server = http://mirror.aur.rocks/$repo/os/$arch
Server = https://mirror.aur.rocks/$repo/os/$arch
Server = http://mirror.rol.ru/archlinux/$repo/os/$arch
Server = https://mirror.rol.ru/archlinux/$repo/os/$arch
Server = http://mirror.yandex.ru/archlinux/$repo/os/$arch
Server = https://mirror.yandex.ru/archlinux/$repo/os/$arch
