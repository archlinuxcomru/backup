Добрый вечер. Имеется проблема на ноутбуке и на стике с полным зависанием системы из-за видеокарты Intel HD Graphics. Оба имеют процессор Bay Trail, на ArchWiki нарыл информацию:

If you are using kernel 4.0.X or above on Baytrail architecture and frequently encounter complete system freezes (especially when watching video or using GFX intensivelly), you should try adding the following kernel option as a workaround, until this bug will be fixed permanently.

intel_idle.max_cstate=1

Только я не понял куда это надо прописать, нашел только информацию для grub, но я использую EFISTUB. Помогите решить эту проблему.
