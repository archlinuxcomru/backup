Вроде запустил, чтобы аналоговый выход выводил через pulseaudio. ( За счет конфига ~/.asoundrc )
~/.asoundrc
pcm.pulse { type pulse }
ctl.pulse { type pulse }
pcm.!default { type pulse }
ctl.!default { type pulse }

Сейчас нужно, чтобы HDMI(т.е цифровой) тоже выводился через Pulseaudio.
