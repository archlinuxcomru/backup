Есть две видеокарточки
1) NVIDIA GeForce 7900GS 256mb
2) NVIDIA GeForce 9500GT 1024GB

Судя из тестов, которые я находил по производительности они одинаковые.

NVIDIA GeForce 7900GS 256mb
+не греется, достаточно холодная
-требует ставить переходник с molex для того, чтобы запитать её
-переходник на гнездо, который неудобно торчит, чтобы я смог поключить монитор

NVIDIA GeForce 9500GT 1024GB
+объем памяти 1Гб
-горячее, чем 7900GS (от 60С и выше при нагрузках)
Какую из них оставить себе?
