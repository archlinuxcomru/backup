AZJIO первая и основная ваша ошибка как и многих новичков что вы не используете вики, а пытаетесь найти простой рецепт в гугле.
В арче одна из лучших вики (используйте английскую версию, локализованные могут запаздывать и устаревать)
Начните с https://wiki.archlinux.org/index.php/Installation_guide
https://wiki.archlinux.org/index.php/General_recommendations

AZJIO
mount /dev/sda7 /mnt

for i in dev sys proc etc/resolv.conf; do sudo mount –bind -v /$i /mnt/$i; done
chroot /mnt

pacman -Ss | grep xf86-video
pacman -S cinnamon

# Добавить пользователя и установить пароль для него
useradd -m user
passwd user
useradd -m root
passwd root

# Создать fstab

pacman -S lightdm
pacman -S nemo

Вот тут уже не по вики потому и ошибки
pacstrap /mnt base base-devel linux linux-firmware
желательно добавить еще консольный редактор для настройки и редактирования ( например nano ) и если интернет через вайфай средство его настройки см. https://wiki.archlinux.org/index.php/Network_configuration/Wireless#Utilities
for i in dev sys proc etc/resolv.conf; do sudo mount –bind -v /$i /mnt/$i; done
chroot /mnt
используйте arch-chroot см. https://wiki.archlinux.org/index.php/Installation_guide#Chroot
pacman -Ss | grep xf86-video
pacman -S cinnamon
pacman -S lightdm
pacman -S nemo

поиск пакета в репозиториях pacman -Ss xf86-video но не факт что этот драйвер вам нужен.
Перед установкой графических программ и графического дисплейного менеджера крайне желательно настроить и проверить графическую систему лучше на каком нибудь простом WM например openbox
https://wiki.archlinux.org/index.php/General_recommendations#Graphical_user_interface
https://wiki.archlinux.org/index.php/Xorg
https://wiki.archlinux.org/index.php/Xinit#xinitrc
https://wiki.archlinux.org/index.php/Openbox
