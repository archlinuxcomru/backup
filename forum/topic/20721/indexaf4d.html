Инструкция установки Arch с помощью Anarchy

1. Сделать образ на флешке, иначе после монтирования образа диск sda окажется недоступен для установки, т.е. не будет предлагаться как место назначение. Загрузку образа проще сделать в Grub2 со следующим конфигом:

menuentry "Anarchy-Z"{
	set isofile="/z/anarchy-1.3.2-x86_64.iso"
	search -f  $isofile -s
	probe -s root_uuid -u $root
	loopback loop $isofile
	linux (loop)/anarchy/boot/x86_64/vmlinuz-linux archisobasedir=anarchy img_dev=/dev/disk/by-uuid/$root_uuid img_loop=$isofile earlymodules=loop
	initrd (loop)/anarchy/boot/intel-ucode.img (loop)/anarchy/boot/amd-ucode.img (loop)/anarchy/boot/x86_64/initramfs-linux.img
}

Образ нужно положить в папку "z" на флешке. Образ не поддерживает файловую систему NTFS, т.е. флешка либо в FAT32 либо Ext4.

Так как в Anarchy есть понятный установщик, который устанавливает Arch, т.е. по сути Arch со скриптом установки, то воспользуемся им.

Во время установки появится меню, в котором можно управлять стрелками клавиатуры вправо, влево, вверх, вниз и Enter, можно Esс для отмены выбора. Установка начнётся только после ответа на все вопросы установщика, т.е. если нет сети вы узнаете об этом в самом конце.

0. Появляется меню выбора языка установщика, выбираем Russian, чтобы всё (почти всё) было на русском.
1. Раскладка - выбираем us (если выбрать ru можно столкнуться с проблемой входа, когда невозможно ввести пользователя или пароль)
2. Три пункта, где выбираем "Старт установки" - "Start the installation".
3. Предлагается настройка сети, жмём "Выйти" (у меня локальная сеть, подхватываемая автоматически). Если нет сети, то ничего не будет установлено, так как все установочные пакеты скачиваются с интернета, локально из iso ничего не используется.
4. Предложено загрузить зеркала, сделать это, чтобы список актуальных на данный момент. Выбрать "Download Rank New Mirror", отработает reflector --latest --sort rate.
5. Предложено выбрать регион (язык системы), выбрать ru-UTF-8
6. Предложено выбрать часовой пояс Asia - Yekaterinburg (тщетно искать Eka...)
7. Выбираем "Ручная разметка" (Manual ...), так как у вас скорее всего не пустой диск и предлагать автоматический вариант опасно, не зная алгоритма работы этой опции.
8. Выбираем раздел для установки, например sda4. Лучше всего приготовить его заранее с помощью менеджера разделов gparted (выделить место, отформатировать в Ext4)
9. Создать корневой каталог. Ответ "Да". Ext4 - "Ок". Спуститься на sda4, в центре кнопка "Записать изменения". По сути указать, что монтировать выбранный раздел (в нашем примере sda4) как корневой "/". Можно пока не создавать Swap-раздел или создать заранее, но не указывать, достаточно будет указать его после в "etc/fstab". Кнопка "Записать изменения".
10. Выбрать Anarchy-Desktop, и там выбрать Cinnamon, в худшем случае XFCE. Кто понимает выберет свой десктоп. Если выбрать последний пункт с текстом "advanced", то там будет много настроек последовательно, в которых можно выбирать варианты по умолчанию: выбор ядра, базовая установка, bach, grub2/суслик, сеть, мультилиб для x32, DHCP, wi-fi, os-prober, оконный менеджер, драйвер, Synaptics, менеджеры входа lightdm.
11. Host - без изменений
12. Пароль root (ввести и потом повторить)
13. Новый пользователь, добавить user (или другое), ввести ещё раз., указать пароль для работы юзера с повышенными привилегиями, повторить пароль.
14. Включить привилегии, ответ "Да". Готово.
15. Установить ПО, ответ "Да". Выбирать ПО - ставить галочки клавишей пробелом.
16. Установить всё. Это когда уже перестаём выбирать и переходим к следующему шагу.
17. У меня размер к установке был 1058 Мб (в качестве ПО были выбраны firefox, geany и вроде gimp, "libre office fresh").
18. Перед установкой ещё раз проверить интернет, лампочки на маршрутизаторе горят/мигают, сетевой кабель два светодиода горит/мигает, если не хочется проходить этот квест дважды и более.
19 После загрузки и установки настройка ядра.

При установке мне показалось, что Cinnamon не правильно установился и я выбрал XFCE, чтобы из него установить Cinnamon в терминале, но, ничего не изменилось, так как Cinnamon обновлённый и аплеты видимо заблокированы от непротестированных настроек.
