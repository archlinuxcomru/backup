Разобрался. При установке делал так:
pvcreate /dev/sda2 # Создание физических томов(PV)
vgcreate arch /dev/sda2 # Создание группы логических томов
lvcreate -l100%FREE root -n arch # Создание логического тома root

После этого я создал файловую систему:
mkfs.ext4 /dev/arch/root # Почему-то это неправильно, после установки приходится править /boot/grub/grub.cfg

Вот правильный способ:
mkfs.ext4 /dev/mapper/arch-root # Тогда все работает по умолчанию и ошибок не пишет

Монтирую:
mount /dev/mapper/arch-root /mnt # после этого устанавливаю систему. Насколько я понял запись /dev/mapper/arch-root равнозначна /dev/arch/root. Почему нужно делать "vgcreate arch /dev/sda2" вместо "vgcreate mapper /dev/sda2" я так и не понял.
