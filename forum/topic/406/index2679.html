Я опять того. В смысле в гугле забанили - из вики выкинули - азбуку отобрали. MPD не могу установить. То есть:
>> mpd
failed to open log file "/home/***/.mpd/mpd.log" (config line 5): Permission denied

Соответственно:
>> 0.failed 
UNIT        LOAD   ACTIVE SUB    JOB DESCRIPTION
mpd.service loaded failed failed     Music Player Daemon

Как бы:
>> cat /etc/mpd.conf
port                    "6600"
music_directory         "/media/wxp/cdd/music"
playlist_directory      "/home/***/.mpd/playlists"
db_file                 "/home/***/.mpd/mpd.db"
log_file                "/home/***/.mpd/mpd.log"
auto_update             "yes"
bind_to_address         "localhost"
audio_output {
         type                    "alsa"
         name                    "My ALSA Device"
         device                  "hw:0,0"     # optional
         format                  "44100:16:2" # optional
}
bind_to_address  "127.0.0.1"
bind_to_address  "/home/***/.mpd/socket"

И эта:
>> sudo id mpd
uid=45(mpd) gid=45(mpd) groups=45(mpd),92(audio),100(users)
>> sudo cat /etc/group
...
audio:x:92:***,mpd
...

Ну и:
>> getfacl /home/***/.mpd
getfacl: Removing leading '/' from absolute path names
# file: home/***/.mpd
# owner: mpd
# group: mpd
user::rwx
group::r-x
other::r-x

Как-то так. Для осеннего поздновато…
