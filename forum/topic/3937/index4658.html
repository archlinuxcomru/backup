Попробуйте загрузиться с live диска, сделать mkinitcpio, указав корнем / на собранном массиве, который перед этим нужно собрать. Я уже точно не помню как там, посмотрите вывод -h для mkinitcpio. Там что-то еще надо указать, а ну да пресет, это теперь linux, указать путь у нему. Сейчас нет времени вспоминать, я уже спать отправился )))

Все таки откопал свой старый багрепорт, когда у меня отвалилось, там пишут:
Resolution:
We assume /dev/md0 is your root and you mdadm –assemble it on the livecd.
mount /dev/md0 /mnt/
mkinitcpio -b /mnt/ -p linux # use /mnt/ as root, build an image using the linux preset
Now your image has /sbin/mdadm.

Подробнее тут: https://bugs.archlinux.org/task/25566 Надеюсь, у вас получится. =)
