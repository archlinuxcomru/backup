Только что поставил i686 из х64.
pacstrap /mnt --arch i686 base
Только в pacman.conf надо закоментировать multilib.
Также при установке пакетов в chroot надо указывать архитектуру. Например
pacman -S --arch i686 grub
Всё. Больше установка ни чем не отличается. Работает.
