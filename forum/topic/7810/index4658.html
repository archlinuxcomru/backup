После доустановки всех компонентов PHP, все стало нормально работать, но только один сайт отпал. Он тербует установки Zend Optimizer. Раньше я решал это установкой его из AUR
aur/zend-optimizer 3.3.3-1 [installed] (2)
Сейчас это не прокатывает. Можете посмотреть, что пишет при входе на saikov.org.ru

Если у вас установлен php 5.3, то Zend Optimizer и не будет работать, поскольку еще не вышла соответствующая версия.
К слову сказать, допил в zend'e идет уже на протяжении нескольких месяцев - и конца этому не видно.

Если выполнить 
php -v
то увидим причину ошибки:
Zend Optimizer requires Zend Engine API version 220060519.
The Zend Engine API version 220090626 which is installed, is newer.
Contact Zend Technologies at http://www.zend.com/ for a later version of Zend Optimizer.
