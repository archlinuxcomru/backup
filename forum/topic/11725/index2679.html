mhd

Приветсвую всех. Недавно начал использовать i3wm. Обнаружил, что при медленной прокрутке в firefox содержимое браузера начинает отображаться плавным волнистым образом (рябь). Для i3wm в chronium такого не наблюдается - изображение чёткое при прокрутке. В gnome3 firefox отображет содержимое при прокрутке без волнистой ряби. Посоветуйте, пожалуйста, как устранить такое поведение firefox для i3wm.

Проверил поведение firefox с fluxbox и c Xmonad в сессии гнома - поведение такое же - рябь. Получается gnome3 лучше всех справляется с отображением в firefox. Может есть какая то настройка графики?

Сейчас проверил ещё на awesome - то же самое поведение firefox при прокрутке

Для начала попробуйте на чистом профиле.
Не поможет - отключите плавную прокрутку.
Не поможет - отключить аппаратное ускорение.

Прокрутку и ускорение найдете в Настройки-> Дополнительные(вкладка "Обшие").
