Добрый день.
Решил вот опробовать Arch Linux.
Установил по инструкции, поставил все нужные мне пакеты и программы.
Настроил доступ в интернет по Wi-Fi через службу Network Manager.
Всё работало хорошо и чёрт меня дёрнул сделать обновления pacman -Syu, после обновления всё работало, перезагрузил систему и пипец, после загрузки DE Mate, NetworkManager подключается по Wi-Fi, работает несколько секунд, потом разрывает соединение и через несколько секунд опять делает соединение и так по кругу.

Пока вышел из положения остановил сервис NetworkManager.service через systemctl и настроил Wi-Fi через wifi-menu.

Вопрос, это я что-то поломал или поломалось после обновления?
Ведь через netctl Wi-Fi работает отлично.
