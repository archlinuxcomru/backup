Lirc (вопрос)

bobart
# 11 лет, 11 месяцев назад
Темы: 38
Сообщения: 2537
Участник с: 28 ноября 2009
Привет. 2.6.35-ARCH & lirc 0.8.6-7, lirc-utils 0.8.6-4
Сразу скажу, лирк работает, пульт настроен, управление smplayer, tvtime (+вырубаю комп, если далеко до мыша) через irexec.
Но, не могу понять: если запускаю лирк как
# lircd --nodaemon --driver=dev/input --device=/dev/input/by-path/pci-0000:05:00.0-event-ir
lircd: error in configfile line 4
lircd: reading of file '/etc/lirc/lircd.conf' failed
lircd: reading of config file failed
lircd: lircd(devinput) ready, using /var/run/lirc/lircd
lircd: accepted new client on /var/run/lirc/lircd
lircd: initializing '/dev/input/by-path/pci-0000:05:00.0-event-ir'
то, как вы видите, получаю “error in configfile line 4”

Вот конфиг, о котором речь:
/etc/lirc/lircd.conf
#
# Parameters for lirc daemon
#
LIRC_DEVICE="/dev/input/by-path/pci-0000:05:00.0-event-ir"
LIRC_DRIVER="devinput"
LIRC_EXTRAOPTS=""
LIRC_CONFIGFILE="/etc/lircd.conf"
Причём, до этого там была ещё одна строка, я просто её убрал для проверки: PIDFILE=“/var/run/lircd.pid” (перед LIRC_DEVICE) - что соответствует действительности. Но в том случае, запуск lircd –nodaemon … так-же выдавал error in configfile line 4, указывая на PIDFILE=“/var/run/lircd.pid”. Короче, что не ставь, всё даёт эррор на 4-ую строку конфига.
При этом, запуск лирка обычным манером, # /etc/rc.d/lircd start проходит нормально, вот куски логов, в подтверждение:
1. тот случай с lircd –nodaemon …
Sep 18 17:06:47 arch lircd: error in configfile line 4
Sep 18 17:06:47 arch lircd: reading of file '/etc/lirc/lircd.conf' failed
Sep 18 17:06:47 arch lircd: reading of config file failed
Sep 18 17:06:47 arch lircd: lircd(devinput) ready, using /var/run/lirc/lircd
Sep 18 17:06:51 arch lircd: accepted new client on /var/run/lirc/lircd
Sep 18 17:06:51 arch lircd: initializing '/dev/input/by-path/pci-0000:05:00.0-event-ir'

2. # /etc/rc.d/lircd start
Sep 18 17:11:49 arch lircd: lircd(devinput) ready, using /var/run/lirc/lircd
Sep 18 17:11:57 arch lircd: accepted new client on /var/run/lirc/lircd
Sep 18 17:11:57 arch lircd: initializing '/dev/input/by-path/pci-0000:05:00.0-event-ir'
Поскольку пульт настроен, работает, то эти фокусы с ошибками не критичны, но всё-же, интересно. Что бы всё это значило?
