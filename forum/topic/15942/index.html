 Новая сборка Windows 10 уже доступна участникам программы тестирования ОС Windows Insider.

Компания Microsoft выпустила новую сборку Windows 10 Insider Preview Build 14316 с поддержкой Linux, которая в настоящее время уже доступна участникам программы тестирования ОС Windows Insider, о чем отмечено на сайте производителя.

Сообщается, что по умолчанию в сборке Linux Ubuntu нет. Чтобы воспользоваться этой функцией, нужно включить в секции "For developers" ("Для разработчиков") раздела "Update & security" ("Обновление и безопасность") системных настроек Windows режим разработчика.

После этого уже в меню "Windows Features" ("Возможности Windows") пользователь должен выбрать пункт "Windows Subsystem for Linux (Beta)" и поставить напротив него галочку.

Сообщается, что инженерами Microsoft разработана целая внутренняя инфраструктура Windows Subsystem for Linux (WSL) для поддержки Ubuntu, которая позволит запустить на ядре Linux образ ОС.

Чтобы получить доступ к функциям Ubuntu, нужно воспользоваться Bash - оболочкой для ввода команд. При первом запуске нужно будет установить Ubuntu.

Известно также, что в новой сборке Windows предусмотрены еще несколько новый функций, среди которых - Cortana - голосовой помощник, предупреждающий, если заряд ноутбука, планшета или смартфона слишком низкий.

Полезным нововведением станет также функция поиска мобильных устройств и подача на них звукового сигнала. При этом на экране компьютера или смартфона будет отображена навигация.

Работа приложений организована без использования контейнеров, виртуализации, отдельной пересборки утилит и даже без использования ядра Linux - запуск родных исполняемых файлов Linux реализован путём включения специальной прослойки, на лету транслирующей системные вызовы Linux в системные вызовы Windows.
