Написал же, если нужен просто запуск и нужен именно NetworkManager, то используй его консольную утилиту nmcli, с помощью которой можно поднять доступное соединение ручками, используя соответствующую команду. Сначала научись подключать ручками, а после можешь или повесить команду подключения на горячую клавишу или написать юнит, полностью автоматизировав процесс подключения - на свой выбор, что удобнее. Привожу пример
Смотрим доступные соединения
nmcli c
NAME                                    UUID                                  TYPE  DEVICE
Megafon RUS  f823f88f-f8e8-4419-9930-9ce4cb2de537  gsm          --
Смотрим статус соединения
nmcli d
DEVICE   TYPE      STATE           CONNECTION
ttyUSB2     gsm       отключено          --
Не подключено. Подключаем
nmcli c up "Megafon RUS"
Соединение успешно активировано (адрес действующего D-Bus: /org/freedesktop/NetworkManager/ActiveConnection/1)
Убеждамся (хотя это и лишнее)
nmcli d
DEVICE   TYPE      STATE           CONNECTION
ttyUSB2    gsm       подключено      Megafon RUS
Отключаем
nmcli c down "Megafon RUS"
Соединение «Megafon RUS» успешно отключено (активный адрес D-Bus: /org/freedesktop/NetworkManager/ActiveConnection/1)

UPD - если я тебя правильно понял и тебе нужно именно это.
