Comment by: fnord0 on Sun, 04 Dec 2011 13:03:14 +0000
I'm going to throw this out there: I had issues with building this package today... the _pacman_get_key() function was returning null (actually, it was returning there is nothing to do). I found that I just needed to download the file nvidia-utils-290.10-1-x86_64.pkg.tar.xz from a pacman mirror, this worked and I was on my way. figured I'd point this out in case someone else runs into a similar issue. also, I had to build the package manually via makepkg, as pacman (nor any of the pacman alternatives) were having it.
fix for me = 'wget ftp://ftp.archlinux.org/extra/os/x86_64/nvidia-utils-290.10-1-x86_64.pkg.tar.xz
then makepkg -si
