в lxpanel есть плагин для отображения раскладки клавы, но, какого-то черта, он не собирается автоматом, в PKGBUILD нужно добавить после make DESTDIR=${startdir}/pkg install, примерно следующее:
# Build XKB plugin
{
pwd
cd src/plugins/xkb
make || return 1
make DESTDIR=${startdir}/pkg install
cd ../../../
}

systemd должен умереть.
