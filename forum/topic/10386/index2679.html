Крупное обновление python переезжает из и скоро окажется на вашем любимом зеркале. Пакет под названием “python” теперь соответствует 3-й версии языка, тогда как 2-я версия поддерживается пакетом “python2”.
Так как /usr/bin/python теперь указывает на бинарник 3-й версии, то могут возникнуть проблемы для программ, требующих 2-ю версию. Для них необходима ссылка на /usr/bin/python2. Все пакеты из официальных репозиториев были соответствующим образом обновлены. Тем не менее, при обнаружении проблем без стеснения сообщайте на баг-треккер.
Пакеты из AUR, как и любый другие сторонние проекты, с большой вероятностью потребуют ручного вмешательства, так как многие программы до сих пор несовместимы с Python 3.
