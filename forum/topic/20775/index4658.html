отдельный настройщик
Я xatk использую, если не смущает второй питон можете попробовать. Работает в любом DE на иксах(не надо по новому хоткеи прописывать), простой текстовый конфиг
class.roxterm = !r = roxterm
class.opera = !o = opera

https://aur.archlinux.org/packages/xatk/
https://xatk.readthedocs.io/en/latest/
Описание первой версии и концепции, во второй добавился запуск по тому же хоткею

не вижу настроек Plank
Большинство программ/утилит в линуксе при запуске с параметром help выведут краткую справку
plank --help
Использование:
plank [ПАРАМЕТР…]

Параметры справки:
-h, –help Показать параметры справки
–help-all Показать все параметры справки
–help-gapplication Показать параметры GApplication
–help-gtk Показать параметры GTK+

Параметры приложения:
-d, –debug Enable debug logging
-v, –verbose Enable verbose logging
-n, –name The name of this dock. Defaults to "dock1".
–preferences Show preferences dialog of the just started or already running instance
-V, –version Show the application's version
–display=ДИСПЛЕЙ Используемый X-дисплей
Более подробную справку можно получить вызвав man для программы
man plank

Используйте их при затруднениях или непонятках
