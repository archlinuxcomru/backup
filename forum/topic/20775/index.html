чисто запуск и всё.
И зачем такое нужно?
Имхо часто запускаемые приложения/действия запускаются хоткеем, этот же хоткей служит для переключения на окно этого приложения или перебора окон если их несколько.
А редко используемые запускаются через лончер с поиском и автодополнением или через меню.
Панель скрываемая же служит и для запуска, взаимодействия с приложеними, индикации

а тупо набор иконок для запуска имхо только для ребенка, пожилых, людей с какими то ограниченными возможностями или для режима киоска/телепульта
