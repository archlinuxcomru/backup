Я эту проблему давно уже решил :)
systemd, в отличии от initscripts, запускает getty на консолях только когда вы туда переключаетесь, поэтому следующей свободной в initscripts оказывается tty7, а в systemd tty2 :)
Я просто чуть поправил вот этот скрипт, который я использую для запуска иксов:
/usr/local/bin/x 
#!/bin/bash
for((i=0;i<6;i++))
do
 P=''
 F=/tmp/.X$i-lock
 [ -f $F ] && read P < $F
 [ -n "$P" -a -d "/proc/$P" ] && continue
 exec startx -- :$i vt$((i+7))
done
echo "Can't start over 6 X-sessions." 1>&2
Теперь этот скрипт однозначно определяет консоль для запуска иксов по номеру “дисплея” плюс 7.
