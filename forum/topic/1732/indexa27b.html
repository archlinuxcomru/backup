а что, это же система инициализации, а не мир(!) … имхо
┌─[ Чт. окт. 25 ] :: [jinn] - [jinnlamp] :: [~]
└─>> systemctl -a | grep syslog
syslog-ng.service         loaded active   running       System Logger Daemon
syslog.socket             loaded active   running       Syslog Socket
syslog.target             loaded active   active        Syslog
┌─[ Чт. окт. 25 ] :: [jinn] - [jinnlamp] :: [~]
└─>> sudo tail -n 5 /var/log/auth.log
[sudo] password for jinn: 
Oct 25 13:03:10 localhost systemd-logind[342]: New session 1 of user jinn.
Oct 25 13:03:10 localhost login: LOGIN ON tty2 BY jinn
Oct 25 13:03:56 localhost sudo:     jinn : TTY=tty2 ; PWD=/home/jinn ; USER=root ; COMMAND=/usr/bin/systemctl stop matrix.service
Oct 25 13:03:56 localhost sudo: pam_unix(sudo:session): session opened for user root by jinn(uid=0)
