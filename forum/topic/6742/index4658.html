Спасибо.
Для добавления репозитория arch-games в конец конфига /etc/pacman.conf добавляем:
[arch-games]
# The Arch Linux Gaming repository project
Server = http://arch.twilightlair.net/games/i686
Server = http://pseudoform.org/arch-games/games/i686
style="color:#008000">В данном репозитории содержатся пакеты многих игр под линукс. Игры там реально на любой вкус.
