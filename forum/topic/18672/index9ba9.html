нечего описывать.
для копирования в рамках tmux используется его буфер.
для других (x приложений) используется clipboard, для копирования в него необходимо определить DISPLAY и добавить одну строчку в tmux
bind-key -T copy-mode-vi Enter send-keys -X copy-pipe-and-cancel "xclip -i -selection clipboard"
