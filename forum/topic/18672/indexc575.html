Вариантов несколько, например, используя xclip.
Но, насколько понял, нужно копировать команды прямо в консоль для запуска, а поэтому опишу общий способ без мышки
- в mcedit копируем нужную команду: F3, выделяем команду, F3, Ctrl+F, Enter
- не выходя из mcedit открываем новый файл, например, ~/comand.txt: Shift+F5 (вставка), сохраняем файл
- копируем следующую команду для вставки в ~/comand.txt (каждая новая команда с новой строки)
Вставка нужной команды в другом tty
- запоминаем номер строки нужной команды в ~/comand.txt (например, 2) и переходим в другую tty
- набираем команду (сразу для запуска)
sed -n '2p' ~/comand.txt | bash
Ошибки не исчезают с опытом - они просто умнеют
