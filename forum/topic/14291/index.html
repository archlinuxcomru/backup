Во первых, dmix в alsa сейчас включен по дефолту. Во-вторых, если стоит пульс, то и конфиг должен быть его:
pcm.!default {
  type pulse
  fallback "sysdefault"
  hint {
    show on
    description "Default ALSA Output (currently PulseAudio Sound Server)"
  }
}

ctl.!default {
  type pulse
  fallback "sysdefault"
}
или типа того.
Можно также в вику заглянуть.
