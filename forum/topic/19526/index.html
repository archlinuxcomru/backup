еще вопрос по i3 статус бару.. Как впилить туда отображение скорости WiFi на загрузку и выгрузку
Варианты есть разные:
1. добавить к выхлопу i3status внешний скрипт, показывающий трафик (примеры на оф. сайте i3status);
2. полностью генерировать строку состояния, т.е. обращаться напрямую к i3bar (на форуме я как-то описывал это);
3. использовать что-то левое типа i3blocks (самый простой вариант).

По первому пункту разобрался. Поставил пакет 'light' из реп , в i3.conf добавил :
bindsym XF86MonBrightnessUp exec light -A 5;
bindsym XF86MonBrightnessDown exec light -U 5;
