У меня так (давно уже)
cat /etc/systemd/system/statiknetwork.service 
[Unit]
Description=Network connectivity (eth0)
Wants=network.target
Before=network.target
BindsTo=sys-subsystem-net-devices-eth0.device
After=sys-subsystem-net-devices-eth0.device

[Service]
Type=oneshot
RemainAfterExit=yes
#EnvironmentFile=/etc/conf.d/[email protected]%i

ExecStart=/usr/bin/ip link set dev eth0 up
ExecStart=/usr/bin/ip addr add 192.168.0.101/24 brd + dev eth0
ExecStart=/usr/bin/ip route add default via 192.168.0.1

ExecStop=/usr/bin/ip addr flush dev eth0
ExecStop=/usr/bin/ip link set dev eth0 down

[Install]
WantedBy=multi-user.target

 vadik, systemd умеет уже и настраивать сеть по умолчанию.
вот пример
/etc/systemd/network/ethernet-static-enp3s0.network
[Match]
Name=enp3s0

[Network]
Description=systemd set enp3s0
Address=192.168.69.12/24
Gateway=192.168.69.1
DNS=192.168.253.3
FallbackDNS=8.8.8.8 8.8.4.4
но нужно включить еще сервисы входящие в сам systemd
systemctl enable systemd-networkd.service
systemctl enable systemd-resolved.service
http://www.freedesktop.org/software/systemd/man/systemd.network.html
