Решил установить систему с подтомами btrfs
До установки разбил диск sda1 на два подтома @ и @home
Установил систему, а затем прописал в fstab
UUID=3d4f1fb1-f1c6-4cea-a925-91548a801a60 / btrfs discard,noatime,nodiratime,compress=lzo,ssd,space_cache,subvol=@ 0 0
UUID=3d4f1fb1-f1c6-4cea-a925-91548a801a60 /home btrfs discard,noatime,nodiratime,compress=lzo,ssd,space_cache,subvol=@home 0 0

Но home не грузится.
Только закоментировав строчку #UUID=3d4f1fb1-f1c6-4cea-a925-91548a801a60 /home btrfs....
vожно загрузиться

Что я не так сделал?
Вроде бы подтома есть, указаны в fstab, почему не грузится то?

[root@archlinux l]# sudo btrfs sub list /
ID 264 gen 897 top level 5 path @home
ID 265 gen 20 top level 5 path @

[root@archlinux l]# sudo btrfs fi show /
Label: '1' uuid: 3d4f1fb1-f1c6-4cea-a925-91548a801a60
Total devices 1 FS bytes used 110.02GiB
devid 1 size 215.76GiB used 114.02GiB path /dev/sda1

[root@archlinux l]#
