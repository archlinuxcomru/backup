VIA VT1720/24, ребуты и отсутствие звука

SmiGes
# 10 лет, 2 месяца назад
Темы: 166
Сообщения: 836
Участник с: 04 августа 2009
вообщем с недавних пор эта дискретная звуковушка после ребута по всей видимости толи как-то неправильно подхватывается ядром, толи чего. Очередной ребут может помочь, но вот буквально только после второго ребута заработала. в dmesg есть о ней следующее:
ice1724: No matching model found for ID 0x12140324
ice1724: Invalid EEPROM version 1
причем сея запись есть в обоих случаях. под оффтопом девайс работает без нареканий.
