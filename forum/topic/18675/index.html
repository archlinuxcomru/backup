И ошибка, похоже, не связана с ключами. И, похоже, давно не обновлялся. Попробуй удали и yaourt и package-query и установи по-новой, ручками, например. так
git clone https://aur.archlinux.org/package-query.git
cd package-query
makepkg -si
cd ..
git clone https://aur.archlinux.org/yaourt.git
cd yaourt
makepkg -si
cd ..
И если будешь ставить ручками, исправь pacman.conf, удали строки, относящиеся к AUR

EDIT 1 - ты приводишь отрывочные логи, не пишешь, что делал и пр., а потому приходится гадать. И совет - гугли по ошибкам.
А если проблема с ключами, то одной команды pacman-key –init не достаточно, смотри Wiki, да и на форуме это не раз обсуждалось.
