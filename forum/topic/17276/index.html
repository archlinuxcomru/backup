Мне пришлось совсем извращаться - скачивать через Opera+VPN каждый из "зарезанных" файерволлом админа пакетов обновления ( благо - их оказалось немного ) и устанавливать их поочередно. Только так решил зависимости для обновления.

Я так понимаю, что мне для полного счастья нужно установить и настроить на своей машине VPN, чтобы вся система ( и менеджер пакетов - главным образом ) тоже "ходили" через VPN, как я - в Opera + VPN?

В настройках менеджера видел загадочную галочку - "Прокси". Это тоже из моей моего случая?

Подскажите, я прав? И дайте ссылок по настройке системы для моего случая, если я прав. )
