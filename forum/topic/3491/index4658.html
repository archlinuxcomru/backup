# ps aux | grep dbus
root       865  0.0  0.0   9468   972 pts/0    S+   12:07   0:00 grep dbus
Т.е. не запущен (видно только саму команду ps aux | grep dbus
Должно быть типа такого
dbus       866  0.0  0.0   3000   836 ?        Ss   Oct15   0:00 /usr/bin/dbus-daemon --system

Вставьте dbus в DAEMONS в файле /etc/rc.conf

PolicyKit.conf должен быть таким (в вики написано - between the <config> tag), но пока пробуйте без него
<?xml version="1.0" encoding="UTF-8"?> <!-- -*- XML -*- -->
<config version="0.1">
    <match action="org.freedesktop.hal.power-management.shutdown">
    <return result="yes"/>
    </match>
    <match action="org.freedesktop.hal.power-management.reboot">
     <return result="yes"/>
    </match>
    <match action="org.freedesktop.hal.power-management.suspend">
     <return result="yes"/>
    </match>
    <match action="org.freedesktop.hal.power-management.hibernate">
     <return result="yes"/>
    </match>
</config>
