Cоздал пользователя и группу 

su - root
groupadd -g 400 dba
useradd -u 400 -g 400 -d /home/oracle -s /bin/bash -c "Oracle Owner" oracle

затем поместил saidmir в эту группу 

sudo gpasswd -a saidmir dba

затем 

groups saidmir
audio optical dba users
groups oracle
dba
