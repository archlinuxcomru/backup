Тем, у кого форум отображается без стилей.

На некоторых серверах cloudflare наблюдаются проблемы с SSL-сертификатами. Поскольку стили форума загружаются по HTTPS с отдельного субдомена static.archlinux.org.ru, при добавлении исключения безопасности требуется добавить исключение также и для этого субдомена, но браузеры добавляют исключение только для того домена, который указан в адресной строке.

Чтобы добавить исключение, можно пройти по ссылке
https://static.archlinux.org.ru/static/css/default.css
и подтвердить исключение.
