Что-то либо я туплю, либо оно где-то рядом...
В tint2 >> system tray висят громкость - volumeicon и уровень зарядки батареи, привязанный к параметрам xfce4-power-manager.
Перелопатил конфиг tint2 и применил отключение tooltips по данному мануалу в секции ~/.config/tint2/tint2rc:
#-------------------------------------
# Tooltip
tooltip = 0
tooltip_show_timeout = 0
tooltip_hide_timeout = 0.1
tooltip_padding = 2 2
tooltip_background_id = 3
tooltip_font_color = #efefef 100
tooltip_font = Noto Sans 8
Но! Данные изненения применяются исключительно к панели задач - task bar (слева).
Сам же system tray (справа), загадочным образом обходит любого рода изменения (шрифты/цвета/размеры/рамки/etc.) в файле конфигурации.
Собственно и сами tooltips продолжают всплывать в трее (справа), в панели задач (слева), после применения приведённых параметров, tooltips (всплывающие сообщения) перестали появляться.
Панель конечно же перезапускал.
