После установки что мне надо сделать, чтобы сменить ядро?
Ядро менять не обязательно, просто для того чтоб собрать модули для этого ядра надо его заголовки.
После установки заголовков для зен ядра переставьте nvidia-390xx-dkms и смотрите что бы сборка была без ошибок, перещагрузитесь и смотрите вывод lspci -kk для карточки должны быть показаны доступные модули nvidia

какой драйвер используется? Просто он показывает сразу 3

все доступные для устройства Kernel modules:nouveau, nvidia_drm, nvidia

используемый Kernel driver in use:
или лог иксов смотреть
например 
cat /var/log/Xorg.0.log |grep LoadModule
[    15.132] (II) LoadModule: "glx"
[    15.162] (II) LoadModule: "nvidia"
[    15.192] (II) LoadModule: "glxserver_nvidia"

Но у вас гибрид и по умолчанию будет на интеле работать
VGA compatible controller: Intel Corporation 2nd Generation Core Processor Family Integrated Graphics Controller (rev 09)
Subsystem: ASUSTeK Computer Inc. Device 1652
Kernel driver in use: i915
нвидию надо включать одним из способов описанным в
https://wiki.archlinux.org/index.php/NVIDIA_Optimus
или на постоянку или для запуска определенных приложений
