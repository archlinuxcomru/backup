Вроде всё верно.
Если подключаетесь по кабелю, закомментируйте в /etc/rc.conf
#wlan0="dhcp"
Ещё:
/etc/ppp/options.xl2tpd
lcp-echo-interval 10
lcp-echo-failure 2
name bobart
remotename tp
ipparam corbina
connect /bin/true
mru 1500 # ничего не
mtu 1500 # упустили?
nodeflate
nobsdcomp
noauth
persist
maxfail 0
nopcomp
noaccomp
defaultroute
Сделайте всё-же файлик, исполняемый:
/etc/ppp/ip-up.d/ip-up-route.sh
#!/bin/sh
PATH="/sbin:/bin:/usr/sbin:/usr/bin"
GATEWAY=`route | grep ^default | awk '{print $2}'`
route del default
route add default dev ppp0
(
echo '#!/bin/sh'
echo 'PATH="/sbin:/bin:/usr/sbin:/usr/bin"'
echo 'route del default'
echo "route add default gateway $GATEWAY"
) >/etc/ppp/ip-down
/etc/rc.d/network restart, затем /etc/rc.d/xl2tpd start
Посмотрите, что выдаст при попытке подключения?
