чтобы в gtk3 неактивные вкладки были чётко разделены? Например, чтобы вокруг каждой вкладки была рамочка, и лучше с закруглёнными верхними углами
..конкретные вопросы: как добиться

Ну немного конкретней:

Для табов править в теме notebook tab
Для подсказок tooltip

Например в теме Clearlooks-Phenix это секции notebook *tab и tooltip* в /usr/share/themes/Clearlooks-Phenix/gtk-3.0/gtk-widgets.css
и цвета в /usr/share/themes/Clearlooks-Phenix/gtk-3.0/gtk.css

Можно добавить настройки в конфиг ~/.config/gtk-3.0/gtk.css

что то вроде
/* Notebooks и  tabs*/

/*notebook →все что ниже панели инструментов (тулбара toolbar)*/
/*Табы*/

notebook tab {
	padding: 3px;
    border-color:@menu_border_color;
/*можно свой цвет задать чтобы сильней выделить
например
border-color:#2F4051
или темы
например
border-color:@border_color*/
}

notebook header.top tab {
/*поля вокруг названия таба*/
padding: 4px 6px 3px 6px;

/*рамки таба*/
	border-width: 1px 3px 1px 3px;

/*радиусы рамки таба*/
	border-radius: 6px 6px 0px 0px;

/*поля слева справа таба , можно отрицательные или положительные значения*/
	margin-left: -3px;
/*или margin-left: 7px;*/
    margin-right: 0px;
}

Настройки в ~/.config/gtk-3.0/gtk.css применяются со всеми темами после перезагрузки приложения на гтк3 ( тунар демоном обычно запускается смотрите команду для перезагрузки)
