А где это посмотреть, там на выбор ничего нет. Просто пользователь и ввод пароля, внизу раскладка.Попытался гном загрузить по другому сначада startx прописал потом gnom-session в нем. Нормально он так и не загрузился но были сообщения что locale не поддерживается. используйте fallback и ‘C’ locale. Причем в консоле стоит нормально locale говорит везде ru_RU. А вот в gnome locale говорит везде ‘C’.
И еще в консоле locale -c писала что не удалось по умолчанию поставить locale в нетокторых опциях мол нет файла или папки: local:
locale -c
Cannot set LC-CTYPE to default local:No such file or directory
Cannot set LC-MESSAGES to default local:No such file or directory
Cannot set LC-ALL to default local:No such file or directory


Всем спасибо проблема решена, я описался в rc.conf вместо ru_RU ru_Ru. и команда
localedef -c -i /usr/share/i18n/locales/ru_RU -f UTF-8 ru_Ru.UTF-8 )))))
