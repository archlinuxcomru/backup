Попробуйте переустановить пакеты:
extra/gnome-icon-theme 3.2.1.2-1 (gnome)
    GNOME icon theme
extra/gnome-icon-theme-extras 3.0.0-2 (gnome)
    GNOME icon theme, extra icons
extra/gnome-icon-theme-symbolic 3.2.2-1 (gnome)
    GNOME icon theme, symbolic icons
Конкретно громкость – файлики audio-volume-*-symbolic.svg, лежащие в /usr/share/icons/gnome/scalable/status/.
