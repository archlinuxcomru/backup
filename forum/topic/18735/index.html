ESC k, ESC x, ESC y, ESC P, ESC M,
ESC g, ESC p, ESC 4, ESC 5, ESC E,
ESC F, ESC !

ESC W, DC4, SO, DC2, SI, ESC w, ESC G, ESC H, ESC T, ESC S, ESC *

Есть какой-то драйвер http://www.openprinting.org/printer/Epson/Epson-Dot_Matrix

Но может быть проще попытаться поставить в dosbox "Слово и дело" (w&d.exe).

Управляющие коды вставляются прямо в текст, там Esc-последовательности, набираются с помощью спец-последовательностей.
Принтер. Esc-последовательность.

С использованием w&d получится некий WYSIWYG.
