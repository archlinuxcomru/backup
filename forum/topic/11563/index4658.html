klaus, какое у тебя значение параметра DiscoverableTimeout = ??? в /etc/bluetooth/main.conf
Желательно, чтобы был 0 - DiscoverableTimeout = 0 (если 0, то Bluetooth будет включенным постоянно).
Если не "0", то bluetooth будет отключаться через определенное этим значением параметра время - для экономии электроэнергии.

PS Советую все-таки пользоваться gnome-bluetooth - единственная нормально работающая утилита
PSS заодно уж проверь PairableTimeout = 0
