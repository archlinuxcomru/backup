часом не прокся?
XferCommand = /path/to/command %u
If set, an external program will be used to download all remote files.
All instances of %u will be replaced with the download URL. If
present, instances of %o will be replaced with the local filename,
plus a “.part” extension, which allows programs like wget to do file
resumes properly.

This option is useful for users who experience problems with built-in
http/ftp support, or need the more advanced proxy support that comes
with utilities like wget.
помнится выбор для различных протоколов (ftp,http etc) производится (по умолчанию) не в конфиге пакмена, нынче, а директива выше , чтото вроде “последнего” слова.
