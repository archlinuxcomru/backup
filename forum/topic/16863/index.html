Решил проблему так
набрал в строке поиска opera://flags чтобы зайти в скрытве настройки opera ,там есть пункт
Приоритет HTML над Flash
Mac, Windows, Linux
Отдавать предпочтение HTML-содержимому, скрывая Flash из списка плагинов.
Было "включено" переставил на "отключено"
После этого всё заработало.
