есть частичное решение

в about:config firefox
browser.display.use_focus_colors;true
browser.display.focus_background_color;#FFFFFF
browser.display.focus_text_color;#000000
здесь можно выставить цвета по вкусу

на активном поле будут задействованы выбранные цвета, то есть не нужно будет выделять вручную текст чтобы его увидеть
