судя по всему звуком управляет Speaker
попробуйте эту команду, прибавит ли она громкость?
amixer set Speaker 5%+
если нет, то попробуйте Speaker заменить на Master
если сработает, то в настройках горячих клавиш в xfce добавьте эту команду для прибавки громкости и нажмите свою мультимедийную клавишу, аналогично с убавлением громкости
amixer set Speaker 5%-
авось сработает
