В Вашем Арче есть файл /etc/X11/Xsession ?
Просто судя по информации в инете, например /etc/gdm/Sessions/fluxbox имеет такой вид:
#!/bin/sh
#
# /etc/gdm/Sessions/fluxbox
#
exec /etc/X11/Xsession /usr/bin/startfluxbox
