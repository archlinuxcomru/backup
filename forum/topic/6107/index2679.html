Подобный вопрос здесь уже был, только для карты от ATI.

Моя видео карта:
VGA compatible controller: Intel Corporation 82G33/G31 Express Integrated Graphics Controller
Проблема та же: при включение эффектов куб, карусель и т.д. появляется окно о невозможности их включения.
Речь идет об эфектах kwin.

xorg.conf

Section "Module" 
        Load  "glx" 
        Load  "extmod" 
        Load  "dri2" 
        Load  "dbe" 
        Load  "dri" 
        Load  "GLcore" 
EndSection 
 
Section "Device" 
        Option "AccelMethod" "UXA" 
        Option     "DRI"        "true"                  # [<bool>] 
        Identifier  "Card0" 
        Driver      "intel" 
        VendorName  "Intel Corporation" 
        BoardName   "82G33/G31 Express Integrated Graphics Controller" 
        BusID       "PCI:0:2:0" 
EndSection 
 
Section "DRI" 
    Mode 0666 
EndSection

Вот вывод kwin –replace

kwin(17586) KWin::CompositingPrefs::detectDriverAndVersion: GL vendor is "Tungsten Graphics, Inc"  
kwin(17586) KWin::CompositingPrefs::detectDriverAndVersion: GL renderer is "Mesa DRI Intel(R) G33 GEM 20100328 2010Q1 x86/MMX/SSE2"  
kwin(17586) KWin::CompositingPrefs::detectDriverAndVersion: GL version is "1.4 Mesa 7.8.2"  
kwin(17586) KWin::CompositingPrefs::detectDriverAndVersion: Detected driver "intel" , version "20100328"  
kwin(17586): OpenGL compositing self-check failed, disabling compositing.  
kwin(17586) KWin::EffectsHandlerImpl::loadEffect: EffectsHandler::loadEffect : Effect  "kwin4_effect_blur"  is not supported
