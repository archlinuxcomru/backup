Это предложил для проверки страбатывания .... если прописать 0, то курсора не будет.

В части проверки .bashrc и .bash_profile …
Будет время, рекомендую провести тестирование запуска оболочки bash
1. Для начала отредактируй файлы .bashrc и .bash_profile
- в начала каждого файла пропиши строчку echo START <имя файла>: echo 'START .bashrc', echo 'START .bash_profile'
- в конец каждого файла пропиши строчку echo END <имя файла>: echo 'END .bashrc', echo 'END .bash_profile'
2. Открой терминал-эмулятор и запусти команду: bash --login -x
3. После завершения команды тестирования ОБЯЗАТЕЛЬНО ЗАПУСКАЕМ КОМАНДУ exit , чтобы выйти из оболочки ...
4. Смотрим вывод: будут показаны все действия, например, смотрим что выполняется у меня
 … пропускаем не нужное … и находим старт .bash_profile, который и запускает .bashrc …
+ echo 'START .bash_profile'
START .bash_profile
+ [[ -f /home/vasek/.bashrc ]]
+ . /home/vasek/.bashrc
++ echo 'START .bashrc'
START .bashrc
… дальше прописано, что делает .bashrc ...
++ echo 'END .bashrc'
END .bashrc
… дальше прописано, что делает .bash_profile …
+ echo -e '\033[?17;7;113c'  ### а вот и запуск формы курсора
+ echo 'END .bash_profile'
END .bash_profile

vall, вообщем пришлось поднять свои старые записи, чтобы вспомнить как правильно записывать параметры формы курсора а grub ... и да, подзабыл, подвела память ... вообщем, если у тебя не работает прописка формы курсора в файлах .bash_profile или .bashrc, то можешь сделать прописку через параметры grub типа vt.cur_default=0x650717 .... меняя первые две цифры можно менять цвет - например, если вместо 65 прописать 99, то голубой заменится на красный ... одно НО - цифры нужно брать не четные. ... проверил, все работает - правда есть один нюанс: в момент работы загрузчика цвет другой, так что не пугайся - после загрузки все устаканивается. Проверяй - даже интересно.

PS - не забываем, что если форма курсора прописана или .bash_profile или .bashrc, то прописка в grub не нужна.
