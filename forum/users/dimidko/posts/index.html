Дистрибутив выбран по рекомендации вики, классический не загружался с usb, но вроде они одинаковые
Note: As of 4/30/2011, Macbook 7,1 models will not boot with the normal iso, but the latest archboot iso should work.

После загрузки с флешки, появляется нормальное меню выбора разрядности устанавливаемой системы
После выбора 64 бит начинается загрузка установщика, и тут начались проблемы
1) Сначала все прошло успешно, появилась надпись “Нажмите enter для входа в консоль и начала установки”, но клавиатура не работала
В вики нашел решение 

Note: Some MacBook users report strange keyboard output such as long delays and character doubling. To fix this problem, boot with following options.
boot: arch noapic irqpoll acpi=force
Дописал параметры в gbub.cfg
После загрузки клавиатура заработала, но
2) Теперь появились проблемы с видео, которые как раз победить не могу
(увеличится при нажатии)


Убрал все параметры из конфига, проблема осталась, но клавиатура работает! Без параметров! Вслепую даже удалось зайти в установщик:

У меня подозрения что проблемы с видеодрайвером, и установка, в отличии от классического образа, не чисто текстовая. Как перевести текущую в текстовый режим с размером окна 640x480 я не нашел. Сейчас разрешение консоли 1366x768.
