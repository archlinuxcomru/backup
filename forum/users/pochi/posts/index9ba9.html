ЗЫ: Решил пока вырубать руками с помощью этого скрипта:
#!/bin/bash
# Small script to set display into standby, suspend or off mode
# 20060301-Joffer
case $1 in
  standby|suspend|off)
    sleep 1; xset dpms force $1
  ;;
  *)
    echo "Usage: $0 standby|suspend|off"
  ;;
esac
Добавил в него “sleep 1”, без этого у меня не работает.
