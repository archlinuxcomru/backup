На примере Network controller: Broadcom Inc. 43255 и двух установленных ядер (linux и linux-lts).

Устанавливаем проприетарные dkms-версии драйвера и выгружаем конфликтующие модули из ядра. Модуль wl должен автоматически загрузить lib80211 или lib80211_crypt_tkip, иначе их придется загружать вручную.
pikaur -S broadcom-wl-dkms dkms linux-headers linux-lts-headers
sudo rmmod b43
sudo rmmod ssb
sudo rmmod brcmsmac
sudo modprobe wl
sudo depmod -a
Крайняя команда обновляет зависимости в ядре. Подробности.
