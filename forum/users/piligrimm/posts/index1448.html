У тебя прокси с авторизацией или нет?

Я в своё время сдела доступ через прокси (если она с авторизацией) таким образом.
Использовал wget.
Конфиг /etc/wgetrc
# You can set the default proxies for Wget to use for http and ftp.
# They will override the value in the environment.
proxy_user=имя пользователя
proxy_password=пароль пользователя
http_proxy = адрес proxy:порт
ftp_proxy = адрес proxy:порт

Далее в /etc/pacman.conf раскоментировать строку: XferCommand = /usr/bin/wget –passive-ftp -c -O %o %u
