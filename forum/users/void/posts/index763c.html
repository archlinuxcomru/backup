fluxbox умеет только управлять окнами, а за меню отвечает уже само приложение. Т.е. если само приложение не предлагает спрятать меню, то, наверно, без грубого напильника эту фишку не реализовать… Хотя в ГНОМе меню можно выносить в отдельную панель в стиле MacOS, но как это работает не имею понтия.

Конкретно для firefox есть адд-он - поищи среди аддонов по слову menubar, там есть который прячет меню по нажатию клавиши и есть который запихивает меню в маленькую иконку.

Если я правильно понял вопрос: в конфиге темы флакса параметр window.handleWidth должен быть установлен в значение больше нуля, тогда внизу слева и справа под окном будет т.н. область grip - за нее хватаешь левой мышкой и тянешь.
