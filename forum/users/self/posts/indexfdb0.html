Установил Arch Linux в качестве дополнительной ОС на нетбук(Asus EEE PC 1000H, видеокарта Intel 945GME). Встала проблема конфигурации Xorg.
При запуске иксов черный экран, без курсора мыши. Пытался поставить по русской инструкции с Arch Wiki. Не получилось :( Гуглил - тоже ничего интересного не нашел(еще, к тому же, у меня с английским плохо).

Прошу помощи у более опытных товарищей.
Dell 1525, карточка Intel 965.
Были такие же проблемы. Добавил вот такие опции:
Section "ServerLayout"
        Identifier     "X.org Configured"
        Screen      0  "Screen0" 0 0
        InputDevice    "Mouse0" "CorePointer"
        InputDevice    "Keyboard0" "CoreKeyboard"
        Option  "AutoAddDevices"        "Off"
        Option  "AutoEnableDevices"     "On"
        Option  "AllowEmptyInput"       "Off"
EndSection
Section "Serverflags"
        Option  "DontVTSwitch"  "Off"
        Option  "DontZap"       "Off"
        Option  "XkbDisable"    "Off"
EndSection
Теперь при запуске startx иксы нормально грузятся. При запуске X - чёрный экран. И не у меня одного. Судя по всему, корявый сам сервер. Либо руки у меня и трёх товарищей, но с этим разбираться пока некогда.
Судя по логу из архива - опция AllowEmptyInput включена по умолчанию, поэтому мышка с клавиатурой не работают. Кроме того, TTF-шрифты либо установлены криво, либо не установлены вовсе.
