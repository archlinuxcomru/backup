у меня с самбой вечно какие-то проблемы были, с доступом и видимостью из системы. Как вариант сделал ручное монтирование, скрипт такой:
---------------
#!/bin/bash
mount_dir="/mnt/netmnt/files" #директория куда монтируем сетевой ресурс(предварительно создаем)
mount_net="//IP/mount_point" #сетевой путь для точки монтирования
local_user="superwoman" #локальный пользователь
smb_user="superwoman1" #пользователь для подключения к самбе серверу
pas="pussywoman" #пароль для подключения к самбе серверу

sudo mount.cifs $mount_net $mount_dir -o username=$smb_user,password=$pas,user,iocharset=utf8,rw,uid=$local_user,noperm
--------------
Нужно запускать с помощью терминала, потому что без него sudo вешает намертво DE.
Это для одного конкретного сервера, если их несколько придеться делать для каждого, зато теперь к нему путь человеческий, работает как локальная папка и быстрее.
