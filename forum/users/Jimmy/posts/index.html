Данный образ использует Syslinux 6.03, но данное приложение включает в себя только установочные файлы для Syslinux 6.03/2014-10-06. Так как новые версии Syslinux не совместимы друг с другом, необходимо, чтобы дополнительныефайлы ('ldlinux.sys' and 'ldlinux.bss') были загружены из интернета: - Выберите 'Да', чтобы скачать файлы из интернета - Выберите 'Нет', чтобы отменить операцию Примечание: Файлы будут загружены в текущий каталог приложения и будут использоваться повторно при необходимости.
