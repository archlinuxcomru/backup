Установи wvdial
 sudo pacman -Su wvdial

Затем пропиши в конфиге /etc/wvdial.conf :
[Dialer Defaults]
Modem = /dev/ttyUSB0  //пропиши вместо этого полный путь к устройству
Baud = 57600
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2
Init3 =
Area Code =
Phone =*99#
Username =mts
Password =mts
Ask Password = 0
Dial Command = ATDT
Stupid Mode = 1
Compuserve = 0
Force Address =
Idle Seconds = 300
DialMessage1 =
DialMessage2 =
ISDN = 0
Auto DNS = 1

после этого в консоли запускай:
sudo wvdial

Можно воспользоваться GUI программами - Kppp -> KDE, GnomePPP -> Gnome
Информацию о USB устройствах выводит команда
lsusb
