Пытаюсь прошить свой adsl модем через tftp. Для этого установил tftp-hpa. Далее в /etc/hosts.allow добавил
tftpd: ALL
in.tftpd: ALL
в /etc/rc.d/tftpd изменил
TFTPD_ARGS="-l -v -v -v -c -s /var/tftpboot"
запустил сервер, далее
$ sudo chmod 777 /var/tftpboot

Преблема 1: не могу залить на него файлы через tftp. только копированием через cp
