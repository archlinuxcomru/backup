When a KF5 stable release of an application is released with KDE Applications,
using the KDE4 version of that application is discouraged as the fixes are
only applied to the KF5 one.

For this reason, on Arch, the KF5 package replaces the KDE4 one. Also, we are
getting rid of kde module prefixes e.g. kdebase-konsole -> konsole.
