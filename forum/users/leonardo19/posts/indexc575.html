talant2000, если не нужны особенные изыски (ввод пароля вручную, вход в разные окружения рабочего стола, в разные учетные записи), я бы посоветовал настроить автологин без экранного менеджера. Сам так давно уже настроил.
Тут обсуждалось: Автологин и автозагрузка любого окружения без DM
Отмечу, что мой ~/.xinitrc выглядит так:
#!/bin/sh

# start some nice programs

if [ -d /etc/X11/xinit/xinitrc.d ] ; then
 for f in /etc/X11/xinit/xinitrc.d/?*.sh ; do
  [ -x "$f" ] && . "$f"
 done
 unset f
fi

exec startxfce4
