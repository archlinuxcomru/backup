Тема начиналась здесь: viewtopic.php?f=30&t=7309

Теперь, поскольку программа стала немного юзабельна выкладываю её в общий доступ.

Gnome Modem Manager - это приложение для Linux, предоставляющее удобный интерфейс для управления подключенными 3G-модемам. Это программа-аналог таких программ как Megafon Internet, MTS Connect и другим для ОС Windows. Основная целевая среда использования этой программы - окружение рабочего стола Gnome, хотя её можно без проблем использовать и в другитх средах.

Программа написана на Vala. Для своей работы использует D-Bus API ModemManager (часть NetworkManager).

Официальный сайт: http://cpu-fun.ru/projects/gnome-modem-manager
Пакет на AUR: https://aur.archlinux.org/packages.php?ID=54542

Все желающие приглашаются к тестированию работы программы.
