И ЧЕРТ меня дернул сегодня установить драйвер на видео карту своего ноутбука! Решил ставить дрова nvidia-340 т.к. на ubuntu стоят и все работает отлично.
Установку делал по wiki:
1. Вышел из иксов;
2. sudo pacman -R xf86-video-nouveau
3. sudo pacman -S nvidia-340xx nvidia-340xx-libgl nvidia-340xx-utils
4. reboot
5. sudo nvidia-xconfig
6. sudo mkinitcpio -p linux
7. sudo reboot
8. ИИИ!! черный экран... с курсором

Далее:
в etc/modprobe.d/modprobe.conf добавил "blacklist nouveau"
bumblebee не устанавливал, поскольку на этом же ноуте второй системой стоит ubuntu и там все работает без bumblebee а управляется непосредственно через nvidia settings

sudo nano /etc/mkinitcpio.conf
MODULES="nvidia"
