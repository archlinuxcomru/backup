Тут никто не писал про fish, оставлю это тут, может быть кому-то пригодится.
Алиасы в Fish Shell добавляются таким образом - "alias {Имя алиаса} {Текст, скрытый за алиасом}", например "alias gdu /home/blituser/binaries/gdu", если есть пробелы то нужно взять текст в кавычки.
При выходе из сессии шелла, несохранённые алиасы стираются, их нужно сохранить командой "funcsave {Имя алиаса}"
Для удаления сохранённых алиасов, нужно удалить файл с именем алиаса в папке ~/.config/fish/functions/
