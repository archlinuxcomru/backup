lumberjack, пропиши в конфиге и скриптах следующее:
[email protected]:~/.i3$ cat config | grep release
bindsym --release Print exec ~/.i3/scrotP.sh
bindsym --release Shift+Print exec ~/.i3/scrotSP.sh
[email protected]:~/.i3$ cat scrotP.sh
scrot '%Y-%m-%d--%s_$wx$h_scrot.png' -e 'mv $f ~/pictures/screenshots/ ; feh ~/pictures/screenshots/$f'
[email protected]:~/.i3$ cat scrotSP.sh
export DISPLAY=:0 ; scrot -s '%Y-%m-%d--%s_$wx$h_scrot.png' -e 'mv $f ~/pictures/screenshots/ ; feh ~/pictures/screenshots/$f'
Без ключа --release работает только первый раз, потом типа "залипает" и не реагирует на повторное нажатие. Помогает только полная перезагрузка, но опять только на один раз.
Для scrot c выделением участка на экране я добавил export DISPLAY=:0
У себя проверил - через добавленные скрипты работает. Если содержимое скриптов напрямую прописать в файл ~/.i3/config, то не работает.
