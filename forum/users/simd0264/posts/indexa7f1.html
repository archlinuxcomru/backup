50baksov, создайте в хоуме файл .asoundrc с таким содержимым
pcm.!default {
        type hw
        card 1
}

ctl.!default {
        type hw
        card 0
}
Хотя у вас 3 устройства, меня 2, если не получится - надо третью строку прописать.
