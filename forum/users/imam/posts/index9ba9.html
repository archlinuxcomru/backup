Я собрал пакеты от книги LFS 6.4 и сделал репозиторий core, также отредактировал установочные скрипы. Плюс сделал один пакет схожий с filesystem и назвал его base-etc.
При установки возникает такая ошибка:
Targets: base-etc-6.4-1 linux-headers-2.6.28.7-1 …
…
Checking package integrity… done
Error: missing package filelist in /mnt/src/core/pkg/base-etc-6.4-1.pkg.tar.gz
Error: missing package filelist in /mnt/src/core/pkg/ linux-headers-2.6.28.7-1.pkg.tar.gz
…
Cleaning up… done
Checking for file conflicts… done
Installing base-etc… done
Installing linux-headers … done
…
Package Installation Complete

При этом все устанавливается и работает нормально, но после установки права на каталог /tmp становятся как на обычную папку, а не drwxrwxrwt.
