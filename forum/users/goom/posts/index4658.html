Отсутствуют устройства /dev/fd*, есть каталог /dev/fd, где находятся
ls -l /dev/fd/
итого 0
lrwx------ 1 root root 64 Дек  7 15:48 0 -> /dev/pts/2
lrwx------ 1 root root 64 Дек  7 15:48 1 -> /dev/pts/2
lrwx------ 1 root root 64 Дек  7 15:48 2 -> /dev/pts/2
lr-x------ 1 root root 64 Дек  7 15:48 3 -> /proc/9142/fd
Что примечательно, на флопоглоте постоянно горит индикатор! При этом никаких звуков не издается. Флопоглот во фряхе работает нормально, индикатор не горит.
