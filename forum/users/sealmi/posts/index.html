Все! Сам разобрался.

Для установки:
1. Выполняем команду - kdesu ./eset_nod32av_64bit_ru.linux
2. После установки, когда спрашиваю разрешение на перезагрузку - отвечаем нет
3. Открываем для редактирования файл /etc/ld.so.preload и удаляем вот эту libesets_pac.so строку
4. Открываем для редактирования файл /etc/rc.local и прописываем вот такую /opt/eset/esets/sbin/esets_daemon строку
5. Перезагружаем комп
