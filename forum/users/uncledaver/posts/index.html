проблема моя просто и банальна - pptp vpn :)
начнем с начала: есть eth0 со статичнским ip
для инета нужен ppp0, поднимающийся сразу после eth0
т.е. ppp0 подключается через eth0
Кусок rc.conf
eth0="eth0 192.168.147.40 netmask 255.255.255.0 broadcast 
192.168.147.2"
INTERFACES=(eth0 ppp0)
#роуты
default="default gw 192.168.147.2"  # шлюз мой
vpn="-host 192.168.147.2 gw 192.168.147.2 dev ppp0" # 192.168.147.2 - адрес впн-сервера
ROUTES=(default vpn)
