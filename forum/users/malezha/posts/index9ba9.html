Основная проблема в том,что на команду modprobe -a vboxguest vboxsf vboxvideo. Error:could not insert ,,vboxvideo": Invalid argument
В офф англ вики:
Install the Guest Additions
Install the virtualbox-guest-utils package:
# pacman -S virtualbox-guest-utils
Manually load the modules with:
# modprobe -a vboxguest vboxsf vboxvideo
Create a *.conf file (e.g. virtualbox.conf) in /etc/modules-load.d/ with these lines:
/etc/modules-load.d/virtualbox.conf
vboxguest
vboxsf
vboxvideo
На данный момент как понимаю самый нормальный способ поставить видео в virtualbox Из за ошибки приходится ставить xf86-video-vesa. Нет просто перетащить не катит , т.к изображение получается в центре. А по краям белый фон. Если перейти в полно экранный режим чёрный фон по бокам разницы в принципе нет) Режим масштабирования полная порнуха)
