KPPP зависает время от времени, в общем как-то не стабильно работает. Почему не знаю.

Лучше поставить wvdial.conf (для мегафона){
[Dialer Defaults]
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0 
Init5 = AT+CGDCONT=1,"IP","internet"
Modem Type = Analog Modem
;New PPPD = yes
Phone = *99#
ISDN = 0
Username = "mgsm"
Password = "mgsm"
Modem = /dev/ttyUSB0
Baud = 460800
Auto Reconnect = on
Abort on Busy = on
Stupid Mode = on
Abort on No Dialtone = off
}
А ещё лучше настроить knetworkmanager - стабильно и удобно!
Для этого нужно установить 3 пакета и перезагрузить систему {
usb_modeswitch + modemmanager + mobile-broadband-provider-info
}
Первый пакет - переключает устройство в режим модема, второй - делает доступной вкладку 3G, третий - файл преднастроек для всех известных провайдеров.

Удачи!
