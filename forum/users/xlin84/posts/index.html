# pacman -S xf86-video-nouveau
# pacman -S nouveau-dri
в файле /etc/modprobe.d/modprobe.conf добавить строчку 
blacklist nvidia
Cоздать файл /etc/X11/xorg.conf.d/10-monitor.conf с этим
Section "Device"
       Identifier "nvidia"
       Driver "nouveau"
       #Driver "nv"
       #Driver "nvidia"
EndSection
Добавить в /etc/mkinitcpio.conf в секцию MODULES и FILES
MODULES="nouveau"
FILES="/etc/modprobe.d/modprobe.conf"
Потом в консоли
 mkinitcpio -p kernel26
перезапуск
