Здравствуйте. Пытаюсь из Raspberry Pi2 + Supstronics X205 Expansion Board (ОС: arch linux arm) сделать точку доступа без интернета.

пробую через wpa_supplicant:

/etc/wpa_supplicant-adhoc.conf
ctrl_interface=DIR=/run/wpa_supplicant GROUP=wheel

# use 'ap_scan=2' on all devices connected to the network
# this is unnecessary if you only want the network to be created when no other networks are available
ap_scan=1

network={
    ssid="MySSID"
    mode=1
    frequency=2432
    proto=RSN
    key_mgmt=WPA-PSK
    pairwise=CCMP
    group=CCMP
    psk="secret passphrase"
}

далее ввожу команду:

 wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant-adhoc.conf -D nl80211,wext

В результате вижу свою сеть в списке существующих, но пункт не активен, и даже попытаться подключиться к ней не могу.
Пробую так:

iwconfig wlan0 mode ad-hoc
iwconfig wlan0 channel 1
iwconfig wlan0 essid RPI
ifconfig wlan0 192.168.1.1 netmask 255.255.255.0 up

тогда сеть становится активной, но подключиться все равно не могу: показывает что пытается подключиться, потом просто прекращается попытка

HELP замучился, не могу найти нормальный вариант создания точки доступа
