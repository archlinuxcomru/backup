В файле /lib/udev/rules.d/69-libmtp.rules есть ли id производителя и id устройства? Например в Вашем случае это: ?? 
ID 1d57:f833 Xenta
Если его в файле 69-libmtp.rules нет (1d57 и f833), то внесите что-то такое:
 # Xenta
ATTR{idVendor}=="1d57", ATTR{idProduct}=="f833", SYMLINK+="libmtp-%k", ENV{ID_MTP_DEVICE}="1", ENV{ID_MEDIA_PLAYER}="1"
У меня для SonyXperia это помогло.

–skipinteg - Не выполнять никаких проверок целостности (контрольной суммы и PGP) для исходных файлов. Это сумма из двух:
–skipchecksums - Не проверять контрольные суммы исходных файлов.
–skippgpcheck - Не проверять подписи PGP исходных файлов. ------------------------------------------- (см. makepkg(8) Manual Page)
