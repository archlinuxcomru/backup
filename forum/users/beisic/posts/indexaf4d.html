ИМХО:
1. Если есть желание получить русифицированный Arch, начните установку с :
 # loadkeys ru
 # setfont UniCyr_8x16
После в /etc/vconsole.conf вписать:
KEYMAP=ru
FONT=UniCyr_8x16
2. Уберите везде лишние пробелы:

# ln -s / usr / share / zoneinfo/Aisa / Vladivostok / etc / localtime
 # ln -sf /usr/share/zoneinfo/Aisa/Vladivostok /etc/localtime
3. Для местного времени использовать:
 hwclock --systohc --localtime

MamkinHaker
12)Обновить зеркала? (надо ли это сейчас)
Русские зеркала "вверх" нужно сделать перед :
 # pacstrap /mnt base linux-lts linux-firmware nano
Иначе, получите, долгую по времени установку системы.
Для автоматического выбора ближайших зеркал можно использовать:
 pacman -S reflector
 reflector --verbose --latest 20 --sort rate --save /etc/pacman.d/mirrorlist
5. Для процессоров AMD установите пакет amd-ucode. Можно добавить в pacstrap.
