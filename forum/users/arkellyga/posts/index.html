Насколько я верно понял информацию из вики,то autologin выполняет автоматический вход при первом запуске системы.Чего хотелось бы избежать,т.к. имею два юзера.Поправьте,если я не прав.И по поводу фигни,цитата из вики:

Enabling interactive passwordless login

LightDM goes through PAM so you must configure the lightdm configuration of PAM:

/etc/pam.d/lightdm

#%PAM-1.0
auth        sufficient  pam_succeed_if.so user ingroup nopasswdlogin
auth        include     system-login
...

You must then also be part of the nopasswdlogin group to be able to login interactively without entering your password:

# groupadd -r nopasswdlogin
# gpasswd -a username nopasswdlogin
