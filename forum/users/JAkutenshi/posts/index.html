Удобный скрипт в /etc/profile:
#autostart systemd default session on tty1
if [[ "$(tty)" == '/dev/tty1' ]]; then
        exec startx
fi

При логине из первого телетайпа стартуют иксы. Как по мне - очень даже удобно без DM. Взято было отсюда: http://tutos.readthedocs.io/en/latest/source/Arch.html#making-system-usable
