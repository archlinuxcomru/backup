запретить можно добавив !snd_usb_audio в секцию MODULES в rc.conf, у меня 3 звучки, я решил проблему прописав
~ $ cat /etc/asound.conf
pcm.!default {
    type hw
    card UAC3553B
}
ctl.!default {
    type hw
    card UAC3553B
}
взяв данные о карте из 
~ $ cat /proc/asound/cards
 0 [V8237          ]: VIA8237 - VIA 8237
                      VIA 8237 with AD1888 at 0x1000, irq 22
 1 [UAC3553B       ]: USB-Audio - UAC3553B
                      Philips UAC3553B at usb-0000:00:10.1-2, full speed
 2 [AudioPCI       ]: ENS1371 - Ensoniq AudioPCI
                      Ensoniq AudioPCI ENS1371 at 0xe400, irq 17
см. http://alsa.opensrc.org/index.php/FAQ026
то есть мне теперь по барабану очередность. интересно для чего надо jack'у hw0???
