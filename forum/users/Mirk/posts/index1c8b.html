 С незапамятных времён при загрузке постоянно выводились сообщения типа:
ACPI Error: Needed type [Reference]...
ACPI Exception: AE_AML_OPERAND_TYPE...
ACPI Error: Method parse/execution failed...
Удалось скрыть добавив в параметры загрузки:
vga=current loglevel=3 rd.systemd.show_status=auto rd.udev.log_priority=3
И:
echo kernel.printk = 3 3 3 3 > /etc/sysctl.d/20-quiet-printk.conf
