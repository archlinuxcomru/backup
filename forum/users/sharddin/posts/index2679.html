добавить drm и kms в загрузку ядра

Вики.

В качестве примера по NVIDIA основное приведу здесь.

Подготовка
Включить интерфейс DRM-KMS и PRIME Synchronization в проприетарном драйвере Nvidia
#Создаём /etc/modprobe.d/zz-nvidia-modeset.conf

options nvidia_drm modeset=1
Перезагрузка и проверка
sudo cat /sys/module/nvidia_drm/parameters/modeset
Y

Установка
Добавляем модули ядра nvidia в initramfs
sudo vim /etc/mkinitcpio.conf

MODULES=(nvidia nvidia_modeset nvidia_uvm nvidia_drm)

Не забывайте запускать mkinitcpio каждый раз после обновления драйвера. См. раздел #Pacman hook для автоматизации данных действий.
sudo mkinitcpio -P
Пользуемся.

По другим видеокартам и драйверам смотрите вики.
