Так, а как можно признаки превратить в диагноз?

Суть в том, что старте mc пытается (пытался когда-то точно) определить имя локального хоста через обращение к dns-серверу.
Это может быть, если cat /etc/hosts не указано имя хоста, который у вас, то есть если имя хоста у вас localhost, и оно не указано в /etc/hosts, mc долго будет пытаться определить его через dns-запрос.
Короче если ping "имя вашего хоста" будет пинговать 127.0.0.1 то проблема не в этом.

Действительно - через hosts имя хоста не указано
cat /etc/hosts
#
# /etc/hosts: static lookup table for host names
#

#<ip-address> <hostname.domain.org> <hostname>
127.0.0.1 localhost.localdomain localhost
::1 localhost.localdomain localhost

# End of file
