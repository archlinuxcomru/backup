Не новичок в линукс, но впервые поставил арч. Не удается настроить сабж. Что-то не могу понять систему настройки беспроводных интерфейсов в нем… Все доступные доки изучил… Видимо, у меня нестандартная ситуация.
Итак, имею сабж (модуль airo). В других дистрах прекрасно удавалась настроить. Модуль в арче есть и определяется. Устройства тоже все видны по ifconfig -a.
У меня в системе имеются 2 сетевых карты:
1) обычная Ethernet - eth0. Отключена в rc.conf
2) беспроводная - для нее в системе присутствуют 2 интерфейса: eth1 и wifi0 (и так должно быть). Не знаю, со всеми ли беспроводными адаптерами так, или только с моей моделью - других не юзал.

Для примера приведу куски конфигов от Slackware 11, с которыми все прекрасно работает, чтоб кто знает - посоветовал, как то же настроить в арче.
файл /etc/rc.d/rc.inet1.conf:
# Config information for eth0:
##IPADDR=“”
##NETMASK=“”
##USE_DHCP=“yes”
##DHCP_HOSTNAME=“”

# Config information for eth1:
IPADDR=“xx.xx.225.8”
NETMASK=“255.255.255.0”
USE_DHCP=“”
DHCP_HOSTNAME=“”

# Default gateway IP address:
GATEWAY=“xx.xx.225.254”


и файл /etc/rc.d/rc.wireless.conf:
# Cisco/Aironet 4800/3x0
# Note : MPL driver only (airo/airo_cs), version 1.3 or later
00:0F:F8:*)
    INFO=“Cisco/Aironet”
    ESSID=“MoyEssid”
    MODE=“Managed”
    KEY=“xxxx-xxxx-xx open”
    ;;

Вывод команд ifconfig и iwconfig в слаке, где все работает - в приложенных файлах.
Заранее спасибо за ответы.
