Если нужно разместить хомяк на отдельном разделе ЖД, то правите фстаб до нужного, и не забудьте создать файловую систему в новом разделе, если ее там нет.
#
# /etc/fstab: static file system information
#
# <file system> <dir>   <type>  <options>       <dump>  <pass>
/dev/sda1       /       ext4    defaults        0       1
/dev/sda2       none    swap    defaults        0       2
/dev/sda3       /home/mazahaka  ext4    defaults        0       2
tmpfs           /tmp    tmpfs   nodev,nosuid    0       0
Как то так? Предварительно создав на sda3 директории
