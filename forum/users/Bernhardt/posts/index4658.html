Взял новый конфиг отсюда: http://www.gentoo-wiki.info/HOWTO_Surround_Sound

pcm.!dmix {
   type plug
   slave {
       pcm surround51
       channels 6
   }
}
pcm.!default {
   type plug
   slave.pcm "dmix"
   slave.channels 6
   route_policy duplicate
}

После перезагрузки в alsamixer включаем каналы и все пашет.
По команде: speaker-test -D surround51 -c 6 -t wav
Звук не нравится, слабый surround (все каналы слабые, кроме центрального (в alsamixer все уровни на макс). Жду других решений, если кто знает.
