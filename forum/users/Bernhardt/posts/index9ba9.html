Может ли помочь, включение Glamour?
Glamor
С новыми версиями открытых драйверов ATI вы можете использовать новый метод ускорения под названием "glamor": это метод ускорения 2D реализуется через OpenGL и будет работать только с видеокартами на чипсете R300 и новее.
 Option	"AccelMethod"           "glamor"

Также есть вариант с wiki (https://wiki.archlinux.org/index.php/ATI):
2D performance (e.g. scrolling) is slow
If you have problem with 2D performance, like scrolling in terminal or browser, you might need to add Option "MigrationHeuristic" "greedy" into the "Device" section of your xorg.conf file.
Bellow is a sample config file /etc/X11/xorg.conf.d/20-radeon.conf:
Section "Device"
        Identifier  "My Graphics Card"
        Driver  "radeon"
        Option  "MigrationHeuristic"  "greedy"
EndSection

Что выбрать, что действительно поможет?
