ip l

eth0 - сетевой интерфейс
192.168.1.2 - статический адрес
192.168.1.1 - шлюз

ip link set dev eth0 up
ip addr add 192.168.1.2/24 dev eth0
ip route add via 192.168.1.1 dev eth0

А если отображает на эту команду:
1: lo: и там набор инфы?

к примеру:
% ip l
1: lo: ... - не надо, это виртуальный сетевой интерфейс
2: enp2s0: ... - надо, сетевой интерфейс для витой пары
3: wlp3s0: ... - не надо, сетевой интерфейс для wifi
4: vboxnet0: ... - не надо, сетевой интерфейс виртуальной машини
из всего надо только enp2s0, у тебя скорее всего будет называться по другому.
