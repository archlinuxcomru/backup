хорошо, комп видит телефон

подключи телефон и покажи:
mtp-detect
===========

подключение с помощью go-mtpfs:
-----
устанавливаем
yaourt -S go-mtpfs-git android-udev

добавляем пользователя в группу fuse
sudo gpasswd -a [user] fuse
[user]- ставим имя своего пользователя, у тебя blade

если пишет что нету группы то сначала создадим её
sudo groupadd fuse

создадим папку куда будем монтировать устройство
mkdir ~/GS3

подключаем телефон к компу
монтируем
go-mtpfs ~/GS3

теперь файловая система телефона доступна по адресу
cd ~/GS3

если надо отмонтировать устройство
fusermount -u ~/GS3
всё, теперь можно отключить телефон

-----
вообще про разные способы работы с MTP написано в ArchWiki
https://wiki.archlinux.org/index.php/MTP

возможно для гнома больше подойдёт gmtp
yaourt -S gmtp

Ругается на ip route add via 172.18.2.7 dev enp0s12 мол RTNETLINK answers: Network is unrtachable
ты наверное маску забыл указать
какая маска твоей сети ?

когда указываешь ip надо еще и маску ставить:
# ip addr add <ip>/<netmask> dev <interface>
к примеру
# ip addr add 192.168.1.2/24 dev eth0
здесь /24 маска сети в простонародье 255.255.255.0
у тебя она может быть другая
