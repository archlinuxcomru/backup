*Можно выполнять с раб.системы*
*названия директорий и разделов условны*
Есть 3 раздела на жестком диске:
- boot
- home
- root

Создаем директорию для монтирование бэкапного диска
 mkdik /mnt/disk

Монтируем его
mount /dev/sdaX /mnt/disk

Архивируем boot- и home-разделы
 tar -cvzpf /mnt/disk/boot.tar.gz /boot
 tar -cvzpf /mnt/disk/home.tar.gz /home

C root-разделом след. заморочка. Если на двух предыдущих было только по одной директории, которые и монтировать не надо было, т.к. в раб. системе они монтируются автоматом, то в root-разделе находятся все остальные директории, поэтому необходимо примонтировать этот раздел к какой-то одной директории, поэтому создаем такой каталог:
mkdir /mnt/ root
Монтируем root-раздел, обозначаемый /dev/sdaX, к созданной директории
mount /dev/sdaX /mnt/root
И:
 tar -cvzpf /mnt/disk/ root.tar.gz /mnt/root

Для восстановления нужно примонтировать диск, примонтировать разделы(root, boot, home) к каталогу и разархивировать.
mount /dev/sdaX /mnt/disk
mkdir /mnt/katalog
mount /dev/sdaX(где sdaX, разделы) /mnt/katalog
tar --same-owner -xvpf /mnt/katalog/<name>.tar.gz

Господа арчеводы! Пожалуйста проверьте на наличие ошибок нижеприведенное, если они есть, поправьте.

КОПИРОВАНИЕ
1. загружаюсь с archlive
2. монтирую жесткий диск для бекапа в созданную для него директорию:
 mkdir /mnt/bakup
mount /dev/sdX /mnt
4. Монтирую жесткий диск с системой:
mount /dev/sdX /mnt
mount /dev/sdX /mnt/home
mount /dev/sdX /mnt/boot
5. Бэкаплю:
sudo tar -cvzpf /mnt/backup/root.tar.gz /mnt
sudo tar -cvzpf /mnt/backup/home.tar.gz /mnt/home
sudo tar -cvzpf /mnt/backup/boot.tar.gz /mnt/boot

ВОССТАНОВЛЕНИЕ( через archlive)
1. Разбиваю диск
2. Создаю каталоги :

 mkdir /mnt/home
mldir /mnt/boot
mount /dev/sdX /mnt
mount /dev/sdX /mnt/boot
mount /dev/sdX /mnt/home
3. Разархивирую бэкап:
tar --same-owner -xvpf /mnt/backup/root.tar.gz -C/
tar --same-owner -xvpf /mnt/backup/home.tar.gz -C/mnt/home
tar --same-owner -xvpf /mnt/backup/boot.tar.gz -C/mnt/boot
4. Корректирую UUID в /etc/fstab
5. Восстанавливаю grub(если загрузочный сектор не линуксовский)
grub-install /dev/sdХ
