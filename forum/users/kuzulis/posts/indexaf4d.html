Внимание! Всем на заметку!
В вики недостоверная информация! У меня по вики VPN не работал!

Недостоверная в части файла options.pptp
1. В нем параметр noauth должен быть в самом конце файла (в самом низу).
2. Не обязательно указывать noauth и в options.pptp и в chap-secrets, достаточно указать только в options.pptp

Мой пример:
options.pptp
lock
refuse-pap
refuse-eap
#refuse-chap
require-chap //<- тут у меня у провайдера CHAP аутоитентификация (у вас может быть другое)
refuse-mschap
nobsdcomp
nodeflate
noauth

/peers/beeline
remotename PPTP
ipparam beeline
pty “pptp vpn.internet.beeline.ru –nolaunchpppd”
name <тут циферки логина>
usepeerdns
persist
maxfail 0
lcp-echo-interval 60
lcp-echo-failure 4
defaultroute
file /etc/ppp/options.pptp
logfile /var/log/vpn.log

chap-secrets
# Secrets for authentication using CHAP
# client server secret IP addresses
<тут циферки логина> <тут пароль> *

Это так, к слову, чтобы мне не забыть самому на будущее.
