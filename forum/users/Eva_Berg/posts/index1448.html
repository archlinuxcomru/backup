раньше была строчка driver : radeon,теперь вообще непонятно какой драйвер
\
если у тебя такая же история как у меня (видяшка на si или cik) то мало заблеклистить радеон. Надо еще включить amdgpu, он по умолчанию выключен в ядре.
из вики
Set module parameters in kernel command line
Set one of the following kernel parameters:
Southern Islands (SI): radeon.si_support=0 amdgpu.si_support=1
Sea Islands (CIK): radeon.cik_support=0 amdgpu.cik_support=1
тогда и блеклистить радеон не надо - он будет отключен сразу, на уровне kernel parameters.
