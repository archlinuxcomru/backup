както раз я сносил glibc
отличное решение для меня было загрузиться с лайфюсб (для этой задачи удобнее archbang) и установить пакет :
1. примонтировать раздел на котором стоит упавший арч (каламбур) в лайв системе
2. доустановить пакеты, и провести прочие манипуляции с pacman
3. отмонтировать раздел(ы) ( у меня все в кашу на одном физическом разделе )
4. Ребутнуться и радоваться жизни 

sudo mount /dev/sdXY /mnt
sudo pacman -Sy pacman --root /mnt
sudo umount /mnt
