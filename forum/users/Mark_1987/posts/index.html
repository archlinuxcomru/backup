Решилась проблема, добавлением в конфиг i3status следующего текста:

battery 0 {
format = "%status %percentage %remaining"
path = "/sys/class/power_supply/BAT1/uevent"
}

По ссылкам, которые ранее тут скинули, назван другой тип батареи. Поэтому всем, кто столкнётся с аналогичной проблемой, перед тем как вбивать строчку в конфиг посмотрите как называется ваша батарея по пути /sys/class/power_supply/. В моём случае это BAT1.
Откликнувшимся огромное спасибо!
