Собственно да, если пакет sudo установлен, тебе даже не надо под рутом заходить, просто там где надо от имени рута перед командой пишешь допустим sudo gedit /folder/file.txt
ну и чтоб пароль не спрашивало добавь себя в файл /etc/sudoers


# User privilege specification
root ALL=(ALL) ALL
your_login ALL=(ALL) ALL <<<

и раскомментируй эти строчки, чтоб выглядело так:

# Uncomment to allow people in group wheel to run all commands
%wheel ALL=(ALL) ALL

# Same thing without a password
%wheel ALL=(ALL) NOPASSWD: ALL

вот и всё, и любое действие от рута будет выполняться просто дописав sudo перед командой без всяких паролей…
