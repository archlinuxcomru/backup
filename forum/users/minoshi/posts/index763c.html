диск /dev/sda:

/dev/sda1 - > /
/dev/sda2 -> swap
/dev/sda3 -> /home
/dev/sda4 -> расширенный, в нем:
/dev/sda5 -> чистый ext3 раздел, использую под разные нужды, например новый дистр потестировать, постоянно не примонтирован
+ 5 гигов свободного неразмеченного места, на котором я и хотел создать новый раздел с помощью gparted.


работаем в иксах под рутом
запускаем xterm, чтобы видеть чего пишет gparted. Запускаем gparted:
[[email protected] ~]# gparted

======================

Доброго времени суток!

С интересом почитал о archlinux, решил поставить. Установка на тестовом компьютере, который подключен к локальной сети с выходом в интернет через отдельный сервер c прошла без проблем. Иксы пока не ставил. При настройке сети воспользовался Вашим wiki.

проблема вот в чем: с данной машины не могу пропинговать интернет-адреса по их имени
т.е., например, "ping http://www.google.ru “ не проходит - получаем ошибку ”ping: unknown host www.google.ru “, а ”ping <ip-адрес гугла>“ идёт без проблем.

Конфигурация:
сервер - altlinux 4.1desktop, динамический ip-adress и адреса DNS-серверов получаем от провайдера. В и-нет выходим через модем. В локальной сети ip 192.168.2.1. имя ”server"
Маршрутизация настроена с помощью альтовского альтератора и работает - проверено на другой машине с linuxmint5: на ней ping того же http://www.google.ru идет нормально, пописывали ip, маску и адреcа dns-серверов через графическую утилиту Networkmanager.

Наша станция - archlinux 2009.08 (скачан с фтп, система обновлена командой pacman -Syu) настройки ниже

/etc/rc.conf
...
HOSTNAME="station"
eth0="eth0 192.168.2.2 netmask 255.255.255.0 broadcast 192.168.2.255"
INTERFACES=(eth0)
gateway="default gw 192.168.2.1"
ROUTES=(gateway)
...

/etc/resolv.conf
# DNS полученные от провайдера
217.116.128.9 
217.116.129.9 
search server

/etc/hosts
127.0.0.1 localhost.localdomain localhost station
192.168.2.2 station
192.168.2.1 server
#репозиторий - 
85.93.146.110 archlinux.freeside.ru

Я понимаю, что проблема где-то с DNS, но где конкретно копать - нет знаний основы. Подскажите, что нужно поправить?
