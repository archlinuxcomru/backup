Кто нить нашел вменяемый способ сокрытия на квантуме titlebar в KDE? Старые расширения не пашут, а на ноуте несказанно бесит лишнее занимаемое этим заголовком пространство :-(

Воспользуйтесь одной из стандартных возможностей настройки всего и вся в KDE. В данном случае: Клик левый верхний угол по иконке лисы - Дополнительно - Специальные параметры приложения - Внешний вид и обход проблем - Скрыть заголовок и границы окна - Применять изначально (либо другое по желанию) - Да. Перезапустить firefox.

Для закрытия можно использовать (помимо Alt+F4):
Первый способ. Параметры системы - Рабочая среда - Поведение рабочей среды - Края экрана - Настройка действий для краёв экрана. Затем ПКМ по квадратам рисунка в нужном месте и активируем необходимое.
Второй способ, который мне тут на форуме настоятельно порекомендовали (за что очень благодарен) - использование easystroke для управления жестами мыши. Замечательная штука! Кстати, с успехом обходит ограничения web-extention по управлению "внутренними" страницами самой лисы.
P.S. Чтобы вернуть всё назад выполнить Alt+F3 — открывает меню действий для активного окна.

Если не получится - придется настраивать загрузку Windows. (С установочного винды)
