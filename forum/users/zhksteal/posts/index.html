Ок, убедили, спасибо.
При установке mariadb следовал мануалу:

Install mariadb, afterwards run the following command before starting the mysqld.service:
# mysql_install_db --user=mysql --basedir=/usr --datadir=/var/lib/mysql
Собственно, под zabbix сервер БД хранится в /var/lib/mysql

Как безболезненно перенести БД, например, в /home/mysql?

Верно ли следующее:


cd /var/lib/
cp -a mysql /home/
rm -rf mysql/*
echo "/home/mysql /var/lib/mysql none bind 0 0" >> /etc/fstab
mount -a
