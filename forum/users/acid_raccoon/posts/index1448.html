Вот пример разметки и монтирования для uefi/gpt:

Разметка диска (cfdisk/gpt/uefi) cfdisk > gpt
1. Раздел esp:/dev/sdx1
Free space > New > 100M > Enter
Выбрать созданный раздел > Type > EFI System
2. Раздел / - /dev/sdx2
3. Раздел /home - /dev/sdx3
4. Выбрать Write > yes :чтобы сохранить изменения.
5. Quit > Enter

2. Форматирование файловых систем:
mkfs.fat -F32 /dev/sdx1 - esp
mkfs.ext4 /dev/sdx2 - /
mkfs.ext4 /dev/sdx3 - /home

3. Монтирование файловых систем:
df -h - выводит список примонтированых файловых систем.

Чтобы установить файлы системы на только что созданные разделы, их нужно сначала примонтировать. Монтировать будем в /mnt. Сначала корень.
mount /dev/sdx2 /mnt - /

mkdir -p /mnt/boot/efi
mount /dev/sdx1 /mnt/boot/efi - esp

mkdir /mnt/home
mount /dev/sdx3 /mnt/home - /home

-p или --parents Создать все директории, которые указаны внутри пути. Если какая-либо директория существует, то предупреждение об этом не выводится.
