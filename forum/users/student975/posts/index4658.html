Проблема такая. coretemp показывает два фиксированных значения температуры для обоих ядер. То есть они всегда такие фиксированные, и после перезагрузки. От нагрузки на CPU (и вообще ни от чего) не зависия.

Может кто-нибудь проверить, что coretemp с этим процессором вообще работает? Для этого:

sudo modprobe coretemp

После чего несколько раз посмотреть:

sudo cat /sys/devices/platform/coretemp.0/temp1_input

Вывод там в тысячных долях градусов по Цельсию. Значение должно немного плавать даже без какой-то особенной нагрузки.
