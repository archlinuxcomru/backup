Я выходил в Астрал и смотрел, что линукс ждёт.
Через 3-6 месяцев будет миграция dbus в systemd. Или, точнее, systemd вытеснит dbus.
Через 9-12 месяцев userspace приложения начнут использовать systemd для общения и взаимодействия.
Через 12-15 месяцев systemd станет контейнером для запуска wayland, классический xorg будет заморожен, userspace ПО будет запускаться только в wayland.
Все многочисленне WM умрут, т.к. там будут запускаться только древние, сильно устаревшие, браузеры.
Через 15-30 месяцев останутся только 3 DE, kde, gnome и unity. Большинство дистров умрёт, так как между ними почти не будет отличий.
После 30 месяцев, остнется только одно DE с вариантом под планшет и десктоп. Т.е. получится 8 винда. Тем временем майкрасофт выпустит уже 10, и про линуксы опять будут говорить, что он отстает от винды на много лет. Впрочем, всем на него уже будет как-то пофиг.

И да, самое важное забыл! На systemd будет завязаны всевозможные лицензионные проверки, а софт будет платным(зато MS портирует свой офис нативно под linux).

Особенно мне доставляет вот этот бред с Арч Вики
Согласно философии The Arch Way (“Путь Arch”) Arch Linux остается легковесным, гибким, простым и максимально UNIX-подобным. После установки предоставляется минимальная среда (без графического интерфейса) для архитектур i686/x86-64: вместо удаления ненужных пакетов, опытный пользователь имеет возможность построить систему, начиная с минимальной основы без каких-либо заранее выбранных шаблонов. Философия дизайна и реализации Arch Linux позволяет легко расширять систему для получения нужных возможностей: от минимальной консольной системы до современной сложной и функциональной настольной среды: именно пользователь решает, какой будет система на базе Arch. Начальная загрузка системы Arch'а во многом напоминает таковую у ОС семейства BSD: все вызовы производятся из единого файла (rc.conf) вместо структуры каталогов System V, содержащей множество символических ссылок для каждого runlevel'а. Конфигурация системы определяется путем редактирования простых текстовых файлов.
