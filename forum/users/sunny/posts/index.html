Поднял иксы, установил проприетарный драйвер nvidia через pacman

pacman -S nvidia

поставил gnome 3.
При попытке зайти гном падает “Oh no! Something has gone wrong!”.
На англофорумах нашёл такой способ 

irst make sure you have the ~/.config/autostart folder, if not create it. Then edit (or create) the gnome-shell.desktop file and insert the following:
[Desktop Entry] 
Type=Application 
Exec=gnome-shell --replace 
Hidden=false 
NoDisplay=false 
X-GNOME-Autostart-enabled=true 
Name[en_US]=Gnome Shell 
Name=Gnome Shell 
Comment[en_US]= 
Comment=

После этого гном не падает, а тупо висит. Нету ничего, тупо обоина. и тока по ctrl+alt+del можно выйти.
Как такое решить?
