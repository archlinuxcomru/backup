зачем понадобилась старая версия реммины?
из-за ругани удалённой винды на сертификат?
Новые версии remmina работают с артефактами при запуске 1С - где-то поля съезжают, где-то просвечиваются окна.
Также при использовании новых версий, после каждого выхода из подключения RDP, для повторного подключения
к удаленному серверу приходится удалять папку freerdp в .config - иначе выдает ошибку соединения (на StackOverflow про это есть упоминание).
Также новые версии remmina быстро теряют подключение если окно свернуть, и далее приходится все заново перезапускать с удалением папки freerdp.

Единственная версия работающая без фризов, артефактов, стабильно держащая подключение и без необходимости каждый раз удаления папки freerdp
для повторного запуска это 1.20-29 из ubuntu.

При подключении же через xfreerdp получается добиться стабильности соединения, но артефакты и необходимость удалять каждый раз папку freerdp
остаётся.
