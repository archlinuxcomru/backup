Пользователям Ubuntu 12.04 можно обойтись без сторонних репозиториев, GNOME 3.4 уже доступен в стандартных PPA.

Пользователи Ubuntu 11.10 могут воспользоваться следующими командами:

sudo add-apt-repository ppa:ricotz/testing
sudo add-apt-repository ppa:gnome3-team/gnome3
sudo apt-get update
sudo apt-get dist-upgrade
sudo apt-get install gnome-shell
После этого нужно выйти из системы и зайти заново, выбрав GNOME Shell.
Браво!
