Пробовал прописать ppp0 в ROUTES

а в etc/ppp/pppoe.conf что прописано?
# Make the PPPoE connection your default route. Set to
# DEFAULTROUTE=no if you don't want this.
DEFAULTROUTE=yes

по поводу автозапуска, ЕМНИП в pppoe-setup был такой пункт, перепроверьте
можно еще прописать в /etc/sudoers
user ALL=(ALL) NOPASSWD: /usr/sbin/pppoe-start, /usr/sbin/pppoe-stop
и сделать кнопку на скрипт 
#!/bin/bash
if
pppoe-status
then
sudo pppoe-stop
else
sudo pppoe-start
fi

кнопочка будет вкл/выкл pppoe
