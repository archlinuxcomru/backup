Grub4dos'ом грузишь parted magic и с помощью gparted создаёшь на флешке второй раздел в ext2 с лейблом ARCH_201005. Монтируешь образ Арча и копируешь содержимое на созданный раздел. Загружаешь Арч grub4dos'ом с первого раздела.

title Parted Magic 5.7
find --set-root /flashdrive.tag
map --heads=0 --sectors-per-track=0 /pmagic/pmagic-5.7.iso (hd32)
map --hook
root (hd32)
chainloader (hd32)
title Arch Linux 2010.05
find --set-root /flashdrive.tag
root (hd0,1)
kernel /boot/vmlinuz26 archisolabel=ARCH_201005 tmpfs_size=75% locale=en_US.UTF-8
initrd /boot/archiso.img

Arch Linux USB Install and Rescue Media.
Grub4dos wiki.
