Та же самая ситуация после обновления. Только не /home страдает, а другой раздел. Плюс иногда fstab не монтирует другой один раздел.

[DEPEND] Dependency failed for File System Check on /dev/disk/by-uuid....
[DEPEND] Dependency failed for /mnt/Media
[DEPEND] Dependency failed for Local File Sistem

mkinitcpio -p linux ошибок не дает.
Проверка fsck этого раздела ошибок не дает.
Комментирование этого раздела в fstab решает проблему. Система грузится без emergency mode.

fstab
# <file system> <dir> <type> <options> <dump> <pass>
# /dev/sda4 LABEL=Arch
UUID=24790963..........................       /               ext4            rw,relatime,data=ordered,discard        0 1

# /dev/sda1
UUID=FC75........        /boot/efi       vfat            rw,relatime,fmask=0022,dmask=0022,codepage=437,iocharset=iso8859-1,shortname=mixed,utf8,errors=remount-ro       0 2

# /dev/sdb1 Photo
UUID=1879f9cc......................      /mnt/Photo      ext4            rw,relatime,data=ordered        0 2

#/dev/sdb2 Media
UUID=8a74220.......................     /mnt/Media      ext4            rw,relatime,data=ordered        0 2

#/dev/sdc1 Music
UUID=1f15c06......................       /mnt/Music      ext4            rw,relatime,data=ordered        0 2

#/dev/sdc2 Films
UUID=b316e58e...................      /mnt/Video      ext4            rw,relatime,data=ordered        0 2
