Если интересно можно разобраться откуда модуль
сделать скриптик helpdbus
#!/usr/bin/env python
import dbus
help (dbus)
И запустить → в секции FILE должен быть путь до модуль
Что-то нету пути:
$ python helpdbus |grep -A 20 FILE
FILE
    (built-in)
catfish запускается.
А, понял:
python2.7 helpdbus |grep -A 20 FILE
FILE
    /usr/lib/python2.7/site-packages/dbus/__init__.py
