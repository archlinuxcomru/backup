At the user level, the heart of Wine's audio support can be found in the ~/.wine/config file.
[WinMM]
; Uncomment the "Drivers" line matching your sound setting.
"Drivers" = "winealsa.drv"    ; for ALSA users
;"Drivers" = "wineoss.drv"     ; default for most common configurations
;"Drivers" = "winearts.drv"    ; for KDE
;"Drivers" = "winejack.drv"    ; for the JACK sound server
;"Drivers" = "winenas.drv"     ; for the NAS sound system
;"Drivers" = "wineaudioio.drv" ; for Solaris machines
;"Drivers" = ""                ; disables sound
"WaveMapper" = "msacm.drv"	 ; do not change !
"MidiMapper" = "midimap.drv"	 ; do not change !
Так что по моему Wine просто выводит звук через выбранную подсистему Linux.
А поддержка железа - уже забота этой подсистемы.
