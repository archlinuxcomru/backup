Возможно система не может сама определить используемую архитектуру.
Тогда просто подставьте сюда нужное сами (вместо правки строчек с репами):
Architecture = auto | i686 | x86_64 | …
The special value “auto” will use the system architecture, provided by in “uname -m”.
If unset, no architecture checks are made.
