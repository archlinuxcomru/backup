Для тех, кто столкнулся с проблемой настройки этого устройства привожу решение.

Чтобы модем распознался как cd-rom:
sudo modprobe -r usb_storage
sudo modprobe usb_storage option_zero_cd=2

Чтобы переключить модем из режима cd-rom в режим модема ( http://www.draisberghof.de/usb_modeswitch/ ):
раскоментируем строки нужного модема в /etc/usb_modeswitch.conf (# AnyDATA ADU-500A, ADU-510A, ADU-510L, ADU-520A)
sudo usb_modeswitch

Затем, чтобы работать со skylink нужно изменить /etc/ppp/options:
mru 296
вместо proxyarp поставь noproxyarp

Дальше просто используем gnome-ppp:
в качестве модема указываем устройство /dev/ttyUSB0
