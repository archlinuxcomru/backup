Сколько потратили
Если раньше ожидалось, что покупка обойдется Microsoft в 5 миллиардов долларов, то теперь оказалось, что корпорация заплатила в полтора раза больше — 7,5 миллиарда. Эту сумму Microsoft отдаст своими акциями.

О чем договорились
После сделки платформа GitHub продолжит работу независимо и останется открытой. По крайней мере, так уверяют в Microsoft.

«Во-первых, мы будем расширять возможности разработчиков на каждом этапе жизненного цикла разработки — от идеи до совместной работы и развертывания в проекта в облаке. GitHub останется открытой платформой, доступной для всех разработчиков. Во-вторых, мы будем развивать использование GitHub корпоративными разработчиками, используя наш отдел продаж, партнерские каналы, а также предлагая доступ к глобальной облачной инфраструктуре и услугам Microsoft. В итоге мы предложим более широкой аудитории инструменты и сервисы Microsoft для разработчиков», — подчеркнул глава Microsoft Сатья Наделла.

Кроме того, генеральным директором GitHub станет вице-президент Microsoft Нэт Фридман. Основатель сервиса Крис Ванстрат, который еще с августа прошлого года никак не мог покинуть пост, займет должность технического специалиста Microsoft.

Все формальности планируют уладить до конца года.

К слову, GitHub пользуются крупнейшие компании мира, в числе которых Apple, Amazon, Google и сам Microsoft. Последний даже разместил здесь исходный код своего приложения Windows File Manager.
