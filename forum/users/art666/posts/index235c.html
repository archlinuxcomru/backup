Не, переменную $5 нужно указывать в скрипте /etc/ppp/ip-up.d/_название_ .sh
Там она будет действовать :) Сделай: poff(если конектился) , создай скрипт с содержанием выше, сделай его исполняемым: sudo chmod +x /etc/ppp/ip-up.d/_название_ .sh
И все запускай конект: pon $Тунель.
А если командывать в терминале, то каждый раз нужно: 
ifconfig
#посмотреть IP после P-t-P у интерфейса ppp0
ip ro del default
ip ro add default via $IP после P-t-P dev ppp0
