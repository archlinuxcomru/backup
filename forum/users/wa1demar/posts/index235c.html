Посмотрите на права /var/lib/mysql, вам туда нельзя.

Скорее всего, у вас не обновлен файл /etc/mysql/my.cnf . Там должно быть:
socket          = /run/mysqld/mysqld.sock
Лучше забекапить/грохнуть файл настроек и скопировать дефолтный из /etc/mysql/my.cnf.pacnew.

ls -al | grep mysql
drwx------  9 mysql   mysql   4096 мар 28 14:08 mysql


socket		= /var/lib/mysql/mysql.sock
