.xinitrc
#!bin/sh
#
# ~/.xinitrc
#
#Executed by startx (run your window manager from here)
if [ -d /etc/X11/xinit/xinitrc.d ]; then
  for f in /etc/X11/xinit/xinitrc.d/*; do
    [ -x "$f" ] && . "$f"
  done
  unset f
fi
#exec gnome-session
#exec startkde
exec startxfce4
# ...or the Window Manager of your choise

Запускаю без демона( пробовал и с gdm и со slim, одна фигня ) inittab не трогал. Пароль ввожу правильно( как ни странно ). При startx мограет экран и выдает ошибку:
