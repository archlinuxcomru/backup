у них вроде сборка неудачная получилась, не ставится бург. Решается просто:
грузишься с лайв-сд чакры и
mkdir /mnt/arch
mount /dev/<рутовый раздел вписАть> /mnt/arch
cd /mnt/arch
mount -o bind /dev dev/
mount -t proc proc proc/
mount -t sysfs sys sys/
mount /dev/<раздел boot монтируем,если есть> boot/
chroot . /bin/bash
(опционально) pacman -Syy pacman -S burg (чтобы нового бурга установить, если старый косячит)
(обязательно) burg-install /dev/sd* (sda если хотите в мбр загрузчик)

Я еще убрал “LOADING INITRAMFS” надпись

1. open (as root) the file /lib/initcpio/init
2. comment out line msg “:: Loading Initramfs”
и еще несколько, начинающихся с msg…
3. rebuild initramfs (mkinitcpio blabla)
all done
PS: when pacman updates the mkinitcpio package, it will overwrite this file.
