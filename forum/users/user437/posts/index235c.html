Конкретно для Вашего телефона официальная разблокировка так не проходит? - http://4pda.ru/forum/index.php?showtopic=260549

На самом деле все просто -

1. региться на том сайте
2. иметь только fastboot, adb и никакие dll не нужны
3. телефон выключен.
2. Зажмите одновременно и держите кнопки питания и уменьшения громкости.
3. При помощи клавиш звука выберите пункт «Fastboot» и нажмите кнопку питания.
4. Соедините ваш аппарат с компьютером при помощи USB-кабеля.

имеем черный экран и какой-нибудь горящий светодиод. Идем в терминал, su

./fastboot oem get_identifier_token

10. Вы увидите один из экранов выше — это ключ. Необходимо нажать правую кнопку мыши, выбрать «пометить», выделить блок и нажать правую кнопку для копирования. Ключ должен начинаться с
<<<< Identifier Token Start >>>>
и заканчиваться
<<<<< Identifier Token End >>>>>
Затем необходимо перейти на официальный сайт (потребуется регистрация), пролистать до Step 10, вставить в поле My Device Identifier Token ваш ключ, и нажать Submit.
11. Спустя несколько секунд на адрес электронной почты, указанный при регистрации, придет сообщение с ключом разблокировки в аттаче — Unlock_code.bin.
Сохраняем файл в папку с фастубтом
скармливаем фастбуту файл -

./fastboot flash unlocktoken Unlock_code.bin
Ничего рискового тут нет. Полностью затрутся пользовательские данные - их можно попытаться сохранить Титаниумом.

ыыы.... давай еще раз с самого начала.
жирные зеленые бочки с глазами андроиды бывают разные. Одних фиг сломаешь, другие сразу отдаются через адб с правами рута.

Внимание вопрос: зачем вам ломать загрузчик? я видел два способа как засунуть левый файл с suid флагом в файловую систему телефона:
1. подменить образ recovery (поставить всенародно любимый clock work mode). делается это обычно с помощью адб +фастбут. и именно эти команды скорее всего будут находиться в скрипте. возможно перед ними будет команда разблокировки загрузчика. (Ибо все делается для домохозяек которым написать два слова в бело черной консоли это адская пытка.)

2. подсунуть необходимые файлы и сделать фиктивное восстановление данных. Скорее всего уже неактуально потому как одно из обновлений "зеленого человечка с мороженкой" явно закрывало эту дырку. В более поздних версиях зеленого ведра рутование телефона идет исключительно первым способом.
