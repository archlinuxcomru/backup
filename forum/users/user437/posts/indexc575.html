Попробуй установить catalyst-total из AUR.
Для этого надо:
1) Добавить в /etc/pacman.conf ВЫШЕ ВСЕХ остальных репозиториев:
[xorg113]
Server = http://catalyst.wirephire.com/repo/xorg113/$arch
2) sudo pacman -Syuu
3) yaourt catalyst-total (ничего больше)
4) sudo systemctl enable atieventsd
5) sudo systemctl enable catalyst-hook
6) убедиться в наличии nomodeset : https://wiki.archlinux.org/index.php/Catalyst#Disable_kernel_mode_setting
7) sudo aticonfig -f --initial
8) reboot
