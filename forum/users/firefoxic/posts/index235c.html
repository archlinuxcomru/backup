Рекомендую попробовать дополнение InstantFox. Совершенно новый уровень поиска. Строка поиска вообще не нужна больше будет (да и убирается она при установки дополнения).
Из возможностей:
  • лёгкое добавление любых поисков находясь на сайте с поисковым движком (полезно добавить поиски на rutracker, lurkmore, яндекс.словари, а также с ресурсов Арча);
  • поиск выделенных слов через контекстное меню;
  • регулировка показа в контекстном меню (можно иметь много поисковых плагинов, но не захламлять контекстное меню), а также положения в нём простым перетаскиванием;
  • задание сокращения для быстрого поиска в AwesomeBar (адресной строке), например rt для rutracker, с возможностью включения/отключения моментального показа результатов (отключение полезно для сайтов не поддерживающих эту фичу).

Дополнение из разряда "Ё-моё! Как же я до этого мучился!"
