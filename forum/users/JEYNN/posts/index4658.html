А по делу дам вам инструкцию:
# pacman -S bbswitch bumblebee nvidia primus xf86-video-intel
# gpasswd -a ПОЛЬЗОВАТЕЛЬ bumblebee
# systemctl enable bumblebeed.service
# reboot
Если у вас 64-битная система, нужно установить еще 32-битные библиотеки (я также устанавливал пакеты lib32-intel-dri, lib32-nvidia-utils, lib32-opencl-nvidia, lib32-primus, lib32-virtualgl). После перезагрузки для проверки на работоспособность можно использовать эти команды:
$ systemctl status bumblebeed
$ optirun --status
