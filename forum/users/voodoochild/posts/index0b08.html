изменить, на арчевом диске, в /etc/fstab
/dev/sda1 / ext3 defaults 0 1
/dev/sda3 swap swap defaults 0 0
/dev/sda4 /home ext3             rw,suid,exec,auto,nouser,async   0  2

/dev/sdb3 swap swap defaults 0 0
/dev/sdb4 /home ext3             rw,suid,exec,auto,nouser,async   0  2

Проблема решилась установкой rfkill
UPD: rfkill помог, но не на долго. Проблема оказалась в маршрутизаторе, вернее в его настройках.
Подключение происходило по стандарту 802.11n, но точка доступа находилась в режиме Hidden SSID.
Выключил Hidden SSID - и все стало работать должным образом.
