Если у вас изначально установлена Windows 8 Pro/Ent то можно делать downgrade. В случае если стоит Basic то по лицензии нельзя. Кстати обычно в таких восстановительных образах активация Windows 7 либо уже активированная или же активация выполняется на этапе установки с помощью файла сертификата ключа. Активировать Windows 7 можно и без Интернета, главно разжиться файлом сертификатом и запустить команду активации с помощью файла.
