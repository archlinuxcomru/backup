Но новые веяния меня как минимум озадачили, на мой взгляд арч уже не способен позиционироваться как “простой” дистр “для знающих”.

Очень интересно, что все видят и, соответственно, требуют “простоты”, забывая, что имеется в виду простота техническая, а это совершенно не подразумевает простоты для людей не желающих учиться, о чем в вики очень внятно и однозначно написано и забывают о “знающих”, что, как минимум, требует понятия о принципах функционирования системы.
Для людей желающих освоить Арч есть очень даже неплохой дистр Manjaro http://blog.manjaro.org/ полностью совместимый с текущим Арчем и позиционируемый как “для новичков”, командный установщик в наличии, правда на ангельском языке, с разными десктопами - пробуйте, может меньше будет вопросов при установке.
Manjaro - дистирибутив, основанный на Arch Linux и ориентированный на начинающих пользователей. Дистрибутив доступен в сборках KDE, GNOME, Cinnamon и XFCE. Выпуски Manjaro основаны на дополнительно протестированных срезах репозитория Arch Linux и полностью совместимы с данным дистрибутивом. Для управления репозиториями в Manjaro используется собственный инструментарий BoxIt, спроектированный по образу Git. Репозиторий поддерживается по принципу непрерывного включения обновлений (rolling), но новые версии проходят дополнительную стадию стабилизации.

9.3.5. Создание собственной раскладки
Если вас не устраивает ни одна из тех раскладок клавиатуры, которые имеются в каталоге /usr/lib/kbd/keytables/i386/qwerty/, можете попробовать подправить ту раскладку, которая ближе всего к вашему идеалу. Попробуем показать, как это делается, на примере выбора клавиши переключения между русской и латинской клавиатурой (этот совет позаимствован у Романа Минакова, [email protected]).
Для переключения между русской и латинской клавиатурой часто используется правая клавиша <Ctrl>...
Почитайте http://rus-linux.net/book1.php?name=book1/gl-09/gl_09_03.html а потом с нами поделитесь раскладкой :)

Самый простой и быстрый метод вам подсказали: dd. Для расширения кругозора всё, что связано с установкой чего-либо с флешки лучше читать на http://greenflash.su, конкретно по арчу http://greenflash.su/forum/2-1097-1. Я делаю так:
1. Копирую папку arch с образа iso в корень флешки.
2. Ставлю на флешку grub4dos.
3. В конфигфайл grub4dos добавляю:
title  Boot Arch x64 Linux\nBoot the Arch Linux live medium. It allows you to install Arch Linux or perform system maintenance.
kernel /arch/boot/X86_64/vmlinuz archisobasedir=arch archisolabel=Метка_вашей_флешки
initrd /arch/boot/X86_64/archiso.img
title  Boot Arch i686 Linux\nBoot the Arch Linux live medium. It allows you to install Arch Linux or perform system maintenance.
kernel /arch/boot/i686/vmlinuz archisobasedir=arch archisolabel=Метка_вашей_флешки
initrd /arch/boot/i686/archiso.img
Грузимся, ставим.
