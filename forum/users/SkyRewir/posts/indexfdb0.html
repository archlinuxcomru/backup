HISTCONTROL="команда"

HISTCONTROL=options1:option2 без скобок
Список опций:
ignorespace не сохранять строки начинающиеся с символа <пробел>
ignoredups не сохранять строки, совпадающие с последней выполненной командой
ignoreboth использовать обе опции ‘ignorespace’ и ‘ignoredups’
erasedups удалять ВСЕ дубликаты команд с истории

Проблему с доступом решил, исправил /etc/httpd/conf/extra/phpmyadmin.conf на

Alias /phpmyadmin "/usr/share/webapps/phpMyAdmin"
<Directory "/usr/share/webapps/phpMyAdmin">
    DirectoryIndex index.php
    AllowOverride All
    Options FollowSymlinks
    Require all granted
</Directory>
Перезагрузил apache

sudo systemctl restart httpd
Перешел по адресу http://localhost/phpmyadmin, выдало ошибку

Error during session start; please check your PHP and/or webserver log file and configure your PHP installation properly.
