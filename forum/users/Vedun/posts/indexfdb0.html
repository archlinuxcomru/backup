Сейчас OpenBox ставил, вообще загрузился серый экран, недождавшись ничего кликнул меню, там некоторые проги и ссылки на несуществующие объекты.

В Openвox по умолчанию загружается стандартное меню и стандартный файл автозапуска
~/.config/openbox/autostart
~/.config/openbox/menu.xml

эти файлы надо править по своему вкусу,
например в автостарт ( autostart ) добавить то что у вас управляет обоями и будет экран не "серый"...

в меню ( menu.xml ) лишнее тоже убирается руками, но есть и пару прог гуи для этого...

раскомментируйте любое из этих (как второе) и добавьте себе в userChrome.css

/* Прозрачность выгруженных вкладок
  .tabbrowser-tab[pending] {
  opacity: 0.3; !important;
} */

/* Серый цвет значков выгруженных вкладок */
  .tabbrowser-tab[pending] {
 filter: grayscale(100%) contrast(10%) !important;
}

/* Цвет выгруженных вкладок
  .tabbrowser-tab[pending] {
 background: darkgrey !important;
} */

/* Цвет шрифта выгруженных вкладок
  .tabbrowser-tab[pending] {
 color: darkgrey !important;
} */

/* Шрифт выгруженных вкладок
  .tabbrowser-tab[pending] {
