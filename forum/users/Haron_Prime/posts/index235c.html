Вот тут можно посмотреть статус зеркал
Сам использую зеркало http://mirrors.n-ix.net/archlinux/ - быстрое (скорость загрузки - 10-12 MiB/s), актуальное (отставание от официального минимально), стабильное
Ну и для подстраховки несколько добавлено
Server = http://mirrors.n-ix.net/archlinux/$repo/os/$arch
Server = https://mirror.js-webcoding.de/pub/archlinux/$repo/os/$arch
Server = https://mirror.f4st.host/archlinux/$repo/os/$arch
Server = http://ftp.nluug.nl/os/Linux/distr/archlinux/$repo/os/$arch
Server = https://archmirror.tomforb.es/$repo/os/$arch
Server = https://mirrors.kernel.org/archlinux/$repo/os/$arch
Server = https://lug.mtu.edu/archlinux/$repo/os/$arch
Server = https://mirrors.ocf.berkeley.edu/archlinux/$repo/os/$arch
