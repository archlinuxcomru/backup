Попробуйте через netctl.

Вроде пробовал уже через него, но сев с утра и не много разобравшись "повнимательнее" сделал! :)
Вот что было нужно:
cd /etc/netctl/examples/
sudo cp wireless-wpa ..
cd ..
sudo nano wireless-wpa
прописываем название роутера и пароль
sudo netctl start wireless-wpa
sudo netctl enable wireless-wpa
sudo pacman -S wpa_actiond
sudo systemctl enable [email protected]

Добрый вечер!

Не могу настроить авто-подключение по Wi-Fi при загрузке i686. Приходится постоянно прописывать
sudo ip link set wlp5s0 up
sudo wpa_supplicant -B -D nl80211 -i wlp5s0 -c /etc/wpa_supplicant/wpa_supplicant.conf
sudo dhcpcd -A wlp5s0
