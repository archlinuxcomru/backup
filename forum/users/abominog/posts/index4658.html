tumanovalex
1. Правильно ли я понял из man modprobe, что эта команда добавляет модули (я так понимаю, что имеются ввиду драйверы) из ядра?
Попробуй отредактировать /etc/rc.conf
Добавь в MODULES=(……….. floppy)
Драйвер флоповода тогда загрузиться автоматом при старте системы и не нужно его будет всякий раз подгружать командой modprobe.

Что касаеться флешек и оптических дисков, то нужно предварительно создать для них точки монтирования. Например /media/cdrom и /media/flash
Затем прописать соответствующие команды в /etc/fstab

у меня это выглядит так:

/dev/cdrom /media/cdrom auto ro,user,noauto,unhide 0 0
/dev/dvd /media/dvd auto ro,user,noauto,unhide 0 0
/dev/fd0 /media/fd0 auto user,noauto 0 0
/dev/sdc1 /media/flash auto user,noauto 0 0


После этого практически любой графический файловый менеджер будет монтировать их по клику мышкой из каталога /media. Ну а если уж хочешь полный автомат на монтирование-размонтирование, то тогда грузи демон HAL
