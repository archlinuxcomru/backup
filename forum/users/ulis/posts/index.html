С установкой плагина нужно согласиться (это плагин есть firmware для принтера). Eсли не скачивается, пробуем скачать и установить руками.
sudo hp-plugin -i
sudo hp-firmware -u
или если совсем плохо
wget -c http://www.openprinting.org/download/printdriver/auxfiles/HP/plugins/hplip-3.10.5-plugin.run
chmod u+x ./hplip-3.10.5-plugin.run
sudo ./hplip-3.10.5-plugin.run
sudo hp-firmware -u
Затем 
 sudo /etc/rc.d/cups restart
Потом открываем в браузере http://localhost:631 и добавляем принтер. Можно использовать какой нибудь фронтенд к CUPS, например в моем случае это Gnome Cups Manager.
Можно использовать альтернативный вариант с драйвером (foo2xqx) приведенный выше.

Ошибка монтирования гномовская.
Я так понимаю сначала раздел монтируется при старте системы согласно fstab, а потом еще раз Гном пытается монтировать этот же раздел.
Может убрать строку /dev/sda2 /media/General ntfs-3g defaults,locale=ru_RU.utf8 0 1 из fstab, и посмотреть в сторону настроек gconf-editor.
