Haron_Prime
пришлось прописать в .xinitrc следующую строку
[[ -f ~/.Xresources ]] && xrdb -merge -I$HOME ~/.Xresources
без неё терминалы не подхватывали настройки
Если посмотреть сейчас в /etc/X11/xinit/xinitrc, то там уже немного по другому.
sysresources=/etc/X11/xinit/.Xresources
userresources=$HOME/.Xresources

if [ -f $sysresources ]; then
    xrdb -merge $sysresources
fi

if [ -f "$userresources" ]; then
    xrdb -merge "$userresources"
fi
Теперь можно и общие для всех пользователей параметры задать...
