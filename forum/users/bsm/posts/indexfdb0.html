lxsession-default-apps

https://wiki.archlinux.org/index.php/LXDE

На данный момент ситуация такова-
-имеется файл /home/archlinux/install/X/conky/rc.conky.sh запуска conky:

#!/bin/sh
#

USER_CONFIG="${HOME}/.conkyrc"   # конфиг. конкретного пользователя
DEFAULT_CONFIG=/conky.conf		    # конфиг общий для всех

[ -f $USER_CONFIG ] && DEFAULT_CONFIG=$USER_CONFIG

sleep 20
conky --config=$DEFAULT_CONFIG
В файле /etc/xdg/lxsession/LXDE/autostart указано:

@lxpanel --profile LXDE
@pcmanfm --desktop --profile LXDE
@xscreensaver -no-splash
@conky --config=/config.conf
@setxkbmap 'us,ru' -option 'grp:ctrl_shift_toggle,grp_led:scroll'

# 15.02.2019 - т.к. не выполняется строка "@conky --config=/config.conf"
#	то сделаем так-
#
@/home/archlinux/install/X/conky/rc.conky.sh
