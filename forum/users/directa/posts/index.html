Вообщем, я создал файлы /etc/asound.conf и ~/.asoundrc (первый, я почему-то думал, у меня существует) и добавил это
pcm.pulse {
    type pulse
}
ctl.pulse {
    type pulse
}
pcm.!default {
    type pulse
}
ctl.!default {
    type pulse
}
