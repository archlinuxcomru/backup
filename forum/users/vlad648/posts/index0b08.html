Можно пока воспользоваться следующим костылём: сделать текст во всех пунктах меню чёрным. Изменить нужно значение fg в Aldabra/gtk-2.0/gtkrc, чтобы получилось вот так:
style "menu_item_white" {
	xthickness	= 2
	ythickness	= 4
	fg[PRELIGHT]	= "#000"
	text[PRELIGHT]	= @selected_fg_color
	engine "murrine" {
		border_shades = {1.5, 1.5}
	}
}
Да, костыль работает, но, как следствие, цвет текста любого выделенного пункта меню стал вместо белого черным. Что ж, пусть лучше будет как есть :)
