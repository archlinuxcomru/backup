В ауре ж инструкция висит пошаговая. Нужно ядро собрать под каталист, потом да - ксорг-максимиз патченный и радоваться жизни.
At last! thankyou snack and the unknow nick. At last! at last!
Here what i do: (guide for dummies)
0.- Download https://www.qdrive.net/download/shareli … XJv8qPsO7X (thanks snack)
1.- tar -xzvf kernel26-NICK.tar.gz
2.- cd kernel-NICK
3.- nano PKGBUILD
4.- Change pkgname=kernel26-NICK for pkgname=kernel26-catalyst. Is the name of the new kernel. Save
5.- makepkg
6.- Wait… wait… wait… end, new kernel is make
7.- As root: pacman -U kernel26-catalyst-2.6.30.4-1-i686.pkg.tar.gz
8.- As root: nano /boot/grub/menu.lst
9.- Add an entry for the new kernel, save.
10.- Reboot
11.- Make from AUR catalyst-utils, catalyst and xorg-server-catalyst-maximize-fix. No need any extra patch except the default. The Unknown symbol flush_tlb_page error is solved with the new kernel (thanks snack). Install all with pacman -U
12.- For create a xorg.conf, in a terminal as root: aticonfig –initial –input=/etc/X11/xorg.conf
13.- Reboot
14.- fglrx load, compiz-fusion work, i play alienarena wonderfull.
