Установите (если ещё не установлен) пакет dkms (Dynamic Kernel Modules System). После этого выполните:

sudo systemctl start dkms

Если хотите, можете прописать "в автозагрузку":

sudo systemctl enable dkms

чтобы модули собирались сами после обновления ядра. Или вручную запускайте первую команду.
