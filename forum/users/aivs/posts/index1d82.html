Здравствуйте, хочу реализовать быстрое переключение между tty экранами нажатием одной кнопки, например F12. у меня задействовано 3 экрана
1) на tty7 - xbmc
2) на tty8 - openbox & tint2
3) на tty9 - firefox с одним сайтом.
Для выполнения команды с помощью F12, нужно установить xbindkeysrc, создать ~/.xbindkeysrc с содержанием:
"script_tty_change.sh"
F12
запустить xbindkeysrc, после этого при нажатии на F12, если экран 7, то он должен переключиться на 8, если 8, то на 9, а если 9 то на 7.
Вот с bash у меня туго помогите пожалуйста, написать мини скрипт!!!
Для переключения экрана используется команда chvt, чтобы переключиться на 8 экран, выполняем команду:
chvt 8

Иксы запускаются у меня на tty7 поэтому первоначальный номер экрана известен
Вот мой алгоритм:

Int x=7

if (x==7);
then
chvt 8;
x=8;
break;

if (x==8);
then
chvt 9;
x=9;
break;

if (x==9);
then
chvt 7;
x=7;
break;

Это смесь С и pascal, а хотелось бы что-то вроде того видеть на bash!! Подскажите!

  one           576  1657
  zero          576   545
  ptrail        587
