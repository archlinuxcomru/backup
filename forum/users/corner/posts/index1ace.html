В данном случае, наверное, проще создать файл с именем карты в /etc/network.d/interfaces.
Например, для карты eth0 будет файл /etc/network.d/interfaces/eth0.
В нем прописать 
PRE_UP='ip link set dev eth0 address 00:0c:11:22:33:44'
или
PRE_UP='ifconfig eth0 hw ether 00:0c:11:22:33:44'
- это если не установлен iproute
