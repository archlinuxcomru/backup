Пропишите в rc.conf
аналогичную строку
предварительно установив hal dbus
Например
DAEMONS=(dbus hal @syslog-ng @network @netfs @crond @alsa @mpd @ntpd @sshd @nscd @libvirtd)
создайте файл ~/.xinitrc
в котором прописан запуск графической оболочки, например в моем случае Gnome
exec ck-launch-session gnome-session

После этого запустите демоны dbus и hal
командами 
sudo /etc/rc.d/dbus start
sudo /etc/rc.d/hal start
или перезагрузитесь
Наберите команду
X -configure
для того чтобы сгенерировать файл xorg.conf.
затем скопируйте его в /etc/X11/xorg.conf
и только после всего этого набирайте 
startx
P.S.
ВНИМАТЕЛЬНО читайте wiki - все это там написано, причем на одной странице
