спасибо за подсказку по клавиатуре, сам бы не догадался если б носом не ткнули.
По поводу gdm (если кому интересно), как временное решение можно прописать в файл /etc/gdm/custom.conf следующие строки (взято с конфига Ubuntu 8.10)
# GDM configuration storage
[daemon]
# Automatic login, if true the first attached screen will automatically logged
# in as user as set with AutomaticLogin key.
AutomaticLoginEnable=true
AutomaticLogin=имя_пользователя
# Timed login, useful for kiosks.  Log in a certain user after a certain amount
# of time.
TimedLoginEnable=true
TimedLogin=имя_пользователя
TimedLoginDelay=10
[xdmcp]
  ......

Ну и естественно можно попробывать поиграться с остальными параметрами.

Была такая ерунда, попробуй проверить правильность UUID в fstab. Малоли может одну букву/цифру неправильно прописал. Если нормально, то при загрузке у тебя будет сообщение об ошибке и команда с которой нужно смонтировать твой раздел (точно не помню, но сам увидиш). Монтируеш раздел с предлагаемыми опциями и перезагружаешся без размонтирования. (При этом флаг корректоного отключения раздела не будет установлен и при следующей загрузке fsck проверит раздел и нормально подмонтирует). Я по крайней мере так победил проблему :) .
