В своё время пользовался archlinux-2009.02-core-i686.iso. Когда понадобилось создать свой установочный образ, из этого, добавив туда установочные файлы из репозитариев (ок. 2 ГГб), то выяснилось что загрузчик в этом образе не совместим с версией sqfs-tools идущей вместе с пакетами включёнными в этот образ. Проблему решил при помощи Ubuntu Live CD 8.0, версия sqfs-tools на этом диске создавала файл ROOT_IMAGE.SQFS, в понятном для загрузчика формате. Вопрос: в этом установочном образе sqfs-tools совместима с загрузчиком? Т.е. можно ли при помощи имеющейся версии sqfs-tools создать ROOT_IMAGE.SQFS, понятный загрузчику “установочного образа 2010.05”?
