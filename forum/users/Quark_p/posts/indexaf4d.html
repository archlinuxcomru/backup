Домашним рассказываем, что торрены для видео ложим в torrent/Video, торренты для книг ложим в torrent/Book, rtorrent сам заберет, скачает и положет куда надо.

Незнаю, может это и не новость, но для себя на оффсайте тинта2 нашел http://code.google.com/p/tint2/wiki/FAQ … %27_button полезную фичу - клик по часам прячет все приложения и показывает раб.стол. Нужно установить wmctrl , создать следующий скрипт show_desktop.sh :

#!/bin/sh
if wmctrl -m | grep “mode: ON”; then
exec wmctrl -k off
else
exec wmctrl -k on
fi

в секции CLOCK добавить
clock_lclick_command = show_desktop.sh

Кстати уже собранный tint2-svn есть в репозитарии archlinuxfr
