Для активации кнопок вык,ребут и т.д надо:
1 - добавить юзера в группу power - gpasswd -a USERNAME power
2 - в .xinitrc что лежит в домашней папке пользователя (если его нет, можно взять в /etc/skel) . добавить строчку exec ck-launch-session startxfce4
3- ну и не забываем запустить hal и добавить его в его в /etc/rc.conf
Более подробно читаем в вики http://wiki.archlinux.org/index.php/Xfce4

У меня стоит на 4 гиговой флешке в NASе, ставил прямо из инсталятора, только добавил в /etc/mkinitcpio.conf
HOOKS=“…usb …”

а в /etc/fstab по советам EEE-PCишников добавил
tmpfs /tmp tmpfs defaults 0 0
tmpfs /var/tmp tmpfs defaults 0 0
tmpfs /var/lock tmpfs defaults 0 0
tmpfs /var/log tmpfs defaults 0 0

про установку на всякие флешки всяких ОС тут можна почитать http://eee-pc.ru/wiki/
