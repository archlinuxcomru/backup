…
mount /dev/fd /mnt/floppy
…
По поводу флоппу.
Сначала надо
#modprobe floppy
затем
#mount /dev/fd0
Монтироваться будет в соответствии с fstab.
В последних версиях в
/media/fl

Пример строки fstab:
/dev/fd0 /media/fl auto user,noauto,utf8 0 0
либо
/dev/fd0 /media/fl vfat user,noauto,utf8 0 0



DE - Desktop environment. Например, KDE, gnome, xfce.
Стандартно в DE есть автомонтирование. Надо только добавить HAL в DAEMONS rc.conf
Для автомонтирования строчка в fstab не нужна. Но не уверен, что через авто можно монтировать флоппи-дискету.
