Поправил pkgbuild для обнаружения imagemagick:

 cmake -DCMAKE_INSTALL_PREFIX=/usr \
        -DPYTHON_LIBRARY=/usr/lib/libpython2.7.so.1.0 \
        -DIMAGEMAGICKLIB_CXX_LIBRARY=/usr/lib/libMagick++-Q16.so.7 \
        -DIMAGEMAGICKLIB_C_LIBRARY=/usr/lib/libMagickCore-Q16.so.7 \
        -DPYTHON_INCLUDE_DIR=/usr/include/python2.7 \
        -DIMAGEMAGICK_INCLUDE_DIR=/usr/include/ImageMagick \
        -DIMAGEMAGICKLIB_INCLUDE_DIR=/usr/include/ImageMagick \
        ..

Оно собирается, родные примеры работают нормально, но python-овый launcher выдаёт ошибку

Magick: Memory allocation failed `Cannot allocate memory' @ fatal/cache.c/AcquirePixelCacheNexus/270.

Кто-нибудь понимает, что эта шифровка означает, и как попросить imagick выделять память.
