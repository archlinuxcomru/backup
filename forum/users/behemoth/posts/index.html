Та же самая ошибка при установке загрузчика grub.
grub> device (sd0) /dev/sda
grub> root (sd0,0)
Filesystem type unknown, partition type 0x7
grub> setup (sd0)
Error17: Cannot mount selected partition
Никаких дуалбутов и нестандартных размещений, сдвинутых разделов нет. sda1 и sda5. Первый - системный, “/”, “/boot”, форматирую в ext4. Второй - логический с ntfs.
Ошибка решена. Появлялась (может просто случайность) при выборе сразу третьего пункта - выбора файловой системы уже существующего раздела. Как решение - пройтись по второму пункту - ручной разметки диска и записать изменения.
